# Vision Agent Skill

## Quick Start

当用户需要以下帮助时使用此技能：
- "分析这张图片"
- "提取图片中的文字"
- "这个错误是什么原因？"
- "解释这个架构图"

## Capabilities

- 图片内容分析和描述
- OCR 文字提取（支持代码、终端输出、文档）
- 错误截图诊断（识别错误类型、定位问题、提供解决方案）
- 技术图表理解（架构图、流程图、UML、ER 图）
- UI 界面分析（布局、设计、交互）
- 数据可视化分析（图表、仪表板）

## Parameters

- `task` (required): 具体的视觉任务描述
  - 示例："描述图片内容"、"提取所有文字"、"分析这个错误"
- `image` (required): 图片文件路径
  - 支持相对路径（相对于工作区）或绝对路径
  - 支持格式：PNG, JPG, JPEG, GIF, WebP, BMP

## Examples

```
用户: 分析这张截图中的错误
→ task: "分析这个错误截图，找出问题并提供解决方案", image: "error.png"

用户: 提取图片中的代码
→ task: "提取图片中的所有代码，保持原有格式", image: "code.png"

用户: 解释这个架构图
→ task: "分析这个架构图，说明各组件之间的关系", image: "architecture.png"
```

## Output Format

提供：
1. 清晰的分析结果
2. 结构化的信息提取
3. 可操作的建议（如适用）
4. 识别出的关键元素

## Notes

- 支持最大图片大小：10MB
- 分析结果基于 GLM-4V 模型
- 对于复杂图片，可能需要多次分析
- OCR 功能支持多种文字和语言
