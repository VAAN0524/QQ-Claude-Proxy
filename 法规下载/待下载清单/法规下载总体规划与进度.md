## 📝 下载注意事项

### 1. 版本确认
- 下载前务必确认标准的最新版本
- 国家标准每5年复审一次
- 注意标准修改单和更正通知

### 2. 文件命名规范
```
格式：标准号-年份 中文名称.扩展名

示例：
├── GB17405-2025 保健食品良好生产规范.pdf
├── GB29922-2025 特殊医学用途配方食品通则.pdf
├── 食品安全总监和食品安全员管理规定（总局令第60号）.pdf
└── ISO22000-2018 食品安全管理体系要求.pdf
```

## 📞 问题与支持

### 常见问题
**Q: 如何确认标准的最新版本？**
A: 访问国家标准全文公开系统，输入标准号查询。

**Q: 某些标准需要付费购买怎么办？**
A: 优先使用免费官方渠道，必要时联系当地标准化研究院。

**Q: 标准更新后如何处理？**
A: 保留旧版本存档，更新工作使用最新版，记录更新日期。
