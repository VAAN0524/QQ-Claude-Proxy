# 食品GMP核心标准详细下载指引

## 一、特殊医学用途配方食品GMP

### GB 29923-2013 特殊医学用途配方食品良好生产规范
**状态**: ⚠️ 需要更新版本检查

#### 官方来源
- 标准全文公开系统：https://openstd.samr.gov.cn/
- 搜索关键词：`GB 29923` 或 `特殊医学用途配方食品良好生产规范`

#### 核心内容框架
```
第一章 范围
第二章 规范性引用文件
第三章 术语和定义
第四章 选址及厂区环境
第五章 厂房和车间
第六章 设备与设施
第七章 卫生管理
第八章 原料和包装材料的要求
第九章 生产过程的食品安全控制
第十章 检验
第十一章 产品的追溯和召回
第十二章 培训
第十三章 管理机构和人员
第十四章 记录和文件管理
```

#### 重点要求
1. **厂房洁净度**：十万级及以上
2. **工艺用水**：符合GB 5749生活饮用水卫生标准
3. **人员要求**：每年健康体检，培训记录完整
4. **检验能力**：具备出厂检验项目的自检能力

## 二、保健食品GMP

### GB 17405-2025 保健食品良好生产规范
**状态**: ✅ 2025年新版发布

#### 官方来源
- 国家卫生健康委员会食品安全标准
- 标准全文公开系统：https://openstd.samr.gov.cn/

#### 与2010版主要变化
| 项目 | 2010版 | 2025版 |
|------|--------|--------|
| 适用范围 | 仅保健食品 | 扩展至功能性食品 |
| 洁净度要求 | 30万级 | 提升至10万级 |
| 微生物限度 | 部分要求 | 全面细化 |
| 电子记录 | 未提及 | 要求电子化追溯 |

#### 关键章节
```
第三章 厂房与设施
  - 3.1 选址要求
  - 3.2 厂区布局
  - 3.3 生产车间分区
  - 3.4 洁净室（区）要求
  - 3.5 仓储设施

第四章 设备
  - 4.1 设备选型
  - 4.2 设备安装
  - 4.3 设备清洗消毒
  - 4.4 设备维护保养

第五章 人员
  - 5.1 人员资质
  - 5.2 健康管理
  - 5.3 培训要求
  - 5.4 人员卫生
```

## 四、HACCP体系

### GB/T 27341-2009 危害分析与关键控制点体系 食品生产企业通用要求

#### 官方来源
- 国家标准全文公开系统
- 推荐性国家标准

#### HACCP七项原则
1. 进行危害分析
2. 确定关键控制点
3. 建立关键限值
4. 建立监控程序
5. 建立纠偏措施
6. 建立验证程序
7. 建立记录保持程序

#### 与GMP的关系
```
GMP（基础）         HACCP（建立在GMP之上）
    │                   │
    ├─ 基础卫生条件      ├─ 危害分析
    ├─ 厂房设施要求      ├─ 关键控制点识别
    ├─ 人员卫生要求      ├─ 关键限值设定
    └─ 基础管理制度      └─ 过程监控体系
```

#### 官方来源
- ISO官方网站：https://www.iso.org/

#### 标准结构
```
1. 范围

3. 术语和定义

5. 领导作用

7. 支持
8. 运行
```

## 下载流程建议

### 方式一：国家标准全文公开系统
```bash
1. 访问 https://openstd.samr.gov.cn/
2. 在搜索框输入标准号（如：GB 17405）
3. 点击"在线阅读"或"下载"
4. 如需下载PDF，可能需要注册账号
```

### 方式二：命令行下载（如果可用）
```bash
# 使用wget或curl
wget "https://openstd.samr.gov.cn/bzgk/gb/newGbInfo?hcno=XXXXXX"
```

### 方式三：第三方标准平台
- 工标网：https://www.csres.com/
- 标准下载网：https://www.bzxz.net/
- ⚠️ 注意：需验证标准是否为最新版本
