# GB 4789.30-2024 食品微生物学检验 单核细胞增生李斯特氏菌检验

## 标准信息

| 项目 | 内容 |
|------|------|
| **标准号** | GB 4789.30-2024 |
| **标准名称** | 食品微生物学检验 单核细胞增生李斯特氏菌检验 |
| **发布日期** | 2024年 |
| **实施日期** | 2024年 |
| **替代标准** | GB 4789.30-2016 |
| **适用范围** | 食品中单核细胞增生李斯特氏菌的检验 |

---

## 1. 范围

本标准规定了食品中单核细胞增生李斯特氏菌（Listeria monocytogenes）的检验方法。

本标准适用于肉制品、乳制品、水产品、即食食品、婴幼儿食品等中单核细胞增生李斯特氏菌的检验。

---

## 2. 设备和材料

### 2.1 设备

- **恒温培养箱**：30℃±1℃、35℃±1℃、37℃±1℃
- **冰箱**：2℃~5℃
- **均质器**
- **显微镜**：相差显微镜
- **无菌吸管、试管、锥形瓶、培养皿**

### 2.2 培养基和试剂

- **李斯特氏菌增菌肉汤（LEB）/ Fraser肉汤**
- **李斯特氏菌选择性增菌肉汤（BLEB）/ Fraser半固体培养基**
- **PALCAM琼脂**
- **牛津琼脂（OXA）**
- **李斯特氏菌显色培养基（ALOA、Rapid'L.mono等）**
- **三糖铁琼脂（TSI）**
- **SIM动力培养基**
- **血琼脂平板**
- **生化鉴定试剂盒或VITEK、API等自动化系统**

---

## 3. 检验程序

```
样品 → 一次增菌（LEB，30℃，24h~48h）
    ↓
二次增菌（Fraser，30℃，24h~48h）
    ↓
分离（PALCAM/OXA/显色培养基，35℃~37℃，24h~48h）
    ↓
挑选可疑菌落（灰黑色、带黑色晕圈/蓝色菌落）
    ↓
初步鉴定（过氧化氢酶、动力、TSI等）
    ↓
确认试验（溶血、CAMP试验、生化鉴定）
    ↓
报告结果
```

---

## 4. 操作步骤

### 4.1 增菌

#### 4.1.1 一次增菌

称取25g样品，加入225mL LEB中，30℃培养24h~48h。

#### 4.1.2 二次增菌

取一次增菌液0.1mL，接种于10mL Fraser肉汤，30℃培养24h~48h（观察到变黑即可）。

---

### 4.2 分离

取二次增菌液划线接种于：
- **PALCAM琼脂**：35℃~37℃，24h~48h
- **牛津琼脂**：35℃~37℃，24h~48h
- **李斯特氏菌显色培养基**：按说明书

---

### 4.3 菌落特征

| 培养基 | 单核细胞增生李斯特氏菌 | 其他李斯特氏菌 | 杂菌 |
|--------|------------------------|----------------|------|
| PALCAM | 灰绿色，有黑色晕圈，凹陷 | 灰绿色，无黑色晕圈 | 被抑制或黄色 |
| 牛津琼脂 | 灰黑色，有黑色晕圈 | 灰黑色，无晕圈 | 被抑制 |
| ALOA | 蓝色菌落，有白色晕圈 | 蓝色菌落，无晕圈 | 其他颜色 |
| 血平板 | β-溶血，狭窄透明 | 弱溶血或无 | 各种溶血 |

---

### 4.4 初步鉴定

#### 4.4.1 形态与染色

| 项目 | 特征 |
|------|------|
| 革兰氏染色 | 阳性（G+） |
| 菌体形态 | 小杆菌，两端钝圆 |
| 排列 | 单个、短链、V形或栅栏状 |
| 鞭毛 | 周身鞭毛，25℃时有动力 |
| 芽孢 | 无 |

#### 4.4.2 生化筛选

| 试验 | 结果 |
|------|------|
| 过氧化氢酶 | + |
| 氧化酶 | - |
| TSI | 产酸不产气，不产生硫化氢 |
| 动力（25℃） | +（伞状生长） |
| 动力（35℃） | -/弱 |
| 七叶苷水解 | + |
| 葡萄糖 | +（产酸不产气） |

---

### 4.5 确认试验

#### 4.5.1 溶血试验

接种血琼脂平板，35℃~37℃培养24h~48h：
- **单核细胞增生李斯特氏菌**：β-溶血，狭窄透明环
- **伊氏李斯特氏菌**：β-溶血，宽大透明环
- **无害李斯特氏菌/威尔斯李斯特氏菌**：无溶血

#### 4.5.2 CAMP试验

| 菌种组合 | 单核细胞增生李斯特氏菌 |
|----------|------------------------|
| 金黄色葡萄球菌 | +（箭头状增强溶血） |
| 马红球菌 | - |

#### 4.5.3 生化鉴定

| 试验 | 单增李斯特 | 伊氏李斯特 | 无害李斯特 | 威尔斯李斯特 | 西尔李斯特 |
|------|------------|------------|------------|--------------|------------|
| 七叶苷 | + | + | + | + | + |
| 甘露醇 | - | + | - | - | - |
| 鼠李糖 | + | - | - | + | - |
| α-甲基-D-甘露糖苷 | + | + | - | - | + |
| 硝酸盐还原 | - | - | - | - | - |
| 明胶液化 | + | + | - | - | - |
| 25℃动力 | + | + | + | + | + |
| 35℃动力 | - | + | - | - | - |
| β-溶血 | + | + | - | - | - |
| CAMP（金葡） | + | - | - | - | + |
| CAMP（马红） | - | + | - | - | - |

---

## 5. 结果报告

### 5.1 检出单核细胞增生李斯特氏菌

- 分离出符合菌落特征的菌株
- 革兰氏阳性小杆菌
- 生化反应符合
- 溶血试验阳性（β-溶血）
- CAMP试验阳性（与金黄色葡萄球菌）

### 5.2 未检出单核细胞增生李斯特氏菌

- 增菌后分离培养无可疑菌落
- 或可疑菌落经鉴定不符合

## 6. 致病性与公共卫生意义

### 6.1 致病性

单核细胞增生李斯特氏菌是重要的食源性病原菌：

| 特点 | 说明 |
|------|------|
| 发病率 | 低，但病死率高（20%~30%） |
| 易感人群 | 孕妇、新生儿、老年人、免疫低下者 |
| 潜伏期 | 3d~70d，平均3周 |
| 感染剂量 | 可能低至100~1000 CFU |

### 6.2 临床表现

| 人群 | 症状 |
|------|------|
| 健康成人 | 发热、头痛、肌痛、胃肠道症状 |
| 孕妇 | 流产、死胎、新生儿感染 |
| 新生儿 | 脓毒症、脑膜炎 |
| 免疫低下者 | 脑膜炎、败血症 |

### 6.3 食品安全风险

**高风险食品**：
- 即食肉制品
- 软质奶酪
- 冷熏鱼类
- 预制沙拉
- 冷冻即食食品

**污染来源**：
- 环境（土壤、水、植物）
- 加工设备（生物膜形成）
- 冷藏存活（0℃~45℃可生长）

## 7. 质量控制

### 7.1 阳性对照
- 单核细胞增生李斯特氏菌ATCC 19111
- 单核细胞增生李斯特氏菌ATCC 19115

### 7.2 阴性对照
- 无害李斯特氏菌ATCC 33090
- 伊氏李斯特氏菌ATCC 19119

### 7.3 培养基质控
- Fraser肉汤加入添加剂后应抑制杂菌
- PALCAM/OXA应促进李斯特氏菌生长并抑制革兰氏阴性菌

---

## 8. 安全注意事项

### 8.1 生物安全

- 单核细胞增生李斯特氏菌为B类病原微生物
- 实验室感染主要通过皮肤接触
- 操作时应穿戴防护用品

### 8.2 操作要点

- **样品处理**：冷藏运输，尽快检验
- **增菌温度**：30℃促进鞭毛表达，利于动力观察
- **动力试验**：25℃培养24h~48h观察
- **鉴别要点**：
  - 与无害李斯特氏菌鉴别（溶血、CAMP）
  - 与伊氏李斯特氏菌鉴别（CAMP、35℃动力）
  - 与棒状杆菌属鉴别（动力、革兰氏形态）

## 9. 附录

### 9.1 李斯特氏菌属分类

| 菌种 | 致病性 |
|------|--------|
| 单核细胞增生李斯特氏菌（L. monocytogenes） | +（主要病原）|
| 伊氏李斯特氏菌（L. ivanovii） | +（主要感染动物）|
| 无害李斯特氏菌（L. innocua） | - |
| 威尔斯李斯特氏菌（L. welshimeri） | - |
| 西尔李斯特氏菌（L. seeligeri） | - |
| 格氏李斯特氏菌（L. grayi） | - |

### 9.2 分子分型

- **血清分型**：1/2a, 1/2b, 1/2c, 3a, 3b, 3c, 4a, 4ab, 4b, 4c, 4d, 4e
- **主要流行株**：
  - 1/2a（欧美常见）
  - 4b（与暴发相关）

### 9.3 检验意义

- **进出口检验**：即食食品必检项目
- **生产过程监控**：环境监测、产品放行
- **流行病学调查**：分子分型溯源
