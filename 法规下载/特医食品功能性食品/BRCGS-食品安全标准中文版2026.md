# BRCGS 食品安全标准（2026版）

## 标准概述

**标准名称**：BRCGS Global Standard for Food Safety (BRCGS 食品安全全球标准)
**原名称**：BRC Food Standard (英国零售商协会食品标准)
**最新版本**：Issue 10 (2024年发布，2026年适用)
**发布机构**：BRCGS (Brand Reputation Compliance Global Standards)
**认证性质**：全球食品安全认证标准（GFSI 认可）

---

## 一、BRCGS 标准简介

### 1.1 历史沿革

| 年份 | 事件 |
|------|------|
| 1998 | BRC 发布第一版食品标准 |
| 2000 | 获得 GFSI（全球食品安全倡议）认可 |
| 2016 | 更名为 BRCGS |
| 2024 | 发布第 10 版标准（Issue 10） |

### 1.2 适用范围

BRCGS 食品安全标准适用于：

- 食品加工企业
- 食品包装材料生产企业
- 食品配料和添加剂生产企业
- 餐饮服务商
- 零售商自有品牌供应商

---

## 二、BRCGS Issue 10 主要更新（2024）

### 2.1 核心变化

1. **食品安全文化**
   - 新增食品安全文化要求（条款 1.1.2）
   - 要求企业制定食品安全文化建设计划
   - 定期评估食品安全文化成熟度

2. **质量管理体系整合**
   - 强化质量管理与食品安全管理的整合
   - 新增产品质量控制要求
   - 增加客户投诉处理要求

3. **食品防御和欺诈**
   - 整合食品防御（Food Defense）评估
   - 强化食品欺诈（Food Fraud）预防
   - 新增脆弱性评估要求

4. **环境管理**
   - 新增环境可持续性承诺
   - 强调废物管理和资源效率
   - 鼓励减少碳足迹

5. **标签和追溯**
   - 强化过敏原管理
   - 增加标签验证要求
   - 提升追溯性测试频率

### 2.2 结构调整

Issue 10 共有 **9 个章节**：

| 章节 | 标题 | 要求 |
|------|------|------|
| **1** | 高层管理承诺 | 食品安全文化、管理层承诺 |
| **2** | 食品安全计划（HACCP） | HACCP 体系要求 |
| **3** | 食品安全与质量管理体系 | 文档、管理评审、内部审核 |
| **4** | 现场标准 | 布局、设备、设施 |
| **5** | 产品控制 | 产品设计、过敏原、追溯 |
| **6** | 过程控制 | 过程控制、温度控制、检测 |
| **7** | 人员管理 | 培训、防护服、健康检查 |
| **8** | 风险评估 | 供应商审核、规格书、检验 |
| **9** | 产品放行 | 检验、数量检查、库存周转 |

## 三、核心要求详解

### 3.1 高层管理承诺（第1章）

#### 3.1.1 食品安全文化

**新增要求**（Issue 10）：
- 制定书面的食品安全文化计划
- 定期评估食品安全文化的成熟度（至少每年）
- 基于评估结果实施改进措施
- 鼓励员工报告食品安全问题

**评估要素**：
- 管理层领导力
- 员工意识和参与度
- 沟通机制
- 持续改进文化

#### 3.1.2 管理体系
- 明确食品安全职责和权限
- 任命食品安全小组组长
- 提供足够的资源
- 定期审查体系绩效

### 3.2 HACCP 食品安全计划（第2章）

#### 3.2.1 HACCP 体系要求

| 要求 | 说明 |
|------|------|
| **HACCP 小组** | 多学科团队，具备相应知识 |
| **产品描述** | 完整描述产品特性、预期用途 |
| **流程图** | 覆盖所有工艺步骤 |
| **危害分析** | 识别所有潜在危害 |
| **CCP 确定** | 科学确定关键控制点 |
| **关键限值** | 可测量的、可验证的限值 |
| **监控程序** | 明确监控方法、频率、人员 |
| **纠正措施** | 规定 CCP 失控时的行动 |
| **验证程序** | 验证 HACCP 体系有效性 |

#### 3.2.2 HACCP 文档要求

- 书面 HACCP 计划
- 危害分析记录
- CCP 监控记录
- 纠正措施记录
- 验证活动记录

### 3.3 现场标准（第4章）

#### 3.3.1 建筑和设施

| 要求 | 标准 |
|------|------|
| **位置** | 远离污染源，环境整洁 |
| **布局** | 合理划分卫生等级区域，防止交叉污染 |
| **地面** | 耐磨损、防滑、易清洁 |
| **墙壁** | 光滑、耐清洗、浅色 |
| **天花板** | 防止冷凝水、霉菌滋生 |
| **门窗** | 密封良好，便于清洗 |
| **照明** | 充足，防护罩完好 |
| **通风** | 良好，控制气流方向 |

#### 3.3.2 设备要求

- **材质**：食品级材料，耐腐蚀
- **设计**：无卫生死角，易清洁
- **安装**：便于操作、清洁、维护
- **校准**：关键设备定期校准

### 3.4 产品控制（第5章）

#### 3.4.1 产品设计

- 成分和配方的安全评估
- 过敏原识别和控制
- 法规符合性验证

#### 3.4.2 过敏原管理

**Issue 10 强化要求**：
- 识别所有过敏原
- 实施过敏原控制措施
- 标签验证（每批次）
- 防止交叉污染
- 员工培训

**主要过敏原类别**：
1. 含麸质的谷物
2. 甲壳纲类动物
3. 蛋类
4. 鱼类
5. 花生
6. 大豆
7. 乳类
8. 坚果
9. 芹菜
10. 芥末
11. 芝麻
12. 二氧化硫和亚硫酸盐
13. 羽扇豆
14. 软体动物

#### 3.4.3 追溯体系

**要求**：
- 前向追溯：成品可追溯到所有原料
- 后向追溯：原料可追溯到所有成品
- 测试频率：每年至少进行一次全追溯测试
- 测试时限：4小时内完成追溯

### 3.5 过程控制（第6章）

#### 3.5.1 操作控制

| 控制点 | 要求 |
|------|------|
| **配方控制** | 严格按照配方生产 |
| **称量** | 使用校准的称量设备 |
| **投料顺序** | 按工艺要求投料 |
| **加工参数** | 控制时间、温度、压力等 |
| **金属检测** | 适当位置设置金属探测器 |
| **X光检测** | 根据产品特性设置 |

#### 3.5.2 温度控制

- 冷藏：0-4°C
- 冷冻：-18°C 或以下
- 热处理：确保达到杀菌温度
- 监控：连续温度监控和记录

### 3.6 人员管理（第7章）

#### 3.6.1 培训要求

| 培训类型 | 要求 |
|------|------|
| **入职培训** | 上岗前完成 |
| **食品安全培训** | 每年至少一次 |
| **岗位技能培训** | 持续进行 |
| **培训效果评估** | 验证培训有效性 |

#### 3.6.2 个人卫生

- 健康检查：上岗前和每年定期检查
- 洗手消毒：进入车间前和操作过程中
- 防护服：穿戴规范，保持清洁
- 个人物品：不得带入生产区
- 伤病处理：报告伤口，适当处理

### 3.7 供应商管理（第8章）

#### 3.7.1 供应商批准

**批准流程**：
1. 评估供应商风险
2. 收集供应商信息
3. 现场审核（高风险供应商）
4. 样品测试
5. 批准列入合格供应商名单

#### 3.7.2 供应商监控

**监控方式**：
| 风险等级 | 审核频率 | 测试频率 |
|---------|----------|----------|
| 高风险 | 每年1次 | 每批次 |
| 中风险 | 每2年1次 | 每季度 |
| 低风险 | 每3年1次 | 每半年 |

## 四、BRCGS 认证等级

### 4.1 认证等级

BRCGS Issue 10 设有 **AA、A、B、C、D** 五个等级：

| 等级 | 评分范围 | 认证有效期 |
|------|----------|-----------|
| **AA** | 95%以上 | 延长至最长有效期 |
| **A** | 90-94% | 标准有效期 |
| **B** | 80-89% | 标准有效期 |
| **C** | 70-79% | 6个月 |
| **D** | 低于70% | 不通过认证 |

### 4.2 评分系统

- **基础条款**：总分 100%
- **额外承诺条款**：加分项
- **不符合项**：
  - **重大不符合**：扣10分/项
  - **一般不符合**：扣5分/项
  - **观察项**：不扣分，但需整改

## 五、BRCGS 认证流程

### 5.1 认证流程图

```
准备阶段 → 体系建立 → 试运行 → 内部审核 → 申请认证 → 正式审核 → 证书发放 → 监督审核
```

### 5.2 认证步骤

| 步骤 | 内容 | 时间 |
|------|------|------|
| **1. 准备** | 了解标准、差距分析 | 1-2个月 |
| **2. 体系建立** | 建立管理体系文件 | 1-2个月 |
| **3. 试运行** | 体系试运行 | 3个月以上 |
| **4. 内部审核** | 内部审核和管理评审 | 1个月 |
| **5. 申请认证** | 选择认证机构、提交申请 | 2-4周 |
| **6. 文件审核** | 评审管理体系文件 | 1-2天 |
| **7. 现场审核** | 现场审核（通常2天） | 2天 |
| **8. 整改** | 不符合项整改 | 4周内 |
| **9. 证书发放** | 审核通过后发放证书 | 审核完成后2-4周 |

### 5.3 证书有效期

- **有效期**：12个月（每年审核一次）
- **延期条件**：获得 A 或 AA 等级
- **延期时限**：最长可延长至下一次审核

## 六、BRCGS 在中国的应用

### 6.1 市场认可

BRCGS 在中国获得广泛认可：

- **国际认可**：GFSI 认可
- **零售商认可**：沃尔玛、家乐福、Tesco 等
- **品牌商认可**：联合利华、雀巢、达能 等

### 6.2 获证企业

中国获证企业类型：
- 出口食品加工企业
- 外资企业国内工厂
- 大型食品集团
- 高端食品制造商

## 八、BRCGS 与其他标准的关系

### 8.1 GFSI 认可标准对比

| 标准 | 发起方 | 适用范围 | 特点 |
|------|--------|----------|------|
| **BRCGS** | 英国零售商协会 | 食品加工、包装 | 要求详细、分级明确 |
| **FSSC 22000** | 基金会 | 食品链全链条 | 基于 ISO 22000 |
| **IFS** | 德国零售商 | 欧洲市场 | 注重细节、要求严格 |
| **SQF** | 食品营销协会 | 北美市场 | 分级认证、灵活性强 |

BRCGS 还发布了其他相关标准：

- **BRCGS 包装材料标准** (Issue 7)
- **BRCGS 仓储配送标准** (Issue 3)
- **BRCGS 消费品标准** (Issue 4)
- **BRCGS 代理商经纪人标准** (Issue 3)

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **GFSI** | 全球食品安全倡议（Global Food Safety Initiative） |
| **食品安全文化** | 组织在食品安全方面的共同价值观、信念和行为规范 |
| **食品防御** | 保护食品免受故意污染的预防措施 |
| **食品欺诈** | 为经济利益而故意替换、添加或篡改食品的行为 |
| **追溯** | 从原料到成品或从成品到原料的追踪能力 |
| **HACCP** | 危害分析与关键控制点 |
