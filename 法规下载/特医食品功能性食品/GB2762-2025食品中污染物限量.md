# GB 2762-2025 食品安全国家标准 食品中污染物限量

**标准号**：GB 2762-2025
**标准名称**：食品安全国家标准 食品中污染物限量
**发布日期**：2025年9月2日
**实施日期**：**2026年9月2日**
**替代标准**：GB 2762-2022
**发布机构**：国家卫生健康委员会、国家市场监督管理总局

---

## 标准简介

本标准规定了食品中铅、镉、汞、砷、镍、铬、亚硝酸盐、硝酸盐、苯并[a]芘、N-二甲基亚硝胺、多氯联苯、3-氯-1,2-丙二醇等污染物的限量指标。

本标准适用于各类食品原料和制成品，不适用于保健食品、特殊医学用途配方食品、婴幼儿配方食品等特殊食品。

---

## 主要污染物限量指标

### 一、重金属污染物

#### 1. 铅（Pb）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 谷物及其制品 | 0.2 |
| 蔬菜及制品 | 0.1-0.3 |
| 水果及制品 | 0.1 |
| 豆类及制品 | 0.2 |
| 肉及肉制品 | 0.5 |
| 水产动物及制品 | 0.5-1.5 |
| 乳及乳制品 | 0.05-0.5 |

#### 2. 镉（Cd）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 谷物及其制品 | 0.1-0.4 |

| 水果及制品 | 0.05 |
| 肉及肉制品 | 0.1 |
| 水产动物及制品 | 0.5-2.0 |

#### 3. 汞（Hg）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 总汞（水产动物及制品） | 0.5 |
| 甲基汞（水产动物及制品） | 0.5 |
| 其他食品 | 0.02-0.05 |

#### 4. 砷（As）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 无机砷（谷物及制品） | 0.2-0.5 |
| 无机砷（蔬菜及制品） | 0.1-0.5 |
| 无机砷（水产动物及制品） | 0.5-1.0 |
| 总砷（其他食品） | 0.5 |

#### 5. 镍（Ni）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 油脂及其制品 | 1.0 |
| 其他食品 | 1.0-5.0 |

#### 6. 铬（Cr）

| 食品类别 | 限量（mg/kg） |
|---------|-------------|
| 谷物及其制品 | 1.0 |
| 蔬菜及制品 | 0.5 |
| 水果及制品 | 0.5 |
| 豆类及制品 | 1.0 |
| 肉及肉制品 | 1.0 |
| 乳及乳制品 | 0.5 |

|---------|-------------------|-------------------|

## 主要修订内容

### 与 GB 2762-2022 相比的主要变化

1. **新增污染物指标**：
   - 增加了镍（Ni）的限量要求
   - 扩大了3-氯-1,2-丙二醇的适用范围

2. **调整限量要求**：
   - 部分食品中铅、镉的限量要求更加严格
   - 调整了部分食品类别的分类和限量

3. **完善食品类别**：
   - 细化了部分食品类别的分类
   - 增加了新型食品类别的限量要求

## 应用指南

### 1. 适用范围

本标准适用于：
- 各类食品原料
- 各类制成品
- 进口食品

本标准不适用于：
- 保健食品
- 特殊医学用途配方食品
- 婴幼儿配方食品
- 婴幼儿辅助食品

### 2. 检验方法

各污染物的检验方法应按照相应国家标准执行：
- GB 5009.12 食品中铅的测定
- GB 5009.15 食品中镉的测定
- GB 5009.17 食品中总汞及有机汞的测定
- GB 5009.11 食品中总砷及无机砷的测定
- GB 5009.123 食品中铬的测定
- GB 5009.33 食品中亚硝酸盐与硝酸盐的测定
- GB 5009.27 食品中苯并[a]芘的测定

### 3. 结果判定

- 检测结果低于或等于限量值的，判定为合格
- 检测结果高于限量值的，判定为不合格
- 复检按照国家相关规定执行

## 实施过渡期

本标准将于**2026年9月2日**正式实施，留有约1年的过渡期：
- 生产企业有充足时间调整原料采购和生产工艺
- 监管部门有充足时间制定配套监管措施
- 市场有充足时间消化现有库存产品
