# 保健食品良好生产规范审查细则（2026版）

## 法规信息
- **发布机构**：国家市场监督管理总局
- **实施日期**：2026年1月1日
- **适用范围**：保健食品生产企业生产许可审查和质量管理
- **依据标准**：GB 17405-2025 保健食品良好生产规范

---

## 第一章 总则

### 第一条 审查目的
规范保健食品生产许可审查工作，保证保健食品生产质量管理，确保保健食品质量安全。

### 第二条 审查依据
本细则依据《中华人民共和国食品安全法》《保健食品注册与备案管理办法》《GB 17405-2025 保健食品良好生产规范》等法律法规和标准制定。

### 第三条 适用范围
本细则适用于保健食品（含特殊医学用途配方食品中的特定全营养配方食品）生产企业的生产许可审查。

### 第四条 审查原则
1. **客观公正原则**：严格按照审查标准进行审查
2. **风险管控原则**：重点审查关键控制点
3. **持续改进原则**：促进企业质量管理体系持续完善

---

## 第二章 机构与人员审查

### 第五条 组织机构审查要点
| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 组织架构 | 是否建立独立的质量管理部门 | 质量管理部门直属企业主要负责人领导 |
| 职责分工 | 各部门职责是否明确，是否有书面文件 | 有明确的组织机构和职责文件 |
| 人员配置 | 是否配备与生产规模相适应的专业技术人员 | 按产品类别和生产规模配备足够人员 |

### 第六条 人员资质审查要点
| 人员类别 | 学历要求 | 专业要求 | 经验要求 |
|---------|---------|---------|---------|
| 企业负责人 | 大专及以上 | 食品、医药、生物等相关专业 | 3年以上食品行业管理经验 |
| 质量负责人 | 本科及以上 | 食品、医药、生物等相关专业 | 3年以上保健食品质量管理经验 |
| 生产负责人 | 大专及以上 | 食品、医药、生物等相关专业 | 3年以上食品生产管理经验 |
| 检验人员 | 中专及以上 | 食品检验相关专业 | 经培训合格后上岗 |

### 第七条 人员培训审查要点
1. 是否制定年度培训计划
2. 是否建立培训档案
3. 培训内容是否包括：法律法规、标准规范、操作规程、卫生知识
4. 是否进行培训考核，考核结果是否合格

### 第八条 人员健康审查要点
1. 是否建立从业人员健康管理制度
2. 从业人员是否持有有效健康证明
3. 是否建立健康档案，记录健康检查情况
4. 患有碍食品安全疾病的人员是否调离接触食品岗位

## 第三章 厂房与设施审查

### 第九条 选址审查要点
1. 厂区周围是否有粉尘、有害气体、放射性物质和其他扩散性污染源
2. 厂区环境是否整洁，路面是否平整硬化
3. 是否有适当的隔离带，防止外界污染

### 第十条 厂区布局审查要点
1. 生产区、辅助生产区、行政生活区是否分开
2. 人流、物流是否分开
3. 原料、半成品、成品是否分开
4. 是否有防止交叉污染的设施

### 第十一条 车间布局审查要点
| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 工艺布局 | 工艺流程是否顺畅，是否有交叉污染 | 工艺流程合理，有效防止交叉污染 |
| 功能分区 | 是否根据生产工艺划分洁净区、一般生产区 | 功能分区明确，有物理隔离 |
| 人流物流 | 人流、物流是否分开 | 有独立的人流、物流通道 |
| 缓冲设施 | 是否设置必要的缓冲间、气闸间 | 在适当位置设置缓冲设施 |

### 第十二条 洁净区审查要点

#### 洁净度级别要求
| 区域 | 洁净度级别 | 监测要求 |
|------|-----------|---------|
| 配料室 | 100,000级 | 每月监测 |
| 混合室 | 100,000级 | 每月监测 |
| 成型室 | 100,000级 | 每月监测 |
| 内包装室 | 100,000级 | 每月监测 |
| 无菌灌装区 | 10,000级 | 每周监测 |
| 关键操作区 | 100级（局部） | 每周监测 |

#### 环境监测项目
1. **悬浮粒子**：每班次监测
2. **沉降菌**：每月监测
3. **浮游菌**：每月监测
4. **表面微生物**：每月监测
5. **温湿度**：连续监测
6. **压差**：连续监测

### 第十三条 仓储设施审查要点
1. 原料库、成品库是否分开设置
2. 是否具备良好的通风、防潮、防虫、防鼠设施
3. 冷库温度是否能控制在2-8℃
4. 是否设置不合格品专区

---

## 第四章 设备审查

### 第十四条 设备选型审查要点
| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 设备材质 | 是否符合食品级要求 | 应当使用不锈钢或其他无毒材料 |
| 设备结构 | 是否便于清洗、消毒 | 无死角，易拆卸清洗 |
| 设备精度 | 是否满足工艺要求 | 配料精度±0.5%，混合均匀度CV≤5% |
| 自动化程度 | 是否具备自动化控制功能 | 关键工序应当自动化 |

### 第十五条 设备安装审查要点
1. 设备安装是否符合工艺要求
2. 设备与地面、墙壁之间是否保持适当距离
3. 管道排列是否整齐，标识是否清晰
4. 电气设备是否符合防爆、防潮要求

### 第十六条 设备维护审查要点
1. 是否建立设备台账
2. 是否制定设备维护保养计划
3. 是否按计划执行维护保养
4. 维护保养记录是否完整

---

## 第五章 卫生管理审查

### 第十七条 卫生制度审查要点
1. 是否建立完善的卫生管理制度
2. 是否划分卫生责任区
3. 是否定期进行卫生检查
4. 检查发现的问题是否及时整改

### 第十八条 清洁消毒审查要点
| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 清洁规程 | 是否制定详细的清洁消毒规程 | 有书面清洁消毒规程 |
| 清洁工具 | 不同洁净区的清洁工具是否专用 | 清洁工具分区使用，标识清晰 |
| 清洁剂消毒剂 | 是否经检验合格 | 有检验报告，不影响产品质量 |
| 清洁验证 | 清洁消毒后是否进行验证 | 有清洁验证记录 |

### 第十九条 虫害控制审查要点
1. 是否建立虫害控制程序
2. 是否定期检查厂区及周边环境
3. 是否设置防鼠板、防蝇灯等设施
4. 发现虫害是否及时采取控制措施

---

## 第六章 原料管理审查

### 第二十条 供应商审查要点
1. 是否建立供应商审核制度
2. 是否对供应商进行定期审核
3. 是否建立合格供应商名录
4. 供应商审核记录是否完整

### 第二十一条 原料验收审查要点
1. 是否建立原料验收标准
2. 每批原料是否进行验收检验
3. 验收内容包括哪些：外观、标识、检验报告
4. 不合格原料是否拒收并记录

### 第二十二条 原料储存审查要点
1. 原料是否分类存放，标识清晰
2. 是否做到先进先出
3. 特殊原料是否按规定条件储存
4. 是否定期检查库存原料质量

---

## 第七章 生产过程审查

### 第二十三条 生产工艺审查要点
1. 是否建立标准的生产工艺规程
2. 工艺参数是否经过验证
3. 关键工艺参数是否实时监控
4. 工艺变更是否经过验证和审批

### 第二十四条 生产操作审查要点
| 审查项目 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 操作规范 | 是否严格按照操作规程进行操作 | 有标准操作规程（SOP） |
| 生产记录 | 是否做好生产记录 | 记录及时、真实、完整 |
| 异常处理 | 发现异常情况是否及时报告并处理 | 有异常处理程序 |
| 污染控制 | 是否防止交叉污染 | 有有效的污染控制措施 |

### 第二十五条 配料审查要点
1. 配料是否采用称量配料方式
2. 称量设备精度是否符合要求
3. 配料是否进行复核
4. 配料记录是否完整

### 第二十六条 混合审查要点
1. 混合设备是否适用
2. 混合时间、转速是否经过验证
3. 是否进行混合均匀度检验
4. 混合记录是否完整

---

## 第八章 质量检验审查

### 第二十七条 检验机构审查要点
1. 是否建立独立的质量检验机构
2. 是否配备与生产规模相适应的检验人员
3. 是否配备必要的检验设备
4. 检验人员是否经过培训

### 第二十八条 检验能力审查要点
| 检验类别 | 检验项目 | 检验依据 |
|---------|---------|---------|
| 原料检验 | 理化指标、微生物指标 | 原料标准 |
| 过程检验 | 关键质量控制点 | 过程控制标准 |
| 成品检验 | 全项检验 | 产品标准 |
| 稳定性考察 | 加速试验、长期试验 | 稳定性试验方案 |

### 第二十九条 检验设备审查要点
1. 检验设备是否满足检验需求
2. 设备是否定期校准
3. 是否有设备校准记录
4. 是否有设备使用和维护记录

---

## 第九章 文件管理审查

### 第三十条 文件体系审查要点
| 文件类型 | 审查内容 | 审查标准 |
|---------|---------|---------|
| 质量手册 | 是否建立质量手册 | 内容完整，覆盖所有要素 |
| 程序文件 | 是否建立必要的程序文件 | 涵盖所有关键过程 |
| 作业指导书 | 是否建立作业指导书 | 操作步骤清晰明确 |
| 记录表单 | 是否建立记录表单 | 记录内容完整准确 |

### 第三十一条 记录管理审查要点
1. 是否建立记录管理制度
2. 记录是否真实、完整、及时
3. 记录是否按规定期限保存
4. 记录填写是否清晰、不得涂改

---

## 第十章 产品追溯与召回审查

### 第三十二条 追溯系统审查要点
1. 是否建立完善的产品追溯系统
2. 产品是否具有唯一标识
3. 批记录是否完整保存
4. 是否能够追溯到原料来源

### 第三十三条 召回制度审查要点
1. 是否建立产品召回制度
2. 是否定期进行召回演练
3. 召回产品是否按规定处理
4. 是否向监管部门报告召回情况

---

## 第十一章 审查判定

### 第三十四条 审查结论
审查结论分为：通过、整改后通过、不通过

### 第三十五条 通过条件
同时满足以下条件判定为通过：
1. 所有关键项目全部符合
2. 一般项目不符合项不超过5项
3. 不符合项已整改或制定整改计划

### 第三十六条 整改后通过条件
符合以下条件判定为整改后通过：
1. 关键项目不符合项不超过2项
2. 一般项目不符合项不超过10项
3. 制定详细的整改计划
4. 整改期限不超过3个月

### 第三十七条 不通过条件
符合以下任一条件判定为不通过：
1. 关键项目不符合项超过2项
2. 一般项目不符合项超过10项
3. 存在严重食品安全隐患
4. 提供虚假材料

## 附录B：保健食品GMP审查评分标准

### B.1 评分方法
- 关键项目：每项10分，不符合不得分
- 一般项目：每项5分，不符合扣分
- 总分100分，80分以上为通过

### B.2 评分等级
| 分数 | 等级 | 结论 |
|------|------|------|
| ≥90分 | 优秀 | 通过 |
| 80-89分 | 良好 | 通过 |
| 70-79分 | 合格 | 整改后通过 |
| <70分 | 不合格 | 不通过 |
