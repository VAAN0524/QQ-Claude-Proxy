# GB 50073-2013 洁净厂房设计规范

## 标准基本信息

- **标准编号**：GB 50073-2013
- **标准名称**：洁净厂房设计规范
- **发布日期**：2013年1月28日
- **实施日期**：2013年9月1日
- **发布部门**：中华人民共和国住房和城乡建设部
- **适用范围**：电子、制药、食品、医疗器械、生物实验室等领域的洁净环境建设

---

## 主要内容章节

### 1. 总则

本规范适用于新建、改建和扩建的洁净厂房设计。洁净厂房的设计应符合节约用地、能源和水资源的原则，并应满足生产工艺对洁净环境的要求。

### 2. 术语

对洁净室、洁净区、悬浮粒子、洁净度等级等关键术语进行了明确定义。

### 3. 空气洁净度等级

规定了洁净室的空气洁净度等级划分，共分为9级（1-9级），其中：

| 等级 | ≥0.5μm粒子最大浓度（个/m³） | ≥5μm粒子最大浓度（个/m³） |
|------|---------------------------|--------------------------|
| 1级 | 10 | - |
| 2级 | 100 | - |
| 3级 | 1,000 | - |
| 4级 | 10,000 | - |
| 5级 | 100,000 | - |
| 6级 | 1,000,000 | - |
| 7级 | - | 10,000 |
| 8级 | - | 100,000 |
| 9级 | - | 1,000,000 |

### 4. 总体设计

#### 4.1 洁净厂房位置选择和总平面布置

- 洁净厂房位置选择应符合当地总体规划要求
- 应远离铁路、码头、飞机场、交通要道等有严重空气污染、振动或噪声干扰的区域
- 应布置在厂区内环境清洁、人流货流不穿越或少穿越的地段

#### 4.2 工艺平面布置和设计综合协调

- 工艺平面布置应合理、紧凑
- 洁净室（区）人员净化、物料净化用室和设施应分别设置
- 洁净室（区）的噪声控制设计应符合现行国家标准

#### 4.3 人员净化和物料净化

- 人员净化用室应包括雨具存放、换鞋、存外衣、更洁净工作服、盥洗、无菌手消毒等房间
- 物料净化设施应根据物料性质、形状和运输方式确定

### 5. 建筑

#### 5.1 一般规定

- 洁净厂房的建筑平面和空间布局应适应生产工艺要求
- 洁净室（区）的内表面应平整、光滑、无裂缝、接口严密

#### 5.2 防火和疏散

- 洁净厂房的耐火等级不应低于二级
- 洁净室（区）的安全疏散设施应符合现行国家标准

#### 5.3 室内装修

- 洁净室（区）的地面、墙面和顶棚的材料应满足不易积尘、易清洁的要求
- 地面应耐磨、耐冲击、防滑、易清洗

### 6. 空气净化

#### 6.1 洁净室压差控制

- 洁净室（区）与周围的空间必须维持一定的压差
- 不同等级的洁净室之间的压差不宜小于5Pa

#### 6.2 气流流型和送风量

- 洁净室（区）的气流流型应根据生产工艺要求确定
- 可采用单向流、非单向流或混合流

#### 6.3 空气净化处理

- 空气净化处理应根据洁净度等级要求采用三级过滤（初效、中效、高效）
- 高效空气过滤器应设置在净化空调系统的末端

### 7. 采暖通风、防排烟

- 洁净室（区）的采暖方式应根据洁净度等级和工艺要求确定
- 洁净厂房的排烟系统应符合现行国家标准
