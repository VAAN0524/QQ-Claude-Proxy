# GB 5413.21-2010 婴幼儿食品和乳品中钙、铁、锌、钠、钾、镁、铜和锰的测定

**标准编号**: GB 5413.21-2010
**实施日期**: 2010年6月1日
**替代标准**: GB/T 5413.21-1997
**发布部门**: 中华人民共和国卫生部

---

## 1. 范围

本标准规定了婴幼儿食品和乳品中钙、铁、锌、钠、钾、镁、铜、锰的测定方法。

本标准适用于婴幼儿食品和乳品中上述8种元素含量的测定。

**方法**：火焰原子吸收光谱法

---

## 2. 原理

试样经干法灰化或湿法消化处理后，制备成酸性溶液，导入原子吸收光谱仪，在火焰中原子化，分别选取各元素的特征谱线，测定其吸光度，与标准系列比较定量。

| 元素 | 特征谱线(nm) | 火焰类型 |
|------|--------------|----------|
| 钙(Ca) | 422.7 | 空气-乙炔（贫燃） |
| 铁(Fe) | 248.3 | 空气-乙炔（贫燃） |
| 锌(Zn) | 213.9 | 空气-乙炔（贫燃） |
| 钠(Na) | 589.0 | 空气-乙炔（贫燃） |
| 钾(K) | 766.5 | 空气-乙炔（贫燃） |
| 镁(Mg) | 285.2 | 空气-乙炔（贫燃） |
| 铜(Cu) | 324.8 | 空气-乙炔（贫燃） |
| 锰(Mn) | 279.5 | 空气-乙炔（贫燃） |

## 5. 分析步骤

### 5.1 试样处理

#### 5.1.1 干法灰化（推荐用于乳粉、固体样品）

```
1. 称取均匀试样1-5g（精确至0.0001g）于瓷坩埚中
2. 在电热板上低温炭化至无烟
3. 移入马弗炉，550℃灰化4-6小时
4. 冷却后取出，加入2mL硝酸(1:1)
5. 在电热板上低温蒸干
6. 再次移入马弗炉，550℃灰化2小时
7. 冷却后用1%硝酸溶解残渣
8. 转移至25mL容量瓶中，用1%硝酸定容
9. 摇匀，如有沉淀需过滤或离心
```

#### 5.1.2 湿法消化（推荐用于液体样品）

```
1. 称取试样2-5g（精确至0.0001g）于消化瓶中
2. 加入10mL硝酸-高氯酸(4:1)混合酸
3. 放置过夜或电热板上低温预消化
4. 加热消化，保持微沸状态
5. 消化至溶液透明或呈淡黄色
6. 继续加热至近干
7. 冷却后用1%硝酸溶解
8. 转移至25mL容量瓶中，用1%硝酸定容
9. 摇匀，如有沉淀需过滤或离心
```

#### 5.1.3 微波消解法

```
1. 称取试样0.2-0.5g于消解罐中
2. 加入5-8mL硝酸
3. 按照微波消解仪程序进行消解
4. 消解完成后冷却
5. 转移至容量瓶，用去离子水定容
6. 摇匀，待测
```

### 5.2 仪器工作条件

**原子吸收光谱仪工作参数**：

| 参数 | 设定值 |
|------|--------|
| 灯电流 | 按说明书推荐值 |
| 狭缝宽度 | 0.2-0.7nm |
| 燃烧器高度 | 按说明书推荐 |
| 空气流量 | 按说明书推荐 |
| 乙炔流量 | 按说明书推荐 |
| 测定方式 | 吸收法 |

**各元素参考条件**：

| 元素 | 波长(nm) | 灯电流(mA) | 狭缝(nm) | 火焰类型 |
|------|----------|------------|----------|----------|
| Ca | 422.7 | 7.5 | 0.5 | 空气-乙炔 |
| Fe | 248.3 | 10 | 0.2 | 空气-乙炔 |
| Zn | 213.9 | 6 | 0.5 | 空气-乙炔 |
| Na | 589.0 | 8 | 0.2 | 空气-乙炔 |
| K | 766.5 | 8 | 0.2 | 空气-乙炔 |
| Mg | 285.2 | 7.5 | 0.5 | 空气-乙炔 |
| Cu | 324.8 | 6 | 0.5 | 空气-乙炔 |
| Mn | 279.5 | 7 | 0.2 | 空气-乙炔 |

### 5.3 标准曲线绘制

```
1. 按浓度由低到高测定标准系列溶液
2. 记录各浓度点的吸光度
3. 以吸光度为纵坐标，浓度为横坐标
4. 绘制标准曲线
5. 计算回归方程和相关系数
```

**相关系数要求**：r ≥ 0.995

### 5.4 试样测定

```
1. 将试样溶液导入原子吸收光谱仪
2. 在各元素的测定条件下测定吸光度
3. 根据标准曲线计算元素含量
```

## 6. 结果计算

### 6.1 计算公式

```
         X = (C × V × f) / m

其中：
X — 试样中元素含量，mg/100g 或 mg/100mL
C — 从标准曲线上查得的元素浓度，mg/L
V — 试样溶液定容体积，mL
f — 稀释倍数
m — 试样质量或体积，g 或 mL
```

### 6.2 结果表示

- 保留两位有效数字
- 报告结果时注明测定方法
- 结果低于检出限时报告"未检出"

## 7. 精密度

在重复性条件下获得的两次独立测定结果的差值应符合：

| 元素含量范围 | 相对偏差(%) |
|--------------|-------------|
| < 1 mg/100g | ≤ 20 |
| 1 - 10 mg/100g | ≤ 15 |
| 10 - 100 mg/100g | ≤ 10 |
| > 100 mg/100g | ≤ 5 |

## 9. 注意事项

### 9.1 样品处理

1. **避免污染**：全过程使用优级纯试剂和高纯水
2. **器皿清洗**：玻璃器皿用硝酸(10%)浸泡过夜，用去离子水冲洗干净
3. **灰化温度**：控制灰化温度，避免元素挥发损失
4. **酸度控制**：测定溶液酸度应与标准溶液一致

### 9.2 干扰消除

**钙、镁测定干扰**：
- 加入镧盐(0.1%)作为释放剂
- 或加入铯盐(0.1%)作为电离抑制剂

**钠、钾测定干扰**：
- 加入铯盐(0.1%)抑制电离干扰
- 或使用贫燃火焰

**背景干扰**：
- 采用背景校正
- 或使用标准加入法

### 9.3 仪器维护

1. **燃烧器清洁**：定期清洁燃烧器狭缝
2. **雾化器调节**：保持最佳雾化状态
3. **点火顺序**：先开空气，后开乙炔
4. **关火顺序**：先关乙炔，后关空气

## 10. 质量控制

### 10.1 空白试验

每批样品应同时做试剂空白，空白值应稳定。

### 10.2 平行试验

每批样品应做平行试验，结果符合精密度要求。

### 10.3 加标回收

每批样品应进行加标回收试验，回收率要求：

| 元素 | 回收率范围 |
|------|------------|
| Ca | 85-115% |
| Fe | 85-115% |
| Zn | 85-115% |
| Na | 90-110% |
| K | 90-110% |
| Mg | 85-115% |
| Cu | 85-115% |
| Mn | 85-115% |

### 10.4 标准物质

每批样品应同时测定有证标准物质，结果应在标准值范围内。

## 11. 常见问题处理

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 灵敏度下降 | 空心阴极灯老化 | 更换新灯 |
| 读数不稳定 | 燃气压力波动 | 检查气路系统 |
| 背景值高 | 试剂不纯 | 使用高纯试剂 |
| 标准曲线线性差 | 标准溶液污染 | 重新配制标准液 |
| 回收率偏低 | 样品消化不完全 | 延长消化时间 |

## 12. 不同样品类型处理要点

### 12.1 乳粉

- 干法灰化，防止结块
- 灰化温度550℃，时间4-6小时
- 钙含量高，需适当稀释

### 12.2 液态乳

- 湿法消化，防止暴沸
- 消化至透明，近干
- 注意蛋白质完全分解

### 12.3 婴幼儿谷类辅助食品

- 干法灰化，预炭化完全
- 注意添加的矿物质强化剂
- 根据配料表预测含量范围

### 12.4 婴幼儿罐装辅助食品

- 湿法消化，高脂肪样品需先脱脂
- 肉类样品消化时间较长
- 注意盐分含量对钠、钾测定的影响

## 附录A: ICP-AES法（参考方法）

对于有条件的实验室，也可采用电感耦合等离子体发射光谱法(ICP-AES)同时测定多种元素。

**优点**：
- 多元素同时测定
- 线性范围宽
- 精密度好

**测定条件**：
- RF功率：1150W
- 等离子气流量：15L/min
- 辅助气流量：0.5L/min
- 雾化气流量：0.7L/min
