# ISO 14001:2015 环境管理体系 要求及使用指南

## 标准信息
- **标准编号**：ISO 14001:2015
- **标准名称**：环境管理体系 要求及使用指南
- **发布日期**：2015年9月15日
- **替代标准**：ISO 14001:2004
- **等同采用**：GB/T 24001-2016

---

## 引言

### 环境管理体系目的
本标准旨在为组织提供一个框架，用于保护环境和应对环境条件变化与社会经济需求变化之间平衡的需要。

### PDCA循环
本标准基于**策划-实施-检查-改进（PDCA）**运行模式：

```
策划（Plan）：建立所需的环境目标和过程，以实现与环境方针相一致的结果
实施（Do）：实施所策划的过程
检查（Check）：依据环境方针，包括承诺、环境目标和运行准则，对过程进行监测和测量，并报告结果
改进（Act）：采取措施持续改进
```

---

## 1 范围

本标准规定了组织可用于提升其环境绩效的环境管理体系的要求。本标准可供寻求以系统的方式管理其环境责任的组织使用，从而为可持续发展的环境支柱做出贡献。

本标准适用于任何规模、类型和性质的组织，并适用于组织基于生命周期观点确定的其能够控制或能够施加影响的环境因素。

## 3 术语和定义

### 3.1 管理体系 management system
组织（3.1.4）用于制订方针、目标（3.2.5）以及实现这些目标的过程（3.3.5）的一组相互关联或相互作用的要素。

### 3.2 环境管理体系 environmental management system
管理体系（3.1.1）的一部分，用于管理环境因素（3.2.2）、履行合规义务（3.2.9），并应对风险和机遇（3.2.11）。

### 3.3 环境方针 environmental policy
由最高管理者（3.1.5）正式发布的组织的环境绩效（3.4.11）的宗旨和方向。

### 3.4 组织 organization
为实现目标（3.2.5），由职责、权限和相互关系构成自身功能的一个人或一组人。

### 3.5 相关方 interested party
能够影响决策或活动、受决策或活动影响，或感觉自身受决策或活动影响的个人或组织（3.1.4）。

### 3.6 环境目标 environmental objective
组织（3.1.4）为实现与环境方针（3.2.3）相一致的具体结果而设定的目标。

### 3.7 污染预防 prevention of pollution
为了降低有害的环境影响（3.4.6）而采用（或综合采用）过程（3.3.5）、惯例、技术、材料、产品、服务或能源以避免、减少或控制任何类型的污染物或废物的产生、排放或废弃。

### 3.8 合规义务 compliance obligations
组织（3.1.4）必须遵守的法律法规要求（3.2.8）和组织必须或选择遵守的其他要求。

## 4 组织所处的环境

### 4.1 组织及其所处的环境
组织应确定与其宗旨相关，并影响其实现环境管理体系预期结果的能力的外部和内部事项。这些事项应包括：
- 受组织影响的或能够影响组织的环境条件
- 相关方的需求和期望

### 4.2 相关方的需求和期望
组织应确定：
1. 与环境管理体系有关的相关方
2. 这些相关方的相关需求和期望（即要求）
3. 这些需求和期望中哪些将成为其合规义务

### 4.3 确定环境管理体系范围
组织应确定环境管理体系的边界和适用性，以确定其范围。

确定范围时，组织应考虑：
1. 4.1中提到的内外部事项
2. 4.2中提到的合规义务
3. 其组织单元、职能和物理边界
4. 其活动、产品和服务
5. 其权限和行使控制的能力

### 4.4 环境管理体系
组织应按照本标准的要求，建立、实施、保持并持续改进环境管理体系，包括所需的过程及其相互作用，以确保其有效性。

## 5 领导作用

### 5.1 领导作用与承诺
最高管理者应通过以下方式证实其对环境管理体系的领导作用和承诺：
1. 对环境管理体系的有效性负责
2. 确保建立环境方针和环境目标
3. 确保环境管理体系要求融入组织的业务过程
4. 确保可获得环境管理体系所需的资源
5. 传达有效的环境管理的重要性
6. 确保环境管理体系实现其预期结果
7. 指导和支持员工为环境管理体系的有效性做出贡献
8. 促进持续改进
9. 支持其他相关管理人员在其职责范围内证实其领导作用

### 5.2 环境方针
最高管理者应建立、实施并保持环境方针，方针应：
1. 适合组织的宗旨和组织所处的环境，包括其活动、产品和服务的性质、规模和环境影响
2. 为制定环境目标提供框架
3. 包括保护环境的承诺，包括污染预防
4. 包括履行其合规义务的承诺
5. 包括持续改进环境管理体系以提高环境绩效的承诺

### 5.3 组织的角色、职责和权限
最高管理者应确保分配相关角色的职责和权限，并在组织内沟通。最高管理者应：
1. 分配职责和权限，以：
   - 确保环境管理体系符合本标准的要求
   - 向最高管理者报告环境管理体系的绩效，包括环境绩效

## 6 策划

### 6.1 应对风险和机遇的措施
#### 6.1.1 总则
组织应建立、实施和保持满足6.1.1至6.1.4要求所需的过程。策划的过程应用于：
1. 管理其重要环境因素
2. 确保合规义务
3. 增加有利机会
4. 实现环境目标

#### 6.1.2 环境因素
组织应在所界定的环境管理体系范围内，确定其活动、产品和服务中能够控制和能够施加影响的环境因素及其相关的环境影响。

确定环境因素时，组织应考虑：
1. 变更，包括已纳入计划的或新的开发，以及新的或修改的活动、产品和服务
2. 异常状况和可合理预见的紧急情况

组织应运用所建立的准则确定那些具有或可能具有重大环境影响的环境因素，即重要环境因素。适当时，组织应：
1. 在各层次和职能间沟通其重要环境因素
2. 更新确定的信息

#### 6.1.3 合规义务
组织应：
1. 确定并获取与其环境因素、合规义务相关的最新信息
2. 确定这些合规义务如何应用于组织
3. 在建立、实施、保持和持续改进其环境管理体系时考虑这些合规义务

#### 6.1.4 措施的策划
组织应策划：
1. 管理其重要环境因素的措施
2. 确保履行合规义务的措施
3. 应对风险和机遇的措施

### 6.2 环境目标及其实现的策划
#### 6.2.1 环境目标
组织应在其相关职能、层次和过程建立环境目标，环境目标应：
1. 与环境方针一致
2. 可测量（可行时）
3. 可监视
4. 得到沟通
5. 适时更新

组织应保持环境目标的文件化信息。

#### 6.2.2 策划实现环境目标
策划如何实现其环境目标时，组织应确定：
1. 要做什么
2. 需要什么资源
3. 由谁负责
4. 何时完成
5. 如何评价结果，包括用于监视实现其可测量的环境目标的进度所需的指标

## 7 支持

### 7.1 资源
组织应确定并提供为建立、实施、保持和持续改进环境管理体系所需的资源。

### 7.2 能力
组织应：
1. 确定在其控制下工作，且其工作影响环境绩效和影响履行合规义务能力的人员的能力
2. 基于适当的教育、培训或经历，确保这些人员是胜任的
3. 确定与环境因素和环境管理体系相关的培训需求
4. 采取措施以获得所必需的能力，并评价所采取措施的有效性

### 7.3 意识
组织应确保在其控制下工作的人员意识到：
1. 环境方针
2. 重要的环境因素和与其相关的实际或潜在的环境影响
3. 其对环境管理体系有效性的贡献，包括改进环境绩效的益处
4. 不符合环境管理体系要求的后果

### 7.4 信息交流
#### 7.4.1 总则
组织应建立、实施并保持与环境管理体系有关的内部和外部信息交流所需的过程，包括：
1. 信息交流的内容
2. 何时进行信息交流
3. 与谁进行信息交流
4. 如何进行信息交流

#### 7.4.2 内部信息交流
组织应：
1. 在组织各层次和职能间就环境管理体系的相关信息进行内部信息交流，适当时，包括环境管理体系变更
2. 确保其信息交流过程能够促使在其控制下工作的人员对持续改进做出贡献

#### 7.4.3 外部信息交流
组织应按其建立的信息交流过程并就其合规义务、环境因素和环境管理体系相关的信息进行外部信息交流，应：
1. 传入信息：予以接收并回应
2. 传出信息：保持一致的真实信息

### 7.5 文件化信息
#### 7.5.1 总则
组织的环境管理体系应包括：
1. 本标准要求的文件化信息
2. 组织确定的实现环境管理体系有效性所需的文件化信息

#### 7.5.2 创建和更新
创建和更新文件化信息时，组织应确保适当的：
1. 标识和说明（如标题、日期、作者或索引编号）
2. 格式（如语言、软件版本、图示）和媒介（如纸质、电子）
3. 评审和批准，以确保适宜性和充分性

#### 7.5.3 文件化信息的控制
环境管理体系及本标准所要求的文件化信息应予以控制，以确保：
1. 在需要的时间和场所均可获得并适用
2. 得到充分的保护（如防止失密、不当使用或完整性受损）

## 8 运行

### 8.1 运行策划和控制
组织应建立、实施、控制和保持满足环境管理体系要求所需的过程，通过：
1. 建立过程的准则
2. 按照准则实施过程控制
3. 保持必要的文件化信息，以确信过程已按策划得到实施

组织应控制策划的变更，评审非预期变更的后果，必要时采取措施减轻任何不利影响。

### 8.2 应急准备和响应
组织应建立、实施并保持对6.1.1中识别的潜在紧急情况的应急准备和响应所需的过程，组织应：
1. 策划对紧急情况的响应
2. 为预防或减轻紧急情况带来的后果而采取措施
3. 针对实际发生的紧急情况做出响应
4. 防止或减轻由紧急情况导致的不利后果
5. 在可行后，对响应措施的有效性进行评审

### 8.3 相关方要求（如适用）
组织应确保从外部供方获得的产品和服务符合要求，应：
1. 确定与产品和服务要求相关的输入
2. 与供方沟通这些要求
3. 确定对外部供方的控制类型和程度

### 8.4 文件控制和记录控制
组织应控制环境管理体系和本标准所要求的文件和记录，以确保其完整性和可用性。

## 9 绩效评价

### 9.1 监视、测量、分析和评价
#### 9.1.1 总则
组织应监视、测量、分析和评价其环境绩效。

组织应确定：
1. 需要监视和测量的内容
2. 监视、测量、分析和评价的方法，以确保有效的结果
3. 组织评价其环境绩效所依据的准则和适当的参数
4. 监视和测量的时机
5. 分析和评价的时机

#### 9.1.2 合规性评价
组织应策划、实施、保持和评价其履行合规义务的过程的频率和方法。

### 9.2 内部审核
#### 9.2.1 总则
组织应按计划的时间间隔实施内部审核，以提供环境管理体系是否符合以下信息：
1. 组织自身环境管理体系的要求
2. 本标准的要求

#### 9.2.2 内部审核方案
组织应建立、实施和保持一个或多个内部审核方案，包括：
1. 审核的频次、方法、职责、策划要求和报告
2. 审核方案的确定应考虑相关过程的环境重要性、影响组织的变化以及以往审核的结果

### 9.3 管理评审
最高管理者应按计划的时间间隔对组织的环境管理体系进行评审，以确保其持续的适宜性、充分性和有效性。

管理评审应包括：
1. 与环境方针符合性的程度
2. 目标指标的实现程度
3. 以往管理评审后续措施的状况
4. 外部相关方的交流，包括抱怨
5. 环境绩效
6. 资源充分性
7. 与环境管理体系有关的内、外部议题的变化

---

## 10 改进

### 10.1 不符合和纠正措施
当发生不符合时，组织应：
1. 对不符合做出响应，适当时：
   - 采取措施控制和纠正不符合
   - 处理后果，包括减轻不利的环境影响
2. 评价消除不符合原因的措施的需求，以防止不符合再次发生或在其他地方发生
3. 实施任何所需的措施
4. 评审所采取的任何纠正措施的有效性
5. 必要时，对环境管理体系进行变更

纠正措施应与不符合所产生的影响的严重程度相适应。

### 10.2 持续改进
组织应持续改进环境管理体系的适宜性、充分性和有效性，以提升环境绩效。

### A.1 结构、PDCA和适用性

#### A.1 PDCA循环
```
        +-----> 持续改进（第10章）
        |            |
        v            |
策划（第6章）<----+
    ^    |
    |    |
    |    v
实施（第7章、第8章）
    |    |
    |    v
检查（第9章）---+
```

### A.2 环境管理体系成功的关键因素
环境管理体系的成功实施需要：
1. 最高管理者的承诺
2. 全员参与
3. 基于风险的思维
4. 与其他管理体系整合
5. 生命周期观点
6. 持续改进

## 附录B（资料性附录）

### B.1 环境因素识别方法

#### B.1.1 输入-输出法
```
输入 → 过程 → 输出
  |      |      |
  v      v      v
资源  活动  产品
能源  服务  排放
        |
        v
      废弃物
```

#### B.1.2 环境因素分类
| 类型 | 示例 |
|------|------|
| 大气排放 | 粉尘、烟气、VOCs |
| 水体排放 | 废水、冷却水 |
| 土地污染 | 废物、泄漏 |
| 原料和自然资源 | 水、能源、原料 |
| 能源使用 | 电、燃料、蒸汽 |
| 能量释放 | 热能、辐射 |
| 废物和副产品 | 一般废物、危险废物 |
