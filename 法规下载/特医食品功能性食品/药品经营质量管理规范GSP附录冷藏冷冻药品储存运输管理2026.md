# 药品经营质量管理规范(GSP)附录
# 冷藏冷冻药品储存运输管理

**发布机关**: 国家药品监督管理局
**施行日期**: 2024年7月1日
**最新修订**: 2026年1月

---

## 第一部分：总则

### 第一条【立法目的】
为规范冷藏、冷冻药品的储存与运输管理，保证药品质量，保障人民群众用药安全、有效，根据《药品经营质量管理规范》，制定本附录。

### 第二条【适用范围】
本附录适用于中华人民共和国境内药品经营企业经营冷藏、冷冻药品的储存和运输活动。

### 第三条【术语定义】
- **冷藏药品**: 指需要2℃～10℃条件下储存和运输的药品
- **冷冻药品**: 指需要-25℃～-15℃条件下储存和运输的药品
- **冷链**: 指冷藏、冷冻药品从生产企业到使用单位的全过程温度控制链
- **温度敏感药品**: 指对温度变化敏感，需要在规定温度范围内储存和运输的药品

---

## 第二部分：储存管理

### 第四条【冷库设置要求】
1. 经营冷藏药品的企业应当设置与其经营规模相适应的冷库
2. 冷库应当具有良好的隔热、密封性能
3. 冷库应当配备备用制冷机组或备用电源
4. 冷库内应当合理划分区域，包括收货区、储存区、发货区

### 第五条【温湿度监测】
1. 冷库应当配备温湿度自动监测系统
2. 监测点应当均匀分布，每个冷库至少3个监测点
3. 监测系统应当能够实时记录、报警和存储数据
4. 监测数据应当至少保存5年

### 第六条【储存要求】
1. 冷藏药品应当储存在2℃～10℃的环境中
2. 冷冻药品应当储存在-25℃～-15℃的环境中
3. 药品应当与地面、墙面、屋顶保持适当距离
4. 药品堆码应当符合规定，保证空气流通

### 第七条【温度异常处理】
1. 当冷库温度超出规定范围时，系统应当自动报警
2. 工作人员应当在规定时间内到达现场处理
3. 应当记录温度异常情况和处理措施
4. 温度异常期间存放的药品应当进行质量评估

### 第八条【备用电源】
1. 冷库应当配备备用发电机组
2. 备用发电机组应当定期维护保养
3. 每年至少进行1次备用电源切换测试
4. 测试记录应当保存至少3年

---

## 第三部分：运输管理

### 第九条【运输设备要求】
1. 运输冷藏、冷冻药品应当使用专用冷藏车或保温箱
2. 冷藏车应当具备独立制冷系统
3. 保温箱应当配备蓄冷剂
4. 运输设备应当定期进行性能验证

### 第十条【温湿度监测】
1. 运输过程中应当对温度进行实时监测
2. 监测设备应当能够记录全程温度数据
3. 温度记录应当随货同行
4. 温度记录应当保存至少5年

### 第十一条【运输操作要求】
1. 装车前应当对运输设备进行预冷
2. 药品装车应当避免温度损失
3. 运输途中应当避免频繁开门
4. 卸货时应当尽量缩短暴露时间

### 第十二条【交接验收】
1. 收货单位应当检查温度记录
2. 温度符合规定的应当及时收货入库
3. 温度不符合规定的应当拒收
4. 拒收情况应当记录并报告

### 第十三条【应急预案】
1. 企业应当制定运输途中温度异常应急预案
2. 应当配备备用冷藏设备
3. 驾驶员应当掌握应急处置方法
4. 应当定期进行应急演练

---

## 第四部分：设备管理

### 第十四条【设备验证】
1. 新建、改建、扩建的冷库应当进行验证
2. 冷藏车、保温箱应当定期验证
3. 温湿度监测系统应当定期校准
4. 验证报告应当保存至少5年

### 第十五条【维护保养】
1. 制冷设备应当定期维护保养
2. 温湿度监测设备应当定期校准
3. 备用发电机组应当定期测试
4. 维护保养记录应当保存至少3年

### 第十六条【设备更新】
1. 设备老化或性能不符合要求的应当及时更新
2. 更新设备应当重新进行验证
3. 报废设备应当符合环保要求

---

## 第五部分：人员管理

### 第十七条【人员资质】
1. 冷链管理相关人员应当经过专业培训
2. 培训内容应当包括冷链知识和操作技能
3. 培训应当有记录并建立培训档案

### 第十八条【人员职责】
1. 冷链管理员负责冷链系统的日常管理
2. 仓储人员负责药品的储存养护
3. 运输人员负责药品的安全运输
4. 质量人员负责冷链质量的监督检查

---

## 第六部分：记录管理

### 第十九条【记录要求】
1. 冷库温度记录应当每日记录
2. 运输温度记录应当随货同行
3. 设备维护保养记录应当完整
4. 异常情况处理记录应当详细

### 第二十条【记录保存】
1. 温度记录应当至少保存5年
2. 设备验证记录应当至少保存5年
3. 维护保养记录应当至少保存3年
4. 异常处理记录应当至少保存5年

---

## 第七部分：附则

### 第二十一条【违规处理】
违反本附录规定的，按照《药品管理法》《药品管理法实施条例》等有关规定处理。

### 第二十二条【施行日期】
本附录自2024年7月1日起施行。

---

## 附件

### 附件一：冷藏冷冻药品储存温度要求

| 药品类别 | 储存温度 | 运输温度 | 备注 |
|---------|---------|---------|------|
| 生物制品 | 2℃～8℃ | 2℃～8℃ | 严禁冻结 |
| 血液制品 | 2℃～8℃ | 2℃～8℃ | 严禁冻结 |
| 疫苗 | 2℃～8℃ | 2℃～8℃ | 严禁冻结 |
| 胰岛素制剂 | 2℃～8℃ | 2℃～8℃ | 严禁冻结 |
| 某些抗生素 | 2℃～8℃ | 2℃～8℃ | 按说明书 |
| 冷冻药品 | -25℃～-15℃ | -25℃～-15℃ | 按说明书 |

### 附件二：冷链设备配置标准

| 设备类型 | 配置要求 | 验证周期 | 备注 |
|---------|---------|---------|------|
| 冷库 | 与规模相适应 | 3年 | 含备用机组 |
| 冷藏车 | 独立制冷系统 | 2年 | 配温度监测 |
| 保温箱 | 配蓄冷剂 | 1年 | 配温度监测 |
| 温度记录仪 | 自动记录 | 1年 | 定期校准 |

### 附件三：温度异常处理流程

1. 温度超标报警 → 2. 现场确认 → 3. 药品隔离 → 4. 质量评估 → 5. 处理决定 → 6. 记录报告
