# GB 28050-2024 食品安全国家标准 预包装食品营养标签通则

**标准编号**: GB 28050-2024
**实施日期**: 2025年1月1日
**替代标准**: GB 28050-2011
**发布部门**: 国家卫生健康委员会 国家市场监督管理总局

---

## 1. 范围

本标准适用于预包装食品营养标签的标示。
本标准不适用于保健食品及预包装特殊膳食用食品的营养标签标示。

---

## 2. 术语和定义

### 2.1 营养标签
预包装食品标签上向消费者提供食品营养信息和特性的说明，包括营养成分表、营养声称和营养成分功能声称。

### 2.2 营养素
食品中具有特定生理作用，能维持机体生长、发育、活动、繁殖以及正常代谢所需的物质，包括蛋白质、脂肪、碳水化合物、矿物质及维生素等。

### 2.3 核心营养素
营养标签中的核心营养素包括蛋白质、脂肪、碳水化合物、钠。

### 2.4 营养成分表
标示食品营养成分名称、含量和占营养素参考值（NRV）百分比的表格。

### 2.5 营养素参考值（NRV）
专用于食品营养标签，用于比较食品营养成分含量高低的参考值。

---

## 3. 基本要求

### 3.1 强制标示内容
所有预包装食品必须标示以下内容：
1. **能量**
2. **蛋白质**
3. **脂肪**
4. **碳水化合物**
5. **钠**

### 3.2 营养成分表达方式
| 项目 | 单位 |
|------|------|
| 能量 | 千焦(kJ) |
| 蛋白质 | 克(g) |
| 脂肪 | 克(g) |
| 碳水化合物 | 克(g) |
| 钠 | 毫克(mg) |

### 3.3 营养素参考值（NRV）

| 营养素 | NRV (每100g/mL) |
|--------|------------------|
| 能量 | 8400 kJ |
| 蛋白质 | 60 g |
| 脂肪 | 60 g |
| 碳水化合物 | 300 g |
| 钠 | 2000 mg |
| 钙 | 800 mg |
| 铁 | 15 mg |
| 锌 | 15 mg |
| 维生素A | 800 μgRE |
| 维生素C | 100 mg |

---

## 4. 营养声称

### 4.1 能量声称

| 声称方式 | 含量要求 |
|----------|----------|
| 无/零能量 | ≤ 17 kJ/100g (mL) |
| 低能量 | ≤ 170 kJ/100g |
| 减少能量 | 与参考食品相比减少25%以上 |

### 4.2 蛋白质声称

| 声称方式 | 含量要求 |
|----------|----------|
| 无/不含蛋白质 | ≤ 0.5g/100g |
| 低蛋白质 | ≤ 5g/100g (mL) |
| 蛋白质来源 | ≥10% NRV/100g |
| 高蛋白 | ≥20% NRV/100g |

### 4.3 脂肪声称

| 声称方式 | 含量要求 |
|----------|----------|
| 无/不含脂肪 | ≤ 0.5g/100g |
| 低脂肪 | ≤ 3g/100g |
| 脂肪来源 | ≥10% NRV/100g |
| 高脂肪 | ≥20% NRV/100g |

### 4.4 钠声称

| 声称方式 | 含量要求 |
|----------|----------|
| 无/不含钠 | ≤ 5mg/100g |
| 极低钠 | ≤ 40mg/100g |
| 低钠 | ≤ 120mg/100g |

---

## 5. 标示要求

### 5.1 位置
营养标签应标示在包装板面或标签的明显位置。

### 5.2 格式
应采用表格形式标示，表头应包括：
- "营养成分表"
- 项目栏
- 含量栏
- NRV%栏

### 5.3 字体
- 表头字体不小于5号字
- 营养成分名称字体不小于6号字
- 含量和NRV%数值字体不小于6号字

### 5.4 修约间隔

| 营养素 | 修约间隔 |
|--------|----------|
| 能量 | 1 kJ |
| 蛋白质、脂肪、碳水化合物 | 0.1 g |
| 钠 | 1 mg |

---

## 6. 豁免标示

以下食品可豁免标示营养标签：
1. 生鲜食品（如包装的生肉、生鱼、生蔬菜和水果、禽蛋等）
2. 乙醇含量≥0.5%的饮料酒类
3. 包装总表面积≤100cm²或最大表面积≤20cm²的食品
4. 现制现售的食品
5. 包装的饮用水
6. 每日食用量≤10g或10mL的预包装食品

---

## 7. 营养成分功能声称

允许使用的营养成分功能声称应符合以下要求：
- 仅可以使用标准中规定的用语
- 声称用语应规范、准确，不得误导消费者
- 应同时标示该营养成分含量

### 7.1 常用功能声称示例

| 营养素 | 功能声称 |
|--------|----------|
| 蛋白质 | 蛋白质是人体生命活动的主要物质基础 |
| 钙 | 钙是人体骨骼和牙齿的主要组成成分 |
| 铁 | 铁是血红细胞形成的重要元素 |
| 锌 | 锌是儿童生长发育必需的元素 |

---

## 8. 检测方法

营养成分检测应采用以下标准方法：

| 营养素 | 检测标准 |
|--------|----------|
| 蛋白质 | GB 5009.5 |
| 脂肪 | GB 5009.6 |
| 碳水化合物 | 计算法 |
| 钠 | GB 5009.91 |
| 能量 | 计算法 |

---

## 9. 过渡期要求

本标准自2025年1月1日起实施，设有2年过渡期：
- 2025年1月1日前生产的食品，可继续销售至保质期结束
- 2027年1月1日后生产的食品，必须符合本标准要求
