## 范围

本标准规定了食品中脂肪的测定方法。

本标准适用于各类食品中脂肪的测定。

**第一法 索氏抽提法**：适用于谷物、肉制品、乳制品等大多数食品，为仲裁法。

**第二法 酸水解法**：适用于各类食品，特别是加工食品和含糖量高的食品。

**第三法 碱水解法**：适用于乳及乳制品。

**第四法 盖勃法**：适用于乳及乳制品的快速测定。

---

## 术语和定义

### 2.1 脂肪 fat

食品中能溶于乙醚、石油醚等有机溶剂的物质的总称，包括甘油三酯、磷脂、固醇等。

### 2.2 粗脂肪 crude fat

用有机溶剂抽提食品所得到的物质，包括脂肪和类脂。

## 第一法：索氏抽提法（仲裁法）

### 3.1 原理

试样用无水乙醚或石油醚等溶剂抽提后，蒸发除去溶剂，称量抽提物的质量，即为脂肪含量。

### 3.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 无水乙醚 | 分析纯，不含过氧化物 |
| 石油醚 | 沸程30℃~60℃ |
| 海砂 | 通过80~100目筛 |
| 无水硫酸钠 | 分析纯 |

### 3.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 索氏抽提器 | - |
| 分析天平 | 感量0.1 mg |
| 电热恒温水浴锅 | 温度控制±2℃ |
| 电热恒温干燥箱 | 105℃±2℃ |
| 滤纸筒 | - |
| 干燥器 | 内附有效干燥剂 |

### 3.4 分析步骤

#### 3.4.1 滤纸筒制备

取滤纸筒，放入105℃干燥箱中干燥2 h，取出放入干燥器中冷却至室温，称量（精确至0.0001 g），直至恒重。

#### 3.4.2 试样处理

- **固体试样**：研磨均匀，准确称取2 g~5 g（精确至0.0001 g），全部移入滤纸筒内
- **半固体或液体试样**：准确称取5 g~10 g（精确至0.0001 g），加入适量海砂混匀，烘干后移入滤纸筒内

#### 3.4.3 抽提

将滤纸筒放入索氏抽提器的抽提筒内，连接已干燥至恒重的接收瓶，加入无水乙醚至瓶内容积的2/3处，于水浴上加热抽提，抽提速度以每分钟回流6~8次为宜，抽提6 h~10 h。

#### 3.4.4 回收溶剂

取下接收瓶，回收溶剂，待接收瓶内溶剂剩1 mL~2 mL时，于水浴上蒸干，再于105℃±2℃干燥箱干燥1 h，放入干燥器中冷却0.5 h后称量。重复操作直至恒重。

### 3.5 结果计算

```
X = (m₁ - m₀) / m × 100
```

其中：
- X：试样中脂肪的含量，单位为克每百克（g/100 g）
- m₁：接收瓶和脂肪的质量，单位为克（g）
- m₀：接收瓶的质量，单位为克（g）
- m：试样的质量，单位为克（g）

**结果保留三位有效数字**。

## 第二法：酸水解法

### 4.1 原理

试样经酸水解后，用有机溶剂抽提脂肪，蒸去溶剂，称量抽提物的质量。

### 4.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 盐酸 | 分析纯，8.3 mol/L |
| 乙醇 | 95% |
| 无水乙醚 | 分析纯 |
| 石油醚 | 沸程30℃~60℃ |

### 4.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 电热恒温水浴锅 | 温度控制±2℃ |
| 分析天平 | 感量0.1 mg |
| 锥形瓶 | 250 mL |
| 分液漏斗 | 250 mL |

### 4.4 分析步骤

#### 4.4.1 试样处理

准确称取2 g~5 g试样（精确至0.0001 g），置于250 mL锥形瓶中，加入50 mL 8.3 mol/L盐酸，混匀。

#### 4.4.2 水解

将锥形瓶置于70℃~80℃水浴中水解1 h，每10 min摇动一次。

#### 4.4.3 抽提

取出锥形瓶，加入25 mL乙醇，混匀。冷却后转移至分液漏斗中，用25 mL无水乙醚洗涤锥形瓶，洗液并入分液漏斗。振摇提取1 min，静置分层。将醚层放入已恒重的锥形瓶中。再用石油醚-无水乙醚（1:1）混合液重复提取2次，合并醚层。

#### 4.4.4 蒸干称重

将醚液蒸发至干，于105℃±2℃干燥箱干燥1 h，放入干燥器中冷却0.5 h后称量。重复操作直至恒重。

### 4.5 结果计算

同3.5。

## 第三法：碱水解法（罗兹-哥特里法）

### 5.1 原理

乳及乳制品经碱水解后，用乙醇-乙醚混合液抽提脂肪，蒸去溶剂，称量抽提物的质量。

### 5.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 氨水 | 分析纯，25% |
| 乙醇 | 95% |
| 无水乙醚 | 分析纯 |
| 石油醚 | 沸程30℃~60℃ |

### 5.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 抽脂瓶 | 100 mL |
| 分析天平 | 感量0.1 mg |
| 电热恒温水浴锅 | 温度控制±2℃ |

### 5.4 分析步骤

#### 5.4.1 试样处理

准确称取1 g~2 g试样（精确至0.0001 g），置于抽脂瓶中。

#### 5.4.2 水解

加入10 mL氨水，混匀，置于60℃水浴中加热15 min，不时振摇。

#### 5.4.3 抽提

取出冷却后，加入10 mL乙醇，混匀。加入25 mL无水乙醚，塞紧瓶塞，振摇1 min。加入25 mL石油醚，振摇30 s。静置分层，将上层醚液移入已恒重的烧瓶中。重复抽提2次，合并醚液。

#### 5.4.4 蒸干称重

将醚液蒸发至干，于105℃±2℃干燥箱干燥1 h，放入干燥器中冷却0.5 h后称量。重复操作直至恒重。

### 5.5 结果计算

同3.5。

## 第四法：盖勃法

### 6.1 原理

乳及乳制品经酸水解后，用异戊醇-乙醇混合液抽提脂肪，离心分离，直接读取脂肪体积。

### 6.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 硫酸 | 相对密度1.60~1.61 |
| 异戊醇 | 分析纯 |
| 乙醇 | 95% |

### 6.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 盖勃乳脂计 | 11 mL |
| 离心机 | 1000 r/min |
| 恒温水浴锅 | 65℃~70℃ |

### 6.4 分析步骤

#### 6.4.1 试样处理

准确吸取10 mL试样，置于盖勃乳脂计中。

#### 6.4.2 水解抽提

加入10 mL硫酸，再加入1 mL异戊醇，塞紧瓶塞，振摇至乳液完全溶解。

#### 6.4.3 离心分离

将乳脂计置于65℃~70℃水浴中保温5 min，取出擦干，放入离心机中以1000 r/min离心5 min。

#### 6.4.4 读取脂肪体积

取出乳脂计，置于65℃~70℃水浴中保温5 min，读取脂肪柱所占的刻度数，即为脂肪含量（g/100 mL）。

### 6.5 结果计算

直接读取脂肪体积即为脂肪含量。

## 方法选择指南

| 食品类别 | 推荐方法 | 说明 |
|----------|----------|------|
| 谷物及制品 | 索氏抽提法 | 仲裁法 |
| 油料作物 | 索氏抽提法 | 仲裁法 |
| 肉制品 | 酸水解法 | 含结合脂肪 |
| 乳及乳制品 | 碱水解法或盖勃法 | 盖勃法为快速法 |
| 蛋制品 | 酸水解法 | - |
| 调味品 | 酸水解法 | 含糖量高 |
| 婴幼儿配方食品 | 酸水解法 | 成分复杂 |
| 保健品 | 索氏抽提法或酸水解法 | 根据剂型选择 |

## 注意事项

1. **索氏抽提法**：
   - 乙醚易燃易爆，使用时应注意防火
   - 抽提时间应充分，确保抽提完全
   - 滤纸筒应提前干燥至恒重

2. **酸水解法**：
   - 水解温度和时间应严格控制
   - 水解后应充分冷却再进行抽提
   - 分层时应小心操作，避免损失醚层

3. **碱水解法**：
   - 氨水浓度应准确
   - 抽提时振摇应充分但不宜过于剧烈
   - 适用于乳及乳制品

4. **盖勃法**：
   - 硫酸密度应准确
   - 水浴温度应控制在65℃~70℃
   - 快速法，适用于常规检测

5. **通用注意事项**：
   - 有机溶剂易挥发，操作应在通风良好的环境中进行
   - 试样应充分混合均匀
   - 恒重操作应一致
   - 注意安全防护
