# 食品生产企业现场检查指南

**发布机关**: 国家市场监督管理总局
**施行日期**: 2024年6月1日
**最新修订**: 2026年1月

---

## 第一部分：总则

### 第一条【制定目的】
为规范食品生产企业现场检查工作，提高检查工作的科学性、规范性和有效性，制定本指南。

### 第二条【适用范围】
本指南适用于：
1. 食品生产许可现场核查
2. 食品生产企业日常监督检查
3. 食品生产企业飞行检查
4. 专项检查和跟踪检查

### 第三条【检查原则】
现场检查应当遵循以下原则：
1. **依法依规**：严格按照法律法规和标准进行检查
2. **客观公正**：以事实为依据，客观公正地评价
3. **风险导向**：关注高风险环节和关键控制点
4. **标本兼治**：既要发现问题，又要帮助改进

---

## 第二部分：检查准备

### 第四条【检查计划】
检查前应当制定检查计划：
1. 确定检查目的和范围
2. 确定检查时间和地点
3. 确定检查组成员和分工
4. 准备检查文件和记录表格

### 第五条【资料准备】
检查前应当收集以下资料：
1. 企业基本信息
2. 生产许可证书
3. 产品标准和技术文件
4. 历次检查记录
5. 投诉举报信息
6. 抽检不合格信息

### 第六条【检查工具】
检查前应当准备以下工具：
1. 检查记录表
2. 照相机或录像设备
3. 温湿度计
4. 洁净度检测设备
5. 快速检测设备（如需要）

---

## 第三部分：检查程序

### 第七条【首次会议】
检查到达企业后，应当召开首次会议：
1. 出示检查证件
2. 说明检查目的、依据和范围
3. 介绍检查组成员和分工
4. 确定企业陪同人员
5. 明确检查纪律和要求

### 第八条【现场检查】
现场检查应当按照以下程序进行：
1. **听取汇报**：听取企业关于生产质量管理情况的汇报
2. **现场查看**：查看厂区环境、厂房设施、设备等
3. **文件审查**：审查质量管理体系文件和记录
4. **人员访谈**：与企业负责人和关键岗位人员交流
5. **现场测试**：对关键环节进行现场测试

### 第九条【末次会议】
现场检查结束后，应当召开末次会议：
1. 通报检查情况
2. 确认检查事实
3. 听取企业陈述
4. 宣布检查结论（如适用）
5. 告知企业权利和申诉途径

---

## 第四部分：检查要点

### 第十条【厂区环境】
检查要点：
1. 厂区周围有无污染源
2. 厂区环境是否整洁
3. 厂区布局是否合理
4. 道路是否硬化、平整
5. 绿化是否符合要求
6. 废弃物处理是否规范

### 第十一条【厂房设施】
检查要点：
1. 厂房建筑结构是否完好
2. 地面是否平整、防滑、易清洁
3. 墙面、顶面是否光滑、不吸水、易清洁
4. 门窗是否密封、易清洁
5. 人流、物流是否分开
6. 洁净区是否符合要求
7. 通风、照明是否良好
8. 卫生设施是否齐全

### 第十二条【设备管理】
检查要点：
1. 设备配置是否满足生产需要
2. 设备材质是否符合食品级要求
3. 设备安装是否合理
4. 设备清洁是否彻底
5. 设备维护保养是否到位
6. 设备标识是否清晰
7. 计量器具是否校准

### 第十三条【物料管理】
检查要点：
1. 供应商是否经过审核
2. 物料验收是否规范
3. 物料储存条件是否符合要求
4. 物料标识是否清晰
5. 物料账卡物是否相符
6. 物料是否有虫害、霉变
7. 过期物料是否隔离

### 第十四条【生产管理】
检查要点：
1. 是否按工艺规程生产
2. 生产记录是否完整规范
3. 清场是否彻底
4. 中间控制是否到位
5. 人员卫生是否符合要求
6. 是否有交叉污染风险

### 第十五条【质量管理】
检查要点：
1. 质量管理体系文件是否完整
2. 检验设备是否齐全
3. 检验方法是否经过验证
4. 检验记录是否完整真实
5. 不合格品是否得到控制
6. 是否有质量事故
7. 是否有客户投诉

### 第十六条【人员管理】
检查要点：
1. 关键岗位人员是否具备资质
2. 人员健康检查是否按规定进行
3. 人员培训是否到位
4. 个人卫生是否符合要求
5. 工作服穿戴是否规范

### 第十七条【验证与确认】
检查要点：
1. 关键工艺是否验证
2. 清洁验证是否开展
3. 设备是否确认
4. 检验方法是否验证
5. 验证报告是否完整

### 第十八条【文件管理】
检查要点：
1. 文件是否齐全
2. 文件是否现行有效
3. 文件修订是否及时
4. 文件分发是否受控
5. 记录填写是否规范

---

## 第五部分：检查方法

### 第十九条【听取汇报】
通过听取企业汇报，了解：
1. 企业基本情况
2. 质量管理体系运行情况
3. 存在的问题和困难

### 第二十条【现场查看】
通过现场查看，发现：
1. 厂房设施的实际情况
2. 设备的运行状态
3. 人员的操作行为
4. 现场的卫生状况

### 第二十一条【文件审查】
通过文件审查，核实：
1. 质量管理体系的完整性
2. 生产活动的合规性
3. 记录的真实性和完整性

### 第二十二条【人员访谈】
通过人员访谈，了解：
1. 人员对质量管理体系的理解
2. 人员对岗位职责的认识
3. 人员对操作规程的掌握

### 第二十三条【现场测试】
通过现场测试，验证：
1. 关键控制点的有效性
2. 人员操作的规范性
3. 设备运行的可靠性

---

## 第六部分：缺陷分类

### 第二十四条【严重缺陷】
符合以下情形之一的，判定为严重缺陷：
1. 质量管理体系失效
2. 生产环境严重不符合要求
3. 使用不合格原料或非法添加
4. 擅自改变生产工艺
5. 检验记录造假
6. 发生质量事故隐瞒不报

### 第二十五条【主要缺陷】
符合以下情形之一的，判定为主要缺陷：
1. 质量管理体系文件不完善
2. 厂房设施维护不到位
3. 设备管理不规范
4. 生产过程控制不严格
5. 检验记录不完整
6. 人员培训不到位

### 第二十六条【一般缺陷】
符合以下情形之一的，判定为一般缺陷：
1. 文件管理不规范
2. 记录填写不规范
3. 现场卫生不达标
4. 标识不清晰
5. 个别人员操作不规范

---

## 第七部分：检查结论

### 第二十七条【通过检查】
同时满足以下条件的，评定为通过检查：
1. 未发现严重缺陷
2. 主要缺陷不超过3项
3. 一般缺陷不超过10项

### 第二十八条【限期整改】
符合以下情形之一的，评定为限期整改：
1. 未发现严重缺陷
2. 主要缺陷不超过5项
3. 一般缺陷不超过20项

### 第二十九条【不通过检查】
符合以下情形之一的，评定为不通过检查：
1. 发现严重缺陷
2. 主要缺陷超过5项
3. 一般缺陷超过20项

---

## 第八部分：检查报告

### 第三十条【报告内容】
检查报告应当包括：
1. 检查基本情况
2. 检查发现的问题
3. 缺陷描述和分类
4. 检查结论
5. 整改要求
6. 检查组签名

### 第三十一条【报告要求】
1. 内容真实、客观
2. 描述准确、具体
3. 结论明确、公正
4. 附必要的照片或证据

---

## 第九部分：附则

### 第三十二条【检查纪律】
检查人员应当遵守以下纪律：
1. 严格遵守保密规定
2. 不得收受企业财物
3. 不得利用检查谋取私利
4. 不得影响企业正常生产

### 第三十三条【施行日期】
本指南自2024年6月1日起施行。
