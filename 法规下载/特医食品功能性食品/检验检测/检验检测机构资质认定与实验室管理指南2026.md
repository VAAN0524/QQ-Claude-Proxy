# 检验检测机构资质认定与实验室管理指南（2026版）

## 文档说明

**创建日期**: 2026-02-28
**适用范围**: 特医食品、功能性食品生产企业实验室
**法规依据**: 检验检测机构资质认定管理办法、RB/T 214、CNAS-CL01等

---

## 一、检验检测机构资质认定（CMA）

### 1.1 法律依据

1. **检验检测机构资质认定管理办法**（2023年修订）
   - 发布部门：国家市场监督管理总局
   - 实施日期：2023年12月1日

2. **检验检测机构资质认定评审准则**（2023年修订）
   - 与RB/T 214-2017衔接
   - 统一评审要求

### 1.2 资质认定类型

| 类型 | 说明 | 适用范围 | 有效期 |
|------|------|----------|--------|
| CMA | 检验检测机构资质认定 | 向社会出具具有证明作用的数据结果 | 6年 |
| CAL | 实验室认可（自愿） | 提升实验室能力 | 5年 |
| CNAS | 实验室认可（国际互认） | 国际互认 | 4年 |

### 1.3 资质认定条件

#### 1.3.1 基本条件

1. **依法成立**：
   - 独立法人
   - 或授权的分支机构
   - 能承担法律责任

2. **人员要求**：
   - 技术负责人：具有中级职称，5年经验
   - 质量负责人：熟悉质量管理
   - 授权签字人：具有中级职称，3年经验
   - 检测人员：专业背景，培训上岗

3. **设施环境**：
   - 固定工作场所
   - 环境满足检测要求
   - 设施设备齐全

4. **管理体系**：
   - 建立管理体系
   - 有效运行
   - 持续改进

#### 1.3.2 能力要求

1. **检测能力**：
   - 依据标准方法
   - 或非标方法（验证）
   - 或自制方法（确认）

2. **仪器设备**：
   - 满足检测要求
   - 定期校准
   - 维护良好

3. **标准物质**：
   - 使用有证标准物质
   - 或次级标准物质
   - 量值溯源

### 1.4 申请流程

```
申请 → 受理 → 评审 → 审批 → 发证
(10日) (20日) (60日) (20日) (10日)
```

#### 1.4.1 申请材料

1. 资质认定申请书
2. 法律地位证明
3. 固定场所证明
4. 技术能力证明
5. 管理体系文件
6. 典型检测报告

#### 1.4.2 评审方式

1. **现场评审**：
   - 首次会议
   - 现场观察
   - 能力验证
   - 文件审查
   - 座谈会
   - 末次会议

2. **书面审查**：
   - 适用于变更、扩项
   - 不需要现场评审

### 1.5 证书管理

1. **证书内容**：
   - 机构名称
   - 地址
   - 检测能力范围
   - 有效期
   - 证书编号

2. **证书使用**：
   - 在认定范围内
   - 出具带CMA标志的报告
   - 不得超范围

3. **变更管理**：
   - 机构变更
   - 地址变更
   - 标准变更
   - 人员变更

4. **延续申请**：
   - 有效期届满3个月前申请
   - 重新评审
   - 符合条件换证

## 二、RB/T 214-2017 应用要求

### 2.1 标准结构

RB/T 214-2017 《检验检测机构资质认定能力评价 检验检测机构通用要求》

#### 2.1.1 标准条款

| 章节 | 内容 | 要点 |
|------|------|------|
| 4 | 要求 | 机构、人员、环境、设备 |
| 4.1 | 机构 | 依法成立、责任明确 |
| 4.2 | 人员 | 能力、培训、监督 |
| 4.3 | 环境 | 设施、环境监控 |
| 4.4 | 设备 | 校准、维护、期间核查 |
| 4.5 | 管理 | 体系文件、内审、管评 |
| 4.6 | 方法 | 方法验证、确认 |
| 4.7 | 样品 | 抽取、处置、记录 |
| 4.8 | 记录 | 信息完整、安全保密 |
| 4.9 | 结果 | 报告、判定、解释 |
| 4.10 | 分包 | 分包选择、监督 |
| 4.11 | 投诉 | 处理、改进 |
| 4.12 | 不符合 | 识别、纠正、预防 |
| 4.13 | 数据控制 | 准确性、完整性 |

### 2.2 关键要求解读

#### 2.2.1 人员（4.2）

1. **能力要求**：
   - 教育：相关专业
   - 培训：岗前培训、持续培训
   - 经验：年限要求
   - 技能：操作能力

2. **授权签字人**：
   - 中级职称以上
   - 3年以上经验
   - 具备专业判断能力
   - 了解检测方法

3. **监督**：
   - 新人员：前6个月持续监督
   - 在培人员：全程监督
   - 监督记录

#### 2.2.2 设备（4.4）

1. **校准**：
   - 周期校准（按计划）
   - 校准机构：具备资质
   - 校准结果：确认

2. **期间核查**：
   - 使用频繁
   - 漂移较大
   - 数据可疑
   - 便携设备
   - 核查方法：比对、标准物质

3. **标识**：
   - 三色标识管理
   * 绿色：合格（准用）
   * 黄色：准用（限用）
   * 红色：停用（禁用）

#### 2.2.3 方法（4.6）

1. **方法选择**：
   - 优先：国家标准、行业标准
   - 其次：国际标准
   - 最后：非标方法（需验证）

2. **方法验证**（标准方法）：
   - 验证内容：
     * 准确度（回收率）
     * 精密度（重复性）
     * 检出限/定量限
     * 线性范围
     * 干扰实验
   - 验证记录

3. **方法确认**（非标方法）：
   - 确认内容：
     * 方法开发
     * 实验室间比对
     * 不确定度评定
   - 客户同意

#### 2.2.4 样品（4.7）

1. **样品接收**：
   - 检查状态
   - 登记记录
   - 唯一标识

2. **样品流转**：
   - 流转记录
   - 状态标识
   * 待检：黄色
   * 在检：绿色
   * 已检：蓝色
   * 留样：红色

3. **样品处置**：
   - 按程序处置
   - 记录完整
   * 留样：保存期限
   * 退样：归还客户
   * 废弃：环保处理

#### 2.2.5 结果报告（4.9）

1. **报告内容**：
   - 标题：检测报告
   - 识别：唯一编号
   - 客户：名称地址
   - 样品：名称描述
   - 方法：标准编号
   - 日期：抽样/接收/检测
   - 结果：检测数据、判定
   - 签字：授权签字人
   - 标志：CMA标志

2. **报告格式**：
   - 规范统一
   - 信息完整
   - 数据准确

3. **报告修改**：
   - 发布后不得随意修改
   - 需要修改：追加文件
   - 记录修改原因

## 三、RB/T 215-2017 食品检验机构要求

### 3.1 特殊要求

#### 3.1.1 人员

1. **专业要求**：
   - 食品、化学、生物相关专业
   - 技术负责人：高级职称，10年经验
   - 授权签字人：高级职称，5年经验

2. **数量要求**：
   - 10人以上
   - 中级职称≥30%
   - 高级职称≥20%

#### 3.1.2 设施

1. **实验室分区**：
   - 理化检验室
   - 微生物检验室（洁净室）
   - 仪器分析室
   - 样品处理室
   - 标准品室
   - 留样室
   - 试剂室

2. **微生物实验室**：
   - 洁净度：万级（局部百级）
   - 缓冲间：一更、二更
   - 阳性对照室：单独设置
   - 压差：正压

3. **试剂储存**：
   - 易制毒：双人双锁
   - 危化品：防爆柜
   - 冷藏试剂：冰箱

#### 3.1.3 设备

1. **基础设备**：
   - 天平（感量0.1mg）
   - pH计
   - 电导率仪
   - 消解仪
   - 旋转蒸发仪

2. **大型仪器**：
   - 气相色谱（GC）
   - 液相色谱（HPLC）
   - 气质联用（GC-MS）
   - 液质联用（LC-MS/MS）
   - 原子吸收/荧光
   - ICP-MS

3. **微生物设备**：
   - 生物安全柜
   - 培养箱
   - 高压灭菌锅
   - 超净工作台

## 四、CNAS-CL01:2018 应用要求

### 4.1 CNAS认可

#### 4.1.1 认可意义

1. **能力证明**：
   - 技术能力
   - 管理水平
   - 国际互认

2. **市场准入**：
   - 招投标要求
   - 客户认可
   - 出口需要

#### 4.1.2 认可流程

```
申请 → 受理 → 文件审查 → 现场评审 → 验证试验 → 认可评定 → 发证
```

### 4.2 关键差异

| 项目 | CMA | CNAS |
|------|-----|------|
| 性质 | 强制 | 自愿 |
| 依据 | RB/T 214 | CNAS-CL01 |
| 有效期 | 6年 | 4年 |
| 范围 | 中国 | 国际互认 |
| 监督 | 年度报告 | 定期监督+复评审 |

### 4.3 CNAS-CL01:2018 要点

#### 4.3.1 公正性（4.1）

1. **识别风险**：
   - 利益冲突
   - 商业压力
   - 内部影响

2. **管理措施**：
   - 声明公正性
   - 保密承诺
   - 人员培训

#### 4.3.2 不确定度（7.6）

1. **评定要求**：
   - 每个项目都要评定
   - 方法验证时评定
   - 报告中注明

2. **评定方法**：
   - GUM法
   - 蒙特卡洛
   - 经验公式

3. **表示方式**：
   - 扩展不确定度：U = k·uc
   - k=2（95%置信水平）
   - 保留2位有效数字

#### 4.3.3 确认方法（7.2）

1. **标准方法**：验证
2. **非标方法**：确认
3. **实验室制定方法**：确认
4. **超出范围**：确认

#### 4.3.4 能力验证（7.7）

1. **参加要求**：
   - 获认可前：参加能力验证
   - 获认可后：每年参加
   - 每个项目：3年内至少1次

2. **不满意处理**：
   - 原因分析
   - 纠正措施
   - 验证有效
   - 再次参加

## 五、质量管理体系

### 5.1 体系文件

#### 5.1.1 文件结构

```
质量手册（第一层次）
↓
程序文件（第二层次）
↓
作业指导书（第三层次）
↓
记录表格（第四层次）
```

#### 5.1.2 质量手册

内容包括：
1. 质量方针
2. 质量目标
3. 组织机构
4. 职责权限
5. 体系要求
6. 程序文件目录

#### 5.1.3 程序文件（必备）

1. 文件控制程序
2. 记录控制程序
3. 内部审核程序
4. 管理评审程序
5. 人员培训程序
6. 设备校准程序
7. 方法验证程序
8. 样品管理程序
9. 不符合工作处理程序
10. 纠正措施程序

### 5.2 内部审核

#### 5.2.1 审核要求

1. **频次**：每年至少1次
2. **覆盖**：全部要素、全部部门
3. **审核员**：具备资格，独立于被审部门

#### 5.2.2 审核流程

```
计划 → 现场 → 报告 → 跟踪验证
```

1. **审核计划**：
   - 目的、范围
   - 审核组
   - 日程安排
   - 审核依据

2. **现场审核**：
   - 首次会议
   - 收集证据
   - 发现不符合
   - 末次会议

3. **审核报告**：
   - 审核概述
   - 不符合项
   - 审核结论
   - 改进建议

4. **跟踪验证**：
   - 整改期限
   - 验证关闭

### 5.3 管理评审

#### 5.3.1 评审要求

1. **频次**：每12个月1次
2. **输入**：
   - 内审结果
   - 客户投诉
   - 能力验证
   - 改进建议
   - 资源需求

3. **输出**：
   - 体系有效性
   - 改进决策
   - 资源需求

## 六、数据完整性

### 6.1 ALCOA原则

| 原则 | 含义 | 要求 |
|------|------|------|
| A - Attributable | 可归属 | 操作可追溯到人 |
| L - Legible | 清晰可读 | 记录清晰、持久 |
| C - Contemporaneous | 同时性 | 记录与操作同时进行 |
| O - Original | 原始性 | 原始记录或副本 |
| A - Accurate | 准确性 | 数据准确、真实 |

### 6.2 电子数据

1. **系统验证**：
   - 安装确认（IQ）
   - 运行确认（OQ）
   - 性能确认（PQ）

2. **权限管理**：
   - 分级授权
   - 密码保护
   - 独立账号

3. **审计追踪**：
   - 记录所有操作
   - 不可删除
   - 定期审查

4. **数据备份**：
   - 定期备份
   - 异地存储
   - 恢复测试

## 七、质量控制

### 7.1 质控手段

1. **空白试验**：每批
2. **平行样**：10%
3. **加标回收**：定期
4. **对照试验**：每批
5. **质控图**：长期监控

### 7.2 能力验证

1. **参加频次**：
   - 主要项目：每年1次
   - 次要项目：3年1次

2. **结果处理**：
   - 满意：继续监控
   - 可疑：原因分析
   - 不满意：纠正措施

### 7.3 实验室间比对

1. **比对目的**：
   - 验证能力
   - 识别问题
   - 持续改进

2. **比对方式**：
   - 样品分割
   - 比对试验
   - 巡回比对
