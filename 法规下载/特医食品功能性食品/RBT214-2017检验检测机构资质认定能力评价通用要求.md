# RB/T 214-2017 检验检测机构资质认定能力评价 检验检测机构通用要求

## 发布信息

- **标准编号**：RB/T 214-2017
- **发布日期**：2017年10月16日
- **实施日期**：2018年5月1日
- **发布机构**：国家认证认可监督管理委员会
- **替代标准**：部分替代《实验室资质认定评审准则》

---

## 前言

本标准按照GB/T 1.1-2009给出的规则起草。

本标准由国家认证认可监督管理委员会提出并归口。

本标准起草单位：国家认证认可监督管理委员会、中国合格评定国家认可委员会等。

本标准为首次发布。

---

## 1 范围

本标准规定了检验检测机构资质认定的通用要求，适用于为司法机关作出的裁决、行政机关作出的行政决定、仲裁机构作出的仲裁决定、为社会经济活动出具具有证明作用的数据和结果的检验检测机构。

## 3 术语和定义

**3.1 检验检测机构**
依法成立，依据相关标准或者技术规范，利用仪器设备、环境设施等技术条件和专业技能，对产品或者法律法规规定的特定对象进行检验检测的专业技术组织。

**3.2 资质认定**
省级以上质量技术监督部门依据有关法律法规和标准、技术规范的规定，对检验检测机构的基本条件和技术能力是否符合法定要求实施的评价许可。

**3.3 能力验证**
利用实验室间比对确定实验室的校准/检测能力。

**3.4 测量审核**
实验室对被测物品进行实际测试，将测试结果与参考值进行比较的活动。

**3.5 方法验证**
提供客观证据证明检验检测方法满足预期用途的要求。

**3.6 方法确认**
提供客观证据证明特定方法满足预期用途的要求。

## 4 要求

### 4.1 机构

**4.1.1 法律地位**

检验检测机构或者其所在的组织应当具有明确的法律地位，能够承担法律责任。

**4.1.2 主体责任**

检验检测机构应当对其出具的检验检测数据、结果负责，并承担相应法律责任。

**4.1.3 独立性**

检验检测机构应当独立、客观、公正地从事检验检测活动，不受来自内外部、商业、财务等方面的干预和影响。

### 4.2 人员

**4.2.1 人员要求**

| 岗位 | 职责 | 资质要求 |
|------|------|----------|
| 技术负责人 | 全面技术工作 | 中级以上职称，5年以上经验 |
| 质量负责人 | 质量管理体系 | 中级以上职称，熟悉质量管理 |
| 授权签字人 | 签发报告 | 中级以上职称，3年以上经验 |
| 检验检测人员 | 检验检测操作 | 专业培训考核合格 |
| 监督人员 | 质量监督 | 熟悉检验检测方法和程序 |

**4.2.2 人员能力**

1. 具有相应的教育背景
2. 具备相应的专业技能
3. 经过必要的培训考核
4. 持续保持专业能力

**4.2.3 人员培训**

1. 制定年度培训计划
2. 实施培训并记录
3. 评价培训效果
4. 建立人员技术档案

**4.2.4 人员监督**

1. 制定监督计划
2. 实施持续监督
3. 记录监督结果
4. 采取必要措施

### 4.3 工作场所与环境

**4.3.1 场所要求**

1. 固定的工作场所
2. 满足检验检测需要
3. 环境条件符合要求
4. 保持良好内务管理

**4.3.2 环境控制**

| 环境因素 | 控制要求 | 监控频率 |
|----------|----------|----------|
| 温度 | 按方法标准要求 | 每日记录 |
| 湿度 | 按方法标准要求 | 每日记录 |
| 洁净度 | 按方法标准要求 | 定期监测 |
| 电磁干扰 | 符合设备要求 | 定期检查 |
| 照明 | 满足工作需要 | 定期检查 |
| 噪声 | 符合职业健康要求 | 定期监测 |

**4.3.3 设施管理**

1. 设施应满足检验检测需要
2. 设施应定期维护保养
3. 关键设施应有备份
4. 设施变更应经过验证

### 4.4 设备设施

**4.4.1 设备配备**

1. 配备满足检验检测需要的设备
2. 设备精度应满足方法要求
3. 设备应经过检定/校准
4. 设备应有明确标识

**4.4.2 设备管理**

| 管理环节 | 管理要求 | 记录要求 |
|----------|----------|----------|
| 采购验收 | 符合规范、验收合格 | 验收记录 |
| 检定校准 | 按计划、有证溯源 | 校准证书 |
| 使用维护 | 操作规范、维护及时 | 使用记录 |
| 期间核查 | 定期核查、确认状态 | 核查记录 |
| 报废处置 | 履行手续、合理处置 | 处置记录 |

**4.4.3 设备标识**

1. **合格标识**（绿色）：经检定/校准合格
2. **准用标识**（黄色）：部分功能可用，有限使用
3. **停用标识**（红色）：停用或待修

**4.4.4 设备档案**

每台设备应建立档案，包括：
1. 名称、型号、编号
2. 制造商、购置日期
3. 使用说明书
4. 检定/校准证书
5. 使用和维护记录
6. 期间核查记录
7. 维修和报废记录

### 4.5 管理体系

**4.5.1 体系要求**

检验检测机构应当建立、实施和保持与其活动范围相适应的管理体系，包括：

1. 管理方针和目标
2. 管理手册和程序文件
3. 作业指导书和记录表格
4. 内部审核和管理评审

**4.5.2 文件控制**

1. 文件应当经过批准
2. 文件应当现行有效
3. 文件应当易于获取
4. 文件应当定期评审

**4.5.3 记录控制**

1. 记录应当真实、准确、完整
2. 记录应当及时填写
3. 记录应当规范清晰
4. 记录应当妥善保存

### 4.6 要求、标书和合同的评审

**4.6.1 评审要求**

在接受客户要求、标书或合同时，应当：

1. 明确客户要求
2. 评估自身能力
3. 确认方法和资源
4. 形成评审记录

**4.6.2 合同修改**

合同的任何修改应当：

1. 与客户沟通确认
2. 重新进行能力评估
3. 通知所有相关人员
4. 保存修改记录

### 4.7 分包

**4.7.1 分包条件**

1. 客户书面同意
2. 分包方具备能力
3. 明确分包责任
4. 保存分包记录

**4.7.2 分包管理**

1. 对分包方进行评估
2. 建立分包方档案
3. 定期评价分包方
4. 向客户报告分包情况

### 4.8 服务和供应品的采购

**4.8.1 供应商管理**

1. 建立供应商档案
2. 评估供应商能力
3. 定期评价供应商
4. 保存供应商记录

**4.8.2 采购验收**

1. 明确采购要求
2. 规定验收标准
3. 实施采购验收
4. 保存验收记录

### 4.9 服务客户

**4.9.1 服务内容**

1. 提供咨询服务
2. 允许客户监督
3. 保护客户信息
4. 处理客户投诉

**4.9.2 客户沟通**

1. 保持有效沟通
2. 及时反馈信息
3. 征求客户意见
4. 改进服务质量

### 4.10 投诉和申诉

**4.10.1 投诉处理**

1. 建立投诉处理程序
2. 及时受理投诉
3. 调查处理投诉
4. 反馈处理结果

**4.10.2 申诉处理**

1. 建立申诉处理程序
2. 客观公正处理
3. 给出处理决定
4. 保存处理记录

### 4.11 不符合工作控制

**4.11.1 不符合识别**

1. 及时识别不符合
2. 评估不符合影响
3. 记录不符合情况
4. 通知相关方

**4.11.2 不符合处理**

1. 停止不符合工作
2. 分析不符合原因
3. 采取纠正措施
4. 验证纠正效果

### 4.12 纠正措施、应对风险和机遇的措施和改进

**4.12.1 纠正措施**

1. 识别不符合原因
2. 制定纠正措施
3. 实施纠正措施
4. 验证措施效果

**4.12.2 风险管理**

1. 识别风险和机遇
2. 评估风险程度
3. 制定应对措施
4. 监控措施效果

**4.12.3 持续改进**

1. 收集改进信息
2. 分析改进机会
3. 实施改进措施
4. 评价改进效果

### 4.13 记录控制

**4.13.1 记录要求**

1. 真实准确
2. 及时填写
3. 规范清晰
4. 妥善保存

**4.13.2 记录保存期限**

| 记录类型 | 保存期限 | 备注 |
|----------|----------|------|
| 技术记录 | 6年 | 根据需要延长 |
| 质量记录 | 6年 | 根据需要延长 |
| 设备记录 | 长期保存 | 直至设备报废 |
| 人员记录 | 长期保存 | 直至人员离职 |

### 4.14 内部审核

**4.14.1 审核要求**

1. 每年至少进行一次
2. 覆盖所有要素和部门
3. 由经过培训的内审员进行
4. 审核员应独立于被审核活动

**4.14.2 审核程序**

1. 制定审核计划
2. 实施现场审核
3. 编写审核报告
4. 跟踪验证整改

### 4.15 管理评审

**4.15.1 评审要求**

1. 最高管理者主持
2. 每年至少进行一次
3. 评价管理体系的适宜性、充分性、有效性
4. 形成评审决议

**4.15.2 评审输入**

1. 政策和目标的适用性
2. 内外部审核结果
3. 客户反馈和投诉
4. 纠正和预防措施
5. 风险和机遇
6. 改进建议

**4.15.3 评审输出**

1. 管理体系改进
2. 资源需求
3. 政策和目标调整
4. 其他相关决定

### 4.16 方法的选择、验证和确认

**4.16.1 方法选择**

优先选择国家标准、行业标准、国际标准，其次选择公认的方法或自行制定的方法。

**4.16.2 方法验证**

使用标准方法前，应当验证：

1. 设备能力满足要求
2. 环境条件符合要求
3. 人员能力满足要求
4. 能够获得正确结果

**4.16.3 方法确认**

使用非标方法前，应当确认：

1. 方法的适用性
2. 方法的可靠性
3. 方法的准确性
4. 方法的稳健性

### 4.17 测量不确定度

**4.17.1 评定要求**

1. 识别不确定度来源
2. 评定各不确定度分量
3. 计算合成不确定度
4. 报告扩展不确定度

**4.17.2 报告要求**

1. 在检测报告中给出测量不确定度
2. 不确定度表示为正值
3. 不确定度不超过两位有效数字
4. 结果和不确定度单位一致

### 4.18 数据信息

**4.18.1 数据管理**

1. 建立数据管理程序
2. 确保数据准确完整
3. 保护数据安全
4. 定期备份数据

**4.18.2 信息系统**

1. 确保系统安全可靠
2. 制定系统操作规程
3. 定期维护系统
4. 备份系统数据

### 4.19 抽样

**4.19.1 抽样计划**

1. 制定抽样计划
2. 明确抽样方法
3. 确定抽样数量
4. 规定抽样部位

**4.19.2 抽样实施**

1. 按照计划抽样
2. 保证样品代表性
3. 做好抽样记录
4. 标识样品状态

### 4.20 样品处置

**4.20.1 样品接收**

1. 检查样品状态
2. 记录样品信息
3. 标识样品编号
4. 存放规定区域

**4.20.2 样品流转**

1. 建立流转记录
2. 样品保持标识
3. 确保样品完好
4. 防止样品混淆

**4.20.3 样品保存**

1. 按要求保存样品
2. 定期检查样品
3. 保存期满处理
4. 记录处理情况

### 4.21 质量控制

**4.21.1 质控方法**

| 质控方法 | 应用场景 | 频率 |
|----------|----------|------|
| 空白试验 | 检测过程 | 每批/每天 |
| 平行样 | 精密度控制 | 10%-20% |
| 加标回收 | 准确度控制 | 定期 |
| 质控样 | 系统控制 | 每批/每天 |
| 能力验证 | 外部质量控制 | 按计划 |

**4.21.2 质控标准**

1. 制定质控计划
2. 建立质控标准
3. 绘制质控图
4. 评价质控结果

### 4.22 能力验证

**4.22.1 参加要求**

1. 每个主要检测领域每年至少参加1次
2. 优先选择CNAS认可的能力验证计划
3. 参加频次满足资质认定要求
4. 保存能力验证记录

**4.22.2 结果处理**

| 结果类型 | 处理措施 |
|----------|----------|
| 满意 | 正常开展检验检测 |
| 可疑 | 原因分析，补充验证 |
| 不满意 | 停止相关项目，整改验证 |

### 4.23 结果报告

**4.23.1 报告要求**

1. 格式规范、信息完整
2. 结论明确、依据充分
3. 数据准确、用语规范
4. 签字盖章、标识正确

**4.23.2 报告内容**

1. 标题和编号
2. 客户信息
3. 样品信息
4. 检测依据
5. 检测结果
6. 检测结论
7. 检测日期
8. 签字盖章

**4.23.3 报告修改**

1. 报告不得随意修改
2. 发现错误应发布新报告
3. 注明原报告作废
4. 保存修改记录

## 5 管理要求

### 5.1 行政管理

**5.1.1 机构资质**

1. 取得营业执照
2. 通过资质认定
3. 在认定范围内执业
4. 接受监督管理

**5.1.2 人员管理**

1. 建立人员档案
2. 实施岗前培训
3. 定期考核评价
4. 保持人员稳定

**5.1.3 信息安全**

1. 建立信息安全制度
2. 保护客户信息
3. 防止信息泄露
4. 定期安全检查

### 5.2 监督管理

**5.2.1 监督检查**

1. 定期监督检查
2. 专项监督检查
3. 投诉处理检查
4. 飞行检查

**5.2.2 违规处理**

| 违规情形 | 处理措施 |
|----------|----------|
| 超范围执业 | 责令改正，情节严重撤销 |
| 出具虚假报告 | 撤销资质，追究责任 |
| 能力验证不满意 | 责令整改，暂停相关项目 |
| 管理混乱 | 责令整改，暂停相关项目 |
