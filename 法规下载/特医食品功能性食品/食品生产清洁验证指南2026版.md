# 食品生产设备清洁验证指南

## 指南基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家药品监督管理局 |
| 发布年份 | 2023年 |
| 适用范围 | 食品、特医食品、保健食品生产设备 |
| 指南性质 | 技术指导原则 |

## 1. 总则

### 1.1 目的

为规范食品生产设备清洁验证工作，确保设备清洁效果满足要求，防止交叉污染，保障产品质量和安全，制定本指南。

### 1.2 适用范围

本指南适用于：
1. 特殊医学用途配方食品生产设备
2. 保健食品生产设备
3. 其他食品生产设备
4. 与食品直接接触的设备、容器、管道

### 1.3 基本原则

| 原则 | 说明 |
|------|------|
| **验证先行** | 清洁方法使用前必须验证 |
| **科学合理** | 验证方案科学可行 |
| **数据可靠** | 验证数据真实可靠 |
| **持续改进** | 持续优化清洁方法 |

## 2. 清洁方法开发

### 2.1 产品特性分析

#### 2.1.1 产品残留风险

| 产品类型 | 主要残留物 | 清洁难度 |
|----------|------------|----------|
| 高蛋白产品 | 蛋白质 | 高（易附着） |
| 高脂肪产品 | 油脂 | 中 |
| 高糖产品 | 糖类 | 低 |
| 粉末产品 | 粉尘 | 中 |
| 特医食品 | 营养成分 | 中-高 |

#### 2.1.2 残留物特性

| 特性 | 说明 | 清洁影响 |
|------|------|----------|
| 溶解性 | 水/溶剂中的溶解能力 | 决定清洁剂选择 |
| 粘附性 | 对设备表面的粘附能力 | 影响清洁力度 |
| 热稳定性 | 加热后的变化 | 影响热清洁效果 |
| 反应性 | 与清洁剂的反应 | 产生副产物 |

### 2.2 清洁剂选择

#### 2.2.1 清洁剂类型

| 类型 | 适用场景 | 优缺点 |
|------|----------|--------|
| **碱性清洁剂** | 油脂、蛋白质 | 去污力强，可能损伤表面 |
| **酸性清洁剂** | 矿物质沉积 | 去除水垢，有腐蚀性 |
| **中性清洁剂** | 日常清洁 | 温和，去污力一般 |
| **酶清洁剂** | 蛋白质残留 | 特异性强，成本高 |
| **消毒剂** | 微生物控制 | 不去除有机物 |

#### 2.2.2 选择原则

| 原则 | 说明 |
|------|------|
| 有效性 | 能有效去除残留物 |
| 安全性 | 对设备、人员安全 |
| 兼容性 | 与设备材质兼容 |
| 经济性 | 成本合理 |
| 环保性 | 环境友好 |

### 2.3 清洁方法确定

#### 2.3.1 手工清洁

| 步骤 | 操作要点 |
|------|----------|
| 1. 预冲洗 | 去除松散残留物 |
| 2. 清洁剂浸泡 | 根据清洁剂要求 |
| 3. 刷洗 | 使用合适工具 |
| 4. 冲洗 | 彻底冲洗清洁剂 |
| 5. 消毒 | 必要时消毒 |
| 6. 最终冲洗 | 去除消毒剂 |

#### 2.3.2 CIP（原位清洗）

| 参数 | 说明 |
|------|------|
| 流速 | 保证湍流（Re>10000） |
| 温度 | 根据清洁剂要求 |
| 浓度 | 按清洁剂说明书 |
| 时间 | 根据验证结果 |
| 程序 | 预冲洗→清洗→中间冲洗→消毒→最终冲洗 |

#### 2.3.3 COP（拆卸清洗）

| 适用设备 | 说明 |
|----------|------|
| 小型设备 | 拆卸后浸泡清洗 |
| 复杂部件 | 拆卸后手工清洗 |
| 管道配件 | 拆卸后清洗消毒 |

### 2.4 清洁参数优化

| 参数 | 优化方法 | 目标 |
|------|----------|------|
| 清洁剂浓度 | 试验确定 | 最低有效浓度 |
| 清洁时间 | 试验确定 | 最短有效时间 |
| 清洁温度 | 试验确定 | 适宜温度 |
| 流速/机械力 | 试验确定 | 足够剪切力 |

## 4. 取样方法

### 4.1 取样位置选择

#### 4.1.1 取样位置确定原则

| 原则 | 说明 |
|------|------|
| 最难清洁部位 | 选择难以清洁的部位 |
| 产品接触表面 | 与产品直接接触的表面 |
| 关键部位 | 影响产品质量的部位 |
| 代表性部位 | 能代表整体清洁效果 |

#### 4.1.2 常见取样位置

| 设备类型 | 取样位置 | 原因 |
|----------|----------|------|
| 混合机 | 搅拌桨、罐壁、出料口 | 难清洁部位 |
| 管道系统 | 弯头、阀门、接头 | 残留物聚集处 |
| 烘干设备 | 加热管、托盘 | 温度高易残留 |
| 包装机 | 充填嘴、封口部位 | 产品残留处 |

### 4.2 取样方法

#### 4.2.1 棉签擦拭法

**适用**：平坦表面、小面积

| 步骤 | 操作要点 |
|------|----------|
| 1. 标记取样区域 | 固定面积（如10×10cm） |
| 2. 准备棉签 | 无菌棉签浸润溶剂 |
| 3. 擦拭 | 定向擦拭，适当压力 |
| 4. 重复擦拭 | 更换棉签，重复2-3次 |
| 5. 洗脱 | 将棉签放入溶剂 |
| 6. 检测 | 按检测方法检测 |

**计算**：
```
回收量 = 检测量 × (溶剂体积/洗脱体积) / (取样面积)
单位：μg/cm² 或 mg/cm²
```

#### 4.2.2 淋洗水法

**适用**：管道、罐体等可冲洗部位

| 步骤 | 操作要点 |
|------|----------|
| 1. 清洁后最后一次淋洗 | 收集淋洗水 |
| 2. 取样 | 取代表性样品 |
| 3. 检测 | 按检测方法检测 |
| 4. 计算 | 根据淋洗水量和表面积计算 |

**计算**：
```
残留浓度 = 检测浓度 × (淋洗水量/总表面积)
单位：mg/L 或 μg/mL
```

#### 4.2.3 冲洗液法

**适用**：不能接触取样的部位

| 步骤 | 操作要点 |
|------|----------|
| 1. 准备冲洗液 | 纯水或适宜溶剂 |
| 2. 冲洗 | 用冲洗液冲洗取样部位 |
| 3. 收集 | 收集冲洗液 |
| 4. 检测 | 按检测方法检测 |

### 4.3 取样回收率

#### 4.3.1 回收率验证

| 要求 | 说明 |
|------|------|
| 回收率 | ≥70% |
| RSD | ≤15% |
| 验证方法 | 在未清洁设备上涂布已知量残留物 |

#### 4.3.2 回收率计算

```
回收率 = (实际检测量/理论涂布量) × 100%
校正后残留量 = 实际检测量/回收率
```

## 5. 检测方法

### 5.1 检测项目

| 检测类型 | 检测项目 | 检测方法 |
|----------|----------|----------|
| **化学残留** | 产品主成分、清洁剂、TOC | HPLC、UV、TOC仪 |
| **微生物** | 菌落总数、指定菌 | 平板计数法 |
| **内毒素** | 细菌内毒素 | LAL法 |
| **可见异物** | 目视检查 | 目视法 |

### 5.2 可接受标准

#### 5.2.1 化学残留标准

| 标准 | 限值 | 说明 |
|------|------|------|
| **残留限度** | ≤10 ppm或≤0.1% | 常用标准 |
| **生物活性限** | ≤日剂量的0.1% | 药品标准 |
| **毒性限** | ≤TD01的0.1% | 基于毒性 |
| **TOC限** | ≤与清洁前基线差值≤50ppb | TOC检测 |

#### 5.2.2 微生物标准

| 标准 | 限值 | 说明 |
|------|------|------|
| 菌落总数 | ≤100 CFU/棉签或≤100 CFU/mL | 洁净区要求 |
| 指定菌 | 不得检出 | 致病菌 |
| **10/100规则** | 洁净区：≤10 CFU/25cm² | 非洁净区：≤100 CFU/25cm² |

#### 5.2.3 目视检查标准

| 检查项目 | 合格标准 |
|----------|----------|
| 可见残留 | 无可见残留物 |
| 清洁度 | 表面光洁、无污渍 |
| 干燥度 | 表面干燥无水迹 |

### 5.3 检测方法

#### 5.3.1 TOC检测

| 参数 | 要求 |
|------|------|
| 仪器 | TOC分析仪 |
| 检测限 | ≤50 ppb |
| 重复性 | RSD≤10% |

#### 5.3.2 HPLC检测

| 参数 | 要求 |
|------|------|
| 仪器 | HPLC（UV/FLD） |
| 专属性 | 专属性好 |
| 检测限 | 符合要求 |

#### 5.3.3 微生物检测

| 参数 | 要求 |
|------|------|
| 培养基 | 营养琼脂等 |
| 培养条件 | 30-35℃，48-72小时 |
| 计数方法 | 平板计数 |

## 6. 验证实施

### 6.1 验证前准备

| 准备工作 | 内容 |
|----------|------|
| 设备准备 | 设备处于可验证状态 |
| 人员培训 | 操作人员培训合格 |
| 文件准备 | 方案批准，记录表格准备 |
| 仪器准备 | 检测仪器校准合格 |

### 6.2 验证实施

#### 6.2.1 验证次数

| 验证类型 | 最少次数 | 说明 |
|----------|----------|------|
| 首次验证 | 3次连续 | 连续3次合格 |
| 再验证 | 3次连续 | 连续3次合格 |

#### 6.2.2 验证顺序

```
最差条件产品生产
    ↓
设备按清洁方法清洁
    ↓
目视检查（合格）
    ↓
取样
    ↓
检测
    ↓
数据分析
    ↓
判定
```

### 6.3 数据分析

#### 6.3.1 数据统计

| 项目 | 说明 |
|------|------|
| 平均值 | 计算平均值 |
| 标准差 | 计算标准差 |
| RSD | 计算相对标准差 |
| 最大值 | 确认最大值 |

#### 6.3.2 结果判定

| 判定 | 条件 |
|------|------|
| 合格 | 所有数据符合标准 |
| 不合格 | 任一数据不符合标准 |

## 8. 清洁失败处理

### 8.1 失败分类

| 类型 | 说明 | 处理 |
|------|------|------|
| 目视不合格 | 可见残留 | 重新清洁 |
| 化学不合格 | 化学残留超限 | 重新清洁+调查 |
| 微生物不合格 | 微生物超标 | 重新清洁+消毒+调查 |

### 8.2 失败调查

| 调查内容 | 说明 |
|----------|------|
| 清洁方法 | 是否按方法执行 |
| 设备状态 | 设备是否完好 |
| 检测方法 | 检测是否准确 |
| 人员操作 | 人员是否培训 |
