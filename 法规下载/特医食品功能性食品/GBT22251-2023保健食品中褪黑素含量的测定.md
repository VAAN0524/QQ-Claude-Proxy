# GB/T 22251-2023 保健食品中褪黑素含量的测定

## 标准基本信息

| 项目 | 内容 |
|------|------|
| 标准号 | GB/T 22251-2023 |
| 标准名称 | 保健食品中褪黑素含量的测定 |
| 英文名称 | Determination of melatonin content in health food |
| 发布日期 | 2023年XX月XX日 |
| 实施日期 | 2024年XX月XX日 |
| 发布机构 | 国家市场监督管理总局 |
| 标准类型 | 推荐性国家标准 |

## 范围

本标准规定了保健食品中褪黑素含量的测定方法。

本标准适用于褪黑素作为原料添加的保健食品（如片剂、胶囊、软胶囊、粉剂、口服液等剂型）中褪黑素含量的测定。

本标准检出限：0.02 mg/g，定量限：0.05 mg/g。

## 原理

试样中的褪黑素经溶剂提取后，采用高效液相色谱法（HPLC）或液相色谱-质谱法（LC-MS）进行分离和定量。

### 方法一：高效液相色谱法（HPLC）

#### 试剂和材料

除非另有说明，所有试剂均为分析纯，水为GB/T 6682规定的一级水。

1. 甲醇（色谱纯）
2. 乙腈（色谱纯）
3. 甲酸（色谱纯）
4. 褪黑素标准品（纯度≥98%）
5. 提取溶液：甲醇-水（70:30，体积比）

#### 仪器和设备

1. 高效液相色谱仪（配紫外检测器或二极管阵列检测器）
2. 分析天平（感量0.1 mg和0.01 mg）
3. 超声波清洗器
4. 涡旋振荡器
5. 离心机（转速≥4000 r/min）
6. 微孔滤膜（0.45 μm）

#### 色谱条件

| 参数 | 条件 |
|------|------|
| 色谱柱 | C18柱（250 mm × 4.6 mm，5 μm）或等效柱 |
| 流动相 | 乙腈-0.1%甲酸溶液（45:55，体积比） |
| 流速 | 1.0 mL/min |
| 检测波长 | 222 nm |
| 柱温 | 30 ℃ |
| 进样量 | 10 μL |

#### 分析步骤

**1. 标准溶液配制**

准确称取褪黑素标准品10 mg（精确至0.01 mg），置于100 mL容量瓶中，用甲醇溶解并定容至刻度，配制成100 μg/mL的标准储备液。

**2. 标准曲线制备**

分别准确吸取标准储备液0.1 mL、0.5 mL、1.0 mL、2.0 mL、5.0 mL、10.0 mL于100 mL容量瓶中，用甲醇定容，配制成浓度分别为0.1 μg/mL、0.5 μg/mL、1.0 μg/mL、2.0 μg/mL、5.0 μg/mL、10.0 μg/mL的标准工作溶液。

**3. 试样制备**

- **固体试样**：取适量试样粉碎，混匀，准确称取约0.5 g（精确至0.1 mg），置于25 mL具塞试管中，加入10 mL提取溶液，涡旋混匀1 min，超声提取15 min，离心5 min（4000 r/min），取上清液过0.45 μm滤膜，待测。

- **液体试样**：准确吸取1.0 mL试样，置于25 mL容量瓶中，用甲醇定容，超声混匀，过0.45 μm滤膜，待测。

- **软胶囊**：取适量软胶囊内容物，准确称取约0.5 g（精确至0.1 mg），按固体试样方法处理。

**4. 测定**

分别取标准工作溶液和试样溶液注入液相色谱仪，测定峰面积，以标准溶液浓度为横坐标，峰面积为纵坐标绘制标准曲线，计算试样中褪黑素的含量。

### 方法二：液相色谱-质谱法（LC-MS）

#### 试剂和材料

1. 甲醇（质谱纯）
2. 乙腈（质谱纯）
3. 甲酸（质谱纯）
4. 褪黑素标准品（纯度≥98%）
5. 同位素内标（d4-褪黑素）

#### 仪器和设备

1. 液相色谱-串联质谱仪（配电喷雾离子源ESI）
2. 分析天平（感量0.1 mg和0.01 mg）
3. 其他同方法一

#### 色谱条件

| 参数 | 条件 |
|------|------|
| 色谱柱 | C18柱（100 mm × 2.1 mm，1.8 μm）或等效柱 |
| 流动相 | A：0.1%甲酸水溶液；B：0.1%甲酸乙腈溶液 |
| 梯度程序 | 0-3 min，30%B；3-8 min，30%-90%B；8-10 min，90%B；10-12 min，90%-30%B；12-15 min，30%B |
| 流速 | 0.3 mL/min |
| 柱温 | 35 ℃ |
| 进样量 | 5 μL |

#### 质谱条件

| 参数 | 条件 |
|------|------|
| 离子源 | 电喷雾离子源（ESI） |
| 扫描模式 | 正离子模式 |
| 监测方式 | 多反应监测（MRM） |
| 母离子 | m/z 233.2 |
| 子离子 | m/z 174.1（定量离子）、m/z 216.2（定性离子） |
| 碰撞能量 | 15 eV、10 eV |

#### 分析步骤

**1. 标准溶液配制**

同方法一，并加入适量内标溶液。

**2. 试样制备**

同方法一，提取前加入内标溶液。

**3. 测定**

采用内标法定量。

## 结果计算

试样中褪黑素的含量按以下公式计算：

```
X = (ρ × V × f) / (m × 1000)
```

式中：
- X：试样中褪黑素的含量，单位为克每百克（g/100g）或毫克每克（mg/g）
- ρ：从标准曲线上查得的试样溶液中褪黑素浓度，单位为微克每毫升（μg/mL）
- V：试样定容体积，单位为毫升（mL）
- f：稀释倍数
- m：试样质量，单位为克（g）

计算结果保留三位有效数字。

## 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的10%。

## 回收率

不同基质中褪黑素的加标回收率范围：

| 基质类型 | 加标水平 | 回收率范围 |
|----------|----------|-----------|
| 片剂 | 50%-150% | 85%-110% |
| 胶囊 | 50%-150% | 85%-110% |
| 软胶囊 | 50%-150% | 80%-110% |
| 粉剂 | 50%-150% | 85%-110% |
| 口服液 | 50%-150% | 90%-110% |

## 附录C 质量控制

1. 每批样品应同时进行空白试验和加标回收试验。
2. 标准曲线相关系数应≥0.999。
3. 平行双样测定结果相对偏差应≤10%。
4. 每个样品至少测定两次，取平均值报告结果。
