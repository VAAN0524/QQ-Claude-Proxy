## 1 范围

本部分规定了根据空气中悬浮粒子浓度划分的洁净室和洁净区空气洁净度等级。

本部分适用于悬浮粒子浓度在 1 粒子/m³ 至 10⁸ 粒子/m³ 范围内的洁净室和洁净区。

## 3 术语和定义

**3.1 洁净室**
空气中悬浮粒子浓度受控，其建造和使用方式使粒子进入、产生和滞留最少的房间。

**3.2 洁净区**
洁净室内的限定空间，其悬浮粒子浓度受控。

**3.3 洁净度**
以单位体积空气中悬浮粒子的数量表示的洁净室（或洁净区）的清洁程度。

**3.4 粒子**
具有给定物理界限的微小物质。

**3.5 悬浮粒子**
漂浮在空气中的固体或液体粒子。

**3.6 粒径**
粒子直径的名义值，系指粒子直径测量值。

**3.7 粒径分布**
不同粒径范围的粒子浓度分布。

**3.8 采样点**
在洁净室或洁净区内进行空气采样的位置。

**3.9 采样量**
每次采样采集的空气体积。

**3.10 置信上限**
在给定置信水平下，某统计量的上限值。

**3.11 UCL描述符**
用于判断95%置信上限是否超过规定等级限值的统计量。

**4.1 洁净度等级**

| ISO等级 | 粒子浓度限值（个/m³） | |
|---------|---------------------|--|
| | ≥0.1μm | ≥0.5μm | ≥1.0μm | ≥5.0μm |
| ISO 1级 | 10 | 2 | — | — |
| ISO 2级 | 100 | 24 | 10 | — |
| ISO 3级 | 1 000 | 237 | 102 | — |
| ISO 4级 | 10 000 | 2 370 | 1 020 | — |
| ISO 5级 | 100 000 | 23 700 | 10 200 | — |
| ISO 6级 | 1 000 000 | 237 000 | 102 000 | 718 |
| ISO 7级 | — | 2 370 000 | 1 020 000 | 7 180 |
| ISO 8级 | — | 23 700 000 | 10 200 000 | 71 800 |
| ISO 9级 | — | 237 000 000 | 102 000 000 | 718 000 |

**4.2 等级说明**

1. ISO 1-5级：单向流洁净室
2. ISO 6-9级：非单向流洁净室
3. 粒径限值表示最大允许浓度
4. "—"表示该粒径粒子不作为控制指标

|---------|---------------|------|
| ISO 3级 | A/B级 | 无菌操作关键区 |
| ISO 5级 | A/B级 | 高风险操作区 |
| ISO 7级 | C级 | 辅助洁净区 |
| ISO 8级 | D级 | 一般操作区 |

## 5 测试方法

**5.1 仪器要求**

**5.1.1** 粒子计数器要求：

| 项目 | 要求 |
|------|------|
| 粒径范围 | 0.1μm-5.0μm |
| 计数效率 | ≥50%（0.1μm） |
| 采样流量 | ≥28.3 L/min（1 ft³/min） |
| 零计数 | ≤1个/5分钟 |
| 校准 | 按ISO 21501-4校准 |

**5.2 采样点设置**

**5.2.1** 采样点最少数量：

| 面积（m²） | 最少采样点数 |
|------------|--------------|
| ＜2 | 1 |
| 2-4 | 2 |
| 4-6 | 3 |
| 6-8 | 4 |
| 8-10 | 5 |
| 10-20 | 6 |
| 20-40 | 7 |
| 40-100 | 8 |
| 100-200 | 9 |
| 200-400 | 10 |
| 400-1000 | 11 |
| 1000-2000 | 12 |

**5.2.2** 采样点位置：

1. 均匀分布在洁净室或洁净区内
2. 避开气流直接吹扫的位置
3. 距离地面0.8m-1.5m高度
4. 工作面高度或关键操作位置

**5.3 采样量要求**

**5.3.1** 最小采样量：

| 洁净度等级 | 最小采样量（L） |
|------------|-----------------|
| ISO 1-2级 | ≥1000 |
| ISO 3级 | ≥300 |
| ISO 4级 | ≥100 |
| ISO 5级 | ≥30 |
| ISO 6级 | ≥10 |
| ISO 7级 | ≥5 |
| ISO 8-9级 | ≥2 |

**5.3.2** 采样时间：

每次采样时间应不少于1分钟，或保证采集到足够数量的粒子（至少20个粒子）。

**5.4 测试条件**

**5.4.1** 空气洁净度等级应进行以下两种状态的测试：

1. **空态**：设施已建成，所有动力设备接通并运行，但无生产设备、材料及人员
2. **静态**：设施已建成，生产设备已安装并能运行，但无人员操作
3. **动态**：设施以规定方式运行，有规定数量的人员进行正常操作

**5.4.2** 测试环境要求：

| 参数 | 要求 |
|------|------|
| 温度 | 18-26℃ |
| 相对湿度 | 45%-65% |
| 压差 | 符合设计要求 |
| 气流组织 | 正常运行 |

## 6 数据处理

**6.1 数据计算**

**6.1.1** 粒子浓度计算：

```
C = N / V
```

式中：
- C：粒子浓度（个/m³）
- N：粒子计数（个）
- V：采样体积（m³）

**6.1.2** 平均浓度计算：

```
C_avg = (C₁ + C₂ + ... + Cₙ) / n
```

式中：
- C_avg：平均粒子浓度（个/m³）
- C₁, C₂, ..., Cₙ：各采样点的粒子浓度（个/m³）
- n：采样点数量

**6.1.3** 标准差计算：

```
SD = √[Σ(Cᵢ - C_avg)² / (n-1)]
```

式中：
- SD：标准差
- Cᵢ：第i个采样点的粒子浓度
- C_avg：平均粒子浓度
- n：采样点数量

**6.2 置信上限计算**

**6.2.1** 95%置信上限：

```
UCL = C_avg + t × SD / √n
```

式中：
- UCL：95%置信上限
- C_avg：平均粒子浓度
- SD：标准差
- n：采样点数量
- t：t分布系数（见下表）

**6.2.2** t分布系数：

| 采样点数 | t值 |
|----------|-----|
| 2 | 6.31 |
| 3 | 2.92 |
| 4 | 2.35 |
| 5 | 2.13 |
| 6 | 2.02 |
| 7-9 | 1.96 |
| ≥10 | 1.96 |

**6.2.3** UCL描述符：

当采样点少于9个时，应计算UCL描述符，判断UCL是否超过等级限值。

## 7 判定标准

**7.1 合格标准**

洁净室或洁净区的空气洁净度等级应符合以下要求：

1. 各采样点的粒子浓度不超过等级限值
2. 平均粒子浓度不超过等级限值
3. 95%置信上限不超过等级限值（当采样点少于9个时）

**7.2 不合格处理**

当测试结果不合格时，应：

1. 检查测试方法和仪器
2. 增加采样点或采样量
3. 进行原因分析
4. 采取纠正措施
5. 重新测试

## 8 监测要求

**8.1 监测频率**

| 洁净度等级 | 监测频率 |
|------------|----------|
| ISO 1-2级 | 连续监测 |
| ISO 3-4级 | 每班监测 |
| ISO 5级 | 每日监测 |
| ISO 6-7级 | 每周监测 |
| ISO 8-9级 | 每月监测 |

**8.2 监测位置**

1. 关键操作区域
2. 产品暴露位置
3. 人员活动频繁区域
4. 气流组织不良位置

**8.3 趋势分析**

应建立粒子浓度趋势分析制度，及时发现异常并采取措施。

## 9 文件记录

**9.1 测试报告**

测试报告应包括以下内容：

1. 洁净室或洁净区名称和编号
2. 测试日期和测试人员
3. 测试状态（空态/静态/动态）
4. 测试方法和仪器
5. 采样点布置图
6. 测试数据记录
7. 计算过程和结果
8. 判定结论
9. 签发日期和签字

**9.2 记录保存**

测试记录应保存至少5年，或根据法规要求确定。

## 附录A（规范性）粒子计数器校准

**A.1 校准周期**

粒子计数器应每年进行一次校准，或根据使用频率确定。

**A.2 校准项目**

| 校准项目 | 要求 |
|----------|------|
| 粒径标定 | ±10% |
| 计数效率 | ±20% |
| 流量校准 | ±5% |
| 零计数 | ≤1个/5分钟 |

**A.3 校准方法**

按照ISO 21501-4规定的校准方法进行。

## 附录B（资料性）采样点布置示例

**B.1 矩形洁净室**

对于矩形洁净室，采样点应均匀布置，形成网格状。

**B.2 圆形洁净室**

对于圆形洁净室，采样点应沿半径和圆周均匀布置。

**B.3 关键区域**

对于关键区域，应增加采样点密度。

## 附录D（资料性）应用指南

**D.1 洁净度等级选择**

选择洁净度等级时应考虑以下因素：

1. 产品要求
2. 工艺要求
3. 人员活动
4. 维护要求
5. 经济性

**D.2 洁净室设计**

洁净室设计应遵循以下原则：

1. 合理确定洁净度等级
2. 优化气流组织
3. 控制污染源
4. 便于维护管理
5. 节能环保
