## 一、总则

### （一）编制目的

为规范保健食品检验与评价工作，保证检验结果科学、准确、公正，促进保健食品产业健康发展，制定本规范。

### （二）适用范围

本规范适用于保健食品的注册检验、备案检验、监督抽检和委托检验。

### （三）基本原则

1. **科学性原则**：采用科学合理的检验方法和评价指标
2. **规范性原则**：严格按照标准操作程序进行检验
3. **公正性原则**：检验结果客观公正，不受任何干扰
4. **保密性原则**：保护企业商业秘密和技术秘密

### （四）检验分类

1. **注册检验**：新产品注册申请时进行的检验
2. **备案检验**：产品备案时进行的检验
3. **抽检检验**：监管部门监督抽检
4. **委托检验**：企业或相关方委托的检验

## 二、检验要求

### （一）检验机构资质

保健食品检验机构应当具备以下条件：

1. **基本条件**

| 项目 | 要求 |
|------|------|
| 机构性质 | 独立法人资格 |
| 资质认定 | CMA资质认定证书 |
| 认可范围 | 包括保健食品全部检验项目 |
| 实验场地 | 符合实验室管理要求 |

2. **人员要求**

| 岗位 | 要求 |
|------|------|
| 技术负责人 | 高级职称，5年以上经验 |
| 质量负责人 | 中级以上职称，熟悉质量管理 |
| 检验人员 | 专业背景，持证上岗 |
| 审核人员 | 中级以上职称，3年以上经验 |

3. **设备要求**

| 设备类别 | 要求 |
|----------|------|
| 理化检验设备 | 满足检验项目需要 |
| 微生物检验设备 | 符合生物安全要求 |
| 毒理学检验设备 | 符合GLP要求 |
| 功能学检验设备 | 符合试验要求 |

### （二）检验样品管理

1. **样品接收**

| 项目 | 要求 |
|------|------|
| 样品数量 | 满足全部检验需要 |
| 样品状态 | 包装完好，标签清晰 |
| 样品保存 | 按要求保存，防止变质 |
| 样品编号 | 唯一编号，状态标识 |

2. **样品留存**

- 留样数量：至少3次检验量
- 留样期限：至保质期后6个月
- 留样条件：与产品标签要求一致

### （三）检验方法选择

1. **国家标准方法**：优先采用国家标准方法
2. **行业标准方法**：无国家标准时采用行业标准
3. **科学文献方法**：需经验证后方可使用
4. **自建方法**：需经方法学验证

## 四、安全性评价

### （一）毒理学试验

#### 1. 试验项目

根据产品情况选择以下毒理学试验：

| 产品类型 | 必做项目 | 选做项目 |
|----------|----------|----------|
| 新产品 | 4项试验 | 遗传毒性、致畸试验 |
| 同类产品 | 急性毒性 | 28天经口毒性 |
| 传统原料 | - | 根据需要确定 |

#### 2. 四项基础试验

（1）**急性经口毒性试验**

| 项目 | 要求 |
|------|------|
| 试验动物 | 小鼠或大鼠 |
| 给药方式 | 灌胃 |
| 观察期限 | 14天 |
| 判定标准 | LD₅₀ > 10 g/kg体重，属实际无毒 |

（2）**三项遗传毒性试验**

| 试验名称 | 检测终点 |
|----------|----------|
| Ames试验 | 基因突变 |
| 微核试验 | 染色体损伤 |
| 精子畸形试验 | 生殖细胞损伤 |

（3）**28天经口毒性试验**

| 项目 | 要求 |
|------|------|
| 试验动物 | 大鼠 |
| 剂量设置 | 高、中、低3个剂量 |
| 观察指标 | 体重、进食、血常规、生化、病理 |
| 判定标准 | 无有害作用水平(NOAEL) |

（4）**致畸试验**

| 项目 | 要求 |
|------|------|
| 试验动物 | 大鼠或兔 |
| 给药时期 | 器官形成期 |
| 观察指标 | 胚胎毒性、胎儿畸形 |

#### 3. 90天经口毒性试验（如需要）

| 项目 | 要求 |
|------|------|
| 试验动物 | 大鼠 |
| 剂量设置 | 高、中、低3个剂量 |
| 观察期限 | 90天 |
| 观察指标 | 全面系统的毒性观察 |

### （二）安全性评价指标

1. **毒理学安全限值**

```
每日允许摄入量(ADI) = NOAEL / 安全系数
安全系数通常取100（10×10）
```

2. **摄入量安全性评价**

```
推荐摄入量安全性 = 每日推荐摄入量 / ADI × 100%
```

要求：< 80% ADI 为安全

## 五、功能学评价

### （一）功能类别及评价方法

#### 1. 增强免疫力功能

**试验项目**：

| 试验 | 方法 |
|------|------|
| 细胞免疫功能 | 小鼠迟发型变态反应试验 |
| 体液免疫功能 | 抗体生成细胞检测试验 |
| 单核-巨噬细胞功能 | 小鼠碳廓清试验 |
| NK细胞活性 | 小鼠NK细胞活性测定 |

**结果判定**：任2项试验结果阳性，可判定该样品具有增强免疫力功能。

#### 2. 辅助降血脂功能

**试验项目**：

| 试验 | 检测指标 |
|------|----------|
| 动物实验 | TC、TG、HDL-C |
| 人体试食 | TC、TG、HDL-C |

**结果判定**：

- 动物实验：模型对照组与正常对照组比较，TC、TG升高有统计学意义
- 试食组与模型对照组比较，TC或TG降低有统计学意义，HDL-C不降低

#### 3. 辅助降血糖功能

**试验项目**：

| 试验 | 方法 |
|------|------|
| 动物实验 | 链脲霉素糖尿病模型 |
| 人体试食 | Ⅱ型糖尿病患者 |

**检测指标**：空腹血糖、餐后血糖、糖化血红蛋白

#### 4. 抗氧化功能

**试验项目**：

| 试验 | 方法 |
|------|------|
| 动物实验 | 脂质过氧化产物、抗氧化酶活性 |
| 人体试食 | MDA、SOD、GSH-Px |

#### 5. 辅助改善记忆功能

**试验项目**：跳台试验、避暗试验、穿梭试验、水迷宫试验

#### 6. 促进排铅功能

**试验项目**：铅中毒模型、血铅、骨铅、肝铅测定

#### 7. 清咽功能

**试验项目**：大鼠慢性咽炎模型

#### 8. 辅助降血压功能

**试验项目**：自发性高血压大鼠模型

#### 9. 改善睡眠功能

**试验项目**：戊巴比妥钠睡眠时间试验

#### 10. 促进泌乳功能

**试验项目**：母鼠哺乳量测定

#### 11. 减肥功能

**试验项目**：动物肥胖模型、人体试食

#### 12. 改善生长发育功能

**试验项目**：幼年动物体重、身长测定

#### 13. 增加骨密度功能

**试验项目**：动物骨密度测定、人体骨密度测定

#### 14. 改善营养性贫血功能

**试验项目**：血红蛋白、红细胞内游离原卟啉、血清铁测定

#### 15. 对化学性肝损伤有辅助保护功能

**试验项目**：CCl₄肝损伤模型

#### 16. 祛痤疮功能

**试验项目**：人体试食试验

#### 17. 祛黄褐斑功能

**试验项目**：人体试食试验

#### 18. 改善皮肤水分功能

**试验项目**：皮肤水分测定

#### 19. 改善皮肤油分功能

**试验项目**：皮肤油分测定

#### 20. 调节肠道菌群功能

**试验项目**：双歧杆菌、乳杆菌、肠球菌、肠杆菌测定

#### 21. 促进消化功能

**试验项目**：胃肠道运动、消化酶活性

#### 22. 通便功能

**试验项目**：小肠运动试验、排便时间、粪便性状

#### 23. 对胃粘膜有辅助保护功能

**试验项目**：胃溃疡模型

### （二）功能学评价原则

1. **动物实验**：至少采用2种动物模型
2. **人体试食**：需要动物实验阳性后进行
3. **剂量设计**：设高、中、低3个剂量组
4. **阳性对照**：设阳性对照组验证试验敏感性
5. **阴性对照**：设阴性对照组排除干扰

### （二）试验条件

|------|---------|------------|----------|
| 加速试验 | 37±2 | 75±5 | 6 |
| 长期试验 | 25±2 | 60±5 | 24-36 |

### （三）检测项目

| 时间点 | 检测项目 |
|--------|----------|
| 0月 | 感官、功效成分、微生物 |
| 1月 | 功效成分、微生物 |
| 2月 | 功效成分、微生物 |
| 3月 | 感官、功效成分、微生物 |
| 6月 | 感官、功效成分、微生物 |
| ... | ... |

### （四）结果判定

功效成分含量不低于标示值的90%，微生物指标符合要求。

## 八、质量控制

### （一）内部质量控制

1. **平行双样**：≥10%样品做平行双样
2. **加标回收**：≥10%样品做加标回收
3. **质控样**：每批样品带质控样
4. **空白试验**：每批样品做空白试验

1. **能力验证**：每年参加能力验证
2. **实验室间比对**：定期参加比对试验
3. **飞行检查**：接受监管部门飞行检查
