# FSSC 22000 食品安全体系认证标准（2026版）

## 标准概述

**标准名称**：FSSC 22000 (Food Safety System Certification 22000)
**最新版本**：FSSC 22000 Version 6.0 (2024年发布，2026年适用)
**发布机构**：FSSC 22000 Foundation
**认证性质**：全球食品安全认证体系
**基础标准**：ISO 22000 + ISO/TS 22002-1 + FSSC 附加要求

---

## 一、FSSC 22000 标准架构

### 1.1 标准组成

FSSC 22000 由三部分组成：

| 组成部分 | 内容 | 作用 |
|---------|------|------|
| **ISO 22000** | 食品安全管理体系要求 | 核心管理体系框架 |
| **ISO/TS 22002-1** | 食品安全前提方案 | 食品制造特定PRP要求 |
| **FSSC 附加要求** | FSSC 22000 附加方案 | 补充认证要求 |

### 1.2 适用范围

FSSC 22000 适用于食品链中的各类组织：

- 农产品生产商
- 动物饲料生产商
- 食品制造商
- 餐饮服务商
- 零售商
- 运输和仓储服务商

---

## 二、FSSC 22000 V6.0 主要更新内容（2024）

### 2.1 核心变化

1. **质量管理整合**
   - 强化质量管理体系与食品安全管理体系的整合
   - 新增产品质量管理要求

2. **食品安全文化**
   - 明确要求建立食品安全文化
   - 增加管理层在食品安全文化建设中的职责

3. **食品防护和欺诈**
   - 加强食品防御（Food Defense）要求
   - 整合食品欺诈（Food Fraud）预防措施

4. **环境管理**
   - 新增环境可持续性要求
   - 强调资源效率和废物管理

5. **标签管理**
   - 强化过敏原标识管理
   - 增加标签验证要求

### 2.2 管理体系要求

#### 2.2.1 领导作用
- 最高管理者必须展示对食品安全管理体系的承诺
- 制定食品安全方针
- 确保食品安全职责和权限得到分配

#### 2.2.2 食品安全文化
- 建立和保持食品安全文化
- 定期评估食品安全文化的有效性
- 鼓励员工报告食品安全问题

#### 2.2.3 风险管理
- 实施基于风险的思维方法
- 管理变更
- 应急准备和响应

---

## 三、前提方案（PRP）要求

### 3.1 ISO/TS 22002-1 核心要求

#### 3.1.1 建筑设计和设施
- **位置选择**：避免周围环境对食品安全造成影响
- **厂区环境**：保持清洁，防止污染源
- **建筑结构**：便于清洁和维护
- **内部布局**：合理划分不同卫生等级区域

#### 3.1.2 设备和工器具
- **材质要求**：食品接触表面符合食品安全要求
- **设计原则**：易于清洁、消毒和维护
- **安装要求**：便于操作、清洁和维护

#### 3.1.3 交叉污染控制
- **物理隔离**：防止交叉污染
- **人流物流**：合理规划人流和物流路线
- **空气流向**：控制空气流动方向

### 3.2 GMP 要求

| 项目 | 要求 |
|------|------|
| **个人卫生** | 健康检查、洗手消毒、着装规范 |
| **培训** | 食品安全培训、岗位技能培训 |
| **清洁消毒** | 清洁程序、消毒程序、验证方法 |
| **虫害控制** | 预防措施、监控程序、处理方法 |
| **化学品管理** | 专用存放、标识清晰、使用记录 |

## 四、HACCP 要求

### 4.1 HACCP 原理

FSSC 22000 要求按照 HACCP 的七项原则建立和实施 HACCP 计划：

1. **危害分析**：识别和评估食品安全危害
2. **确定关键控制点**：识别可以控制危害的关键步骤
3. **建立关键限值**：设定 CCP 的可接受限值
4. **建立监控程序**：监控 CCP 是否符合关键限值
5. **建立纠正措施**：当监控显示 CCP 失控时采取的措施
6. **建立验证程序**：验证 HACCP 体系的有效性
7. **建立记录保持程序**：保持适当的记录

### 4.2 HACCP 计划要素

- 产品描述
- 预期用途
- 流程图
- 危害分析
- CCP 确定
- 关键限值
- 监控系统
- 纠正措施
- 验证程序
- 记录保持

## 五、FSSC 22000 认证流程

### 5.1 认证申请

1. **申请前准备**
   - 了解 FSSC 22000 标准要求
   - 评估现有体系与标准的差距
   - 制定实施计划

2. **选择认证机构**
   - 选择经 FSSC 22000 批准的认证机构
   - 确认认证范围和认证费用

### 5.2 体系实施

1. **体系建立**
   - 建立食品安全管理体系文件
   - 实施 HACCP 计划
   - 执行前提方案

2. **体系运行**
   - 试运行至少 3 个月
   - 完成内部审核和管理评审
   - 实施纠正和预防措施

### 5.3 认证审核

| 阶段 | 内容 | 时长 |
|------|------|------|
| **文件审核** | 评估管理体系文件的符合性 | 1-2天 |
| **第一阶段审核** | 评估体系策划和准备情况 | 1-2天 |
| **第二阶段审核** | 评估体系实施的有效性 | 2-5天 |
| **证书发放** | 审核通过后发放证书 | 审核完成后2-4周 |

### 5.4 证书维持

- **监督审核**：每年进行一次监督审核
- **复评审核**：每 3 年进行一次完整的复评审核
- **证书有效期**：3 年

## 六、FSSC 22000 在中国的应用

### 6.1 与国家标准的关系

| FSSC 22000 | 中国标准 |
|------------|----------|
| ISO 22000 | GB/T 22000-2024 |
| ISO/TS 22002-1 | GB 14881-2025 |
| HACCP 原理 | GB/T 27341-2024 |

### 6.2 GFSI 认可

FSSC 22000 获得 **全球食品安全倡议（GFSI）** 认可，是全球范围内广泛接受的食品安全认证体系之一。

### 6.3 中国认证认可

FSSC 22000 在中国获得**中国国家认证认可监督管理委员会（CNCA）** 认可，可在中国合法开展认证活动。

## 十、FSSC 22000 与其他标准的关系

### 10.1 GFSI 认可标准比较

| 标准 | 适用范围 | 特点 |
|------|----------|------|
| **FSSC 22000** | 食品链全链条 | 基于 ISO 22000，体系完整 |
| **BRCGS** | 食品加工、包装 | 英国标准，要求详细 |
| **IFS** | 欧洲市场为主 | 欧洲标准，注重细节 |
| **SQF** | 北美市场为主 | 美国标准，分级认证 |

### 10.2 与 ISO 22000 的关系

FSSC 22000 是在 ISO 22000 基础上增加了：
- 行业特定的前提方案（PRP）
- FSSC 附加要求
- GFSI 认可要求

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **食品安全** | 食品在按照预期用途进行制备和/或食用时不会伤害消费者的保证 |
| **前提方案 (PRP)** | 为保持卫生环境所必需的基本条件和活动 |
| **关键控制点 (CCP)** | 能够进行控制并防止、消除或降低食品安全危害的步骤 |
| **关键限值**：区分可接受与不可接受的判定标准 |
| **纠正措施**：为消除已发现的不合格或其他不期望情况的原因所采取的措施 |
| **验证**：通过提供客观证据对规定要求已得到满足的认定 |
