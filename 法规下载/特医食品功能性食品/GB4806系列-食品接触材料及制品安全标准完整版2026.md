# GB4806 食品接触材料及制品安全标准（完整版 2026）

## 标准概述

**标准系列名称**：GB 4806 食品接触用金属材料及制品、食品接触用塑料材料及制品、食品接触用橡胶材料及制品等
**发布机构**：国家卫生健康委员会、国家市场监督管理总局
**标准性质**：强制性国家标准（GB）
**适用范围**：各类食品接触材料及制品的生产、检验和销售

## 二、GB 4806.1 通用安全要求（2022）

### 2.1 基本要求

| 要求类别 | 具体要求 |
|---------|----------|
| **基本原则** | 食品接触材料及制品在正常或可预见的使用条件下，不应迁移或释放出达到对人体健康构成危害或造成食品感官特性发生不可接受变化的物质 |
| **感官要求** | 迁移到食品或食品模拟物中的物质不应使食品产生异常色泽、异味、浑浊 |
| **标签标识** | 产品应有清晰、持久、易识别的标签标识 |

### 2.2 技术要求

#### 2.2.1 迁移限量

| 物质类别 | 迁移限量要求 |
|---------|-------------|
| **总迁移量** | ≤ 10 mg/dm² 或 ≤ 60 mg/kg |
| **重金属（以铅计）** | ≤ 1 mg/kg |
| **特定物质迁移量** | 符合各分标准具体要求 |
| **初级芳香胺** | 不得检出（检出限 0.01 mg/kg） |

#### 2.2.2 其他要求

- **残留物限量**：符合各分标准要求
- **降解要求**：可降解材料需符合降解率要求
- **微生物限量**：纸和纸板、竹木制品需符合微生物限量

### 2.3 标签标识要求

| 标识内容 | 要求 |
|---------|------|
| **产品名称** | 明确标识 |
| **材质类型** | 标明主要材质 |
| **遵循标准** | 标明执行标准编号 |
| **使用注意事项** | 如有特殊要求，应标明 |
| **生产者信息** | 名称、地址、联系方式 |

## 三、GB 4806.7 塑料材料及制品（2023）

### 3.1 适用范围

适用于以食品接触用塑料树脂为主要原料生产的各类食品接触用塑料材料及制品。

### 3.2 原料要求

#### 3.2.1 树脂要求

- 应使用 **GB 4806.6** 或相关公告中允许使用的树脂
- 不得使用回收料作为食品接触材料（特定情况除外）

#### 3.2.2 添加剂要求

- 应符合 **GB 9685** 及相关公告要求
- 不得使用未经批准的添加剂

### 3.3 技术要求

#### 3.3.1 感官要求

| 项目 | 要求 |
|------|------|
| **色泽** | 正常 |
| **异味** | 无异味 |
| **污物** | 无肉眼可见的外来杂质 |

#### 3.3.2 理化指标

| 项目 | 指标 |
|------|------|
| **总迁移量** | ≤ 10 mg/dm²（接触水性/酸性/油性食品） |
| | ≤ 60 mg/kg（容积） |
| **高锰酸钾消耗量** | ≤ 10 mg/kg |
| **重金属（以铅计）** | ≤ 1 mg/kg |
| **脱色试验** | 阴性 |

#### 3.3.3 特定物质迁移量

| 物质 | 迁移限量 |
|------|----------|
| **己内酰胺** | ≤ 15 mg/kg |
| **对苯二甲酸** | ≤ 7.5 mg/kg |
| **乙二醇** | ≤ 30 mg/kg |
| **双酚A（BPA）** | ≤ 0.05 mg/kg（用于婴幼儿食品接触材料不得使用） |
| **氯乙烯** | ≤ 1.0 mg/kg（不得用于婴幼儿食品接触材料） |
| **偏二氯乙烯** | ≤ 6 mg/kg |

### 3.4 特殊用途要求

| 用途 | 特殊要求 |
|------|----------|
| **婴幼儿食品接触** | 禁止使用双酚A、氯乙烯等特定物质 |
| **微波加热** | 应标注"可用于微波炉"或类似说明 |
| **高温使用** | 应标注最高使用温度 |

## 四、GB 4806.8 纸和纸板材料及制品（2022）

### 4.1 适用范围

适用于各类食品接触用纸和纸板材料及制品，包括：

- 包装用纸、纸板
- 纸袋、纸盒
- 纸杯、纸碗
- 纸吸管、纸托
- 其他纸质食品接触制品

### 4.2 技术要求

#### 4.2.1 感官要求

| 项目 | 要求 |
|------|------|
| **色泽** | 正常 |
| **异味** | 无异味 |
| **污物** | 无霉变、无异物 |

#### 4.2.2 理化指标

| 项目 | 指标 |
|------|------|
| **总迁移量** | ≤ 10 mg/dm² 或 ≤ 60 mg/kg |
| **重金属（以铅计）** | ≤ 1 mg/kg |
| **甲醛** | ≤ 1 mg/kg |
| **荧光性物质** | 不得检出 |
| **脱色试验** | 阴性（印刷部分） |

#### 4.2.3 微生物限量

| 项目 | 指标 |
|------|------|
| **大肠菌群** | 不得检出 |
| **沙门氏菌** | 不得检出 |
| **霉菌** | ≤ 50 CFU/g |
| **酵母菌** | ≤ 50 CFU/g |

### 4.3 荧光增白剂要求

- 食品接触面不得检出荧光增白剂
- 非食品接触面使用荧光增白剂需确保不迁移

## 五、GB 4806.9 金属材料及制品（2023）

### 5.1 适用范围

适用于各类食品接触用金属材料及制品，包括：

- 不锈钢制品
- 铝及铝合金制品
- 铁及铁合金制品
- 其他金属制品

### 5.2 技术要求

#### 5.2.1 感官要求

| 项目 | 要求 |
|------|------|
| **表面** | 光滑、无毛刺、无裂纹 |
| **涂层** | 均匀、无脱落 |
| **异味** | 无异味 |

#### 5.2.2 理化指标

**不锈钢制品：**

| 项目 | 指标 |
|------|------|
| **铅（Pb）** | ≤ 0.01 mg/kg |
| **铬（Cr）** | ≤ 2.0 mg/kg |
| **镍（Ni）** | ≤ 0.5 mg/kg |
| **镉（Cd）** | ≤ 0.005 mg/kg |
| **砷（As）** | ≤ 0.01 mg/kg |

**铝制品：**

| 项目 | 指标 |
|------|------|
| **锌（Zn）** | ≤ 5 mg/kg |
| **铅（Pb）** | ≤ 0.01 mg/kg |
| **镉（Cd）** | ≤ 0.005 mg/kg |
| **砷（As）** | ≤ 0.01 mg/kg |

#### 5.2.3 涂层要求

- 涂层应均匀、牢固
- 涂层迁移量符合要求
- 不得使用含铅涂料

## 七、GB 4806.10 涂料及涂层（2022）

- 金属容器涂料
- 烘烤器具涂层

- 其他食品接触用涂料及涂层

### 7.2 技术要求

#### 7.2.1 感官要求

| 项目 | 要求 |
|------|------|
| **外观** | 涂层均匀、光滑、无脱落 |

| 项目 | 指标 |
|------|------|

| **高锰酸钾消耗量** | ≤ 10 mg/kg |

- **高温涂层**：需标注最高使用温度

## 九、GB 4806.12 复合材料及制品（2022）

### 9.1 适用范围

适用于由两种或两种以上材料复合而成的食品接触材料及制品，包括：

- 纸塑复合
- 铝塑复合
- 多层复合
- 其他复合材料

### 9.2 技术要求

- 应符合各组成材料的相关标准要求
- 复合层间粘结牢固，不分离
- 总迁移量按最严格的要求执行

### 10.2 技术要求

#### 10.2.1 感官要求

| 项目 | 要求 |
|------|------|

| **异味** | 无异味 |

#### 10.2.2 理化指标

| 项目 | 指标 |
|------|------|

## 十一、迁移试验方法

### 11.1 食品模拟物选择

| 食品类型 | 模拟物 | 使用条件 |
|---------|--------|----------|
| **水性食品（pH>4.5）** | 10% 乙醇或水 | 常温或高温 |
| **酸性食品（pH≤4.5）** | 3% 乙酸 | 常温或高温 |
| **酒精食品** | 10%、20% 或 50% 乙醇 | 根据酒精含量选择 |
| **油脂食品** | 橄榄油、95% 乙醇或异辛烷 | 根据实际选择 |

### 11.2 迁移试验条件

| 使用条件 | 试验条件 |
|---------|----------|
| **冷藏/冷冻** | 5°C/10 天 |
| **室温** | 25°C/10 天 |
| **高温（70°C以上）** | 70°C/2 小时 或 100°C/1 小时 |
| **高温短时（100°C以上）** | 100°C/30 分钟 或 121°C/15 分钟 |

## 十二、企业合规要点

### 12.1 食品接触材料生产企业

1. **原料控制**
   - 使用符合标准的原料
   - 建立原料验收制度
   - 做好原料追溯记录

2. **生产控制**
   - 符合良好生产规范（GMP）
   - 控制生产环境卫生
   - 做好生产记录

3. **检验控制**
   - 按标准要求进行出厂检验
   - 建立检验记录档案
   - 定期进行型式检验

4. **标签标识**
   - 按标准要求标注产品信息
   - 标明执行标准编号
   - 标注使用注意事项

### 12.2 食品生产企业

1. **采购控制**
   - 采购符合标准的食品接触材料
   - 要求供应商提供合格证明
   - 做好供应商审核

2. **使用控制**
   - 按照标识要求使用
   - 注意使用温度限制
   - 定期检查材料状态

3. **验证检测**
   - 定期进行迁移试验
   - 做好检测记录
   - 发现问题及时处理

## 十三、常见问题解答

### 13.1 标准适用

| 问题 | 答案 |
|------|------|
| **GB 4806 与 GB 9685 的关系** | GB 4806 规定产品标准，GB 9685 规定添加剂使用标准 |
| **新标准何时实施** | 各标准实施日期不同，详见标准前言 |
| **旧标准还能用吗** | 应使用最新版本的标准 |

### 13.2 检测要求

| 问题 | 答案 |
|------|------|
| **检测频率** | 出厂检验每批进行，型式检验每年至少1次 |
| **检测机构** | 应选择有资质的第三方检测机构 |
| **检测周期** | 一般7-15个工作日 |

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **食品接触材料及制品** | 在正常使用条件下，各种已经或预期可能与食品或食品添加剂接触、或其成分可能转移到食品中的材料和制品 |
| **总迁移量** | 从食品接触材料及制品中迁移到食品或食品模拟物中的所有非挥发性物质的总量 |
| **特定迁移量** | 从食品接触材料及制品中迁移到食品或食品模拟物中的特定物质的量 |
| **食品模拟物** | 用于模拟食品特性的试验介质 |
