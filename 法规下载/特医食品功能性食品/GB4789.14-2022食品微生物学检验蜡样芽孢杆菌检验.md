# GB 4789.14-2022 食品微生物学检验 蜡样芽孢杆菌检验

## 标准信息

| 项目 | 内容 |
|------|------|
| **标准号** | GB 4789.14-2022 |
| **标准名称** | 食品微生物学检验 蜡样芽孢杆菌检验 |
| **发布日期** | 2022年 |
| **实施日期** | 2022年 |
| **替代标准** | GB 4789.14-2014 |
| **适用范围** | 食品中蜡样芽孢杆菌的检验 |

---

## 1. 范围

本标准规定了食品中蜡样芽孢杆菌（Bacillus cereus）的检验方法。

本标准适用于粮谷类、乳制品、肉制品、婴幼儿食品等中蜡样芽孢杆菌的检验。

---

## 2. 设备和材料

### 2.1 设备

- **恒温培养箱**：30℃±1℃、36℃±1℃
- **冰箱**：2℃~5℃
- **恒温水浴箱**
- **显微镜**
- **均质器**
- **无菌吸管、试管、锥形瓶、培养皿**

### 2.2 培养基和试剂

- **胰酪大豆胨琼脂（TSA）**
- **甘露醇卵黄多粘菌素琼脂（MYP）**
- **酚红葡萄糖肉汤**
- **硝酸盐肉汤**
- **溶菌酶肉汤**
- **V-P反应培养基**
- **蜡样芽孢杆菌显色培养基**

---

## 3. 检验程序

```
样品 → 前处理（均质）
    ↓
计数法：平板涂布（MYP，30℃，24h）
     或
增菌法：增菌（TSA，30℃，18h~24h）
    ↓
挑选可疑菌落（粉红色菌落，有沉淀环）
    ↓
确认试验（形态观察、生化试验）
    ↓
报告结果
```

---

## 4. 操作步骤

### 4.1 平板计数法

#### 4.1.1 样品处理

称取25g样品，加入225mL无菌蛋白胨-吐温80稀释液中，均质制成1:10稀释液。

#### 4.1.2 平板涂布

取系列稀释液0.1mL，涂布于MYP平板，30℃培养24h。

#### 4.1.3 菌落计数

**典型菌落特征**：
- 蜡样芽孢杆菌：粉红色（不发酵甘露醇），周围有乳白色沉淀环（卵磷脂酶阳性）
- 菌落大小：3mm~5mm
- 形态：圆形、凸起、边缘整齐

---

### 4.2 增菌法

取1:10稀释液1mL，接种于TSA平板，30℃培养18h~24h。

---

### 4.3 确认试验

#### 4.3.1 形态观察

| 项目 | 特征 |
|------|------|
| 革兰氏染色 | 阳性（G+） |
| 菌体形态 | 大杆菌，两端钝圆 |
| 芽孢 | 椭圆形，位于中央或次末端 |
| 芽孢囊 | 不膨大 |
| 鞭毛 | 周身鞭毛，有动力 |
| 菌落 | 灰白色、不透明、表面粗糙 |

#### 4.3.2 生化试验

| 试验项目 | 蜡样芽孢杆菌 |
|----------|--------------|
| 氧化酶 | - |
| 过氧化氢酶 | + |
| 硝酸盐还原 | + |
| V-P试验 | + |
| 溶菌酶耐受 | + |
| 葡萄糖产酸 | + |
| 葡萄糖产气 | - |
| 木糖 | - |
| 阿拉伯糖 | -/d |
| 甘露醇 | - |
| 明胶液化 | + |
| 卵磷脂酶 | + |
| 酪蛋白分解 | + |
| 淀粉水解 | + |
| 运动力 | + |

#### 4.3.3 与类似菌鉴别

| 项目 | 蜡样芽孢杆菌 | 苏云金芽孢杆菌 | 蕈状芽孢杆菌 |
|------|--------------|----------------|--------------|
| 菌落形态 | 粗糙、扩展 | 粗糙、有β-溶血 | 光滑 |
| V-P试验 | + | + | - |
| 卵磷脂酶 | + | + | - |
| 溶血 | 弱/无 | 明显 | - |
| 蛋白毒素结晶 | - | + | - |
| 生长温度 | 10℃~45℃ | 10℃~45℃ | 4℃~37℃ |

## 5. 结果计算与报告

### 5.1 计数法结果

**蜡样芽孢杆菌总数（CFU/g）** = 典型菌落数 × 稀释倍数

### 5.2 报告格式

- **检出**：报告蜡样芽孢杆菌计数，如"100 CFU/g"
- **未检出**：报告"<10 CFU/g"（最低检测限）

### 5.3 判定标准

参考标准：
- **婴幼儿食品**：<100 CFU/g（部分国家要求<50 CFU/g）
- **即食食品**：<10^3 CFU/g~10^4 CFU/g
- **其他食品**：<10^5 CFU/g

---

## 6. 致病性

### 6.1 食物中毒

**中毒类型**：
1. **腹泻型**：摄入活菌，产生肠毒素
2. **呕吐型**：预先形成的耐热肠毒素

**中毒剂量**：
- 腹泻型：10^5~10^8 CFU
- 呕吐型：10^5~10^8 CFU（或毒素）

### 6.2 毒素

| 毒素类型 | 特性 | 症状 |
|----------|------|------|
| 腹泻型毒素（NHE、HBL） | 蛋白质，不耐热 | 腹痛、腹泻 |
| 呕吐型毒素（Cereulide） | 小肽环状物，耐热 | 恶心、呕吐 |

### 6.3 临床表现

- **潜伏期**：
  - 呕吐型：0.5h~6h
  - 腹泻型：8h~16h
- **病程**：自限性，24h~36h
- **症状**：
  - 呕吐型：恶心、呕吐为主
  - 腹泻型：腹痛、水样腹泻为主

## 7. 质量控制

### 7.1 阳性对照
- 蜡样芽孢杆菌ATCC 11778
- 蜡样芽孢杆菌ATCC 14579

### 7.2 阴性对照
- 枯草芽孢杆菌ATCC 6633
- 大肠埃希氏菌ATCC 25922

### 7.3 培养基质控
- MYP平板应促进蜡样芽孢杆菌生长，抑制杂菌
- 30℃培养24h应出现典型菌落

---

## 8. 安全注意事项

### 8.1 生物安全
- 蜡样芽孢杆菌为条件致病菌
- 操作时注意防护

### 8.2 操作要点
- 样品处理时避免芽孢飞溅
- 热处理试验可用于区分蜡样芽孢杆菌（耐热）与类似菌
- 注意与苏云金芽孢杆菌鉴别（后者产生伴胞晶体）

---

## 9. 附录

### 9.1 检验意义

- **粮谷类食品**：米饭、米粉、面条等蜡样芽孢杆菌食物中毒主要来源
- **乳制品**：巴氏杀菌不能杀死芽孢
- **婴幼儿食品**：严格限量要求

### 9.2 预防措施

- **食品加工**：
  - 快速冷却（10℃~55℃危险带）
  - 低温贮存（<4℃）
  - 彻底加热（>75℃）

- **加工环境**：
  - 控制加工环境卫生
  - 防止交叉污染
