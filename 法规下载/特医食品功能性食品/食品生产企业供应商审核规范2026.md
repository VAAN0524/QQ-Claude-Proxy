# 食品生产企业供应商审核规范

**发布机关**: 国家市场监督管理总局
**施行日期**: 2024年5月1日
**最新修订**: 2026年1月

---

## 第一部分：总则

### 第一条【制定目的】
为规范食品生产企业供应商审核工作，确保采购物料的质量安全，制定本规范。

### 第二条【适用范围】
本规范适用于：
1. 食品生产企业对供应商的审核
2. 特殊医学用途配方食品生产企业对供应商的审核
3. 保健食品生产企业对供应商的审核
4. 乳制品生产企业对供应商的审核

### 第三条【审核原则】
供应商审核应当遵循以下原则：
1. **风险导向**：根据物料风险等级确定审核频次和深度
2. **客观公正**：客观评价供应商的质量保证能力
3. **持续改进**：促进供应商持续改进质量管理

---

## 第二部分：供应商分类

### 第四条【物料分类】
根据物料对产品质量的影响程度，将物料分为三类：

| 类别 | 定义 | 示例 |
|------|------|------|
| A类（关键物料） | 直接影响产品质量安全的主要原料 | 特医食品蛋白原料、维生素、矿物质等 |
| B类（重要物料） | 对产品质量有重要影响的辅料 | 食品添加剂、加工助剂等 |
| C类（一般物料） | 对产品质量影响较小的包装材料等 | 外包装材料、标签等 |

### 第五条【供应商分类】
根据审核结果，将供应商分为四类：

| 类别 | 条件 | 管理要求 |
|------|------|----------|
| 优秀供应商 | 审核评分≥90分 | 优先采购，减少审核频次 |
| 合格供应商 | 75分≤审核评分<90分 | 正常采购，正常审核 |
| 观察供应商 | 60分≤审核评分<75分 | 限制采购，增加审核频次 |
| 不合格供应商 | 审核评分<60分 | 停止采购，限期整改 |

## 第三部分：审核内容

### 第六条【资质审核】
审核供应商是否具备以下资质：
1. 营业执照
2. 生产许可证（如适用）
3. 产品注册/备案凭证（如适用）
4. 出入境检验检疫证明（进口物料）
5. 有机产品认证（如适用）
6. 其他必要的资质文件

### 第七条【质量管理体系审核】
审核供应商的质量管理体系：
1. 是否建立质量管理体系
2. 质量方针和目标是否明确
3. 组织机构和职责是否清晰
4. 人员资质和培训是否到位
5. 文件管理是否规范

### 第八条【生产现场审核】
审核供应商的生产现场：
1. 厂区环境是否整洁
2. 厂房设施是否符合要求
3. 生产设备是否完好
4. 物料管理是否规范
5. 生产过程是否受控
6. 检验能力是否满足要求

### 第九条【质量控制审核】
审核供应商的质量控制：
1. 是否设立质检部门
2. 检验设备是否齐全
3. 检验方法是否经过验证
4. 检验记录是否完整
5. 不合格品是否得到控制

### 第十条【产品审核】
审核供应商的产品质量：
1. 产品质量标准是否符合要求
2. 产品检验报告是否合格
3. 历史供货质量是否稳定
4. 是否有质量事故

### 第十一条【服务审核】
审核供应商的服务能力：
1. 供货能力是否满足需求
2. 交货期是否准确
3. 售后服务是否及时
4. 沟通是否顺畅

### 第十二条【合规性审核】
审核供应商的合规性：
1. 是否遵守法律法规
2. 是否有违规记录
3. 是否有诚信问题
4. 是否有环保问题

---

## 第四部分：审核程序

### 第十三条【审核准备】
1. 确定审核组长和审核员
2. 制定审核计划
3. 准备审核文件和记录表格
4. 通知供应商审核时间

### 第十四条【审核实施】
1. 召开首次会议
2. 现场查看
3. 文件审查
4. 人员访谈
5. 召开末次会议

### 第十五条【审核报告】
审核完成后，应当形成审核报告：
1. 审核概况
2. 审核发现的问题
3. 审核评分
4. 审核结论
5. 整改要求

### 第十六条【整改验证】
1. 供应商应当在规定时限内完成整改
2. 提交整改报告
3. 审核组验证整改效果

---

## 第五部分：审核评分

### 第十七条【评分标准】
供应商审核采用百分制评分：

| 审核项目 | 标准分 | 评分要点 |
|----------|--------|----------|
| 资质文件 | 10分 | 资质齐全有效 |
| 质量管理体系 | 20分 | 体系健全，运行有效 |
| 生产现场 | 20分 | 现场管理规范 |
| 质量控制 | 20分 | 检验能力满足要求 |
| 产品质量 | 15分 | 质量稳定，合格率高 |
| 服务能力 | 10分 | 供货及时，服务良好 |
| 合规性 | 5分 | 无违规记录 |

### 第十八条【评分细则】

1. **资质文件（10分）**
   - 资质齐全有效：10分
   - 资质齐全但有部分过期：6分
   - 资质不齐全：3分
   - 无资质：0分

2. **质量管理体系（20分）**
   - 体系健全，运行有效：20分
   - 体系基本健全，运行基本有效：15分
   - 体系不健全：8分
   - 无质量管理体系：0分

3. **生产现场（20分）**
   - 现场管理规范：20分
   - 现场管理基本规范：15分
   - 现场管理一般：8分
   - 现场管理混乱：0分

4. **质量控制（20分）**
   - 检验能力完全满足要求：20分
   - 检验能力基本满足要求：15分
   - 检验能力不足：8分
   - 无检验能力：0分

5. **产品质量（15分）**
   - 质量稳定，合格率100%：15分
   - 质量较稳定，合格率≥98%：12分
   - 质量一般，合格率≥95%：8分
   - 质量不稳定，合格率<95%：0分

6. **服务能力（10分）**
   - 供货及时，服务良好：10分
   - 供货基本及时，服务一般：7分
   - 供货不及时，服务差：3分

7. **合规性（5分）**
   - 无违规记录：5分
   - 有轻微违规：2分
   - 有严重违规：0分

---

## 第六部分：审核频次

### 第十九条【审核频次】
根据物料类别和供应商类别确定审核频次：

| 供应商类别\物料类别 | A类物料 | B类物料 | C类物料 |
|---------------------|---------|---------|---------|
| 优秀供应商 | 每年1次 | 每2年1次 | 每3年1次 |
| 合格供应商 | 每年1次 | 每2年1次 | 每3年1次 |
| 观察供应商 | 每年2次 | 每年1次 | 每2年1次 |

### 第二十条【临时审核】
出现以下情况时，应当进行临时审核：
1. 供应商发生重大变更
2. 供应商出现质量问题
3. 供应商被监管部门处罚
4. 客户投诉供应商物料

---

## 第七部分：合格供应商管理

### 第二十一条【合格供应商名单】
企业应当建立合格供应商名单：
1. 通过审核的供应商列入合格供应商名单
2. 从合格供应商名单中采购物料
3. 定期更新合格供应商名单

### 第二十二条【供应商档案】
应当为每个合格供应商建立档案：
1. 供应商基本信息
2. 资质文件
3. 审核报告
4. 供货记录
5. 质量记录
6. 投诉记录

### 第二十三条【供应商绩效评价】
定期对供应商进行绩效评价：
1. 产品质量合格率
2. 交货及时率
3. 服务满意度
4. 问题响应速度
5. 持续改进情况

---

## 第八部分：附则

### 第二十四条【记录保存】
供应商审核相关记录应当保存至供应商合作关系终止后不少于3年。

### 第二十五条【施行日期】
本规范自2024年5月1日起施行。
