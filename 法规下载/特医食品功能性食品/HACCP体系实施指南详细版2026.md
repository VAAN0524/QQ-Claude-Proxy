# HACCP体系实施指南详细版（2026）

## 指南信息
- **指南名称**：危害分析与关键控制点体系实施指南
- **发布机构**：国家市场监督管理总局
- **实施日期**：2026年1月1日
- **适用范围**：食品生产企业建立和实施HACCP体系
- **依据标准**：GB/T 27341-2024、CAC/RCP 1-1969

---

## 第一章 HACCP概述

### 第一条 HACCP定义
HACCP（Hazard Analysis and Critical Control Point）即危害分析与关键控制点，是一个国际公认的、科学的、系统的食品安全管理方法，通过识别、评估和控制食品安全危害，确保食品安全。

### 第二条 HACCP七项原则
1. **原则1**：进行危害分析
2. **原则2**：确定关键控制点（CCP）
3. **原则3**：建立关键限值
4. **原则4**：建立关键控制点监控程序
5. **原则5**：建立纠偏措施
6. **原则6**：建立验证程序
7. **原则7**：建立记录保持程序

### 第三条 HACCP应用前提
HACCP体系的建立和实施应建立在良好操作规范（GMP）和卫生标准操作程序（SSOP）的基础上。

## 第二章 HACCP体系建立准备

### 第四条 管理承诺
最高管理者应：
1. 承诺建立、实施、保持和改进HACCP体系
2. 任命HACCP小组组长
3. 提供必要的资源（人力、财力、设备、信息）
4. 定期评审HACCP体系

### 第五条 组建HACCP小组
#### 小组组成
| 成员 | 专业要求 | 职责 |
|------|---------|------|
| 组长 | 食品科学、质量管理 | 协调HACCP体系建立和实施 |
| 质量管理人员 | 质量管理、检验 | 危害分析、验证 |
| 生产管理人员 | 食品工艺、工程 | CCP监控、纠偏 |
| 设备维护人员 | 机械、电气 | 设备验证、维护 |
| 采购人员 | 供应链管理 | 供应商控制 |
| 检验人员 | 食品检验 | 监控、验证 |

#### 小组职责
1. 制定HACCP计划
2. 协调HACCP体系的实施
3. 监督HACCP体系的运行
4. 组织HACCP体系的验证
5. 负责HACCP体系的改进

### 第六条 产品描述
应对每种产品（或产品类别）进行全面描述：

#### 产品描述要素
| 项目 | 描述内容 | 示例 |
|------|---------|------|
| 产品名称 | 产品的正式名称 | 特殊医学用途婴儿配方食品 |
| 原料成分 | 所有原料和辅料 | 乳清蛋白粉、植物油、维生素等 |
| 产品特性 | 物理、化学特性 | 粉末状、水分≤5%、脂肪≥25% |
| 加工方式 | 主要加工工艺 | 喷雾干燥、干混 |
| 包装形式 | 包装材料和方式 | 铝塑复合袋、充氮包装 |
| 贮存条件 | 贮存温度和湿度 | 常温、阴凉干燥处 |
| 保质期 | 保质期长短 | 24个月 |
| 预期用途 | 食用对象和方式 | 0~6个月婴儿、冲调后食用 |
| 销售方式 | 销售渠道和对象 | 药店、母婴店 |

### 第七条 绘制工艺流程图
#### 流程图要求
1. 流程图应覆盖从原料接收到成品销售的全过程
2. 流程图应清晰、准确、完整
3. 流程图应经过现场确认

#### 工艺流程图示例（特殊医学用途配方食品）
```
原料验收→外包装清理→贮存→配料→预处理→混合→均质→杀菌→浓缩→喷雾干燥→筛分→包装→金属探测→检验→成品贮存→销售
```

#### 流程图确认
1. HACCP小组应现场验证流程图的准确性
2. 流程图如有变更，应及时更新并重新确认
3. 流程图确认应由HACCP小组成员签字确认

## 第三章 危害分析（原则1）

### 第八条 危害识别
应识别所有从原料种植、收获、加工、包装、贮存、运输到销售全过程中可能存在的生物、化学和物理危害。

#### 生物性危害
| 危害类型 | 来源 | 潜在后果 |
|---------|------|---------|
| 细菌 | 原料、人员、环境 | 食物中毒、腐败 |
| 病毒 | 人员、水源 | 病毒感染 |
| 寄生虫 | 原料 | 寄生虫感染 |
| 霉菌及其毒素 | 原料、贮存 | 真菌毒素中毒 |

#### 化学性危害
| 危害类型 | 来源 | 潜在后果 |
|---------|------|---------|
| 天然毒素 | 原料 | 急性/慢性中毒 |
| 农药残留 | 原料 | 慢性中毒 |
| 兽药残留 | 原料 | 过敏、中毒 |
| 重金属 | 原料、设备 | 慢性中毒 |
| 食品添加剂 | 添加剂使用 | 超量危害 |

#### 物理性危害
| 危害类型 | 来源 | 潜在后果 |
|---------|------|---------|
| 金属碎片 | 设备磨损 | 损伤 |
| 玻璃 | 包装、灯具 | 损伤 |
| 石头、沙粒 | 原料 | 牙齿损伤 |
| 塑料 | 包装、设备 | 损伤 |

### 第九条 危害评估
对识别出的危害进行评估，确定：
1. 危害的严重性（高、中、低）
2. 危害发生的可能性（高、中、低）
3. 是否为显著危害

#### 危害评估矩阵
| 严重性\可能性 | 高 | 中 | 低 |
|-------------|-----------|-----------|---------|
| 高 | 显著危害 | 显著危害 | 需评估 |
| 中 | 显著危害 | 需评估 | 不显著 |
| 低 | 需评估 | 不显著 | 不显著 |

### 第十条 危害控制措施
针对每个显著危害，确定控制措施：
1. 改进工艺或设备
2. 调整原料、配方
3. 制定操作规程
4. 建立监控程序

## 第四章 确定关键控制点（原则2）

### 第十一条 CCP判定方法
使用CCP决策树判断：

#### CCP判定树
```
问题1：针对该危害是否有控制措施？
→ 否：不是CCP，应修改工艺或产品
→ 是：进入问题2

问题2：该步骤是否专门用于消除或降低危害到可接受水平？
→ 是：CCP
→ 否：进入问题3

问题3：已识别的危害是否可能超过可接受水平或增加到不可接受水平？
→ 否：不是CCP
→ 是：进入问题4

问题4：后续步骤能否消除危害或降低到可接受水平？
→ 是：不是CCP
→ 否：CCP
```

### 第十二条 常见CCP示例

#### 特医食品生产常见CCP
| 工序 | 危害 | 关键限值 | 监控方法 |
|------|------|---------|---------|
| 原料验收 | 生物、化学、物理 | 符合原料标准 | 检验合格证 |
| 杀菌 | 致病菌 | 杀菌温度×时间 | 温度、时间记录 |
| 金属探测 | 金属异物 | Fe≥φ2.0mm，非Fe≥φ2.5mm | 金属探测器测试块 |
| 微量营养素添加 | 营养素过量 | ±5%规定量 | 称量记录 |

## 第五章 建立关键限值（原则3）

### 第十三条 关键限值的确定
关键限值（CL）是区分安全与不安全的界限，应：
1. 可测量
2. 可操作
3. 基于科学依据
4. 得到验证

#### 关键限值来源
1. 食品安全国家标准
2. 科学文献
3. 实验数据
4. 专家意见
5. 法规要求

### 第十四条 操作限值
为确保不超出关键限值，应设定操作限值（OL）：
- 操作限值应比关键限值更严格
- 操作限值为监控提供预警

```
示例：
关键限值（CL）：杀菌温度85℃
操作限值（OL）：杀菌温度87℃（预警）
```

## 第六章 建立监控程序（原则4）

### 第十五条 监控内容
对每个CCP应监控：
1. **监控对象**：监控什么（温度、时间、pH等）
2. **监控方法**：如何监控（测量、观察）
3. **监控频率**：监控频次（连续、间歇）
4. **监控人员**：谁来监控（培训合格的人员）

### 第十六条 监控方法
| 监控类型 | 适用情况 | 示例 |
|---------|---------|------|
| 连续监控 | 关键参数 | 温度自动记录 |
| 间歇监控 | 非关键参数 | 每小时检查一次 |

### 第十七条 监控记录
监控记录应包括：
1. 产品名称和批次
2. CCP点名称
3. 监控日期和时间
4. 监控对象实测值
5. 关键限值
6. 操作人员签名
7. 异常情况记录

---

## 第七章 建立纠偏措施（原则5）

### 第十八条 纠偏措施类型
1. **调整工艺**：使CCP恢复控制
2. **隔离产品**：受影响的产品隔离存放
3. **产品评估**：评估受影响产品的安全性
4. **产品处置**：返工、销毁或降级使用
5. **记录和报告**：记录纠偏过程

### 第十九条 纠偏措施程序
```
发现偏离→立即停止生产→隔离受影响产品→查找偏离原因→采取纠偏措施→验证恢复控制→评估受影响产品→产品处置→记录→报告
```

### 第二十条 产品评估
受影响产品应评估：
1. 偏离持续时间
2. 偏离程度
3. 受影响产品数量
4. 评估产品安全性

---

## 第八章 建立验证程序（原则6）

### 第二十一条 验证内容
1. **确认**：HACCP计划是否科学、完整
2. **验证**：HACCP体系是否有效运行
3. **审核**：HACCP体系的符合性和有效性

### 第二十二条 验证方法
| 验证方法 | 频率 | 内容 |
|---------|------|------|
| CCP监控设备校准 | 按规定 | 设备准确性 |
|针对性取样检测|按计划|产品安全性|
|CCP记录审核|每日|记录完整性|
|HACCP体系审核|每年|体系有效性|

### 第二十三条 验证记录
验证记录应包括：
1. 验证目的和范围
2. 验证方法和程序
3. 验证结果和结论
4. 验证人员签名
5. 验证日期

---

## 第九章 建立记录保持程序（原则7）

### 第二十四条 记录类型
1. **HACCP计划和支持文件**：体系建立文件
2. **CCP监控记录**：CCP日常监控
3. **纠偏措施记录**：偏离处理记录
4. **验证记录**：体系验证记录
5. **培训记录**：人员培训记录

### 第二十五条 记录保存期限
| 记录类型 | 保存期限 | 说明 |
|---------|---------|------|
| CCP监控记录 | 产品保质期后1年 | 不少于2年 |
| 纠偏措施记录 | 永久保存 | 便于追溯 |
| 验证记录 | 永久保存 | 体系评价依据 |
| 培训记录 | 3年 | 人员资质证明 |

### 第二十六条 记录要求
1. 记录应及时、准确、真实
2. 记录应清晰可辨，不得涂改
3. 记录应经相关人员审核
4. 记录应妥善保存，防止损坏、丢失

---

## 第十章 HACCP计划的实施与保持

### 第二十七条 培训
应制定培训计划，培训内容包括：
1. HACCP原理和知识
2. 公司HACCP计划
3. 个人职责和操作要求
4. 记录和报告要求

### 第二十八条 HACCP计划评审
1. **年度评审**：每年至少一次
2. **变更评审**：发生重大变更时
3. **特殊情况评审**：出现严重偏离时

评审内容：
1. 体系运行的符合性
2. 体系运行的有效性
3. 改进的机会

### 第二十九条 HACCP计划更新
以下情况应更新HACCP计划：
1. 原料或产品配方改变
2. 加工工艺或设备改变
3. 包装或贮存条件改变
4. 销售方式或消费对象改变
5. 科学认识改变
6. 法规标准变化
