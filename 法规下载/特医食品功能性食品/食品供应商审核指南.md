# 食品生产企业供应商审核指南

## 指南基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家市场监督管理总局 |
| 发布年份 | 2024年 |
| 适用范围 | 食品生产企业供应商管理 |
| 指南性质 | 技术指导原则 |

## 1. 总则

### 1.1 目的

指导食品生产企业对供应商进行审核，确保供应商具备持续提供合格物料和服务的能力，保障产品质量和安全。

### 1.2 适用范围

本指南适用于：
1. 原料供应商审核
2. 包装材料供应商审核
3. 辅料供应商审核
4. 服务供应商审核
5. 其他供应商审核

### 1.3 基本原则

| 原则 | 说明 |
|------|------|
| **基于风险** | 根据风险确定审核深度 |
| **全面评估** | 全面评估供应商能力 |
| **持续改进** | 促进供应商持续改进 |
| **双赢合作** | 建立长期合作关系 |

## 2. 供应商分类

### 2.1 按物料重要性分类

| 类别 | 说明 | 审核要求 |
|------|------|----------|
| **关键供应商** | 影响产品质量安全 | 现场审核 |
| **重要供应商** | 间接影响产品质量 | 现场或远程审核 |
| **一般供应商** | 不直接影响产品质量 | 文件审核 |

### 2.2 按物料类型分类

| 物料类型 | 审核重点 |
|----------|----------|
| **原料** | 质量控制、生产管理 |
| **包装材料** | 材质安全、卫生控制 |
| **辅料** | 成分控制、纯度 |
| **耗材** | 质量稳定性 |

### 2.3 按供应商规模分类

| 规模 | 审核重点 |
|------|----------|
| 大型供应商 | 管理体系、持续改进 |
| 中型供应商 | 质量控制、生产能力 |
| 小型供应商 | 质量控制、合规性 |

## 6. 审核发现

### 6.1 发现分类

| 类别 | 定义 | 示例 |
|------|------|------|
| **不符合项** | 不符合标准要求 | 无检验记录 |
| **观察项** | 有待改进 | 标识不够清晰 |
| **建议项** | 改进建议 | 可以改进的地方 |

### 6.2 不符合项分级

| 级别 | 定义 | 示例 |
|------|------|------|
| **严重不符合** | 严重影响产品质量安全 | 无质量体系 |
| **一般不符合** | 影响产品质量安全 | 部分记录不全 |
| **轻微不符合** | 对产品质量影响较小 | 标识不规范 |

### 6.3 问题分析

| 分析方法 | 说明 |
|----------|------|
| **5Why** | 连续问5次为什么 |
| **鱼骨图** | 因果分析 |
| **风险分析** | 风险评估 |

## 7. 审核报告

### 7.1 报告内容

| 章节 | 内容 |
|------|------|
| 1. 基本信息 | 供应商名称、地址、联系方式 |
| 2. 审核概述 | 审核目的、范围、标准 |
| 3. 审核发现 | 审核发现详情 |
| 4. 不符合项 | 不符合项清单 |
| 5. 优点 | 供应商优点 |
| 6. 结论 | 审核结论 |
| 7. 改进建议 | 改进建议 |

### 7.2 审核结论

| 结论 | 说明 | 后续措施 |
|------|------|----------|
| **推荐** | 符合要求 | 可以合作 |
| **有条件推荐** | 基本符合，需整改 | 整改后审核 |
| **不推荐** | 不符合要求 | 不推荐合作 |

### 7.3 评分系统

| 评分 | 等级 | 结论 |
|------|------|------|
| 90-100分 | A | 优秀 |
| 80-89分 | B | 良好 |
| 70-79分 | C | 合格 |
| 60-69分 | D | 需改进 |
| <60分 | E | 不合格 |

## 8. 整改验证

### 8.1 整改要求

| 不符合级别 | 整改时限 |
|------------|----------|
| 严重不符合 | 1个月 |
| 一般不符合 | 2个月 |
| 轻微不符合 | 3个月 |

### 8.2 整改验证

| 验证方式 | 说明 |
|----------|------|
| 文件验证 | 提交整改文件 |
| 现场验证 | 现场验证整改效果 |
| 跟踪验证 | 后续跟踪验证 |

### 8.3 整改关闭

| 条件 | 说明 |
|------|------|
| 原因分析 | 原因分析清楚 |
| 纠正措施 | 纠正措施有效 |
| 预防措施 | 预防措施到位 |
| 验证合格 | 验证合格 |

## 9. 持续监控

### 9.1 定期审核

| 供应商等级 | 审核频率 |
|------------|----------|
| A级 | 每2-3年 |
| B级 | 每1-2年 |
| C级 | 每年 |
| D/E级 | 审核后决定 |

### 9.2 绩效监控

| 监控指标 | 目标 |
|----------|------|
| 来料合格率 | ≥98% |
| 交付及时率 | ≥95% |
| 服务响应 | 及时响应 |
| 持续改进 | 持续改进 |

### 9.3 供应商分级

| 等级 | 条件 | 优惠政策 |
|------|------|----------|
| 优秀 | A级+持续改进 | 优先采购、宽松付款 |
| 合格 | B级/C级 | 正常合作 |
| 观察 | D级 | 限制采购 |
| 淘汰 | E级 | 终止合作 |

|------|------|
