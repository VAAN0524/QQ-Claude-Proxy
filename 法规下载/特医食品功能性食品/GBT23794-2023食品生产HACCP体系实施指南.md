# GB/T 23794-2023 食品生产HACCP体系实施指南

**标准编号**: GB/T 23794-2023
**实施日期**: 2024年6月1日
**替代标准**: GB/T 27341-2009部分内容
**发布部门**: 国家市场监督管理总局

---

## 1. 范围

本标准提供了食品生产企业危害分析与关键控制点(HACCP)体系的实施指南。

本标准适用于各类食品生产企业建立、实施、保持和改进HACCP体系。

## 3. 术语和定义

### 3.1 HACCP (Hazard Analysis and Critical Control Point)
危害分析与关键控制点，对食品安全显著危害进行识别、评估和控制的体系。

### 3.2 危害 (Hazard)
食品中生物的、化学的或物理的因素所引起的、可能对健康产生不良影响的状况。

### 3.3 显著危害 (Significant Hazard)
必须由HACCP体系控制的、可能发生、一旦发生将对消费者导致不可接受的健康风险的危害。

### 3.4 关键控制点 (CCP, Critical Control Point)
能够进行控制，并且该控制对防止、消除某一食品安全危害或将其降低到可接受水平所必需的某一步骤。

### 3.5 关键限值 (Critical Limit)
区分可接受与不可接受的标准，是CCP上预防、消除或降低危害至可接受水平的最大值或最小值。

### 3.6 操作限值 (Operating Limit)
比关键限值更严格的限值，由操作人员使用，用以降低偏离关键限值风险的标准。

### 3.7 监控 (Monitoring)
为评估CCP是否处于控制之内，对控制参数进行有计划的连续观察或测量。

### 3.8 纠偏措施 (Corrective Action)
当监控结果表明CCP失控时，采取的措施。

### 3.9 确认 (Validation)
获得证据，证明HACCP计划的各要素是有效的。

### 3.10 验证 (Verification)
除监控外，用以确定HACCP体系是否按HACCP计划有效运行的活动。

## 4. HACCP七大原理

### 原理1：进行危害分析
识别和评估各环节的生物、化学、物理危害。

### 原理2：确定关键控制点(CCP)
根据危害分析结果，确定关键控制点。

### 原理3：建立关键限值
对每个CCP确定关键限值。

### 原理4：建立监控程序
对CCP进行监控，以证实其处于受控状态。

### 原理5：建立纠偏措施
当监控显示CCP失控时，采取纠偏措施。

### 原理6：建立验证程序
验证HACCP体系的有效性。

### 原理7：建立记录保持程序
保持准确的记录。

## 5. HACCP体系建立步骤

### 步骤1：组建HACCP小组

**小组成员应包括**：
- 质量负责人（组长）
- 生产技术负责人
- 质量检验人员
- 设备维护人员
- 卫生管理人员
- 采购/仓储人员
- 外部专家（可选）

**HACCP小组职责**：
| 职责 | 内容 |
|------|------|
| 体系策划 | 制定HACCP计划 |
| 危害分析 | 识别和评估危害 |
| CCP确定 | 确定关键控制点 |
| 文件编制 | 编制HACCP文件 |
| 培训 | 组织HACCP培训 |
| 验证 | 验证体系有效性 |

### 步骤2：产品描述

**产品描述应包括**：

| 项目 | 内容 |
|------|------|
| 产品名称 | 产品的全称 |
| 原辅料 | 所有原辅料清单 |
| 配料 | 产品配方 |
| 加工方式 | 热处理、冷加工等 |
| 包装形式 | 罐装、袋装、瓶装等 |
| 贮存条件 | 常温、冷藏、冷冻 |
| 保质期 | 产品的保质期限 |
| 预期用途 | 消费者食用方式 |
| 目标人群 | 婴幼儿、普通成人、特殊人群等 |

### 步骤3：识别预期用途

**预期用途考虑因素**：
- 是否将进一步加工
- 是否加热后食用
- 是否直接食用
- 适用人群
- 特殊要求（如清真、犹太食品等）

### 步骤4：绘制流程图

**流程图内容**：
- 原料接收
- 贮存
- 加工
- 包装
- 贮存
- 运输
- 销售

**流程图验证**：
- HACCP小组现场验证
- 确保覆盖所有工序
- 确认准确性

### 步骤5：现场确认流程图

**确认内容**：
- 工序是否完整
- 操作顺序是否正确
- 是否有遗漏或重复

## 6. 危害分析（原理1）

### 6.1 危害识别

**三大类危害**：

| 类型 | 常见危害 | 举例 |
|------|----------|------|
| **生物危害** | 细菌 | 沙门氏菌、金黄色葡萄球菌 |
| | 病毒 | 甲肝病毒、诺如病毒 |
| | 寄生虫 | 旋毛虫、绦虫 |
| | 真菌毒素 | 黄曲霉毒素 |
| **化学危害** | 天然毒素 | 河豚毒素、组胺 |
| | 农兽药残留 | 有机磷、抗生素 |
| | 重金属 | 铅、镉、汞 |
| | 食品添加剂 | 超范围/超量使用 |
| | 清洁剂残留 | 消毒剂、洗涤剂 |
| **物理危害** | 异物 | 玻璃、金属、塑料、石子 |
| | 放射性物质 | 核污染 |

### 6.2 危害评估

**评估因素**：
1. 危害的严重性
2. 危害发生的可能性
3. 是否有后续控制措施
4. 对健康的影响程度

**评估工具**：
- 决策树
- 风险矩阵

**风险矩阵示例**：

| 可能性\严重性 | 轻微 | 中等 | 严重 |
|--------------|------|------|------|
| 经常发生 | 中风险 | 高风险 | 高风险 |
| 偶尔发生 | 低风险 | 中风险 | 高风险 |
| 极少发生 | 低风险 | 低风险 | 中风险 |

### 6.3 危害分析工作单

| 工序 | 潜在危害 | 是否显著 | 判断依据 | 预防措施 | 是否CCP |
|------|----------|----------|----------|----------|---------|
| 原料接收 | 病原菌 | 是 | 原料可能污染 | 供应商控制、验收检验 | 是 |
| 原料贮存 | 病原菌生长 | 是 | 温度控制不当 | 温度控制、时间管理 | 是 |
| 热处理 | 病原菌存活 | 是 | 加热不足 | 温度/时间控制 | 是 |
| 金属探测 | 金属异物 | 是 | 设备磨损 | 金属探测器 | 是 |

## 7. 确定关键控制点（原理2）

### 7.1 CCP判断树

```
          [有显著危害?]
               |
          ┌──No──┐
          |       |
       [结束]   [Yes]
               |
    [有预防措施?]
          |
     ┌──No──┐
     |      |
  [结束]  [Yes]
          |
  [该步骤专用于控制?]
          |
     ┌──No──┐
     |      |
  [后序步骤控制]  [Yes]
          |       |
     ┌────┘      [Yes]
     |            |
[该点是否CCP?] [该点是否CCP?]
     |            |
    [Yes]       [No]
```

### 7.2 常见CCP

| 加工类型 | 常见CCP |
|----------|---------|
| 原料接收 | 原料验收(农残、重金属) |
| 贮存 | 冷藏/冷冻温度控制 |
| 热加工 | 加热温度/时间 |
| 冷却 | 冷却速度 |
| 包装 | 封口完整性 |
| 金属检测 | 金属探测 |
| 水处理 | 水质控制 |

## 8. 建立关键限值（原理3）

### 8.1 关键限值来源

| 来源 | 例子 |
|------|------|
| 科学文献 | 杀灭温度和时间 |
| 法规标准 | 食品安全国家标准 |
| 实验数据 | 热穿透试验 |
| 专家指南 | FDA、CAC指南 |

### 8.2 常见关键限值

| CCP | 关键限值 |
|-----|----------|
| 热加工 | 中心温度≥75℃，持续15秒 |
| 冷却 | 中心温度从60℃降至10℃，≤4小时 |
| 冷藏贮存 | 温度≤4℃ |
| 冷冻贮存 | 温度≤-18℃ |
| 酸度调节 | pH≤4.6 |
| 水分活度 | aw≤0.85 |
| 金属探测 | Fe≥Φ1.5mm，非Fe≥Φ2.0mm，SUS≥Φ2.5mm |

### 8.3 操作限值

| CCP | 关键限值 | 操作限值 |
|-----|----------|----------|
| 热加工 | 75℃ | 80℃ |
| 冷藏 | 4℃ | 2℃ |
| 冷冻 | -18℃ | -20℃ |

### 9.2 监控方法

## 10. 建立纠偏措施（原理5）

### 10.1 纠偏措施类型

| 类型 | 内容 |
|------|------|
| 隔离 | 隔离受影响产品 |
| 评估 | 评估产品安全性 |
| 处置 | 返工、销毁、降级 |
| 原因分析 | 分析偏离原因 |
| 预防 | 防止再次发生 |

### 10.2 纠偏措施流程

```
[发现偏离]
    |
[隔离产品]
    |
[评估影响]
    |
[处置产品]
    |
[分析原因]
    |
[采取预防措施]
    |
[验证效果]
```

### 10.3 纠偏措施示例

| CCP偏离 | 纠偏措施 |
|---------|----------|
| 热加工温度不足 | 延长加热时间或产品重新加热 |
| 冷藏温度超标 | 检查设备，评估产品，必要时销毁 |
| 发现金属异物 | 追溯同批次产品，全部隔离检查 |
| pH偏高 | 调整酸度，延长发酵时间 |

## 11. 建立验证程序（原理6）

### 11.1 验证内容

| 验证类型 | 内容 | 频率 |
|----------|------|------|
| 确认 | HACCP计划科学性 | 最初、每年、变更时 |
| 审核 | HACCP体系执行情况 | 每季度、半年 |
| 检查 | CCP控制效果 | 每月、季度 |
| 检验 | 产品检测 | 每批次 |

### 11.2 验证方法

| 方法 | 内容 |
|------|------|
| 文件审核 | 检查HACCP计划、记录 |
| 现场检查 | 检查实际操作 |
| 产品检测 | 微生物、理化检测 |
| 设备校准 | 计量器具校准 |

### 11.3 验证人员

| 人员 | 职责 |
|------|------|
| 内审员 | 内部体系审核 |
| HACCP组长 | HACCP计划确认 |
| 外部专家 | 独立验证 |

## 13. 不同食品类型HACCP要点

### 13.1 特医食品

**重点关注**：
- 原料安全性（营养素、污染物）
- 生产过程控制（防止交叉污染）
- 微生物控制
| 关键控制点 | 关键限值 |
|-----------|----------|
| 原料验收 | 符合药典/食品级标准 |
| 金属检测 | 符合相应标准 |

### 13.2 保健食品

**重点关注**：
- 原料安全性（功效成分、农残）
- 生产过程控制
| 关键控制点 | 关键限值 |
|-----------|----------|
| 原料验收 | 功效成分含量、农残、重金属 |
| 配料 | 准确性 |
| 金属检测 | 符合相应标准 |

### 13.3 乳制品

**重点关注**：
- 原料奶质量
- 杀菌温度/时间
| 关键控制点 | 关键限值 |
|-----------|----------|
| 原料奶验收 | 菌落总数、体细胞、抗生素 |
| 杀菌 | 温度、时间 |
| 冷链 | 温度≤4℃ |

## 15. HACCP体系实施常见问题

| 问题 | 原因 | 解决措施 |
|------|------|----------|
| CCP确定过多 | 危害分析不准确 | 重新进行危害分析 |
| 关键限值不合理 | 缺乏科学依据 | 参考法规标准，进行试验 |
| 监控记录不完整 | 培训不足 | 加强培训，明确责任 |
| 纠偏措施无效 | 未分析根本原因 | 使用根本原因分析工具 |

---

**标准实施要点**：
1. HACCP应建立在GMP/SSOP基础上
2. HACCP是动态体系，需要持续改进
3. 全员参与是成功的关键
4. 定期验证确保体系有效性
5. 记录是体系运行的证据
