# 计算机化系统管理规范

## 规范基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家药品监督管理局 |
| 发布年份 | 2023年 |
| 适用范围 | 药品、特医食品、保健食品生产企业 |
| 规范性质 | 技术指导原则 |

## 1. 总则

### 1.1 目的

为规范药品、特殊医学用途配方食品、保健食品生产企业计算机化系统管理，确保系统运行稳定、数据真实可靠，保障产品质量和安全，制定本规范。

### 1.2 适用范围

本规范适用于：
1. 生产执行系统（MES）
2. 实验室信息管理系统（LIMS）
3. 仓储管理系统（WMS）
4. 质量管理系统（QMS）
5. 设备监控系统（SCADA）
6. 其他影响产品质量的计算机化系统

### 1.3 基本原则

| 原则 | 说明 |
|------|------|
| **风险评估** | 基于风险评估确定系统管理策略 |
| **生命周期管理** | 覆盖系统全生命周期 |
| **验证合格** | 系统必须经过验证合格 |
| **数据完整** | 确保数据真实、准确、完整 |
| **持续改进** | 持续改进系统性能 |

## 2. 系统分类

### 2.1 GAMP 5分类

| 类别 | 说明 | 验证策略 |
|------|------|----------|
| **第1类** | 基础设施软件 | 供应商资质确认 |
| **第2类** | 标准软件（已配置） | 基于供应商文档验证 |
| **第3类** | 定制软件（已配置） | 全生命周期验证 |
| **第4类** | 定制软件（已定制） | 全生命周期验证 |
| **第5类** | 组合系统 | 综合验证策略 |

### 2.2 影响分类

| 类别 | 影响 | 管理要求 |
|------|------|----------|
| **直接影响** | 影响产品质量、安全、有效性 | 最严格管理 |
| **间接影响** | 不直接影响但支持系统 | 基本管理 |
| **无影响** | 与产品质量无关 | 常规管理 |

## 5. 数据管理

### 5.1 数据完整性

| 要求 | 实施措施 |
|------|----------|
| 可归因 | 用户管理、电子签名 |
| 清晰可读 | 数据格式、显示 |
| 同步记录 | 自动记录、时间戳 |
| 原始记录 | 原始数据保留 |
| 准确 | 输入验证、范围检查 |
| 完整 | 必填项、完整性检查 |
| 一致 | 逻辑检查、跨系统核对 |
| 持久 | 备份、归档 |
| 可获取 | 检索功能、权限控制 |

### 5.2 数据备份

| 备份类型 | 频率 | 保留时间 |
|----------|------|----------|
| 完全备份 | 每周 | 1个月 |
| 增量备份 | 每日 | 1周 |
| 差异备份 | 每日 | 1周 |
| 异地备份 | 每月 | 1年 |

### 5.3 数据恢复

| 要求 | 目标 |
|------|------|
| RTO（恢复时间目标） | ≤4小时 |
| RPO（恢复点目标） | ≤1天 |
| 数据完整性 | 100% |
| 恢复测试 | 每季度 |

## 6. 安全管理

### 6.1 物理安全

| 措施 | 要求 |
|------|------|
| 机房门禁 | 授权进入 |
| 视频监控 | 24小时监控 |
| 消防设施 | 消防报警、灭火 |
| 环境控制 | 温度、湿度、洁净度 |
| 电源保障 | UPS、备用电源 |

### 6.2 网络安全

| 措施 | 要求 |
|------|------|
| 网络隔离 | 生产网络与办公网络隔离 |
| 防火墙 | 配置防火墙规则 |
| 入侵检测 | 部署入侵检测系统 |
| 病毒防护 | 安装防病毒软件 |
| 漏洞修复 | 及时修补漏洞 |

### 6.3 应用安全

| 措施 | 要求 |
|------|------|
| 用户管理 | 独立账户、权限分级 |
| 密码策略 | 复杂度、定期更换 |
| 电子签名 | 唯一、不可伪造 |
| 审计追踪 | 记录所有操作 |
| 权限控制 | 最小权限原则 |

### 6.4 数据安全

| 措施 | 要求 |
|------|------|
| 数据加密 | 敏感数据加密 |
| 传输加密 | SSL/TLS加密 |
| 存储加密 | 数据库加密 |
| 备份加密 | 备份数据加密 |
| 脱敏 | 非生产环境脱敏 |

### 7.3 故障处理

#### 7.3.1 故障分级

| 级别 | 影响 | 响应时间 |
|------|------|----------|
| 一级 | 严重影响生产 | 立即 |
| 二级 | 影响部分功能 | 1小时 |
| 三级 | 轻微影响 | 4小时 |

#### 7.3.2 处理流程

```
故障发现
    ↓
故障报告
    ↓
故障定位
    ↓
故障修复
    ↓
恢复验证
    ↓
故障总结
```

### 7.4 变更管理

#### 7.4.1 变更分类

| 类型 | 说明 | 审批要求 |

| 一般变更 | 小的功能调整 | 中级审批 |
| 紧急变更 | 故障紧急修复 | 事后审批 |

#### 7.4.2 变更流程

```
变更申请
    ↓
风险评估
    ↓
审批
    ↓
变更计划
    ↓
变更实施
    ↓
变更验证
    ↓
变更关闭
```
