# 食品从业人员健康管理制度指南

> 制定依据：《中华人民共和国食品安全法》《食品生产经营监督检查管理办法》
> 适用范围：食品生产企业、食品销售企业、餐饮服务企业、保健食品生产企业、特医食品生产企业

---

## 一、总则

### 1.1 制定目的
为规范食品从业人员健康管理，防止传染病通过从业人员传播，保障食品安全，保护消费者身体健康，根据《中华人民共和国食品安全法》等法律法规，制定本指南。

### 1.2 适用范围
本指南适用于在中华人民共和国境内从事食品生产经营活动的企业和从业人员，包括：
- 食品生产企业
- 食品销售企业
- 餐饮服务企业
- 保健食品生产企业
- 特医食品生产企业
- 食品相关产品生产企业

### 1.3 基本原则
从业人员健康管理应当遵循以下原则：
1. 预防为主原则
2. 定期检查原则
3. 动态管理原则
4. 档案管理原则

---

## 二、健康检查要求

### 2.1 从业人员定义
从业人员是指从事食品生产经营活动的所有人员，包括：
1. 从事食品采购、验收、贮存、运输的人员
2. 从事食品加工制作的人员
3. 从事食品销售服务的人员
4. 从事食品检验检测的人员
5. 从事食品质量安全管理的人员
6. 清洁、消毒、杀虫灭害等与食品相关的人员

### 2.2 健康证明要求
食品从业人员必须取得有效的健康证明，方可从事食品生产经营活动。

**健康证明内容应当包括：**
1. 从业人员基本信息
2. 体检日期
3. 体检结论（合格/不合格）
4. 发证机构
5. 有效期限

**健康证明的有效期：**
- 健康证明有效期为1年
- 有效期满前应当重新进行体检

### 2.3 健康检查项目
从业人员健康检查应当包括以下项目：

| 检查项目 | 检查内容 | 检查依据 |
|---------|---------|---------|
| 既往病史 | 甲型病毒性肝炎、戊型病毒性肝炎等传染病病史 | 询问调查 |
| 肝功能 | 肝功能检查，排除肝脏疾病 | 实验室检查 |
| 胸部X光检查 | 排除活动性肺结核 | 影像学检查 |
| 皮肤检查 | 排除化脓性或渗出性皮肤病 | 体格检查 |
| 粪便检查 | 排除伤寒、痢疾等肠道传染病 | 实验室检查 |
| 其他项目 | 根据需要增加其他检查项目 | 按需检查 |

### 2.4 禁忌疾病
患有以下疾病的人员不得从事接触直接入口食品的工作：
1. 霍乱
2. 细菌性和阿米巴性痢疾
3. 伤寒和副伤寒
4. 病毒性肝炎（甲型、戊型）
5. 活动性肺结核
6. 化脓性或者渗出性皮肤病
7. 其他有碍食品安全的疾病

---

## 三、健康管理制度

### 3.1 建立健康档案
企业应当为每个从业人员建立健康档案，包括以下内容：

**档案基本信息：**
1. 姓名、性别、出生日期
2. 身份证号码
3. 职位/工种
4. 入职时间

**健康检查记录：**
1. 健康检查日期
2. 检查项目及结果
3. 健康证明编号
4. 健康证明有效期

**日常健康监测：**
1. 每日健康状况询问
2. 疾病报告记录
3. 请假记录
4. 复工体检记录

### 3.2 健康证明管理

**取得健康证明：**
1. 新入职人员应当在上岗前取得健康证明
2. 在职人员应当在健康证明有效期届满前1个月重新体检
3. 健康证明遗失的，应当及时补办

**健康证明查验：**
1. 企业应当定期查验从业人员健康证明
2. 发现健康证明过期的，应当立即停止其工作
3. 健康检查不合格的，不得安排接触直接入口食品的工作

**健康证明存档：**
1. 健康证明复印件应当归入个人健康档案
2. 健康档案应当至少保存至从业人员离职后2年
3. 鼓励使用电子档案管理健康证明

### 3.3 每日健康晨检
企业应当建立每日晨检制度，由指定人员负责检查从业人员健康状况。

**晨检内容包括：**
1. 从业人员精神状态
2. 有无发热、腹泻、呕吐等症状
3. 手部有无化脓性感染
4. 其他有碍食品安全的症状

**晨检记录：**
1. 每日晨检情况应当记录
2. 发现异常情况的处置应当记录
3. 晨检记录应当至少保存6个月

### 3.4 疾病报告制度
从业人员应当主动报告以下情况：
1. 被诊断为上述禁忌疾病之一的
2. 出现发热、腹泻、呕吐等症状
3. 手部出现化脓性感染的
4. 其他可能影响食品安全的情况

**报告程序：**
1. 从业人员发现身体不适应当立即向直接上级报告
2. 企业接到报告后应当立即停止其接触直接入口食品的工作
3. 安排其进行体检或治疗
4. 确认合格后方可恢复工作

---

## 四、培训教育制度

### 4.1 卫生知识培训
企业应当定期组织从业人员进行食品安全卫生知识培训。

**培训内容包括：**
1. 食品安全法律法规
2. 食品安全基础知识
3. 个人卫生要求
4. 清洁消毒方法
5. 交叉污染预防
6. 食源性疾病预防

### 4.2 培训频次和记录
- 新入职人员应当在入职时接受培训
- 在职人员每年至少接受1次培训
- 培训应当做好记录，记录应当至少保存2年

### 4.3 培训效果评估
- 培训后应当进行考核
- 考核不合格的应当重新培训
- 考核结果应当记录存档

---

## 五、个人卫生要求

### 5.1 个人卫生规范
从业人员应当遵守以下个人卫生要求：

**基本要求：**
1. 保持良好的个人卫生习惯
2. 勤洗手、勤剪指甲
3. 勤换洗工作服
4. 不留长指甲、不涂指甲油
5. 不佩戴饰物（戒指、手镯等）

**头发要求：**
1. 头发应当清洁整齐
2. 接触直接入口食品的人员应当戴发网或发帽
3. 头发不得外露

### 5.2 洗手消毒
从业人员应当在以下情况下洗手消毒：
1. 开始工作前
2. 处理食物前
3. 上厕所后
4. 处理生食后
5. 处理垃圾后
6. 触摸不洁物品后
7. 其他需要洗手的情况

**洗手方法：**
1. 用流动水冲洗双手
2. 使用洗手液或肥皂
3. 揉搓双手至少20秒
4. 冲洗干净
5. 用一次性纸巾或干手器烘干

---

## 六、管理职责

### 6.1 企业负责人职责
1. 负责建立从业人员健康管理制度
2. 保障健康检查和培训所需经费
3. 确保制度有效实施

### 6.2 食品安全管理人员职责
1. 组织从业人员健康检查
2. 管理从业人员健康档案
3. 组织卫生知识培训
4. 监督个人卫生执行情况
5. 处理健康相关问题

### 6.3 从业人员职责
1. 遵守健康管理制度
2. 主动进行健康检查
3. 报告身体健康状况
4. 保持良好个人卫生
5. 参加卫生知识培训

---

## 七、监督检查

### 7.1 自查要求
企业应当定期对从业人员健康管理进行自查：
1. 每月至少自查1次
2. 自查内容包括健康证明、个人卫生、疾病报告等
3. 自查发现问题的应当及时整改

### 7.2 记录保存
以下记录应当妥善保存：
1. 健康检查记录：至少保存2年
2. 培训记录：至少保存2年
3. 晨检记录：至少保存6个月
4. 疾病报告记录：至少保存2年

---

## 八、特殊要求

### 8.1 保健食品生产企业
除基本要求外，还应满足：
1. 建立更严格的健康检查制度
2. 对接触原辅料、包装材料的人员重点管理
3. 对从事检验检测的人员定期培训

### 8.2 特医食品生产企业
除基本要求外，还应满足：
1. 建立完整的健康追溯体系
2. 对关键岗位人员加强健康管理
3. 健康档案保存期限延长至离职后5年

---

## 九、应急处理

### 9.1 从业人员突发疾病处理
1. 立即停止其工作
2. 安排就医检查
3. 追踪密切接触者健康状况
4. 对相关工作场所进行消毒

### 9.2 疫情期间管理
疫情期间应当加强：
1. 每日体温检测
2. 健康码核查
3. 加强个人防护
4. 减少人员聚集
5. 做好环境消毒

---

### 10.1 食品从业人员健康管理制度（模板）

```

第二条 本制度适用于本企业所有从业人员。

第十二条 培训应当做好记录。

第五章 个人卫生

第十四条 工作时应当穿戴清洁工作服。

第六章 附则
```

## 附录B：健康检查机构查询

从业人员健康检查应当到具备相应资质的医疗机构进行，包括：
1. 当地疾病预防控制中心
2. 具备体检资质的公立医院
3. 具备体检资质的民营体检机构
4. 当地卫生健康部门指定的其他机构
