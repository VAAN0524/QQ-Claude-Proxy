## 范围

本标准规定了食品中铅的测定方法。

本标准适用于各类食品中铅的测定。

**第一法 石墨炉原子吸收光谱法**：适用于各类食品中低含量铅的测定，检出限为0.005 mg/kg。

**第二法 电感耦合等离子体质谱法（ICP-MS）**：适用于各类食品中铅的测定，检出限为0.001 mg/kg。

**第三法 火焰原子吸收光谱法**：适用于含铅量较高的食品，检出限为0.1 mg/kg。

---

## 术语和定义

### 2.1 铅 lead

一种重金属元素，具有蓄积性毒性，对人体神经、血液、消化等系统有危害。

### 2.2 石墨炉原子吸收光谱法 graphite furnace atomic absorption spectrometry

利用石墨炉作为原子化器，通过测定原子蒸气对特征辐射的吸收来测定元素含量的方法。

## 第一法：石墨炉原子吸收光谱法

### 3.1 原理

试样经消解处理后，注入石墨炉原子化器中，在高温下铅原子化，吸收283.3 nm共振线，在一定浓度范围内，其吸光度与铅含量成正比，与标准系列比较定量。

### 3.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 硝酸 | 优级纯 |
| 高氯酸 | 优级纯 |
| 过氧化氢 | 优级纯 |
| 磷酸二氢铵 | 基体改进剂，20 g/L |
| 硝酸钯 | 基体改进剂，1 g/L |
| 铅标准储备液 | 1000 mg/L |
| 去离子水 | 18.2 MΩ·cm |

### 3.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 石墨炉原子吸收光谱仪 | 配备自动进样器 |
| 铅空心阴极灯 | - |
| 微波消解仪 | - |
| 分析天平 | 感量0.1 mg |
| 电热板 | - |
| 聚四氟乙烯消解罐 | - |

### 3.4 仪器工作条件

| 参数 | 条件 |
|------|------|
| 波长 | 283.3 nm |
| 狭缝 | 0.5 nm |
| 灯电流 | 7.5 mA |
| 干燥温度 | 120℃，30 s |
| 灰化温度 | 800℃，20 s |
| 原子化温度 | 2100℃，3 s |
| 净化温度 | 2500℃，3 s |

### 3.5 分析步骤

#### 3.5.1 试样预处理

- **固体试样**：粉碎混匀，过40目筛
- **液体试样**：摇匀备用
- **含油脂试样**：先提取脂肪后测定

#### 3.5.2 试样消解

**微波消解法**：
准确称取0.3 g~0.5 g试样（精确至0.0001 g），置于微波消解罐中，加入5 mL硝酸、2 mL过氧化氢，密闭消解。消解程序：升温至120℃，保持5 min；升温至180℃，保持15 min。消解完全后，转移至25 mL容量瓶中，用水定容至刻度，摇匀备用。

**湿法消解法**：
准确称取1 g~2 g试样（精确至0.0001 g），置于锥形瓶中，加入10 mL硝酸+高氯酸（4:1）混合酸，加盖浸泡过夜。置于电热板上加热消解，至溶液澄清透明，继续加热至近干。冷却后加入0.5 mol/L硝酸溶解残渣，转移至25 mL容量瓶中定容，摇匀备用。

#### 3.5.3 标准曲线绘制

准确吸取0、0.5、1.0、2.0、4.0、8.0 ng/mL铅标准使用液，分别注入石墨炉，测定吸光度，绘制标准曲线。

#### 3.5.4 试样测定

取试样消解液，按标准曲线相同条件测定吸光度，从标准曲线上查出铅含量。

### 3.6 结果计算

```
X = (ρ - ρ₀) × V / m × 1000
```

其中：
- X：试样中铅的含量，单位为毫克每千克（mg/kg）
- ρ：测定液中铅的质量浓度，单位为纳克每毫升（ng/mL）
- ρ₀：空白液中铅的质量浓度，单位为纳克每毫升（ng/mL）
- V：试样消解液定容体积，单位为毫升（mL）
- m：试样质量，单位为克（g）

**结果保留两位有效数字**。

## 第二法：电感耦合等离子体质谱法（ICP-MS）

### 4.1 原理

试样经消解处理后，由电感耦合等离子体质谱仪测定，以元素特定质量数定性，采用外标法定量。

### 4.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 硝酸 | 优级纯，经过亚沸蒸馏 |
| 过氧化氢 | 优级纯 |
| 铅标准储备液 | 1000 mg/L |
| 内标溶液 | 铑或铼，1 mg/L |
| 氩气 | 高纯，99.999% |
| 去离子水 | 18.2 MΩ·cm |

### 4.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 电感耦合等离子体质谱仪 | - |
| 微波消解仪 | - |
| 分析天平 | 感量0.1 mg |
| 聚四氟乙烯消解罐 | - |

### 4.4 仪器工作条件

| 参数 | 条件 |
|------|------|
| RF功率 | 1550 W |
| 等离子气流量 | 15 L/min |
| 辅助气流量 | 0.9 L/min |
| 雾化气流量 | 1.0 L/min |
| 采样深度 | 8 mm |
| 测定同位素 | ²⁰⁸Pb、²⁰⁶Pb |

### 4.5 分析步骤

#### 4.5.1 试样消解

同3.5.2。

#### 4.5.2 标准曲线绘制

配制0、0.5、1.0、5.0、10.0、50.0 ng/mL铅标准系列溶液，加入内标溶液，按仪器工作条件测定，以信号强度比（铅/内标）对铅浓度绘制标准曲线。

#### 4.5.3 试样测定

取试样消解液，加入内标溶液，按标准曲线相同条件测定。

### 4.6 结果计算

同3.6。

## 第三法：火焰原子吸收光谱法

### 5.1 原理

试样经消解处理后，喷入空气-乙炔火焰，在高温下铅原子化，吸收283.3 nm共振线，在一定浓度范围内，其吸光度与铅含量成正比，与标准系列比较定量。

### 5.2 试剂和材料

| 试剂 | 规格 |
|------|------|
| 硝酸 | 优级纯 |
| 高氯酸 | 优级纯 |
| 铅标准储备液 | 1000 mg/L |
| 甲基异丁基酮（MIBK） | 分析纯 |
| 去离子水 | 18.2 MΩ·cm |

### 5.3 仪器和设备

| 设备 | 规格 |
|------|------|
| 火焰原子吸收光谱仪 | - |
| 铅空心阴极灯 | - |
| 分析天平 | 感量0.1 mg |
| 电热板 | - |

### 5.4 仪器工作条件

| 参数 | 条件 |
|------|------|
| 波长 | 283.3 nm |
| 狭缝 | 0.5 nm |
| 灯电流 | 7.5 mA |
| 空气流量 | 10 L/min |
| 乙炔流量 | 2.5 L/min |
| 燃烧器高度 | 7 mm |

### 5.5 分析步骤

#### 5.5.1 试样消解

同3.5.2。

#### 5.5.2 萃取浓缩（必要时）

对于铅含量较低的样品，可采用MIBK萃取浓缩。

#### 5.5.3 标准曲线绘制

配制0、0.5、1.0、2.0、4.0、6.0 μg/mL铅标准系列溶液，按仪器工作条件测定吸光度，绘制标准曲线。

#### 5.5.4 试样测定

取试样消解液，按标准曲线相同条件测定吸光度。

### 5.6 结果计算

```
X = (ρ - ρ₀) × V / m × 1000
```

其中：
- X：试样中铅的含量，单位为毫克每千克（mg/kg）
- ρ：测定液中铅的质量浓度，单位为微克每毫升（μg/mL）
- ρ₀：空白液中铅的质量浓度，单位为微克每毫升（μg/mL）
- V：试样消解液定容体积，单位为毫升（mL）
- m：试样质量，单位为克（g）

**结果保留两位有效数字**。

## 方法选择指南

| 食品类别 | 推荐方法 | 说明 |
|----------|----------|------|
| 粮食、蔬菜、水果 | 石墨炉原子吸收光谱法 | 含量低 |
| 肉制品、鱼类 | 火焰原子吸收光谱法 | 含量较高 |
| 乳制品 | 石墨炉原子吸收光谱法或ICP-MS | 含量低 |

| 茶叶 | 火焰原子吸收光谱法 | 含量高 |
| 保健品 | ICP-MS | 多元素同时测定 |

## 质量控制

### 8.1 空白试验

每批样品应同时做试剂空白，空白值应低于方法检出限。

### 8.2 平行试验

每批样品应进行平行试验，平行样相对偏差应≤20%。

### 8.3 加标回收试验

每批样品应进行加标回收试验，回收率应在85%~110%之间。

### 8.4 标准物质分析

定期分析有证标准物质，测定结果应在标准值范围内。

## 注意事项

1. **试剂纯度**：
   - 应使用优级纯或更高纯度的试剂
   - 实验用水应为去离子水或超纯水

2. **消解操作**：
   - 消解过程中应防止样品溅出
   - 湿法消解应先低温后高温
   - 微波消解应严格按照程序操作

3. **仪器维护**：
   - 定期清洁石墨管和雾化器
   - 更换空心阴极灯时应预热30 min
   - ICP-MS应定期进行质量校准和调谐

4. **安全防护**：
   - 铅是有毒元素，操作时应注意个人防护
   - 实验室应保持良好通风
   - 废液应集中处理，不得直接排放

5. **避免污染**：
   - 使用前玻璃器皿应用硝酸（1:1）浸泡24 h
   - 避免使用金属器具
   - 实验环境应保持清洁
