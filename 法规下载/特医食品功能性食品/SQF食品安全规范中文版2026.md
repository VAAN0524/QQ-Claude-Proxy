## 1. 概述

### 1.1 SQF体系介绍

SQF（Safe Quality Food）是全球食品安全倡议（GFSI）认可的食品安全管理体系认证标准，涵盖从初级生产到食品零售的全供应链。

### 1.2 SQF认证级别

| 级别 | 名称 | 要求 |
|------|------|------|
| **SQF Level 1** | 基础级别 | 食品安全基础 |
| **SQF Level 2** | 认证级别 | HACCP + 食品安全 |
| **SQF Level 3** | 综合级别 | HACCP + 质量 |

### 1.3 适用范围

SQF规范适用于：
1. 特殊医学用途配方食品
2. 保健食品
3. 婴幼儿配方食品
4. 其他食品和饮料
5. 食品包装材料
6. 食品储存和分销

## 5. 关键控制点（CCP）

### 5.1 常见CCP

| CCP | 关键限值 | 监控 | 纠偏措施 |
|-----|----------|------|----------|
| **金属检测** | 无金属片 | 每批 | 拒留产品 |
| **致病菌控制** | 阴性 | 每批 | 拒留/返工 |
| **温度控制** | 特定温度 | 连续 | 调整/隔离 |
| **水分活度** | aw≤0.6 | 每批 | 调整/隔离 |
| **pH控制** | 特定pH | 每批 | 调整/隔离 |
| **过敏原控制** | 无交叉污染 | 每批 | 清洗/隔离 |

### 5.2 CCP监控要求

| 要求 | 说明 |
|------|------|
| 监控方法 | 在线监测或人工检测 |
| 监控频率 | 足以及时发现偏离 |
| 监控人员 | 培训合格人员 |
| 监控记录 | 完整记录监控数据 |

## 16. SQF认证流程

### 16.1 认证申请

| 步骤 | 内容 |
|------|------|
| 1. 选择认证机构 | 选择SQF认可的认证机构 |
| 2. 提交申请 | 填写申请表 |
| 3. 签订合同 | 确定审核范围和费用 |

### 16.2 认证准备

| 步骤 | 内容 |
|------|------|
| 1. 体系建立 | 建立SQF体系 |
| 2. 体系运行 | 运行至少3个月 |
| 3. 内部审核 | 内部审核和改进 |
| 4. 申请审核 | 提交审核申请 |

### 16.3 认证审核

| 阶段 | 内容 | 时长 |
|------|------|------|
| 文件审核 | 审核体系文件 | 1-2天 |
| 现场审核 | 现场检查和记录审核 | 2-5天 |
| 不符合整改 | 整改不符合项 | 按要求 |

### 16.4 认证决定

| 结果 | 说明 |
|------|------|
| 通过 | 颁发证书 |
| 不通过 | 整改后重新审核 |
| 暂缓 | 部分通过，整改 |
