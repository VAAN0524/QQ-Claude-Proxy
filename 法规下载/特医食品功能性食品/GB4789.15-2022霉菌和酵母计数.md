## 范围

本部分规定了食品中霉菌和酵母的计数方法。

本部分适用于各类食品中霉菌和酵母的计数。

---

## 术语和定义

### 2.1 霉菌 moulds

菌丝体发达，不产生大型子实体的真菌，多数可形成分支的菌丝体。

### 2.2 酵母 yeasts

单细胞真菌，通常以出芽方式繁殖。

### 2.3 酵母菌落形成单位 yeast colony forming unit，YCFU

在固体培养基上，由单个酵母细胞繁殖形成的、肉眼可见的微生物集团。

## 设备和材料

### 3.1 设备

- 恒温培养箱：28℃±1℃
- 冰箱：2℃~5℃
- 恒温水浴箱：46℃±1℃
- 天平：感量0.1 g
- 均质器
- 振荡器
- 无菌吸管：1 mL、10 mL
- 无菌锥形瓶：250 mL、500 mL
- 无菌培养皿：直径90 mm
- 无菌试管：16 mm×160 mm
- 显微镜
- 放大镜或菌落计数器
- 均质袋

### 3.2 培养基和试剂

- 马铃薯葡萄糖琼脂（PDA）
- 孟加拉红（虎红）琼脂
- 氯硝胺18%甘油琼脂（DG18）
- 无菌生理盐水
- 无菌蒸馏水
- 75%乙醇

## 检验程序

```
检样 → 25g(mL)样品+225mL稀释液 → 均质 → 10倍系列稀释
                                    ↓
                    选择2-3个适宜稀释度 → 各取1mL → 加入无菌培养皿
                                    ↓
                    每皿加入15-20mL PDA或孟加拉红琼脂（46℃）→ 转动混合
                                    ↓
                    凝固后翻转平板 → 28±1℃培养5-7天 → 菌落计数
                                    ↓
                                    报告结果
```

---

## 操作步骤

### 5.1 样品的稀释

5.1.1 以无菌操作取检样25 g（或mL），放入含有225 mL无菌稀释液的无菌均质袋或均质杯内。

5.1.2 均质1 min~2 min，制成1:10的样品匀液。

5.1.3 用1 mL无菌吸管吸取1:10样品匀液1 mL，沿管壁缓慢注入含有9 mL稀释液的无菌试管中，振摇试管使其混合均匀，制成1:100的样品匀液。

5.1.4 按上述操作顺序，制作10倍系列稀释样品匀液。每递增稀释一次，换用1支1 mL无菌吸管。

### 5.2 接种

5.2.1 根据对样品污染状况的估计，选择2~3个适宜稀释度，每个稀释度接种2个无菌培养皿。

5.2.2 同时分别吸取1 mL样品匀液注入2个无菌培养皿内。

5.2.3 及时将15 mL~20 mL冷却至46℃的孟加拉红琼脂或马铃薯葡萄糖琼脂培养基倾注培养皿，并转动培养皿使其混合均匀。

### 5.3 培养

5.3.1 待琼脂凝固后，翻转平板。

5.3.2 置28℃±1℃恒温培养箱内培养。

5.3.3 培养5 d~7 d，必要时延长至14 d。

### 5.4 菌落计数

#### 5.4.1 霉菌菌落计数

观察平板，依据菌落形态（大小、颜色、质地等）计数霉菌菌落。

常见霉菌菌落特征：
- **曲霉属**：菌落较大，绒状或絮状，呈绿色、黑色、黄色等
- **青霉属**：菌落较小，绒状或絮状，呈绿色、灰绿色
- **毛霉属**：菌落生长迅速，呈白色或灰色
- **根霉属**：菌落呈白色或灰色，有假根

#### 5.4.2 酵母菌落计数

酵母菌落通常较小，圆润凸起，表面光滑，呈乳白色或粉红色。

#### 5.4.3 计数原则

- 选取菌落数在15 CFU~150 CFU之间的平板计数
- 如果两个稀释度都在范围内，取平均值计算
- 扩散生长的霉菌菌落不计入

## 结果计算与报告

### 6.1 计算方法

#### 6.1.1 若只有一个稀释度平板的菌落数在适宜计数范围内

计算该稀释度的两个平板平均菌落数，再乘以相应稀释倍数。

#### 6.1.2 若有两个连续稀释度的平板菌落数在适宜计数范围内

按以下公式计算：

```
N = ΣC / (n1 + 0.1 × n2) × d
```

其中：
- N：样品中霉菌和酵母总数
- ΣC：所有适宜计数范围内平板菌落数之和
- n1：第一个适宜稀释度的平板数
- n2：第二个适宜稀释度的平板数
- d：稀释因子（第一个适宜稀释度）

### 6.2 结果报告

#### 6.2.1 报告单位

- 固体样品：CFU/g
- 液体样品：CFU/mL

#### 6.2.2 报告示例

| 样品类型 | 检验结果 | 报告方式 |
|----------|----------|----------|
| 固体 | 12500 CFU/g | 12000 CFU/g 或 1.2×10⁴ CFU/g |
| 液体 | 850 CFU/mL | 850 CFU/mL |
| 未检出 | <10 | <10 CFU/g（或CFU/mL） |

## 结果解释

### 7.1 霉菌和酵母的卫生学意义

- 霉菌和酵母是评价食品卫生质量的指示菌
- 反映食品在生产、储存、运输过程中的卫生状况
- 某些霉菌可产生真菌毒素，对人体健康有害
- 酵母过多可导致食品变质

### 7.2 卫生标准参考

| 食品类别 | 霉菌限量 (CFU/g 或 CFU/mL) | 酵母限量 (CFU/g 或 CFU/mL) |
|----------|---------------------------|---------------------------|
| 粮食及其制品 | ≤ 10² ~ 10⁴ | - |
| 糕点、面包 | ≤ 10² ~ 10³ | - |
| 果汁饮料 | ≤ 10 ~ 10² | ≤ 10 ~ 10² |
| 婴幼儿配方食品 | ≤ 50 | ≤ 50 |
| 调味品 | ≤ 10² ~ 10³ | - |

## 质量控制

### 8.1 培养基质量控制

每批培养基使用前应进行质量控制：
- 无菌性试验：培养基应无菌生长
- 阳性对照：接种已知菌株（如黑曲霉、酿酒酵母），应生长良好

### 8.2 操作质量控制

- 每批检验应设置空白对照
- 定期参加能力验证或实验室间比对

### 8.3 设备维护

- 培养箱温度定期校准
- 天平定期检定
- 均质器定期维护

## 附录B（资料性）常见产毒霉菌

### B.1 黄曲霉

- 产毒种类：黄曲霉毒素
- 污染食品：花生、玉米、坚果等
- 毒性：强致癌性

### B.2 赭曲霉

- 产毒种类：赭曲霉毒素A
- 污染食品：谷物、咖啡、葡萄干等
- 毒性：肾毒性、致癌性

### B.3 伏马菌素产生菌

- 产毒种类：伏马菌素
- 污染食品：玉米及其制品
- 毒性：神经毒性、致癌性

## 注意事项

1. 霉菌孢子容易飞扬，操作时应注意防止交叉污染。

2. 培养时间较长，应定期观察菌落生长情况。

3. 某些霉菌可能产生色素，影响计数，应注意辨别。

4. 对于扩散生长的霉菌，可在菌落扩散边缘计数，或使用倒置显微镜计数。

5. 某些食品（如含防腐剂）可能抑制霉菌生长，可添加中和剂。

6. 霉菌和酵母同时计数时，应分别记录。

7. 操作过程应严格无菌操作。

8. 检验过程中产生的废弃物应经灭菌后处理。
