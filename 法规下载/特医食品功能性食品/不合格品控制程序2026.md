# 不合格品控制程序

**发布机关**: 国家市场监督管理总局
**施行日期**: 2024年1月1日
**最新修订**: 2026年1月

---

## 第一部分：总则

### 第一条【制定目的】
为规范不合格品的识别、隔离、评审和处置，防止不合格品非预期使用或交付，制定本程序。

### 第二条【适用范围】
本程序适用于：
1. 进货检验发现的不合格原辅料、包装材料
2. 生产过程中发现的不合格中间产品
3. 成品检验发现的不合格成品
4. 库存发现的不合格产品
5. 客户退货的不合格产品
6. 抽检发现的不合格产品

### 第三条【职责分工】
1. **质量管理部门**：负责不合格品的归口管理
2. **检验部门**：负责不合格品的识别和隔离
3. **生产部门**：负责生产过程中不合格品的报告和处理
4. **采购部门**：负责不合格原辅料的退货或索赔
5. **销售部门**：负责不合格成品的召回和处理

---

## 第二部分：不合格品的识别与报告

### 第四条【识别依据】
不合格品的识别依据：
1. 国家法律法规和标准
2. 企业内控标准
3. 合同约定的质量要求
4. 产品技术文件

### 第五条【不合格分类】
根据不合格的严重程度，分为：

| 类别 | 定义 | 示例 |
|------|------|------|
| A类不合格（严重） | 危及人身安全或健康，或主要性能指标达不到标准要求 | 微生物超标、重金属超标、含有禁用物质 |
| B类不合格（主要） | 影响产品使用性能，或严重影响外观 | 主要营养成分不符合标示值、感官指标严重不合格 |
| C类不合格（次要） | 对产品性能影响较小，或外观有轻微瑕疵 | 标签标注不规范、包装轻微破损 |

### 第六条【报告要求】
发现不合格品时，应当立即报告：
1. 填写《不合格品报告单》
2. 报告内容包括：品名、规格、批号、数量、不合格项目、不合格类别、发现日期等
3. 通知质量管理部门组织评审

---

## 第三部分：不合格品的隔离

### 第七条【隔离标识】
不合格品应当立即隔离：
1. 移至指定的不合格品存放区
2. 挂红色"不合格"标识牌
3. 填写《不合格品隔离记录》

### 第八条【隔离要求】
不合格品隔离区应当：
1. 与合格品存放区明显分开
2. 有明显的警示标识
3. 防止误用、误发
4. 记录完整

---

## 第四部分：不合格品的评审

### 第九条【评审组织】
质量管理部门收到不合格品报告后，应当组织评审：
1. A类不合格：由质量负责人组织评审
2. B类不合格：由质量管理部门负责人组织评审
3. C类不合格：由检验部门负责人组织评审

### 第十条【评审内容】
评审内容包括：
1. 不合格的事实描述
2. 不合格的原因分析
3. 不合格的影响评估
4. 处置意见建议

### 第十一条【评审结论】
评审结论应当包括：
1. 不合格品的处置方式
2. 是否需要采取纠正措施
3. 是否需要通知客户或相关方
4. 是否需要报告监管部门

---

## 第五部分：不合格品的处置

### 第十二条【处置方式】
不合格品的处置方式：

| 处置方式 | 适用条件 | 处置要求 |
|----------|----------|----------|
| 返工 | 可通过返工达到规定要求 | 制定返工方案，重新检验 |
| 退货 | 供应商责任，可退货 | 联系供应商退货 |
| 报废 | 不能返工或返工不经济 | 审批后销毁，做好记录 |
| 让步接收 | 轻微不合格，不影响使用 | 评审批准，记录完整 |
| 改作他用 | 可降级使用 | 评估安全性，明确标识 |

### 第十三条【返工控制】
1. 返工应当制定返工方案
2. 返工方案应当经过质量管理部门批准
3. 返工产品应当重新检验
4. 返工记录应当完整

### 第十四条【让步接收控制】
1. 让步接收应当进行风险评估
2. 让步接收应当经质量负责人批准
3. 让步接收应当通知相关方（如适用）
4. 让步接收记录应当完整

### 第十五条【报废处理】
1. 报废应当填写《报废申请单》
2. 报废应当经质量负责人批准
3. 报废应当监督销毁
4. 报废应当拍照或录像留存

---

## 第六部分：不合格品的统计分析

### 第十六条【统计要求】
1. 建立不合格品统计台账
2. 每月统计不合格品情况
3. 每季度进行不合格品分析

### 第十七条【分析内容】
1. 不合格品数量和不合格率
2. 不合格项目的分布
3. 不合格原因的分类
4. 不合格趋势分析

### 第十八条【分析报告】
1. 编制《不合格品统计分析报告》
2. 报告应当包括：统计数据、分析结论、改进建议
3. 报告应当提交管理层评审

---

## 第七部分：纠正与预防措施

### 第十九条【纠正措施】
对于重复发生或严重的不合格，应当采取纠正措施：
1. 分析不合格的根本原因
2. 制定纠正措施计划
3. 实施纠正措施
4. 验证纠正措施的有效性
5. 修订相关文件（如需要）

### 第二十条【预防措施】
根据不合格品统计分析结果，确定预防措施的需求：
1. 识别潜在的不合格因素
2. 分析潜在不合格的原因
3. 制定预防措施计划
4. 实施预防措施
5. 验证预防措施的有效性

---

## 第八部分：客户退货处理

### 第二十一条【退货接收】
1. 销售部门接到客户退货通知后，应当填写《客户退货单》
2. 退货产品应当隔离存放
3. 质量管理部门应当组织检验

### 第二十二条【退货检验】
1. 核对退货产品信息
2. 确认不合格事实
3. 分析不合格原因
4. 出具检验报告

### 第二十三条【退货处置】
根据检验结果处置：
1. 产品合格：重新入库或重新发货
2. 产品不合格：按不合格品处置程序处理

---

## 第九部分：抽检不合格处理

### 第二十四条【监管部门抽检】
产品被监管部门抽检发现不合格时：
1. 立即启动召回程序（如需要）
2. 分析不合格原因
3. 采取纠正措施
4. 向监管部门提交整改报告

### 第二十五条【企业自检】
企业自检发现不合格时：
1. 立即停止相关产品的生产
2. 隔离已生产的同批产品
3. 查明不合格原因
4. 采取纠正措施

---

## 第十部分：记录管理

### 第二十六条【记录要求】
不合格品控制过程中的所有记录应当：
1. 及时、真实、准确填写
2. 字迹清晰、无涂改
3. 签名齐全
4. 妥善保管

### 第二十七条【记录保存】
不合格品相关记录应当保存至产品保质期后1年，不得少于5年。

---

## 第十一部分：附则

### 第二十八条【相关文件】
1. 《不合格品报告单》
2. 《不合格品隔离记录》
3. 《不合格品评审记录》
4. 《返工记录》
5. 《报废申请单》
6. 《客户退货单》
7. 《不合格品统计台账》
8. 《不合格品统计分析报告》

### 第二十九条【施行日期】
本程序自2024年1月1日起施行。

## 附件：不合格品控制流程图

```
发现不合格
    ↓
填写《不合格品报告单》
    ↓
立即隔离（红色标识）
    ↓
质量管理部门组织评审
    ↓
评审结论
    ├─→ 返工 → 制定返工方案 → 返工实施 → 重新检验 → 合格放行
    ├─→ 退货 → 联系供应商 → 退货处理
    ├─→ 报废 → 填写《报废申请单》 → 批准 → 监督销毁
    ├─→ 让步接收 → 风险评估 → 批准 → 记录
    └─→ 改作他用 → 安全评估 → 明确标识 → 记录
    ↓
统计分析
    ↓
制定纠正/预防措施
    ↓
验证措施有效性
    ↓
文件修订（如需要）
```
