# 食品供应商审核规范详细版（2026版）

## 法规信息
- **规范名称**：食品供应商审核规范
- **发布机构**：国家市场监督管理总局
- **实施日期**：2026年1月1日
- **适用范围**：食品生产企业对供应商的审核管理

---

## 第一章 总则

### 第一条 审核目的
为规范食品生产企业供应商审核工作，保证采购原料、辅料和包装材料的质量安全，根据《中华人民共和国食品安全法》及相关法规，制定本规范。

### 第二条 审核原则
1. **风险控制原则**：根据原料风险程度确定审核频次和深度
2. **全程控制原则**：审核覆盖供应商质量管理体系全流程
3. **客观公正原则**：审核过程和评价应客观、公正
4. **持续改进原则**：促进供应商质量管理体系持续改进

### 第三条 适用范围
本规范适用于食品生产企业对以下供应商的审核：
1. 原料供应商
2. 辅料供应商
3. 食品添加剂供应商
4. 食品相关产品供应商
5. 包装材料供应商

---

## 第二章 供应商分类

### 第四条 风险分类
根据原料特性和对产品质量安全的影响程度，将供应商分为三类：

| 类别 | 风险程度 | 审核重点 | 审核频率 |
|------|---------|---------|---------|
| A类 | 高风险 | 现场审核+年度审核 | 每年至少1次 |
| B类 | 中风险 | 现场审核+周期审核 | 每2年至少1次 |
| C类 | 低风险 | 文件审核+抽样检验 | 每3年至少1次 |

### 第五条 A类供应商判定
符合以下条件之一的，判定为A类供应商：
1. 动物源性原料供应商
2. 食品添加剂供应商
3. 特殊营养食品原料供应商
4. 食品接触材料供应商
5. 历史上有质量问题的供应商

### 第六条 B类供应商判定
符合以下条件之一的，判定为B类供应商：
1. 植物源性原料供应商
2. 通用辅料供应商
3. 包装材料供应商

### 第七条 C类供应商判定
符合以下条件之一的，判定为C类供应商：
1. 劳保用品供应商
2. 清洁用品供应商
3. 检验耗材供应商

---

## 第三章 首次审核

### 第八条 审核申请
首次供应原料前，供应商应提交以下资料：
1. 营业执照
2. 生产许可证或经营许可证
3. 产品质量标准
4. 第三方检测报告（近3个月）
5. 其他相关证明文件

### 第九条 资料审核
对供应商提交的资料进行审核，确认：
1. 证照齐全且在有效期内
2. 生产经营范围包含所供应产品
3. 产品质量符合相关标准
4. 检测报告真实有效

### 第十条 现场审核
A类和B类供应商应进行现场审核，C类供应商可只进行文件审核。

#### 现场审核内容
| 审核项目 | 审核内容 | 审核要点 |
|---------|---------|---------|
| 管理职责 | 质量管理组织机构 | 是否有独立的质量管理部门 |
| 厂房设施 | 厂房布局、卫生设施 | 布局合理，卫生设施完善 |
| 设备管理 | 设备配置、维护保养 | 设备适宜，维护到位 |
| 人员管理 | 人员资质、健康证明 | 人员持证上岗，健康证明有效 |
| 原料管理 | 原料验收、储存 | 原料验收规范，储存条件适宜 |
| 生产过程 | 工艺控制、卫生管理 | 工艺稳定，卫生管理规范 |
| 检验能力 | 检验设备、检验记录 | 设备齐全，记录完整 |

### 第十一条 审核结论
审核结论分为：合格、有条件合格、不合格

| 审核结论 | 条件要求 |
|---------|---------|
| 合格 | 所有项目符合要求 |
| 有条件合格 | 存在轻微不符合项，限期整改 |
| 不合格 | 存在严重不符合项，不予通过 |

### 第十二条 样品检验
审核合格后，应对产品进行抽样检验：
1. 按标准抽样
2. 检验项目根据产品特性确定
3. 检验合格后可纳入合格供应商名录

---

## 第四章 日常监督

### 第十三条 定期审核
按供应商风险分类确定的频率进行定期审核：
1. A类供应商：每年至少1次
2. B类供应商：每2年至少1次
3. C类供应商：每3年至少1次

### 第十四条 专项审核
出现以下情况时，应进行专项审核：
1. 产品质量出现异常
2. 供应商发生重大变更
3. 收到客户投诉
4. 监管部门通报问题
5. 其他需要审核的情况

### 第十五条 变更管理
供应商发生以下变更时，应报告采购方：
1. 生产地址变更
2. 生产工艺变更
3. 原料来源变更
4. 关键设备变更
5. 质量负责人变更

采购方应根据变更内容评估风险，决定是否进行专项审核。

### 第十六条 进货检验
每批进货应进行检验或验证：
1. 查验合格证明
2. 检查外观和包装
3. 按标准抽样检验
4. 记录检验结果

---

## 第五章 审核实施

### 第十七条 审核组组成
1. 审核组由2~3名审核员组成
2. 审核员应具备相应的专业能力
3. 审核员不得与供应商存在利益关系
4. 审核员应遵守审核纪律

### 第十八条 审核计划
审核前应制定审核计划，包括：
1. 审核目的和范围
2. 审核准则
3. 审核组成员
4. 审核日程安排
5. 审核方法

### 第十九条 审核方法
1. **查阅文件**：查阅质量手册、程序文件、作业指导书、记录等
2. **现场观察**：观察生产现场、设备、卫生状况等
3. **人员访谈**：与管理人员、操作人员进行访谈
4. **产品追溯**：随机抽取批次进行追溯检查

### 第二十条 不符合项分类
| 不符合类型 | 定义 | 示例 |
|-----------|------|------|
| 严重不符合 | 严重影响产品质量安全 | 使用非食品级原料 |
| 一般不符合 | 个别条款不符合要求 | 记录不规范 |
| 观察项 | 需要改进的问题 | 标识不清晰 |

### 第二十一条 审核报告
审核结束后，应编写审核报告，包括：
1. 审核概述
2. 审核发现
3. 不符合项
4. 审核结论
5. 整改要求

---

## 第六章 审核后续管理

### 第二十二条 整改验证
供应商应在规定期限内完成整改，提交整改报告。审核组应对整改情况进行验证：
1. 文件验证：查阅整改记录
2. 现场验证：现场确认整改效果
3. 产品验证：对产品进行抽样检验

### 第二十三条 供应商分级
根据审核结果，对供应商进行分级：

| 等级 | 分数 | 管理措施 |
|------|------|---------|
| 优秀 | ≥90分 | 优先采购，减少审核频次 |
| 良好 | 80~89分 | 正常采购，正常审核 |
| 合格 | 70~79分 | 正常采购，加强监督 |
| 不合格 | <70分 | 停止采购，限期整改 |

### 第二十四条 动态管理
建立供应商动态管理机制：
1. 定期评估供应商绩效
2. 根据绩效调整采购策略
3. 对优秀供应商给予优惠条件
4. 对不合格供应商暂停或终止合作

### 第二十五条 档案管理
建立供应商档案，包括：
1. 基本信息资料
2. 审核报告和记录
3. 检验报告和记录
4. 整改验证记录
5. 绩效评价记录

档案保存期限不少于3年。

---

## 第七章 审核要点

### 第二十六条 管理职责审核要点
- [ ] 质量方针和目标是否明确
- [ ] 质量管理组织机构是否健全
- [ ] 各部门职责是否明确
- [ ] 质量负责人是否具备相应资格

### 第二十七条 厂房设施审核要点
- [ ] 厂房选址是否合理
- [ ] 厂房布局是否防止交叉污染
- [ ] 卫生设施是否完善
- [ ] 仓储设施是否适宜

### 第二十八条 设备管理审核要点
- [ ] 设备配置是否满足生产需要
- [ ] 设备材质是否符合食品级要求
- [ ] 设备维护保养是否到位
- [ ] 设备校准是否有效

### 第二十九条 人员管理审核要点
- [ ] 人员是否持证上岗
- [ ] 健康证明是否有效
- [ ] 培训是否充分
- [ ] 卫生意识是否良好

### 第三十条 原料管理审核要点
- [ ] 供应商是否经过审核
- [ ] 原料验收是否规范
- [ ] 原料储存是否适宜
- [ ] 追溯体系是否完善

### 第三十一条 生产过程审核要点
- [ ] 工艺规程是否完整
- [ ] 关键控制点是否监控
- [ ] 卫生管理是否到位
- [ ] 记录是否完整

### 第三十二条 检验能力审核要点
- [ ] 检验设备是否齐全
- [ ] 检验方法是否正确
- [ ] 检验记录是否完整
- [ ] 检验报告是否规范

---

## 第八章 记录要求

### 第三十三条 审核记录
审核过程应做好记录，包括：
1. 审核计划
2. 审核检查表
3. 审核发现记录
4. 不符合项报告
5. 审核报告

### 第三十四条 记录要求
1. 记录应及时、准确、真实
2. 记录应清晰可辨，不得涂改
3. 记录应经相关人员签字确认
4. 记录应按规定期限保存
