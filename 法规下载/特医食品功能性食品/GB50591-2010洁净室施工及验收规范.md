# GB 50591-2010 洁净室施工及验收规范

## 标准信息
- **标准编号**：GB 50591-2010
- **标准名称**：洁净室施工及验收规范
- **发布部门**：中华人民共和国住房和城乡建设部
- **发布日期**：2010年7月15日
- **实施日期**：2011年2月1日
- **主编单位**：中国电子工程设计院

---

## 1 总则

### 1.1 编制目的
为统一洁净室及洁净区（以下简称"洁净室"）的施工及验收要求，保证工程质量，制定本规范。

### 1.2 适用范围
本规范适用于新建、改建、扩建的洁净室及相关受控环境的施工及验收。

### 1.3 洁净室定义
洁净室是指对空气洁净度、温度、湿度、压力、噪声等参数根据需要实行控制的封闭性空间，其房间结构、装备和使用均具有减少该区域内污染物介入、产生和滞留的功能。

---

## 2 术语

### 2.1 洁净度 cleanroom class
根据单位体积空气中大于或等于某粒径粒子的数量限值确定的洁净室等级。

### 2.2 洁净区 clean zone
洁净室内部具有划定空间和特定洁净度的区域。

### 2.3 粒子 particle
尺寸一般为0.1μm~5μm的固态和液态物质。

### 2.4 悬浮粒子 airborne particle
悬浮在空气中的粒子。

### 2.5 气流流型 air pattern
洁净室内空气的流动形态和分布。

---

## 3 基本规定

### 3.1 施工资质
1. 洁净室施工单位应具备相应的施工资质
2. 施工人员应经过专业培训并考核合格
3. 特种作业人员应持证上岗

### 3.2 施工准备
| 项目 | 要求 |
|------|------|
| 设计文件审查 | 施工前应进行设计交底和图纸会审 |
| 施工组织设计 | 编制专项施工方案，经审批后实施 |
| 材料设备检验 | 所有材料设备进场前应进行检验 |
| 施工环境控制 | 施工现场应保持清洁，控制灰尘 |

### 3.3 施工原则
1. 先处理后细装
2. 先暗后明
3. 先上后下
4. 先内后外
5. 防止交叉污染

## 4 建筑结构施工

### 4.1 墙板施工

#### 材料要求
| 材料类型 | 适用范围 | 技术要求 |
|---------|---------|---------|
| 彩钢夹芯板 | 一般洁净区 | 厚度≥50mm，燃烧性能A级 |
| 不锈钢板 | 高洁净度区 | 表面粗糙度Ra≤0.8μm |
| 铝合金板 | 中等洁净度区 | 表面阳极氧化处理 |
| 矿棉板 | 一般辅助区 | 厚度≥12mm |

#### 安装要点
1. 板材安装应垂直、平整，垂直度偏差≤2mm/m
2. 拼接缝隙应均匀，缝隙宽度≤2mm
3. 板缝应用密封胶密封，密封应连续、无气泡
4. 墙板与地面、吊顶交接处应采用圆弧过渡，R≥50mm

### 4.2 吊顶施工

#### 吊顶龙骨安装
1. 龙骨间距应符合设计要求，一般≤1200mm
2. 龙骨应水平，水平度偏差≤2mm/m
3. 龙骨应固定牢固，吊点间距≤1200mm
4. 龙骨表面应进行防腐蚀处理

#### 吊顶板安装
1. 吊顶板安装应平整，平整度偏差≤2mm/m
2. 板缝应密封，不得有渗漏
3. 灯具、风口等设备安装处应加强处理
4. 检修口应设置密封装置

### 4.3 地面施工

#### 环氧自流平地面
**施工工艺流程**：
```
基层处理 → 底涂施工 → 中涂施工 → 面涂施工 → 养护
```

**技术要求**：
- 基层含水率≤8%
- 底涂渗透深度≥1mm
- 面涂层厚度≥2mm
- 表面平整度≤2mm/m

#### PVC防静电地板
**技术要求**：
- 表面电阻：10⁶~10⁹Ω
- 系统电阻：10⁶~10⁸Ω
- 耐电压：≥3000V
- 耐磨性：≤0.005g/cm²

## 5 净化空调系统施工

### 5.1 风管制作安装

#### 材料要求
| 风管类型 | 材料要求 | 适用范围 |
|---------|---------|---------|
| 镀锌钢板 | 厚度0.5~1.2mm | 一般洁净区 |
| 不锈钢板 | 厚度0.5~1.0mm | 高洁净度区、腐蚀性环境 |
| 无机玻璃钢 | 厚度4~8mm | 低洁净度区 |

#### 制作要求
1. 风管应采用机械加工，咬口缝应密封
2. 风管内表面应光滑，不得有横向拼接缝
3. 风管尺寸偏差：长边≤1000mm时，偏差≤1mm；长边>1000mm时，偏差≤2mm
4. 法兰连接应采用密封垫密封

#### 安装要求
1. 风管安装应水平或垂直，偏差≤2mm/m
2. 风管支吊架间距：水平管≤3m，垂直管≤4m
3. 风管穿墙处应设套管，套管应与风管密封
4. 风管安装后应进行清洁和密封检查

### 5.2 高效过滤器安装

#### 安装前检查
1. 外观检查：无破损、无变形
2. 性能检查：扫描检漏合格
3. 记录检查：有出厂合格证和性能测试报告

#### 安装要求
1. 安装方向应正确，气流方向与箭头标识一致
2. 密封垫应完整、压缩均匀，压缩量5~8mm
3. 边框应平整，与框架接合严密，不得有渗漏
4. 安装后应进行完整性检查（PAO扫描）

### 5.3 净化空调设备安装

#### 风机过滤单元（FFU）
| 项目 | 技术要求 |
|------|----------|
| 安装平整度 | ≤1mm/m |
| 风量偏差 | ±5% |
| 噪声 | ≤57dB(A) |
| 振动 | ≤3μm |

#### 层流罩
1. 安装应水平，水平度偏差≤1mm/m
2. 风口风速：0.35~0.45m/s（100级）
3. 送风面均匀性：≥80%
4. 噪声：≤65dB(A)

## 6 电气与照明施工

### 6.1 电气配线

#### 配线要求
1. 洁净区不应明敷电线，应采用穿管或线槽敷设
2. 穿线管应采用不锈钢或镀锌钢管
3. 管口应光滑，穿线后应密封
4. 接线盒应设置在非洁净区

#### 照明安装
| 洁净度级别 | 照度要求 | 照明方式 |
|-----------|---------|---------|
| 100级 | ≥500lux | 嵌入式、密闭型 |
| 1000~100000级 | ≥300lux | 吊挂式、格栅型 |
| >100000级 | ≥200lux | 一般照明 |

### 6.2 防静电接地

#### 接地要求
1. 洁净室应设置防静电接地系统
2. 接地电阻：≤10Ω
3. 每个设备、工作台、地面都应可靠接地
4. 接地线应采用黄绿双色线

## 7 工艺管道施工

### 7.1 管道材质选择
| 介质类型 | 推荐材质 | 连接方式 |
|---------|---------|---------|
| 纯化水 | 316L不锈钢 | 氩弧焊 |
| 注射用水 | 316L不锈钢 | 氩弧焊+抛光 |
| 纯蒸汽 | 316L不锈钢 | 氩弧焊 |
| 压缩空气 | 304不锈钢 | 法兰或焊接 |
| 真空 | 304不锈钢 | 法兰或焊接 |

### 7.2 管道安装要求
1. 管道应平直，坡度应符合设计要求
2. 管道支架间距：DN15~DN25为1.5m，DN32~DN50为2.0m
3. 阀门应便于操作和维护
4. 管道标识应清晰、完整

### 7.3 管道试压吹洗
1. **压力试验**：水压试验压力为工作压力的1.5倍，保持30min
2. **吹洗**：用洁净压缩空气或氮气吹扫，直到出口无杂物
3. **酸洗钝化**：不锈钢管道焊接后应进行酸洗钝化

## 8 验收要求

### 8.1 验收程序

#### 第一阶段：竣工验收
1. 外观检查
2. 设备单机试运转
3. 系统联动调试
4. 性能检测

#### 第二阶段：综合性能评定
1. 洁净度检测
2. 静压差检测
3. 风量风速检测
4. 温湿度检测
5. 照度检测
6. 噪声检测
7. 照明检测

### 8.2 洁净度检测

#### 检测状态
- **空态**：设施已建成，所有动力接通并运行，但无设备、材料和人员
- **静态**：设施已建成，设备就位并运行，但无人员
- **动态**：设施以规定方式运行，有规定数量的人员在场

#### 采样要求
| 洁净度级别 | 最少采样点数 | 最少采样量(L/点) |
|-----------|-------------|----------------|
| 5级（100级） | ≥5 | ≥20 |
| 6级（1000级） | ≥3 | ≥2 |
| 7级（10000级） | ≥2 | ≥2 |
| 8级（100000级） | ≥2 | ≥2 |

#### 检测方法
1. 使用激光粒子计数器
2. 采样点应均匀布置，避开回风口
3. 采样高度：工作面高度（0.8~1.5m）
4. 采样时间：每点至少2min

### 8.3 静压差检测
1. 洁净区与非洁净区之间压差：≥10Pa
2. 不同级别洁净区之间压差：≥5Pa
3. 检测仪器：微压计或电子压力计
4. 测点布置：每个房间至少1个测点

### 8.4 风量风速检测

#### 风口风速检测
1. 检测仪器：热式风速仪
2. 测点布置：每个风口至少5个测点
3. 风速偏差：±15%

#### 换气次数检测
1. 通过总送风量计算
2. 换气次数要求：
   - 100级：≥360次/h
   - 1000级：≥180次/h
   - 10000级：≥25次/h
   - 100000级：≥15次/h

### 8.5 温湿度检测
| 洁净度级别 | 温度范围 | 相对湿度范围 |
|-----------|---------|------------|
| 100~10000级 | 20~24℃ | 45~60% |
| 100000级 | 18~28℃ | 50~65% |

### 8.6 照度检测
1. 检测仪器：照度计
2. 测点布置：每个房间至少5个测点，均匀分布
3. 检测时机：无自然光干扰时
4. 照度均匀度：≥0.7

### 8.7 噪声检测
1. 检测仪器：声级计（A计权）
2. 测点布置：每个房间至少1个测点，高度1.2m
3. 背景噪声应低于检测值10dB以上
4. 噪声限值：≤65dB(A)

## 9 验收标准

### 9.1 合格判定

#### 主控项目
1. 洁净度必须符合设计要求
2. 静压差必须符合设计要求
3. 高效过滤器不得有渗漏
4. 温湿度必须符合设计要求

#### 一般项目
1. 风量风速偏差≤±15%
2. 照度≥设计值
3. 噪声≤设计值
4. 外观质量符合要求

### 9.2 验收文件
1. 施工图纸及设计变更文件
2. 材料设备合格证和检验报告
3. 施工记录和检验记录
4. 调试报告和检测报告
5. 竣工图纸
6. 使用维护手册

## 附录B：高效过滤器检漏方法

### B.1 PAO扫描检漏法

#### 原理
使用PAO（聚α烯烃）作为气溶胶，在高效过滤器上游发尘，下游用粒子计数器扫描检测是否有泄漏。

#### 方法
1. 在过滤器上游发生PAO气溶胶
2. 调节上游浓度，使其达到20~80μg/L
3. 用粒子计数器在过滤器下游扫描
4. 扫描速度：≤5cm/s
5. 扫描距离：距过滤器表面1~3cm
6. 泄漏判定：下游浓度≥0.01%上游浓度

#### 检测点
- 过滤器与边框连接处
- 滤纸拼接处
- 可疑部位
