# 食品生产许可审查通则（2026完整版）

---

## 第一章 总则

### 第一条 目的依据

为规范食品生产许可审查工作，加强食品生产监督管理，保证食品安全，根据《中华人民共和国食品安全法》《食品生产许可管理办法》等法律法规，制定本通则。

### 第二条 适用范围

本通则适用于食品生产许可的审查工作，包括：
1. 食品生产许可证的申请审查
2. 食品生产许可证的变更审查
3. 食品生产许可证的延续审查

### 第三条 审查原则

食品生产许可审查应当遵循以下原则：
1. **公开、公平、公正**原则
2. **科学规范**原则
3. **风险分级**原则
4. **优化服务**原则

---

## 第二章 申请与受理

### 第四条 申请条件

申请食品生产许可，应当符合以下条件：

| 条件类别 | 具体要求 |
|----------|----------|
| 主体资格 | 具有营业执照等合法主体资格 |
| 生产场所 | 具有与生产的食品品种、数量相适应的生产场所 |
| 设备设施 | 具有与生产的食品品种、数量相适应的设备设施 |
| 人员配备 | 具有食品安全管理人员和专业技术人员 |
| 管理制度 | 建立完善的食品安全管理制度 |
| 检验能力 | 具有出厂检验能力 |

### 第五条 申请材料

申请食品生产许可应当提交以下材料：

1. **基本材料**：
   - 食品生产许可申请书
   - 营业执照复印件
   - 食品生产加工场所及其周围环境平面图
   - 食品生产加工场所各功能区间布局平面图
   - 工艺设备布局图
   - 食品生产工艺流程图

2. **证明材料**：
   - 食品生产主要设备、设施清单
   - 食品安全管理制度目录
   - 食品安全专业技术人员、管理人员信息
   - 产品执行标准

3. **特殊食品**：
   - 特医食品：注册证书
   - 保健食品：注册证书或备案凭证
   - 婴幼儿配方食品：注册证书

### 第六条 受理审查

许可机关收到申请材料后，应当在5个工作日内完成以下审查：

1. **形式审查**：
   - 材料是否齐全
   - 格式是否符合要求
   - 内容是否完整

2. **受理决定**：
   - 材料齐全、符合要求的：受理申请
   - 材料不齐全的：一次性告知补正
   - 不符合要求的：不予受理并说明理由

## 第三章 审查程序

### 第七条 审查组织

许可机关受理申请后，应当组织审查组进行现场审查。

**审查组组成**：
- 组长：1名（由审查员担任）
- 组员：2-3名（审查员或专家）
- 特殊情况可聘请技术专家

### 第八条 审查时限

| 环节 | 时限要求 |
|------|----------|
| 材料审查 | 受理后5个工作日内 |
| 现场审查 | 受理后10个工作日内完成 |
| 审查报告 | 现场审查后3个工作日内提交 |
| 许可决定 | 受理后20个工作日内作出 |

### 第九条 审查内容

#### 9.1 生产场所审查

| 审查项目 | 审查要点 |
|----------|----------|
| 选址 | 远离污染源，环境清洁 |
| 布局 | 功能分区明确，人流物流分开 |
| 车间 | 结构合理，易于清洗消毒 |
| 仓储 | 原料、成品、包装材料分开存放 |
| 卫生设施 | 更衣、洗手、消毒设施完备 |

#### 9.2 设备设施审查

| 审查项目 | 审查要点 |
|----------|----------|
| 生产设备 | 与生产规模相适应，完好有效 |
| 检验设备 | 满足出厂检验需要 |
| 清洗消毒 | 设备设施齐全有效 |
| 通风照明 | 符合生产要求 |
| 供排水 | 水质符合要求，排水通畅 |

#### 9.3 设备布局审查

| 审查项目 | 审查要点 |
|----------|----------|
| 工艺布局 | 符合工艺流程，避免交叉污染 |
| 设备安装 | 便于操作、清洗、维护 |
| 物料流向 | 单向流动，无回流 |

#### 9.4 人员审查

| 审查项目 | 审查要点 |
|----------|----------|
| 企业负责人 | 熟悉食品安全法律法规 |
| 食品安全总监 | 配备并经培训考核 |
| 食品安全员 | 配备并经培训考核 |
| 检验人员 | 具备相应检验能力 |
| 从业人员 | 健康证明、培训记录 |

#### 9.5 管理制度审查

| 制度类别 | 审查要点 |
|----------|----------|
| 食品安全管理制度 | 完整、可操作、落实到位 |
| 过程控制制度 | 覆盖生产全过程 |
| 检验制度 | 检验项目和频次符合要求 |
| 不合格品控制 | 明确处理流程 |
| 追溯召回制度 | 可有效追溯和召回 |

## 第四章 审查判定

### 第十条 审查结论

现场审查结论分为：**符合、基本符合、不符合**

| 结论 | 条件 |
|------|------|
| 符合 | 所有审查项目均符合要求 |
| 基本符合 | 一般项目存在少量问题，可在限期内整改 |
| 不符合 | 关键项目不符合或一般项目问题较多 |

### 第十一条 关键项目

以下项目为关键项目（**任一不符合则审查结论为不符合**）：

1. **生产场所**：
   - 厂区环境严重污染
   - 车间选址不当
   - 功能分区严重缺失

2. **设备设施**：
   - 生产设备严重不足
   - 检验设备缺失
   - 清洗消毒设施缺失

3. **人员**：
   - 未配备食品安全总监
   - 从业人员无健康证明
   - 无检验人员

4. **管理制度**：
   - 未建立食品安全管理制度
   - 无出厂检验制度
   - 无追溯制度

### 第十二条 一般项目

除关键项目外的其他项目为一般项目。

### 第十三条 整改要求

1. **基本符合**结论：
   - 规定整改时限（一般不超过20个工作日）
   - 提交整改报告
   - 整改复核

2. **不符合**结论：
   - 不予许可
   - 告知不合格原因
   - 整改后重新申请

## 第五章 许可决定

### 第十四条 许可决定

许可机关根据审查结论作出许可决定：

1. **符合**或**基本符合（经整改合格）**：作出准予许可决定
2. **不符合**或**基本符合（逾期未整改或整改不合格）**：作出不予许可决定

### 第十五条 许可证内容

食品生产许可证载明以下事项：

| 事项 | 说明 |
|------|------|
| 许可证编号 | 统一编码规则 |
| 生产者名称 | 与营业执照一致 |
| 社会信用代码 | 与营业执照一致 |
| 法定代表人 | 与营业执照一致 |
| 生产地址 | 实际生产场所地址 |
| 食品类别 | 按分类目录填写 |
| 许可明细 | 具体品种明细 |
| 有效期 | 5年 |

### 第十六条 许可证编号

许可证编号格式：**SC + 14位阿拉伯数字**

| 位数 | 含义 |
|------|------|
| 第1-2位 | 省份代码 |
| 第3-4位 | 市代码 |
| 第5-6位 | 区县代码 |
| 第7位 | 食品类别代码 |
| 第8-14位 | 顺序码 |

**食品类别代码**：
- 1：粮食加工品
- 2：食用油、油脂及其制品
- 3：调味品
- 4：肉制品
- 5：乳制品
- 6：饮料
- 7：方便食品
- 8：饼干
- 9：罐头
- 10：冷冻饮品
- 24：特殊医学用途配方食品
- 27：保健食品
- 28：婴幼儿配方食品

## 第六章 变更与延续

### 第十七条 许可变更

**需要变更的情形**：
1. 生产场所迁址
2. 生产设备布局变化
3. 生产工艺设备变化
4. 食品类别增加

**变更程序**：
1. 提交变更申请
2. 提交变更材料
3. 组织现场审查（如需要）
4. 作出变更决定

### 第十八条 许可延续

**延续申请**：
- 时间：有效期届满30个工作日前
- 材料：延续申请书、营业执照等

**审查方式**：
- 风险等级低的：书面审查
- 风险等级高的：现场审查

**延续有效期**：
- 符合条件的：延续5年

## 第七章 附则

### 第十九条 特殊食品

特殊食品（特医食品、保健食品、婴幼儿配方食品）的审查除符合本通则外，还应当符合相应的审查细则要求。

### 第二十条 风险分级

根据食品风险程度实行分级审查：

| 风险等级 | 审查要求 |
|----------|----------|
| 高风险 | 全项目现场审查 |
| 中风险 | 全项目现场审查 |
| 低风险 | 部分项目现场审查或书面审查 |

### 第二十一条 审查员管理

1. **审查员资质**：
   - 经培训考核合格
   - 取得审查员证书
   - 定期继续教育

2. **审查员职责**：
   - 严格按照审查标准审查
   - 客观公正出具审查意见
   - 对审查结果负责

3. **审查员纪律**：
   - 不得利用职务谋取私利
   - 不得泄露企业商业秘密
   - 回避与自身有利害关系的审查

### 第二十二条 信用管理

1. **失信行为**：
   - 提供虚假材料
   - 隐瞒真实情况
   - 拒绝接受监督检查

2. **失信惩戒**：
   - 不予许可或撤销许可
   - 纳入失信名单
   - 加严审查

### B.1 整改时限

| 审查结论 | 整改时限 |
|----------|----------|
| 基本符合（一般问题） | 10个工作日 |
| 基本符合（较多问题） | 20个工作日 |
