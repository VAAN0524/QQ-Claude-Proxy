# 记录管理规范

## 规范基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 企业质量管理部门 |
| 适用范围 | 食品生产企业记录管理 |
| 规范性质 | 企业内部管理规范 |

## 1. 总则

### 1.1 目的

规范记录管理，确保记录真实、准确、完整、及时，为产品质量追溯、质量分析和体系改进提供依据。

### 1.2 适用范围

本规范适用于：
1. 生产记录
2. 检验记录
3. 质量记录
4. 设备记录
5. 培训记录
6. 其他质量相关记录

### 1.3 基本原则

| 原则 | 说明 |
|------|------|
| **真实性** | 记录必须真实 |
| **准确性** | 记录必须准确 |
| **完整性** | 记录必须完整 |
| **及时性** | 记录必须及时 |
| **可追溯性** | 记录必须可追溯 |

## 2. 记录分类

### 2.1 按内容分类

| 类别 | 示例 |
|------|------|
| **生产记录** | 批生产记录、投料记录、工艺记录 |
| **检验记录** | 原料检验、过程检验、成品检验 |
| **设备记录** | 运行、维护、校准记录 |
| **质量记录** | 偏差、变更、CAPA记录 |
| **人员记录** | 培训、健康档案 |
| **销售记录** | 销售、发货记录 |

### 2.2 按重要性分类

| 类别 | 说明 | 示例 |
|------|------|------|
| **关键记录** | 直接影响产品质量 | 批记录 |
| **重要记录** | 间接影响产品质量 | 设备维护记录 |
| **一般记录** | 支持性记录 | 会议记录 |

## 3. 记录填写要求

### 3.1 基本要求

| 要求 | 说明 | 正确示例 | 错误示例 |
|------|------|----------|----------|
| **使用蓝黑墨水** | 使用蓝黑色墨水 | 蓝黑钢笔 | 铅笔、可擦笔 |
| **字迹清晰** | 字迹清晰易读 | 工整书写 | 潦草、模糊 |
| **及时填写** | 及时填写记录 | 操作完成后立即填写 | 补记、回忆记录 |
| **准确无误** | 数据准确 | 实际数据 | 估计、编造 |
| **签名完整** | 签全名 | 张三 | 张、ZS |
| **日期完整** | 完整日期 | 2024-02-28 | 24/2/28 |

### 3.2 修改规范

| 要求 | 说明 |
|------|------|
| **单线划去** | 单线划去错误内容 |
| **保持可读** | 划去后仍可读 |
| **正确内容** | 在上方写上正确内容 |
| **签名** | 签名并注明日期 |
| **原因** | 简要说明修改原因 |

**示例**：
```
温度：25°C  30°C
          张三 2024-02-28 原记录笔误
```

### 3.3 空白处理

| 情况 | 处理方式 |
|------|----------|
| 不适用 | 填写"N/A"或"不适用" |
| 无数据 | 填写"0"或划单横线"一" |
| 未执行 | 划单横线"一"并注明原因 |
| 空白行 | 划单横线"一"并签名 |

## 4. 记录审核

### 4.1 审核要求

| 审核类型 | 审核内容 | 审核时机 |
|----------|----------|----------|
| **复核** | 数据准确性 | 记录完成后立即复核 |
| **审核** | 合规性 | 定期审核 |
| **批准** | 最终批准 | 重要记录需批准 |

### 4.2 审核内容

| 审核项目 | 审核内容 |
|----------|----------|
| 完整性 | 所有项目是否填写完整 |
| 准确性 | 数据是否准确 |
| 合规性 | 是否符合要求 |
| 及时性 | 是否及时填写 |
| 规范性 | 是否符合填写规范 |

### 4.3 审核流程

```
记录填写
    ↓
自检（填写人）
    ↓
复核（复核人）
    ↓
审核（审核人）
    ↓
归档
```

## 7. 记录保存

### 7.1 保存条件

| 要求 | 说明 |
|------|------|
| **环境** | 防潮、防火、防虫、防鼠 |

| **温湿度** | 适宜的温湿度 |
| **安全** | 防盗、防损坏 |

### 7.2 保存期限

| 记录类型 | 保存期限 | 备注 |
|----------|----------|------|
| 批生产记录 | 产品有效期后1年 | 最短不少于5年 |
| 批检验记录 | 产品有效期后1年 | 最短不少于5年 |
| 设备验证记录 | 设备使用年限 | 或设备报废后3年 |
| 校准记录 | 5年 | |
| 维护记录 | 3年 | |
| 培训记录 | 在职期间+3年 | |
| 偏差处理记录 | 3年 | 重大偏差长期保存 |
| 变更控制记录 | 永久 | |
| 质量记录 | 3年 | |

### 7.3 保存方式

| 方式 | 说明 |
