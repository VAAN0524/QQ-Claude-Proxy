### 5.2 标准版本说明

| 标准类型 | 版本规则 |
|---------|----------|
| **国家标准（GB）** | 强制性标准，年份为最新版本 |
| **推荐标准（GB/T）** | 推荐性标准，年份为最新版本 |

### 5.3 快速查找

1. **按产品类型查找**：
   - 特医食品 → 搜索"特医""特殊医学用途"
   - 保健食品 → 搜索"保健食品"
   - 功能性食品 → 搜索"功能性"

2. **按标准类型查找**：
   - 国家标准 → 搜索"GB"
   - 国际标准 → 搜索"ISO""FSSC""BRCGS""IFS"
   - 行业标准 → 搜索对应行业标准代码

3. **按功能查找**：
   - GMP → 搜索"良好生产规范"
   - HACCP → 搜索"HACCP""危害分析"
   - 检验方法 → 搜索"检验""测定"

## 七、企业合规要点

### 7.1 特医食品企业

| 要点 | 要求 |
|------|------|
| **产品注册** | 按照注册管理办法提交完整资料 |
| **生产许可** | 取得食品生产许可证 |
| **GMP符合性** | 符合GB 29923良好生产规范 |
| **临床试验** | 遵循临床试验质量管理规范 |
| **标签标识** | 符合GB 13432特医食品标签要求 |

### 7.2 保健食品企业

| 要点 | 要求 |
|------|------|
| **产品注册/备案** | 按照注册与备案管理办法 |
| **生产许可** | 取得食品生产许可证 |
| **GMP符合性** | 符合GB 17405良好生产规范 |
| **功能评价** | 遵循功能评价方法 |
| **标签标识** | 符合相关标签要求 |

### 7.3 食品接触材料企业

| 要点 | 要求 |
|------|------|
| **产品标准** | 符合GB 4806系列要求 |
| **生产许可** | 取得工业产品生产许可证 |
| **检验检测** | 按标准进行型式检验 |
| **标签标识** | 标明材质和执行标准 |
