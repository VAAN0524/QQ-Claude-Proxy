# GB 4789 食品微生物学检验标准（完整版 2026）

## 标准概述

**标准系列名称**：GB 4789 食品微生物学检验
**发布机构**：国家卫生健康委员会、国家市场监督管理总局
**标准性质**：强制性国家标准（GB）
**适用范围**：食品中微生物的检验方法

## 二、GB 4789.1 总则（2024版）

### 2.1 实验室基本要求

| 要求类别 | 具体要求 |
|---------|----------|
| **环境设施** | 洁净度符合要求，有良好的通风和排气设施 |
| **设备仪器** | 定期校准和维护，确保准确可靠 |
| **培养基和试剂** | 使用合格培养基，按规定条件保存 |
| **人员要求** | 具备相应的专业知识和操作技能 |

### 2.2 样品采集与处理

#### 2.2.1 采样原则

| 原则 | 说明 |
|------|------|
| **代表性** | 采样应具有代表性 |
| **随机性** | 随机采样，避免人为选择 |
| **无菌操作** | 严格无菌操作，防止污染 |
| **样品数量** | 满足检验和复检需要 |

#### 2.2.2 样品处理

- 固体样品：均质处理
- 液体样品：充分混匀
- 包装样品：外包装消毒后无菌开启

### 2.3 检验方法选择

| 检验项目 | 首选方法 | 备选方法 |
|---------|----------|----------|
| **菌落总数** | 平板计数法 | Petrifilm™ 测试片法 |
| **大肠菌群** | MPN 法 | 平板计数法、测试片法 |
| **沙门氏菌** | 常规培养法 | PCR 法、ELISA 法 |
| **金黄色葡萄球菌** | Baird-Parker 平板法 | 血浆凝固酶试验 |
| **霉菌酵母** | 平板计数法 | 测试片法 |

## 三、GB 4789.2 菌落总数测定（2022版）

### 3.1 原理

食品样品经过处理，在一定条件下（如培养基、培养温度、培养时间、计数方法等）培养后，所得每 g（mL）检样中形成的微生物菌落总数。

### 3.2 培养基和试剂

| 材料 | 要求 |
|------|------|
| **平板计数琼脂（PCA）** | 符合 GB 4789.28 要求 |
| **稀释液** | 磷酸盐缓冲液或生理盐水 |
| **无菌生理盐水** | 0.85% - 0.90% 氯化钠溶液 |

### 3.3 检验程序

```
样品处理 → 10倍系列稀释 → 选择2-3个稀释度 → 接种平板 → 培养（36±1°C，48±2h）→ 菌落计数 → 计算
```

### 3.4 菌落计数方法

| 菌落数 | 计数规则 |
|--------|----------|
| **30-300** | 计数该稀释度的所有菌落 |
| **<30** | 记录为"少于30×稀释倍数" |
| **>300** | 记录为"多于300×稀释倍数" |
| **蔓延生长** | 记录为"蔓延" |

### 3.5 结果计算

```
菌落总数(CFU/g或mL) = 平均菌落数 × 稀释倍数
```

### 3.6 报告方式

| 菌落总数范围 | 报告方式 |
|-------------|----------|
| **<100** | 按实数报告 |
| **100-10,000** | 以两位有效数字报告 |
| **≥10,000** | 以科学计数法报告 |

## 四、GB 4789.3 大肠菌群计数（2022版）

### 4.1 原理

大肠菌群是指在 37°C 培养能分解乳糖产酸产气、需氧或兼性厌氧的革兰氏阴性无芽孢杆菌。

### 4.2 方法选择

| 方法 | 适用范围 | 检测限 |
|------|----------|--------|
| **MPN 法** | 各类食品 | 3.6 MPN/g 或 mL |
| **平板计数法** | 各类食品 | 10 CFU/g 或 mL |
| **Petrifilm™ 测试片法** | 各类食品 | 10 CFU/g 或 mL |

### 4.3 MPN 法（多管发酵法）

#### 4.3.1 培养基

| 培养基 | 用途 |
|--------|------|
| **月桂基硫酸盐胰蛋白胨（LST）肉汤** | 初发酵 |
| **煌绿乳糖胆盐（BGLB）肉汤** | 复发酵 |
| **EC 肉汤** | 粪大肠菌群验证 |

#### 4.3.2 检验程序

```
样品处理 → 接种LST肉汤 → 37°C培养24-48h → 观察产气 → 转种BGLB肉汤 → 37°C培养48h → 观察产气 → 查MPN表
```

### 4.4 结果报告

```
大肠菌群 MPN/100g(mL) = MPN值 × 稀释倍数
```

## 五、GB 4789.4 沙门氏菌检验（2022版）

### 5.1 原理

沙门氏菌是肠道病原菌，通过选择性增菌、分离培养和生化鉴定进行检测。

### 5.2 培养基和试剂

| 培养基 | 用途 |
|--------|------|
| **缓冲蛋白胨水（BPW）** | 前增菌 |
| **四硫磺酸钠煌绿（TTB）肉汤** | 选择性增菌 |
| **亚硒酸盐胱氨酸（SC）肉汤** | 选择性增菌 |
| **亚硫酸铋（BS）琼脂** | 分离培养基 |
| **HE 琼脂** | 分离培养基 |
| **XLD 琼脂** | 分离培养基 |
| **三糖铁（TSI）琼脂** | 生化试验 |
| **赖氨酸铁（LI）琼脂** | 生化试验 |

### 5.3 检验程序

```
前增菌（BPW，8-18h）→ 选择性增菌（TTB/SC，24h）→ 分离培养（BS/HE/XLD，24h）→
挑选可疑菌落 → 生化鉴定 → 血清学分型 → 报告
```

### 5.4 生化鉴定

| 生化试验 | 沙门氏菌反应 |
|---------|-------------|
| **三糖铁琼脂** | 斜面产碱/底层产酸，H2S 多为阳性 |
| **尿素酶试验** | 阴性 |
| **氰化钾试验** | 阴性 |
| **赖氨酸脱羧酶** | 阳性 |
| **β-半乳糖苷酶** | 阴性 |
| **V-P 试验** | 阴性 |

### 5.5 血清学分型

使用沙门氏菌 O 抗原和 H 抗原血清进行分型鉴定。

## 六、GB 4789.10 金黄色葡萄球菌检验（2022版）

### 6.1 原理

金黄色葡萄球菌产生的凝固酶可使兔血浆凝固，可通过 Baird-Parker 平板分离，血浆凝固酶试验确认。

### 6.2 培养基和试剂

| 培养基 | 用途 |
|--------|------|
| **Baird-Parker 琼脂** | 选择性分离 |
| **血琼脂** | 分离 |
| **胰酪大豆胨琼脂（TSA）** | 纯化 |
| **兔血浆** | 凝固酶试验 |
| **脑心浸出液肉汤（BHI）** | 增菌 |

### 6.3 检验程序

```
样品处理 → 接种Baird-Parker平板 → 37°C培养24-48h → 挑选可疑菌落（黑色、有透明圈）→
纯化 → 血浆凝固酶试验 → 报告
```

### 6.4 血浆凝固酶试验

| 方法 | 判定标准 |
|------|----------|
| **试管法** | 3 mL 兔血浆 + 0.3 mL 菌液，37°C培养6h，血浆凝固为阳性 |
| **玻片法** | 1 滴兔血浆 + 1 环菌液，混匀，1min内出现凝块为阳性 |

### 6.5 结果报告

```
金黄色葡萄球菌 CFU/g(mL) = 典型菌落数 × 稀释倍数
```

## 七、GB 4789.15 霉菌和酵母计数（2022版）

### 7.1 原理

霉菌和酵母在适宜培养基上培养后，计数形成的菌落。

### 7.2 培养基

| 培养基 | 用途 |
|--------|------|
| **马铃薯葡萄糖琼脂（PDA）** | 霉菌和酵母计数 |
| **孟加拉红琼脂** | 霉菌和酵母计数 |
| **氯硝胺孟加拉红琼脂（DRBC）** | 霉菌和酵母计数 |

### 7.3 检验程序

```
样品处理 → 10倍系列稀释 → 选择2-3个稀释度 → 接种平板 → 28°C培养5-7天 → 菌落计数 → 计算
```

### 7.4 菌落特征

| 类型 | 菌落特征 |
|------|----------|
| **霉菌** | 菌丝状、绒毛状、棉絮状、有颜色 |
| **酵母** | 乳白色、光滑、湿润、边缘整齐 |

## 八、GB 4789.30 单核细胞增生李斯特氏菌检验（2024版）

### 8.1 原理

单增李斯特氏菌是重要的食源性致病菌，通过选择性增菌、分离培养和生化鉴定进行检测。

### 8.2 培养基

| 培养基 | 用途 |
|--------|------|
| **李氏增菌肉汤（LEB）** | 增菌 |
| **PALCAM 琼脂** | 分离培养 |
| **牛津琼脂（OXA）** | 分离培养 |
| **TSI 琼脂** | 生化试验 |

### 8.3 检验程序

```
样品处理 → LEB增菌（30°C，24-48h）→ 接种PALCAM/OXA平板 → 37°C培养24-48h →
挑选可疑菌落 → 生化鉴定 → 溶血试验 → 动物试验 → 报告
```

## 九、GB 4789.26 商业无菌检验（2024版）

### 9.1 适用范围

适用于罐头食品和其他密封包装食品的商业无菌检验。

### 9.2 检验原理

商业无菌是指产品经过杀菌处理后，不含致病性微生物或常温下能繁殖的非致病性微生物。

### 9.3 检验方法

#### 9.3.1 保温试验

| 产品类型 | 保温条件 |
|---------|----------|
| **低酸食品（pH>4.6）** | 37°C，10天 |
| **酸食品（pH≤4.6）** | 35°C，10天 |
| **酸性食品（pH≤4.0）** | 30°C，10天 |

#### 9.3.2 微生物检验

- 开启保温后膨胀的样品
- 染色镜检
- 需氧菌和厌氧菌培养

### 9.4 结果判定

| 检验结果 | 判定 |
|---------|------|
| **保温后无膨胀，微生物阴性** | 商业无菌 |
| **保温后膨胀，微生物阳性** | 非商业无菌 |
| **保温后膨胀，微生物阴性** | 需进一步分析原因 |

## 十、GB 4789.40 克罗诺杆菌属检验（2024版）

### 10.1 适用范围

适用于婴幼儿配方食品和婴幼儿谷基辅助食品中克罗诺杆菌属（阪崎肠杆菌）的检验。

### 10.2 培养基

| 培养基 | 用途 |
|--------|------|
| **缓冲蛋白胨水（BPW）** | 前增菌 |
| **肠杆菌增菌肉汤（EE）** | 增菌 |
| **VRBGA 琼脂** | 分离培养 |
| **DFI 琼脂** | 分离培养 |
| **API 20E 或 Vitek** | 鉴定 |

### 10.3 检验程序

```
样品处理（100g + 900mL BPW）→ 前增菌（37°C，18h）→ 转种EE肉汤（37°C，18-24h）→
分离培养（VRBGA/DFI，37°C，24h）→ 挑选可疑菌落 → 生化鉴定 → 报告
```

### 10.4 结果判定

| 检验结果 | 报告方式 |
|---------|----------|
| **检出克罗诺杆菌** | 检出/100g |
| **未检出克罗诺杆菌** | 未检出/100g |

## 十一、快速检测方法

### 11.1 分子生物学方法

| 方法 | 检测项目 | 优势 |
|------|----------|------|
| **PCR 法** | 沙门氏菌、李斯特氏菌等 | 快速、灵敏 |
| **实时荧光 PCR** | 多种致病菌 | 定量、快速 |
| **基因芯片** | 多种致病菌同时检测 | 高通量 |

### 11.2 免疫学方法

| 方法 | 检测项目 | 优势 |
|------|----------|------|
| **ELISA 法** | 多种致病菌 | 快速、操作简单 |
| **胶体金法** | 多种致病菌 | 现场快速检测 |

### 11.3 测试片法

| 测试片 | 检测项目 | 优势 |
|--------|----------|------|
| **Petrifilm™** | 菌落总数、大肠菌群、霉菌酵母 | 操作简单、节省空间 |

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **菌落形成单位（CFU）** | 在培养基上形成一个可见菌落的微生物单位 |
| **最大可能数（MPN）** | 基于统计学原理，用多管法估算的微生物数量 |
| **选择性培养基** | 含有抑制剂，只允许特定微生物生长的培养基 |
| **鉴别培养基** | 含有指示剂，可鉴别不同微生物的培养基 |
| **前增菌** | 在非选择性培养基中使受伤的微生物恢复 |
| **选择性增菌** | 在选择性培养基中使目标菌增殖并抑制杂菌 |
