# IFS 国际食品标准（2026版）

## 标准概述

**标准名称**：IFS Food (International Featured Standards Food)
**中文名称**：IFS 国际食品标准
**最新版本**：IFS Food Version 8 (2023年发布，2026年适用)
**发布机构**：IFS GmbH（德国）
**认证性质**：全球食品安全认证标准（GFSI 认可）

---

## 一、IFS 标准简介

### 1.1 背景与发展

| 年份 | 事件 |
|------|------|
| 2002 | IFS 首次发布，由德国、法国零售商发起 |
| 2003 | 获得 GFSI（全球食品安全倡议）认可 |
| 2019 | 发布第7版（IFS Food Version 7） |
| 2023 | 发布第8版（IFS Food Version 8） |

### 1.2 标准系列

IFS 标准（International Featured Standards）包含多个标准：

| 标准名称 | 适用范围 |
|---------|----------|
| **IFS Food** | 食品加工企业（主要标准） |
| **IFS HPC** | 家居和个人护理用品 |
| **IFS Logistics** | 物流、运输、仓储 |
| **IFS Broker** | 经销商、贸易商 |
| **IFS Cash & Carry** | 批发、自选市场 |
| **IFS Global Markets** | 初级加工企业 |

#### 2.1.2 新增要求

2. **质量管理体系整合（第2章）**
   - 强化质量管理与食品安全管理整合
   - 新增质量指标监控要求
   - 增强客户满意度管理

3. **食品防御与欺诈（第7章）**
   - 整合食品防御评估
   - 强化食品欺诈预防
   - 脆弱性评估和预防措施

4. **产品防护强化**
   - 异物检测要求更严格
   - 金属检测/X光检测评估
   - 玻璃和硬塑料管理

5. **可持续性关注**
   - 鼓励环境保护措施
   - 强调资源效率

## 三、核心要求详解

### 3.1 高层管理承诺（第1章）

#### 3.1.1 食品安全文化

**新增要求（Version 8）**：

| 要求 | 说明 |
|------|------|
| **文化计划** | 制定书面的食品安全文化建设计划 |
| **文化评估** | 定期评估食品安全文化成熟度（至少每年） |
| **改进措施** | 基于评估结果实施改进 |
| **员工参与** | 鼓励员工报告食品安全问题 |
| **管理层示范** | 管理层以身作则，展示食品安全承诺 |

**文化评估维度**：
- 领导力和承诺
- 员工意识和参与
- 沟通和反馈
- 培训和 competence
- 持续改进

#### 3.1.2 管理体系

- **食品安全方针**：明确、可测量、沟通到位
- **职责和权限**：明确分配并沟通
- **食品安全小组**：多学科团队，具备必要知识
- **资源提供**：确保足够的人力、财力、物力资源

### 3.2 HACCP 食品安全计划（第4章）

#### 3.2.1 HACCP 体系要求

IFS 要求完全符合 CAC/RCP 1-1969（HACCP 体系及其应用准则）：

| HACCP 原理 | 要求 |
|-----------|------|
| **原理1** | 危害分析：识别所有潜在危害 |
| **原理2** | 确定关键控制点（CCP） |
| **原理3** | 建立关键限值 |
| **原理4** | 建立监控程序 |
| **原理5** | 建立纠正措施 |
| **原理6** | 建立验证程序 |
| **原理7** | 建立记录保持程序 |

#### 3.2.2 HACCP 文档要求

- **HACCP 计划**：书面文件，涵盖所有产品
- **产品描述**：完整描述产品特性和预期用途
- **流程图**：覆盖所有工艺步骤
- **危害分析**：科学、系统、完整
- **CCP 确定记录**：决策树或其他科学方法
- **关键限值**：可测量、可验证
- **监控记录**：实时、准确、完整
- **纠正措施记录**：详细、可追溯
- **验证记录**：定期验证有效性

### 3.3 生产现场标准（第5章）

#### 3.3.1 建筑和设施

| 要求 | 标准 |
|------|------|
| **位置选择** | 远离污染源，周边环境整洁 |
| **厂区布局** | 合理划分卫生等级区域 |
| **建筑结构** | 光滑、耐清洗、防渗透 |
| **地面** | 耐磨损、防滑、有坡度、易排水 |
| **墙壁** | 浅色、光滑、易清洗 |
| **天花板** | 防止冷凝水、霉菌、脱落 |
| **门窗** | 密封良好、便于清洗 |
| **照明** | 充足、防护罩完好 |
| **通风** | 良好、控制气流方向 |
| **排水** | 畅通、防倒灌、易清洁 |

#### 3.3.2 设备要求

- **材质**：食品接触材料符合要求
- **设计**：无卫生死角、易清洁
- **安装**：便于操作、清洁、维护
- **维护**：预防性维护计划
- **校准**：关键测量设备定期校准

#### 3.3.3 卫生设施

| 设施 | 要求 |
|------|------|
| **洗手设施** | 充足、方便、配有洗手液和干手器 |
| **消毒设施** | 必要时设置脚消毒池 |
| **更衣室** | 分区设置、有足够储物空间 |
| **卫生间** | 数量充足、保持清洁 |
| **废弃物存放** | 密闭、标识清晰、及时清理 |

### 3.4 产品防护（第7章）

#### 3.4.1 食品防御（Food Defense）

**要求**：
- 进行食品防御评估
- 识别潜在威胁和脆弱点
- 制定预防措施
- 建立应急响应程序

**评估重点**：
- 外部进入控制
- 内部人员管理
- 水源安全
- 关键原料保护
- 成品保护

#### 3.4.2 食品欺诈（Food Fraud）

**要求**：
- 进食品欺诈脆弱性评估
- 识别高风险原料和供应商
- 实施预防措施
- 建立供应商审计机制
- 使用检测技术验证

**高风险原料**：
- 橄榄油、蜂蜜、牛奶
- 香料、果汁、谷物
- 高价值肉类和海鲜

#### 3.4.3 异物管理

**要求**：
- **评估**：评估异物风险
- **预防**：实施预防措施
- **检测**：设置金属检测/X光检测
- **控制**：玻璃和硬塑料管理

### 3.5 过程控制（第5章）

#### 3.5.1 卫生标准操作程序（SSOP）

| 程序 | 要求 |
|------|------|
| **清洁消毒** | 书面程序，规定频率、方法、责任人 |
| **虫害控制** | 专业公司、定期检查、记录完整 |
| **化学品管理** | 专用存放、标识清晰、使用记录 |
| **废物管理** | 及时清理、标识清晰、防止污染 |
| **交叉污染控制** | 物理隔离、人流物流分开、空气流向控制 |

#### 3.5.2 加工过程控制

| 控制点 | 要求 |
|------|------|
| **配方管理** | 严格按照配方生产 |
| **称量控制** | 使用校准的称量设备 |
| **投料顺序** | 按工艺要求投料 |
| **加工参数** | 控制、监控、记录 |
| **温度控制** | 关键步骤连续监控 |
| **时间控制** | 确保加工时间符合要求 |

### 3.6 人员管理（第8章）

#### 3.6.1 培训要求

| 培训类型 | 要求 | 频率 |
|---------|------|------|
| **入职培训** | 上岗前完成 | 新员工入职时 |
| **食品安全培训** | 基础知识、HACCP、个人卫生 | 每年至少1次 |
| **岗位技能培训** | 岗位操作、质量控制 | 持续进行 |
| **培训效果评估** | 验证培训有效性 | 每次培训后 |

#### 3.6.2 个人卫生

**要求**：
- **健康检查**：上岗前和每年定期检查
- **洗手消毒**：进入车间前和操作过程中
- **防护服**：穿戴规范、保持清洁
- **个人行为**：禁止吸烟、饮食、佩戴首饰
- **伤病处理**：报告伤口、适当处理

### 3.7 产品信息与追溯（第10章）

#### 3.7.1 标签管理

**要求**：
- **准确性**：标签信息准确无误
- **合规性**：符合法规要求
- **验证**：每批次标签验证
- **过敏原**：清晰标识所有过敏原

#### 3.7.2 追溯体系

**要求**：
- **前向追溯**：成品可追溯到所有原料
- **后向追溯**：原料可追溯到所有成品
- **测试频率**：每年至少全追溯测试1次
- **测试时限**：4小时内完成追溯
- **记录保持**：保持所有追溯记录

### 3.8 供应商管理（第11章）

#### 3.8.1 供应商批准

**批准流程**：
1. 风险评估
2. 信息收集
3. 现场审核（高风险供应商）
4. 样品测试
5. 批准列入合格供应商名单

#### 3.8.2 供应商监控

| 风险等级 | 审核频率 | 测试频率 |
|---------|----------|----------|
| **高风险** | 每年1次 | 每批次 |
| **中风险** | 每2年1次 | 每季度 |
| **低风险** | 每3年1次 | 每半年 |

## 四、IFS 认证评分体系

### 4.1 评分等级

IFS Food Version 8 设有 **A、B、C、D** 四个等级：

| 等级 | 评分范围 | 认证有效期 |
|------|----------|-----------|
| **A** | 75%以上 | 标准有效期（12个月） |
| **B** | 60-74% | 标准有效期（12个月） |
| **C** | 低于60% | 6个月 |
| **D** | 有重大不符合项 | 不通过认证 |

### 4.2 不符合项分类

| 类别 | 描述 | 处理要求 |
|------|------|----------|
| **重大不符合（MA）** | 严重违反标准要求，影响食品安全 | 28天内整改并验证 |
| **一般不符合（Mi）** | 偏离标准要求，但非严重 | 审核后整改并验证 |
| **观察项（OB）** | 建议改进项 | 建议改进，非强制 |

### 4.3 评分规则

- **基础分**：100%
- **扣分规则**：
  - 重大不符合：扣50%
  - 一般不符合：扣20%
  - 观察项：不扣分

## 五、IFS 认证流程

### 5.1 认证步骤

| 步骤 | 内容 | 时间 |
|------|------|------|
| **1. 准备** | 了解标准、差距分析 | 1-2个月 |
| **2. 体系建立** | 建立管理体系文件 | 1-2个月 |
| **3. 试运行** | 体系试运行 | 3个月以上 |
| **4. 内部审核** | 内部审核和管理评审 | 1个月 |
| **5. 申请认证** | 选择认证机构、提交申请 | 2-4周 |
| **6. 文件审核** | 评审管理体系文件 | 1-2天 |
| **7. 现场审核** | 现场审核 | 通常1-3天 |
| **8. 整改** | 不符合项整改 | 28天内 |
| **9. 证书发放** | 审核通过后发放证书 | 审核完成后2-4周 |

### 5.2 证书有效期

- **有效期**：12个月
- **监督审核**：每年进行一次
- **复评审核**：每3年进行一次完整审核

- 对欧出口食品加工企业
- 欧资企业国内工厂
- 高端食品制造商
- 婴幼儿食品生产企业

## 八、IFS 与其他标准的关系

### 8.1 GFSI 认可标准对比

| 标准 | 发起方 | 适用范围 | 特点 |
|------|--------|----------|------|
| **IFS** | 德国、法国零售商 | 食品加工 | 欧洲标准、详细严格 |
| **BRCGS** | 英国零售商协会 | 食品加工、包装 | 分级明确、英国标准 |
| **FSSC 22000** | 基金会 | 食品链全链条 | 基于 ISO 22000 |
| **SQF** | 美国食品营销协会 | 北美市场 | 分级认证、灵活 |

### 8.2 IFS 标准选择指南

| 企业类型 | 推荐标准 |
|---------|----------|
| **对欧出口企业** | IFS Food |
| **对英出口企业** | BRCGS |
| **北美市场** | SQF |
| **全球市场** | FSSC 22000 |
