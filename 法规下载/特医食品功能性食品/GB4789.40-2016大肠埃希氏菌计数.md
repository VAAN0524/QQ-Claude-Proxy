# GB 4789.40-2016 食品安全国家标准 食品微生物学检验 大肠埃希氏菌计数

## 标准信息
- **标准编号**：GB 4789.40-2016
- **标准名称**：食品安全国家标准 食品微生物学检验 大肠埃希氏菌计数
- **发布日期**：2016年12月23日
- **实施日期**：2017年6月23日
- **替代标准**：GB/T 4789.40-2008
- **归口单位**：国家卫生健康委员会
- **起草单位**：国家食品安全风险评估中心

---

## 1 范围

本标准规定了食品中大肠埃希氏菌（Escherichia coli）的计数方法。

本标准适用于食品中大肠埃希氏菌的计数。

---

## 2 术语和定义

### 2.1 大肠埃希氏菌 Escherichia coli
亦称大肠杆菌，是肠杆菌科埃希氏菌属的代表菌，主要寄居于人和动物的肠道内，是肠道正常菌群的重要组成部分。部分菌株具有致病性，可引起腹泻、出血性结肠炎等食源性疾病。

### 2.2 最可能数 most probable number；MPN
基于泊松分布，利用统计学原理估计样品中活菌密度的方法。

## 3 设备和材料

### 3.1 设备
| 设备名称 | 规格要求 |
|---------|---------|
| 恒温培养箱 | 36℃±1℃ |
| 冰箱 | 2℃~8℃ |
| 恒温水浴箱 | 44.5℃±0.5℃ |
| 天平 | 感量0.1g |
| 均质器 | 8000 r/min~10000 r/min |
| 振荡器 | 可调节转速 |
| 显微镜 | 10×~100× |
| 吸管 | 1mL、10mL（具0.1mL刻度） |
| 试管 | 18mm×180mm |
| 平皿 | 90mm直径 |
| 广口瓶或锥形瓶 | 500mL |

### 3.2 培养基和试剂
#### 3.2.1 EC肉汤（EC Broth）
**成分**：
- 胰蛋白胨：20.0g
- 3号胆盐（或混合胆盐）：1.12g
- 乳糖：5.0g
- 磷酸氢二钾（K₂HPO₄）：4.0g
- 磷酸二氢钾（KH₂PO₄）：1.5g
- 氯化钠：5.0g
- 蒸馏水：1000mL

**制法**：
1. 将各成分加入蒸馏水中
2. 加热溶解，校正pH至6.9±0.2
3. 分装于带有发酵倒管的试管中，每管10mL
4. 121℃高压灭菌15min

#### 3.2.2 结晶紫中性红胆盐琼脂（VRBA）
**成分**：
- 胰蛋白胨：7.0g
- 酵母膏粉：3.0g
- 3号胆盐：1.5g
- 氯化钠：5.0g
- 乳糖：10.0g
- 中性红：0.03g
- 结晶紫：0.002g
- 琼脂：15.0g~18.0g
- 蒸馏水：1000mL

**制法**：
1. 将各成分（除中性红、结晶紫外）加入蒸馏水中
2. 加热溶解，校正pH至7.4±0.2
3. 加入中性红、结晶紫溶液
4. 分装，121℃高压灭菌15min
5. 冷却至45℃~50℃，倾注平皿

#### 3.2.3 缓冲蛋白胨水（BPW）
**成分**：
- 蛋白胨：10.0g
- 氯化钠：5.0g
- 磷酸氢二钠：9.0g
- 磷酸二氢钾：1.5g
- 蒸馏水：1000mL

**制法**：
1. 将各成分加入蒸馏水中
2. 加热溶解，校正pH至7.2±0.2
3. 分装，121℃高压灭菌15min

## 4 检验程序

```
样品（25g/mL）
    ↓
加入BPW 225mL，均质
    ↓
10倍系列稀释
    ↓
选择2个~3个适宜稀释度
    ↓
接种EC肉汤（3管×3稀释度）
    ↓
36℃±1℃培养24h
    ↓
接种VRBA平板（划线）
    ↓
36℃±1℃培养18h~24h
    ↓
挑取可疑菌落（紫红色带或不带胆汁酸沉淀环）
    ↓
生化试验
    ↓
结果计算
```

---

## 5 操作步骤

### 5.1 样品处理
1. 以无菌操作取检样25g（或25mL），放入225mL BPW中
2. 用拍击式均质器均质1min~2min，或用乳钵研磨制备成1:10的样品匀液
3. 用1mL无菌吸管或微量移液器吸取1:10样品匀液1mL，注入含有9mL BPW的试管内
4. 振摇试管混匀，制成1:100的样品匀液
5. 重复操作，制备10倍系列稀释样品匀液

### 5.2 接种和培养
1. 根据对样品污染状况的估计，选择2个~3个适宜稀释度的样品匀液
2. 每个稀释度接种3管EC肉汤，每管接种1mL
3. 同时接种3管作为空白对照
4. 轻轻混匀，置于36℃±1℃恒温培养箱内培养24h
5. 观察各管产气情况

### 5.3 分离培养
1. 将产气管划线接种于VRBA平板上
2. 置于36℃±1℃恒温培养箱内培养18h~24h
3. 观察平板上菌落形态

### 5.4 菌落形态特征
在VRBA平板上，大肠埃希氏菌呈现以下特征：
- **典型菌落**：紫红色，菌落周围有胆汁酸沉淀环
- **非典型菌落**：粉红色或淡红色，菌落周围无胆汁酸沉淀环

### 5.5 生化鉴定
#### 5.5.1 靛基质试验
1. 将可疑菌落接种于蛋白胨水培养基中
2. 置于36℃±1℃恒温培养箱内培养24h
3. 沿管壁加入柯凡克试剂（Kovacs reagent）0.5mL
4. 轻轻摇动试管，观察液面颜色变化
5. **阳性结果**：出现红色环状物
6. **阴性结果**：无红色环状物出现

#### 5.5.2 甲基红试验（MR试验）
1. 将可疑菌落接种于磷酸盐葡萄糖胨水培养基（MR-VP培养基）中
2. 置于36℃±1℃恒温培养箱内培养48h
3. 吸取培养物1mL于试管中
4. 加入甲基红指示剂2滴~3滴
5. 观察颜色变化
6. **阳性结果**：呈鲜红色
7. **阴性结果**：呈黄色

#### 5.5.3 维-佩试验（VP试验）
1. 将可疑菌落接种于MR-VP培养基中
2. 置于36℃±1℃恒温培养箱内培养48h
3. 取培养物1mL于试管中
4. 加入VP试剂甲液（6%α-萘酚乙醇溶液）1mL
5. 加入VP试剂乙液（40%氢氧化钾溶液）0.4mL
6. 充分振摇，观察4h内结果
7. **阳性结果**：出现红色
8. **阴性结果**：无红色出现

#### 5.5.4 柠檬酸盐利用试验
1. 将可疑菌落接种于西蒙氏柠檬酸盐培养基斜面上
2. 置于36℃±1℃恒温培养箱内培养24h~48h
3. 观察培养基颜色变化
4. **阳性结果**：培养基由绿色变为蓝色
5. **阴性结果**：培养基颜色不变

### 5.6 结果判定
根据生化试验结果，按IMViC反应模式判定：
| 靛基质(I) | 甲基红(M) | 维-佩(V) | 柠檬酸盐(C) | 结果判定 |
|-----------|-----------|----------|------------|----------|
| + | + | - | - | 典型大肠埃希氏菌 |
| - | + | - | - | 大肠埃希氏菌非典型株 |
| + | + | - | + | 大肠埃希氏菌变异株 |
| 其他组合 | | | | 非大肠埃希氏菌 |

## 6 结果计算与报告

### 6.1 MPN法结果计算
1. 根据阳性管数，查MPN检索表
2. 计算每g（或mL）样品中大肠埃希氏菌的MPN值

### 6.2 平板计数法结果计算
1. 计数平板上紫红色菌落
2. 按下列公式计算：
   N = ΣC / (n₁ + 0.1n₂) × d
   式中：
   N：样品中大肠埃希氏菌数，CFU/g或CFU/mL
   ΣC：所有平板菌落数之和
   n₁：第一稀释度接种的平板数
   n₂：第二稀释度接种的平板数
   d：稀释倍数

### 6.3 结果报告
1. 根据计算结果，按GB/T 8170修约至两位有效数字
2. 以CFU/g或CFU/mL为单位报告结果
3. 当结果低于检测限时，报告"<10 CFU/g（或mL）"

## 7 质量控制

### 7.1 对照试验
每批试验应设置以下对照：
- **阳性对照**：使用大肠埃希氏菌标准菌株（如ATCC 25922）
- **阴性对照**：使用非大肠埃希氏菌菌株（如产气肠杆菌）
- **空白对照**：使用无菌水代替样品

### 7.2 培养基质量控制
- 使用前检查培养基外观、pH值
- 每批培养基应进行无菌性试验和促生长试验

### 7.3 仪器设备校准
- 天平、培养箱、水浴箱应定期校准
- 培养箱温度均匀性应定期验证

---

## 8 生物安全

### 8.1 实验室要求
1. 实验室应符合生物安全二级（BSL-2）要求
2. 配备生物安全柜
3. 配备高压灭菌器

### 8.2 个人防护
1. 实验人员应穿戴实验服、手套、口罩
2. 操作时应避免产生气溶胶
3. 实验结束后应彻底消毒

### 8.3 废物处理
1. 所有废弃物应经高压灭菌后处理
2. 污染的物品应立即消毒或灭菌
