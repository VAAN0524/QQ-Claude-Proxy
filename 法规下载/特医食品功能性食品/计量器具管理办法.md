# 食品生产企业计量器具管理办法

## 办法基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家市场监督管理总局 |
| 发布年份 | 2023年 |
| 适用范围 | 食品生产企业计量器具管理 |
| 法规性质 | 部门规章 |

## 1. 总则

### 1.1 目的

为规范食品生产企业计量器具管理，确保测量数据准确可靠，保障产品质量和安全，根据《中华人民共和国计量法》及相关法律法规，制定本办法。

### 1.2 适用范围

本办法适用于：
1. 特殊医学用途配方食品生产企业
2. 保健食品生产企业
3. 其他食品生产企业
4. 食品相关产品生产企业

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| **计量器具** | 单独地或连同辅助设备一起用以进行测量的器具 |
| **强制检定** | 由法定计量检定机构对特定计量器具进行的检定 |
| **校准** | 在规定条件下，为确定计量器具示值与对应计量标准复现值之间关系的一组操作 |
| **计量确认** | 为确保计量器具满足预期使用要求所需的一组操作 |

## 2. 管理职责

### 2.1 企业负责人

1. 对计量工作负全面责任
2. 建立计量管理制度
3. 配备计量管理人员
4. 保障计量经费

### 2.2 计量管理部门

1. 制定计量管理制度
2. 建立计量器具台账
3. 组织检定/校准
4. 监督计量器具使用
5. 处理计量纠纷

### 2.3 使用部门

1. 正确使用计量器具
2. 做好日常维护保养
3. 按期送检
4. 发现异常及时报告

## 3. 计量器具分类

### 3.1 强制检定计量器具

| 类别 | 器具示例 | 检定周期 | 检定机构 |
|------|----------|----------|----------|
| 贸易结算 | 电子秤、台秤、案秤 | 1年 | 法定计量检定机构 |
| 安全防护 | 压力表、压力变送器 | 6个月-1年 | 法定计量检定机构 |
| 医疗卫生 | 温度计、血压计 | 按规程 | 法定计量检定机构 |
| 环境监测 | 噪声仪、照度计 | 1年 | 法定计量检定机构 |

### 3.2 非强制检定计量器具

| 类别 | 器具示例 | 校准周期 |
|------|----------|----------|
| 长度类 | 游标卡尺、钢卷尺 | 1年 |
| 温度类 | 数字温度计、温度记录仪 | 1年 |
| 湿度类 | 温湿度计、露点仪 | 1年 |
| 压力类 | 真空表、差压变送器 | 1年 |
| 流量类 | 流量计、液位计 | 1-2年 |
| 质量类 | 天平、砝码 | 1年 |
| 时间类 | 秒表、计时器 | 1年 |
| 电学类 | 万用表、钳形电流表 | 1年 |

### 3.3 A/B/C分类管理

| 分类 | 定义 | 管理要求 |
|------|------|----------|
| **A类** | 关键质量控制点的计量器具 | 强制检定或严格校准 |
| **B类** | 一般质量控制点的计量器具 | 周期校准 |
| **C类** | 指示性计量器具 | 一次性检定或功能检查 |

## 4. 入厂验收

### 4.1 验收项目

| 验收项目 | 验收内容 | 合格标准 |
|----------|----------|----------|
| 外观检查 | 完好性、铭牌、标识 | 完好、清晰 |
| 随机文件 | 说明书、合格证、型式评价证书 | 齐全 |
| 计量性能 | 示值误差、重复性 | 符合技术要求 |
| 法定要求 | CMC标志、检定证书 | 符合规定 |

### 4.2 验收流程

```
到货通知
    ↓
外观检查
    ↓
随机资料检查
    ↓
计量性能验收
    ↓
验收合格 → 入账、编号、标识
验收不合格 → 退货/换货
```

### 4.3 首次检定/校准

新购计量器具投入使用前必须进行首次检定或校准：
- 强制检定器具：送法定计量检定机构检定
- 非强制检定器具：企业自校或委托校准实验室

## 5. 周期检定/校准

### 5.1 周期确定

| 因素 | 短周期 | 长周期 |
|------|--------|--------|
| 使用频率 | 频繁使用 | 偶尔使用 |
| 环境条件 | 恶劣 | 良好 |
| 稳定性 | 差 | 好 |
| 准确度要求 | 高 | 一般 |
| 历史数据 | 容易超差 | 稳定 |

### 5.2 周期表示例

| 器具名称 | 型号规格 | 测量范围 | 准确度等级 | 检定/校准周期 | 执行机构 |
|----------|----------|----------|------------|----------------|----------|
| 电子天平 | XX型 | 0-200g | 0.1mg | 1年 | 法定机构 |
| 压力表 | XX型 | 0-1.6MPa | 1.6级 | 6个月 | 法定机构 |
| 数字温度计 | XX型 | -50-200℃ | 0.5℃ | 1年 | 企业自校 |
| 游标卡尺 | 0-150mm | 0.02mm | 1年 | 企业自校 |

### 5.3 检定/校准计划

每年12月制定下年度检定/校准计划：

| 计划内容 | 说明 |
|----------|------|
| 器具清单 | 名称、型号、编号 |
| 检定月份 | 安排检定时间 |
| 执行机构 | 法定机构或企业自校 |
| 预算 | 检定费用预算 |

### 5.4 检定/校准实施

#### 5.4.1 外部检定/校准

1. 提前1个月联系检定机构
2. 送检或预约现场检定
3. 取得检定/校准证书
4. 更新计量标识

#### 5.4.2 企业自校

1. **自校条件**：
   - 配备计量标准器
   - 有自校规程
   - 有持证校准人员

2. **自校流程**：
   ```
   准备工作
       ↓
   检查标准器
       ↓
   按规程校准
       ↓
   记录数据
       ↓
   判定合格/不合格
       ↓
   签发校准证书
       ↓
   更新标识
   ```

### 5.5 校准不合格处理

| 不合格程度 | 处理措施 |
|------------|----------|
| 轻微超差 | 调整后重新校准 |
| 严重超差 | 维修后重新校准 |
| 无法修复 | 报废、更新 |

## 6. 使用管理

### 6.1 使用要求

| 要求 | 说明 |
|------|------|
| 持证上岗 | 操作人员经培训合格 |
| 正确使用 | 按操作规程使用 |
| 定期维护 | 做好日常维护保养 |
| 环境要求 | 满足使用环境条件 |
| 禁止行为 | 不得擅自调整、拆卸 |

### 6.2 现场标识

| 标识类型 | 颜色 | 含义 |
|----------|------|------|
| 合合格证 | 绿色 | 在检定有效期内 |
| 准用证 | 黄色 | 降级使用或限用 |
| 停用证 | 红色 | 超期或故障停用 |
| 封存 | 灰色 | 长期不用，完好封存 |

### 6.3 计量异常处理

```
发现异常
    ↓
停止使用
    ↓
粘贴停用标识
    ↓
报告计量部门
    ↓
原因分析
    ↓
维修/校准
    ↓
验证合格
    ↓
恢复使用
```

## 7. 记录档案

### 7.1 台账管理

建立计量器具台账，包含：

| 信息项 | 说明 |
|--------|------|
| 器具编号 | 唯一编号 |
| 名称型号 | 准确标识 |
| 测量范围 | 使用范围 |
| 准确度等级 | 精度等级 |
| 制造厂家 | 供应商信息 |
| 出厂编号 | 原厂编号 |
| 购置日期 | 入厂时间 |
| 使用部门 | 责任部门 |
| 安装位置 | 具体位置 |
| 检定周期 | 检定间隔 |
| 下次检定 | 到期时间 |
| 状态 | 在用/停用/报废 |

### 7.2 档案管理

#### 7.2.1 一器一档

每件计量器具建立档案，包含：
1. 采购合同、发票
2. 说明书、合格证
3. 验收记录
4. 历次检定/校准证书
5. 使用记录
6. 维修记录
7. 报废记录

#### 7.2.2 保存期限

| 记录类型 | 保存期限 |
|----------|----------|
| 台账 | 长期 |
| 检定证书 | 3个周期 |
| 校准记录 | 3个周期 |
| 维修记录 | 器具报废后3年 |
| 报废记录 | 永久 |

## 8. 不确定度评定

### 8.1 评定要求

关键测量过程应进行测量不确定度评定：
1. 识别不确定度来源
2. 评定各分量不确定度
3. 计算合成不确定度
4. 确定扩展不确定度
5. 评定测量能力

### 8.2 评定方法

| 方法 | 适用场合 |
|------|----------|
| A类评定 | 统计分析方法 |
| B类评定 | 非统计分析方法 |
| GUM法 | 常规测量 |
| 蒙特卡洛法 | 复杂模型 |

### 8.3 典型不确定度分量

| 来源 | 评定方法 |
|------|----------|
| 计量器具 | B类（检定/校准证书） |
| 测量重复性 | A类（实验统计） |
| 环境因素 | B类（估计） |
| 人员操作 | A类（实验统计） |
| 方法误差 | B类（文献） |

## 9. 质量监控

### 9.1 期间核查

对使用中的计量器具进行期间核查：

| 器具类型 | 核查周期 | 核查方法 |
|----------|----------|----------|
| 关键测量器具 | 3-6个月 | 使用核查标准 |
| 稳定性差的器具 | 1-3个月 | 使用核查标准 |
| 稳定性好的器具 | 1年 | 使用核查标准 |

### 9.2 比对验证

参与计量比对：
1. 实验室间比对
2. 能力验证
3. 内部比对

### 9.3 统计控制

对关键测量过程实施统计控制：
1. 绘制控制图
2. 分析趋势
3. 预警处理

## 10. 人员培训

### 10.1 培训内容

| 培训类型 | 培训内容 | 培训对象 |
|----------|----------|----------|
| 计量基础 | 计量法律法规、术语 | 全体计量人员 |
| 专业培训 | 专业计量知识、技能 | 专业计量人员 |
| 器具操作 | 具体器具操作方法 | 使用人员 |
| 法规培训 | 最新计量法规、标准 | 管理人员 |

### 10.2 培训要求

| 人员类型 | 培训频率 | 考核方式 |
|----------|----------|----------|
| 计量管理人员 | 每2年 | 理论考试 |
| 计量检定员 | 每3年 | 理论+实操 |
| 使用人员 | 上岗前+复训 | 实操考核 |

### 10.3 资质要求

| 岗位 | 资质要求 | 发证机构 |
|------|----------|----------|
| 计量检定员 | 计量检定员证 | 省级市场监管部门 |
| 校准人员 | 校准人员证 | 计量协会 |
| 内审员 | 计量体系内审员证 | 认证机构 |

## 11. 计量体系确认

### 11.1 ISO 10012

鼓励企业按照ISO 10012《测量管理体系 测量过程和测量设备的要求》建立计量体系。

### 11.2 确认内容

| 确认项目 | 确认内容 |
|----------|----------|
| 计量法制 | 满足计量法制要求 |
| 计量技术 | 测量能力满足要求 |
| 计量管理 | 管理体系有效运行 |

### 11.3 确认流程

```
体系建立
    ↓
体系运行
    ↓
内部审核
    ↓
管理评审
    ↓
外部确认（自愿）
```

## 12. 违规处理

| 违规行为 | 处理措施 |
|----------|----------|
| 使用不合格计量器具 | 停止使用、罚款 |
| 伪造检定证书 | 追究法律责任 |
| 擅自调整计量器具 | 责令改正、罚款 |
| 逾期不检定 | 责令检定、罚款 |

---

**办法说明**：
- 本办法为部门规章
- 企业应结合实际制定实施细则
- 强制检定计量器具必须严格按规定执行
