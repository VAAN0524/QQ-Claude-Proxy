## 范围

本标准规定了食品中水分的测定方法。

本标准适用于各类食品中水分的测定。

**第一法 直接干燥法**：适用于谷物、蔬菜、水果、肉制品等绝大多数食品。

**第二法 减压干燥法**：适用于糖类、味精等易分解的食品。

**第三法 蒸馏法**：适用于含较多挥发性物质的食品，如香辛料等。

**第四法 卡尔·费休法**：适用于含水量较低的食品，如脱水蔬菜、油脂等。

---

## 术语和定义

### 2.1 水分 moisture

食品在105℃±2℃或减压条件下干燥至恒重所失去的质量。

### 2.2 恒重 constant weight

试样经干燥，前后两次称量质量差不超过2 mg。

---

## 第一法：直接干燥法

### 3.1 原理

利用食品中水分可以物理性质地蒸发，将样品加热至105℃，使水分蒸发，以样品失重来计算水分含量。

### 3.2 仪器和设备

| 设备 | 规格 |
|------|------|
| 电热恒温干燥箱 | 105℃±2℃ |
| 分析天平 | 感量0.1 mg |
| 扁形称量瓶 | 50 mm×30 mm |
| 干燥器 | 内附有效干燥剂 |
| 组织捣碎机 | - |

### 3.3 分析步骤

#### 3.3.1 称量瓶预处理

取洁净称量瓶，置于105℃±2℃干燥箱中，瓶盖斜支于瓶边，加热0.5 h~1.0 h，取出盖好，置干燥器内冷却0.5 h，称量，并重复干燥至恒重。

#### 3.3.2 试样制备

- **固体试样**：切碎或磨细，混合均匀
- **半固体或液体试样**：先在水浴上低温蒸干，再于105℃干燥
- **高脂肪试样**：先经脱脂处理

#### 3.3.3 测定

称取2 g~10 g试样（精确至0.0001 g），放入已恒重的称量瓶中，试样厚度不超过5 mm。置于105℃±2℃干燥箱中，瓶盖斜支于瓶边，干燥2 h~4 h后，盖好取出，放入干燥器内冷却0.5 h后称量。再放入干燥箱中干燥1 h左右，取出，放入干燥器内冷却0.5 h后再称量。重复操作至恒重。

### 3.4 结果计算

```
X = (m1 - m2) / (m1 - m3) × 100
```

其中：
- X：试样中水分的含量，单位为克每百克（g/100 g）
- m1：称量瓶和试样的质量，单位为克（g）
- m2：称量瓶和试样干燥后的质量，单位为克（g）
- m3：称量瓶的质量，单位为克（g）

**结果保留三位有效数字**。

## 第二法：减压干燥法

### 4.1 原理

利用真空条件下水分沸点降低的原理，在较低温度下将样品中的水分蒸发，以样品失重来计算水分含量。

### 4.2 仪器和设备

| 设备 | 规格 |
|------|------|
| 真空干燥箱 | 温度可控制在70℃±2℃，真空度0.09 MPa以上 |
| 分析天平 | 感量0.1 mg |
| 扁形称量瓶 | 50 mm×30 mm |
| 干燥器 | 内附有效干燥剂 |

### 4.3 分析步骤

#### 4.3.1 试样制备

同3.3.2。

#### 4.3.2 测定

称取2 g~5 g试样（精确至0.0001 g），放入已恒重的称量瓶中。置于真空干燥箱中，在70℃±2℃、0.09 MPa以上真空度条件下干燥5 h后，放气取出，放入干燥器内冷却0.5 h后称量。重复操作至恒重。

### 4.4 结果计算

同3.4。

## 第三法：蒸馏法

### 5.1 原理

利用食品中水分与有机溶剂（如甲苯、二甲苯）共同蒸馏，水分与溶剂形成共沸物被蒸出，冷凝后收集水分，根据水分体积计算含量。

### 5.2 仪器和设备

| 设备 | 规格 |
|------|------|
| 水分蒸馏器 | - |
| 电热套 | - |
| 分析天平 | 感量0.1 mg |

### 5.3 试剂

- 甲苯或二甲苯：化学纯，使用前先加水充分振摇后静置，弃去水层，蒸馏后使用
- 无水硫酸钠：化学纯

### 5.4 分析步骤

#### 5.4.1 试样制备

准确称取适量试样（含水量约2 mL~5 mL），放入水分蒸馏器的烧瓶中。

#### 5.4.2 测定

加入约50 mL甲苯或二甲苯，连接蒸馏装置，加热蒸馏，控制蒸馏速度约2滴/秒。蒸馏至水分全部蒸出（接收管中水分体积不再增加）。读取接收管中水的体积。

### 5.5 结果计算

```
X = V / m × 100
```

其中：
- X：试样中水分的含量，单位为克每百克（g/100 g）
- V：接收管中水的体积，单位为毫升（mL）（按1 mL = 1 g计）
- m：试样的质量，单位为克（g）

**结果保留三位有效数字**。

## 第四法：卡尔·费休法

### 6.1 原理

利用碘氧化二氧化硫时需要定量的水参与反应的原理，测定样品中的水分含量。

```
I₂ + SO₂ + 3C₅H₅N + H₂O → 2C₅H₅N·HI + C₅H₅N·SO₃
```

### 6.2 仪器和设备

| 设备 | 规格 |
|------|------|
| 卡尔·费休水分测定仪 | - |
| 分析天平 | 感量0.1 mg |
| 微量注射器 | 10 μL~100 μL |

### 6.3 试剂

- 卡尔·费休试剂：按仪器说明书配制
- 无水甲醇：优级纯
- 水标准溶液：准确称取适量水，用无水甲醇配制成已知浓度的水标准溶液

### 6.4 分析步骤

#### 6.4.1 卡尔·费休试剂标定

按仪器说明书操作，用水标准溶液标定卡尔·费休试剂的滴定度T（mg/mL）。

#### 6.4.2 试样测定

根据试样含水量，准确称取适量试样（精确至0.0001 g），迅速放入测定仪的反应瓶中，按仪器说明书进行测定。

### 6.5 结果计算

```
X = V × T / m × 100
```

其中：
- X：试样中水分的含量，单位为克每百克（g/100 g）
- V：消耗卡尔·费休试剂的体积，单位为毫升（mL）
- T：卡尔·费休试剂的滴定度，单位为毫克每毫升（mg/mL）
- m：试样的质量，单位为克（g）

**结果保留三位有效数字**。

## 方法选择指南

| 食品类别 | 推荐方法 | 说明 |
|----------|----------|------|
| 谷物、豆类 | 直接干燥法 | - |
| 肉制品 | 直接干燥法 | 先于60℃~80℃烘干大部分水分，再于105℃干燥 |
| 乳制品 | 直接干燥法 | - |
| 蔬菜、水果 | 直接干燥法 | 需切碎 |
| 糖类、蜂蜜 | 减压干燥法 | 温度过高会分解 |
| 调味品 | 直接干燥法或减压干燥法 | 根据成分选择 |
| 香辛料 | 蒸馏法 | 含挥发性物质 |
| 油脂 | 卡尔·费休法 | 含水量低 |
| 脱水食品 | 卡尔·费休法 | 含水量低 |

## 注意事项

1. **直接干燥法**：
   - 适用于不含挥发性物质的食品
   - 干燥温度和时间应根据样品性质适当调整
   - 含糖量高的样品易结焦，应先低温干燥

2. **减压干燥法**：
   - 适用于高温下易分解的食品
   - 真空度应控制在0.09 MPa以上
   - 注意真空泵的维护保养

3. **蒸馏法**：
   - 适用于含挥发性物质的食品
   - 有机溶剂易燃，使用时应注意防火
   - 接收管应预先用溶剂润湿

4. **卡尔·费休法**：
   - 适用于含水量低的食品
   - 试剂有毒，应妥善保管
   - 仪器应定期校准

5. **通用注意事项**：
   - 样品应充分混合均匀
   - 称量瓶应提前恒重
   - 冷却时间应一致
   - 操作过程中应防止样品吸潮
