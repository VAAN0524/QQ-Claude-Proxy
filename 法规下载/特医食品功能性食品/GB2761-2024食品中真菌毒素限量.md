# GB 2761-2024 食品中真菌毒素限量

## 标准基本信息

| 项目 | 内容 |
|------|------|
| 标准号 | GB 2761-2024 |
| 标准名称 | 食品安全国家标准 食品中真菌毒素限量 |
| 发布日期 | 2024年XX月XX日 |
| 实施日期 | 2025年XX月XX日 |
| 发布机构 | 国家卫生健康委员会 国家市场监督管理总局 |
| 代替标准 | GB 2761-2017 |

## 适用范围

本标准规定了食品中黄曲霉毒素、脱氧雪腐镰刀菌烯醇、展青霉素、赭曲霉毒素A、玉米赤霉烯酮、伏马毒素、T-2毒素、HT-2毒素的限量要求。

本标准适用于各类食品。

## 术语和定义

### 真菌毒素
真菌在生长繁殖过程中产生的次级有毒代谢产物。

### 黄曲霉毒素
黄曲霉和寄生曲霉产生的二呋喃香豆素衍生物，主要包括B1、B2、G1、G2、M1、M2等。

### 脱氧雪腐镰刀菌烯醇
又称呕吐毒素，主要由禾谷镰刀菌产生的单端孢霉烯族毒素。

## 限量要求

### 1. 黄曲霉毒素限量

#### 1.1 黄曲霉毒素B1限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 花生及其制品 | 20 |
| 玉米及其制品 | 20 |
| 大豆油、花生油、玉米油 | 10 |
| 其他植物油 | 10 |
| 婴幼儿配方食品 | 0.5 |
| 特殊医学用途配方食品 | 0.5 |
| 婴幼儿辅助食品 | 0.5 |
| 乳及乳制品 | 0.5 |
| 保健食品 | 5 |
| 其他谷物及其制品 | 5 |

#### 1.2 黄曲霉毒素M1限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 生乳 | 0.5 |
| 乳制品（按乳量折算） | 0.5 |
| 特殊医学用途婴儿配方食品 | 0.025 |
| 婴儿配方食品 | 0.025 |
| 较大婴儿和幼儿配方食品 | 0.025 |

#### 1.3 总黄曲霉毒素限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 坚果及籽类（直接食用的） | 10 |
| 调味品 | 5 |
| 干果及果蔬制品 | 5 |

### 2. 脱氧雪腐镰刀菌烯醇限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 小麦、大麦、玉米及其制品 | 1000 |
| 粮食制品（如面粉、玉米粉） | 1000 |
| 特殊医学用途配方食品 | 500 |
| 婴幼儿配方食品 | 200 |

### 3. 展青霉素限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 苹果、山楂制品 | 50 |
| 枣制品 | 50 |
| 其他果蔬制品 | 50 |

### 4. 赭曲霉毒素A限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 谷物及其制品 | 5 |
| 葡萄干、葡萄汁 | 10 |
| 葡萄酒 | 2 |
| 咖啡豆 | 5 |
| 保健食品 | 5 |

### 5. 玉米赤霉烯酮限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 小麦、玉米及其制品 | 60 |
| 特殊医学用途配方食品 | 30 |
| 婴幼儿配方食品 | 20 |

### 6. 伏马毒素限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 玉米及其制品（FB1+FB2） | 4000 |
| 玉米制品（如玉米粉、玉米糁） | 2000 |
| 特殊医学用途配方食品 | 1000 |

### 7. T-2毒素和HT-2毒素限量

| 食品类别 | 最大残留量（μg/kg） |
|----------|---------------------|
| 小麦、大麦、燕麦及其制品 | 500 |
| 特殊医学用途配方食品 | 200 |
| 婴幼儿谷物辅助食品 | 100 |

## 其他要求

1. 食品生产经营者应采取控制措施，使食品中真菌毒素含量尽可能达到最低水平。

2. 食品中真菌毒素超过本标准限量时，不得用于食品生产加工、销售和餐饮服务。

3. 对于特殊医学用途配方食品和婴幼儿配方食品，应采用严格的质量控制措施。

## 附录A 食品分类说明

本标准食品分类采用GB 2760-2024中的食品分类系统。

**注**：特殊医学用途配方食品的分类按照GB 29922执行。
