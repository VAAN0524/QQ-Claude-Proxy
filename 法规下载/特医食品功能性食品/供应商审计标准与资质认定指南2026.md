# 供应商审计标准与资质认定指南（2026年版）

## 发布信息

- **编制机构**：中国食品工业协会
- **发布日期**：2026年X月X日
- **适用范围**：特医食品、功能性食品生产企业
- **文件编号**：CFIA-SQA-2026-001

---

## 前言

本指南为特殊医学用途配方食品和功能性食品生产企业提供供应商质量审计和资质认定的标准化方法，确保供应链的质量安全。

本指南依据《食品安全法》及其实施条例、GB 14881《食品生产通用卫生规范》、ISO 9001《质量管理体系要求》等法规标准编制。

---

## 第1章 总则

**1.1 目的**

建立科学、规范的供应商审计和资质认定体系，确保供应商能够持续提供符合要求的产品和服务。

**1.2 适用范围**

本指南适用于特医食品、功能性食品生产企业的以下供应商：
- 原辅料供应商
- 包装材料供应商
- 设备供应商
- 检验服务供应商
- 仓储物流供应商
- 其他相关服务供应商

**1.3 基本原则**

1. **风险导向**：根据供应商风险等级确定审计频次和深度
2. **客观公正**：采用科学方法，确保审计结果客观准确
3. **持续改进**：推动供应商质量管理体系持续改进
4. **合规性**：确保审计活动符合法律法规要求

---

## 第2章 供应商分类

**2.1 按风险等级分类**

| 风险等级 | 定义 | 审计频率 |
|----------|------|----------|
| 高风险 | 直接影响产品安全的关键物料供应商 | 每年至少1次 |
| 中风险 | 可能影响产品质量的重要物料供应商 | 每2年至少1次 |
| 低风险 | 对产品质量影响较小的一般物料供应商 | 每3年至少1次 |

**2.2 按物料类别分类**

| 物料类别 | 典型物料 | 风险等级 |
|----------|----------|----------|
| 特殊医学用途配方原料 | 蛋白质、脂肪、碳水化合物等 | 高 |
| 功能性成分原料 | 维生素、矿物质、膳食纤维等 | 高 |
| 食品添加剂 | 防腐剂、抗氧化剂、甜味剂等 | 中 |
| 包装材料 | 复合膜、金属罐、玻璃瓶等 | 中 |
| 辅助材料 | 清洁剂、润滑油等 | 低 |

**2.3 按供应商性质分类**

| 供应商性质 | 审计重点 |
|------------|----------|
| 生产商 | 生产过程控制、质量管理体系 |
| 贸易商 | 供应商管理、追溯体系 |
| 服务商 | 资质能力、服务质量 |

## 第3章 供应商资质要求

**3.1 基本资质要求**

**3.1.1** 所有供应商应具备以下基本资质：

1. 营业执照（统一社会信用代码）
2. 生产许可证或经营许可证
3. 产品质量标准
4. 产品检验报告
5. 产品合格证明

**3.1.2** 特殊类别供应商的附加资质：

| 供应商类别 | 附加资质要求 |
|------------|--------------|
| 食品生产供应商 | 食品生产许可证（SC） |
| 进口供应商 | 进口货物报关单、检验检疫证明 |
| 药品级原料供应商 | 药品生产许可证、GMP证书 |
| 检测服务供应商 | CMA证书、CNAS认可证书 |
| 物流服务供应商 | 道路运输经营许可证 |

**3.2 质量管理体系要求**

**3.2.1** 供应商应建立并实施以下质量管理体系：

1. ISO 9001质量管理体系
2. FSSC 22000食品安全体系
3. HACCP危害分析与关键控制点体系
4. 或其他等效的质量管理体系

**3.2.2** 体系认证要求：

| 认证体系 | 适用范围 | 最低要求 |
|----------|----------|----------|
| ISO 9001 | 所有供应商 | 鼓励认证 |
| FSSC 22000 | 食品生产企业 | 高风险供应商必须 |
| HACCP | 食品生产企业 | 中高风险供应商必须 |
| ISO 22000 | 食品供应链企业 | 鼓励认证 |

**3.3 技术能力要求**

**3.3.1** 生产能力：

1. 满足采购需求的生产能力
2. 稳定的生产设备
3. 合理的生产布局
4. 必要的应急生产能力

**3.3.2** 检验能力：

1. 基本的检验设备和设施
2. 合格的检验人员
3. 完善的检验规程
4. 可靠的检验记录

**3.3.3** 研发能力（适用于关键原料供应商）：

1. 研发团队和设施
2. 新产品开发能力
3. 技术创新能力
4. 问题解决能力

## 第4章 审计准备

**4.1 审计计划**

**4.1.1** 年度审计计划应包括：

1. 审计供应商名单
2. 审计时间安排
3. 审计组成员
4. 审计重点内容
5. 审计资源预算

**4.1.2** 审计计划编制流程：

```
供应商风险评估 → 确定审计优先级 → 制定审计计划 → 审批 → 通知供应商
```

**4.2 审计组组成**

**4.2.1** 审计组基本配置：

| 审计类型 | 审计组组成 | 人数 |
|----------|------------|------|
| 现场审计 | 审计组长 + 审计员 | 2-4人 |
| 文件审计 | 审计员 | 1-2人 |
| 远程审计 | 审计员 | 1人 |

**4.2.2** 审计组成员要求：

1. 具备相关专业知识
2. 接受过审计培训
3. 具有审计经验
4. 独立公正

**4.3 审计文件准备**

**4.3.1** 审计清单应包括：

1. 审计计划
2. 审计检查表
3. 审计评分标准
4. 审计记录表
5. 审计报告模板

**4.3.2** 供应商应准备的资料：

1. 营业执照及相关资质证明
2. 质量手册和程序文件
3. 产品标准和技术文件
4. 检验记录和报告
5. 不合格品处理记录
6. 客户投诉处理记录
7. 纠正预防措施记录

## 第5章 审计实施

**5.1 审察项目**

**5.1.1** 管理体系审计：

| 审计要素 | 审计要点 | 评分标准 |
|----------|----------|----------|
| 管理职责 | 组织架构、职责权限、管理承诺 | 0-10分 |
| 文件控制 | 文件体系、文件管理、记录控制 | 0-10分 |
| 资源管理 | 人力资源、基础设施、工作环境 | 0-15分 |
| 产品实现 | 生产策划、过程控制、产品放行 | 0-25分 |
| 测量分析 | 监控测量、数据分析、持续改进 | 0-20分 |
| 合规性 | 法规符合、证照齐全、合规经营 | 0-20分 |

**5.1.2** 现场审查项目：

| 审查区域 | 审查要点 | 不合格示例 |
|----------|----------|------------|
| 厂区环境 | 周边环境、厂区布局、环境卫生 | 厂区有污染源、垃圾堆积 |
| 仓储区域 | 物料存放、标识管理、温湿度控制 | 物料混放、标识不清 |
| 生产区域 | 设备设施、卫生控制、人员操作 | 设备锈蚀、卫生差 |
| 检验区域 | 检验设备、检验能力、记录管理 | 设备未校准、记录不完整 |
| 办公区域 | 文件管理、记录保存 | 文件混乱、记录缺失 |

**5.1.3** 产品质量审计：

| 审计内容 | 审计要点 | 合格标准 |
|----------|----------|----------|
| 原辅料验收 | 供应商合格证明、检验报告 | 100%合格 |
| 生产过程控制 | 关键控制点、工艺参数 | 符合要求 |
| 成品检验 | 检验项目、检验方法 | 符合标准 |
| 不合格品控制 | 识别、隔离、处理 | 100%控制 |
| 追溯能力 | 批次管理、记录追溯 | 100%可追溯 |

**5.2 审计方法**

**5.2.1** 文件审查：

1. 查阅质量管理体系文件
2. 检查记录的完整性和准确性
3. 评估文件的可操作性
4. 验证文件的实际执行情况

**5.2.2** 现场观察：

1. 生产现场观察
2. 设备设施检查
3. 人员操作观察
4. 卫生状况评估

**5.2.3** 人员访谈：

1. 管理层访谈
2. 关键岗位人员访谈
3. 操作人员访谈
4. 质量人员访谈

**5.2.4** 产品抽检：

1. 随机抽取样品
2. 现场检验或送检
3. 对比历史检验数据
4. 评估检验能力

**5.3 审计发现**

**5.3.1** 不符合项分类：

| 不符合类别 | 定义 | 示例 |
|------------|------|------|
| 严重不符合 | 严重影响产品质量或安全 | 无证生产、使用违禁原料 |
| 一般不符合 | 对质量有一定影响 | 记录不完整、标识不清 |
| 观察项 | 可能发展成为不符合 | 设备老化、文件需要更新 |

**5.3.2** 不符合项处理：

1. 现场确认不符合项
2. 评估不符合项影响
3. 确定整改期限
4. 制定整改措施
5. 验证整改效果

## 第6章 审计报告

**6.1 报告内容**

**6.1.1** 审计报告应包括：

1. 供应商基本信息
2. 审计范围和依据
3. 审计时间和人员
4. 审察发现和评分
5. 不符合项描述
6. 审计结论和建议
7. 审计人员签字

**6.2 评分标准**

**6.2.1** 综合评分：

| 评分区间 | 等级 | 结论 |
|----------|------|------|
| 90-100分 | 优秀 | 推荐为优先供应商 |
| 75-89分 | 良好 | 认定为合格供应商 |
| 60-74分 | 合格 | 有条件认可，限期整改 |
| 低于60分 | 不合格 | 不予认可 |

**6.2.2** 分项评分：

| 审计项目 | 权重 | 合格线 |
|----------|------|--------|
| 管理体系 | 20% | ≥60分 |
| 现场管理 | 30% | ≥60分 |
| 质量控制 | 30% | ≥60分 |
| 产品质量 | 20% | ≥60分 |

**6.3 审计结论**

**6.3.1** 审计结论类型：

1. **合格**：综合得分≥75分，无严重不符合项
2. **有条件合格**：综合得分60-74分，或存在一般不符合项
3. **不合格**：综合得分<60分，或存在严重不符合项

**6.3.2** 审计建议：

1. **推荐合作**：综合得分≥90分
2. **可以合作**：综合得分75-89分
3. **限期整改后合作**：综合得分60-74分
4. **不予合作**：综合得分<60分

## 第7章 供应商管理

**7.1 供应商批准**

**7.1.1** 批准条件：

1. 审计结论为合格或有条件合格
2. 不符合项已整改完成
3. 样品检验合格
4. 签订质量协议

**7.1.2** 批准流程：

```
审计 → 整改验证 → 评定 → 批准 → 签约 → 采购
```

**7.2 供应商监控**

**7.2.1** 日常监控指标：

| 监控指标 | 监控频率 | 评判标准 |
|----------|----------|----------|
| 交货准时率 | 每月 | ≥95% |
| 产品合格率 | 每批 | ≥99% |
| 价格竞争力 | 每季度 | 市场平均水平 |
| 服务响应 | 按需 | 24小时内响应 |
| 重大质量事故 | 随时 | 零容忍 |

**7.2.2** 质量问题处理：

| 问题类别 | 处理措施 |
|----------|----------|
| 一般质量问题 | 通知整改，跟踪验证 |
| 严重质量问题 | 暂停采购，限期整改 |
| 重大质量事故 | 终止合作，启动应急程序 |

**7.3 供应商评价**

**7.3.1** 年度评价内容：

1. 质量表现
2. 交货表现
3. 服务表现
4. 价格水平
5. 技术支持

**7.3.2** 评价结果应用：

1. 优秀供应商：扩大合作，授予荣誉
2. 合格供应商：继续合作
3. 不合格供应商：终止合作

**7.4 供应商发展**

**7.4.1** 供应商支持：

1. 技术指导
2. 培训支持
3. 管理经验分享
4. 联合改进

**7.4.2** 供应商激励：

1. 优先采购权
2. 延长付款期限
3. 联合研发
4. 战略合作

## 第8章 资质认定

**8.1 CMA资质认定**

**8.1.1** CMA（中国计量认证）基本要求：

1. 具有独立法人资格
2. 固定的工作场所
3. 满足要求的检测设备
4. 合格的检测人员
5. 完善的质量管理体系

**8.1.2** CMA认定流程：

```
申请 → 资料审查 → 现场评审 → 整改验证 → 审批发证
```

**8.1.3** CMA证书管理：

- 有效期：6年
- 需要定期监督评审
- 需要能力验证

**8.2 CNAS认可**

**8.2.1** CNAS（中国合格评定国家认可委员会）认可要求：

1. 符合ISO/IEC 17025要求
2. 建立完善的质量管理体系
3. 具备相应的技术能力
4. 通过实验室认可评审

**8.2.2** CNAS认可流程：

```
申请 → 文件审查 → 现场评审 → 验证试验 → 认可决定 |
```

**8.2.3** CNAS认可有效期：

- 有效期：5年
- 需要定期监督评审
- 需要能力验证

**8.3 其他资质**

**8.3.1** 食品生产许可（SC）：

1. 申请条件
2. 申请流程
3. 证书管理

**8.3.2** 出口备案：

1. 备案条件
2. 备案流程
3. 备案管理
