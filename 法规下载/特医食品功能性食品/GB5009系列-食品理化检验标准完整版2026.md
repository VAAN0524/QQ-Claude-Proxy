# GB 5009 食品理化检验标准（完整版 2026）

## 标准概述

**标准系列名称**：GB 5009 食品安全国家标准 食品理化检验
**发布机构**：国家卫生健康委员会、国家市场监督管理总局
**标准性质**：强制性国家标准（GB）
**适用范围**：食品中理化成分的检验方法

## 二、基础成分检验详解

### 2.1 GB 5009.3 水分的测定（2024版）

#### 2.1.1 测定方法

| 方法 | 适用范围 | 原理 |
|------|----------|------|
| **直接干燥法** | 谷类、豆类等 | 101-105°C 烘干至恒重 |
| **减压干燥法** | 糖、味精等 | 低温减压烘干 |
| **蒸馏法** | 含挥发性成分食品 | 蒸馏出水分 |
| **卡尔·费休法** | 各种食品 | 碘量滴定法 |

#### 2.1.2 直接干燥法

```
试样处理 → 称量瓶称重 → 101-105°C烘干2-4h → 冷却称重 →
再烘干1h → 冷却称重 → 直至恒重（两次差值≤2mg）→ 计算
```

#### 2.1.3 结果计算

```
X = (m1 - m2) / (m1 - m3) × 100

式中：
X - 水分含量，g/100g
m1 - 称量瓶和试样烘干前质量，g
m2 - 称量瓶和试样烘干后质量，g
m3 - 称量瓶质量，g
```

### 2.2 GB 5009.5 蛋白质的测定（2024版）

#### 2.2.1 测定方法

| 方法 | 适用范围 | 原理 |
|------|----------|------|
| **凯氏定氮法** | 各种食品 | 消化、蒸馏、滴定 |
| **分光光度法** | 大豆制品等 | 双缩脲反应 |
| **燃烧法** | 各种食品 | Dumas 法 |

#### 2.2.2 凯氏定氮法

```
试样消化（硫酸-硫酸铜-硫酸钾）→ 蒸馏（硼酸吸收）→
滴定（盐酸标准溶液）→ 计算
```

#### 2.2.3 结果计算

```
X = (V1 - V2) × c × 0.0140 / m × F × 100

式中：
X - 蛋白质含量，g/100g
V1 - 试样滴定消耗盐酸标准溶液体积，mL
V2 - 空白滴定消耗盐酸标准溶液体积，mL
c - 盐酸标准溶液浓度，mol/L
0.0140 - 氮的毫摩尔质量，g/mmol
m - 试样质量，g
F - 氢换算为蛋白质的系数（一般6.25）
```

### 2.3 GB 5009.6 脂肪的测定（2024版）

#### 2.3.1 测定方法

| 方法 | 适用范围 | 原理 |
|------|----------|------|
| **索氏抽提法** | 各类食品 | 乙醚或石油醚抽提 |
| **酸水解法** | 各类食品 | 酸水解后抽提 |
| **碱水解法** | 乳制品 | 碱水解后抽提 |
| **盖勃法** | 乳制品 | 专用仪器 |

#### 2.3.2 索氏抽提法

```
试样处理 → 滤纸筒包样 → 索氏抽提器抽提（6-8h）→
蒸发溶剂 → 烘干称重 → 计算
```

### 6.1 根据检验项目选择

| **高脂肪样品** | 脱肪处理 |
| **高蛋白样品** | 蛋白沉淀 |

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **检出限** | 能可靠检测出的最低浓度或含量 |
| **定量限** | 能准确定量的最低浓度或含量 |
| **回收率** | 加标样品测定值与理论值的比值 |
| **精密度** | 同一样品多次测定结果的一致程度 |
| **准确度** | 测定结果与真值的接近程度 |
| **不确定度** | 表征合理地赋予被测量之值的分散性 |
