# 保健食品功能评价方法

**发布机关**: 国家市场监督管理总局
**国家卫生健康委员会**
**施行日期**: 2023年版施行，2026年修订
**标准编号**: GB 16740-2024 保健食品标准

---

## 第一部分：总则

### 第一条【评价目的】
为规范保健食品功能评价，保证保健食品功能声称的科学性、真实性，制定本方法。

### 第二条【适用范围】
本方法适用于保健食品功能评价的全过程，包括功能学评价、安全性评价、功效成分评价等。

### 第三条【评价原则】
1. **科学性原则**: 评价方法应当科学、合理
2. **规范性原则**: 评价程序应当规范、标准
3. **可重复性原则**: 评价结果应当可重复
4. **伦理原则**: 人体试食试验应当符合伦理要求

## 第三部分：评价方法

3. 给药剂量设置
4. 实验周期确定
5. 观察指标选择
6. 统计分析方法

4. 给药方案确定
5. 观察指标选择

5. 致癌试验

4. 吸收利用度研究

## 第四部分：具体功能评价方法

### 第九条【增强免疫力功能评价】

#### 动物实验
1. **细胞免疫功能测定**
   - 迟发型变态反应试验
   - 淋巴细胞转化试验
   - 抗体生成细胞检测

2. **体液免疫功能测定**
   - 血清溶血素测定
   - 抗体生成细胞检测

3. **单核-巨噬细胞功能测定**
   - 碳粒廓清试验
   - 腹腔巨噬细胞吞噬鸡红细胞试验

4. **NK细胞活性测定**

#### 人体试食试验
1. 细胞免疫功能测定
2. 体液免疫功能测定
3. 非特异性免疫功能测定

### 第十条【抗氧化功能评价】

#### 动物实验
1. **过氧化脂质测定**
   - MDA含量测定
   - LPO含量测定

2. **抗氧化酶活性测定**
   - SOD活性测定
   - GSH-Px活性测定
   - CAT活性测定

3. **抗氧化物质测定**
   - GSH含量测定

#### 人体试食试验
1. 血清MDA含量测定
2. 血清SOD活性测定
3. 血清GSH-Px活性测定
4. 血清GSH含量测定

### 第十一条【辅助降血脂功能评价】

#### 动物实验
1. **血清TC测定**
2. **血清TG测定**
3. **血清HDL-C测定**
4. **血清LDL-C测定**

#### 人体试食试验
1. 血清TC测定
2. 血清TG测定
3. 血清HDL-C测定
4. 血清LDL-C测定

### 第十二条【辅助降血糖功能评价】

#### 动物实验
1. **空腹血糖测定**
2. **糖耐量试验**
3. **糖化血红蛋白测定**

#### 人体试食试验
1. 空腹血糖测定
2. 餐后2小时血糖测定
3. 糖化血红蛋白测定

### 第十三条【改善睡眠功能评价】

#### 动物实验
1. **直接睡眠法**
2. **延长戊巴比妥钠睡眠时间法**
3. **戊巴比妥钠阈下剂量催眠实验**
4. **巴比妥钠睡眠潜伏期实验**

#### 人体试食试验
1. **睡眠时间测定**
2. **入睡时间测定**
3. **睡眠质量评价**
4. **匹兹堡睡眠质量指数(PSQI)**

### 第十四条【缓解视疲劳功能评价】

#### 人体试食试验
1. **视力检查**
2. **屈光度检查**
3. **眼部症状检查**
4. **眼部不适症状评分**

## 第五部分：评价标准

### 第十五条【结果判定】
1. **阳性结果**: 动物实验或人体试食试验结果与阴性对照组比较，差异有显著性(P<0.05)
2. **阴性结果**: 动物实验或人体试食试验结果与阴性对照组比较，差异无显著性(P>0.05)
3. **可疑结果**: 部分指标阳性，部分指标阴性

### 第十六条【综合评价】
1. 动物实验和人体试食试验均阳性，判定功能成立
2. 动物实验阳性，人体试食试验阴性，需重新进行试验
3. 动物实验阴性，功能不成立

---

## 第六部分：质量控制

### 第十七条【实验条件】
1. 实验动物应当符合国家标准
2. 实验室应当符合GLP要求
3. 检测仪器应当经过校准
4. 检测试剂应当符合要求

### 第十八条【人员资质】
1. 实验人员应当经过专业培训
2. 实验人员应当熟悉实验方法
3. 数据分析应当由专业人员进行

### 第十九条【数据管理】
1. 原始数据应当完整保存
2. 数据应当真实可靠
3. 统计分析应当科学规范
4. 报告应当客观真实

---

## 第七部分：附则

### 第二十条【功能调整】
国家根据科学研究和监管需要，可以对保健食品功能进行调整。

### 第二十一条【方法修订】
国家根据科学技术发展，可以对本评价方法进行修订。

### 第二十二条【施行日期】
本方法自发布之日起施行。
