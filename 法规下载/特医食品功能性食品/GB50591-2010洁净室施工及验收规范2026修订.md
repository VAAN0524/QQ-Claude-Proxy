# GB 50591-2010 洁净室施工及验收规范（2026年修订版）

## 发布信息

- **主编部门**：中华人民共和国住房和城乡建设部
- **批准部门**：中华人民共和国住房和城乡建设部
- **施行日期**：2011年2月1日（2026年修订确认）
- **统一书号**：GB 50591-2010
- **标准类别**：国家标准

---

## 1 总则

**1.0.1** 为加强洁净室及相关受控环境（以下简称"洁净室"）的施工管理，统一施工验收标准，确保工程质量，制定本规范。

**1.0.2** 本规范适用于新建、改建和扩建的洁净室及相关受控环境的施工及验收。

**1.0.3** 洁净室的施工及验收除应符合本规范外，尚应符合国家现行有关标准的规定。

**1.0.4** 洁净室工程施工及验收应遵循以下原则：
1. 设计合理、技术先进、经济适用
2. 施工规范、质量可靠、安全环保
3. 验收科学、数据准确、结论可靠

---

## 2 术语

**2.0.1 洁净室**
空气中悬浮粒子浓度受控，其建造和使用方式使粒子进入、产生和滞留最少的房间。

**2.0.2 洁净度**
洁净室空气中悬浮粒子浓度的程度。

**2.0.3 单向流**
沿单一方向流动、流速均匀、没有涡流的气流。

**2.0.4 非单向流**
流速不均匀、有涡流的气流。

**2.0.5 洁净区**
洁净室内悬浮粒子浓度受控的特定区域。

**2.0.6 气流流型**
洁净室内空气流动的形态和分布。

**2.0.7 压差**
相邻区域之间的静压差。

**2.0.8 悬浮粒子**
悬浮在空气中的固体和液体粒子。

**2.0.9 微粒**
直径在0.1μm-5μm范围内的悬浮粒子。

**2.0.10 过滤器**
用于去除空气中悬浮粒子的装置。

## 3 基本规定

**3.0.1 洁净室施工应具备以下条件**：

1. 设计文件齐全并经审查通过
2. 施工方案编制完成并经审批
3. 施工人员经培训考核合格
4. 施工机具和检测仪器准备就绪
5. 材料和设备检验合格

**3.0.2 洁净室施工质量控制**：

| 质量控制阶段 | 控制内容 | 控制方法 |
|--------------|----------|----------|
| 施工准备 | 图纸会审、方案编制、人员培训 | 审查、检查、考核 |
| 材料设备 | 进场检验、储存保管 | 检验、核查、记录 |
| 施工过程 | 工序质量、隐蔽工程 | 巡检、专检、旁站 |
| 竣工验收 | 性能检测、综合评定 | 检测、验证、评审 |

**3.0.3 洁净室施工顺序**：

```
场地准备 → 结构施工 → 装修施工 → 设备安装 → 系统调试 → 性能验证 → 竣工验收
```

## 4 风管及配件

**4.1 材料要求**

**4.1.1** 风管材料应符合以下要求：

| 风管类型 | 材料要求 | 适用场景 |
|----------|----------|----------|
| 普通风管 | 镀锌钢板 | 低级别洁净区 |
| 高效送风管 | 不锈钢板 | 高级别洁净区 |
| 特殊风管 | 铝板、聚氨酯板 | 特殊要求场所 |

**4.1.2** 风管板材厚度应符合下表要求：

| 风管直径或长边尺寸 | 钢板厚度（mm） |
|---------------------|----------------|
| ≤200 | 0.5 |
| 201-500 | 0.75 |
| 501-1120 | 1.0 |
| 1121-2000 | 1.2 |
| ＞2000 | 1.5 |

**4.2 加工制作**

**4.2.1** 风管加工应符合以下要求：

1. 风管尺寸偏差：±2mm
2. 风管对角线偏差：≤3mm
3. 风管口平整度偏差：≤1mm/m
4. 法兰平整度偏差：≤2mm

**4.2.2** 风管加固应符合以下要求：

| 风管边长（mm） | 加固间距（mm） | 加固方式 |
|----------------|----------------|----------|
| ≤630 | — | 不需加固 |
| 631-1250 | ≤1000 | 角钢加固 |
| 1251-2000 | ≤800 | 角钢加固 |
| ＞2000 | ≤600 | 双角钢加固 |

**4.3 安装要求**

**4.3.1** 风管安装位置偏差：

| 项目 | 允许偏差（mm） |
|------|----------------|
| 坐标位置 | ±10 |
| 标高 | ±5 |
| 水平度 | 3/全长 |
| 垂直度 | 2/全长 |

**4.3.2** 风管支吊架安装：

| 风管边长（mm） | 支吊架间距（mm） |
|----------------|------------------|
| ≤400 | ≤4000 |
| 401-1000 | ≤3000 |
| 1001-2000 | ≤2500 |
| ＞2000 | ≤2000 |

## 5 风管配件

**5.1 风阀**

**5.1.1** 风阀安装应符合以下要求：

1. 风阀应安装在便于操作的位置
2. 风阀安装方向应与气流方向一致
3. 风阀启闭应灵活可靠
4. 风阀应有开度指示

**5.2 风口**

**5.2.1** 高效过滤器风口安装要求：

1. 安装平整度偏差：≤1mm/m
2. 与风管连接应严密
3. 边缘密封应完整
4. 框架应固定牢固

**5.3 静压箱**

**5.3.1** 静压箱制作安装要求：

1. 静压箱应采用镀锌钢板或不锈钢板制作
2. 静压箱内壁应光滑无毛刺
3. 静压箱导流板应均匀分布
4. 静压箱强度应能承受设计压力

## 6 空气处理设备

**6.1 空气处理机组**

**6.1.1** 空气处理机组安装要求：

| 项目 | 允许偏差 |
|------|----------|
| 坐标位置 | ±10mm |
| 标高 | ±5mm |
| 水平度 | 1/1000 |
| 垂直度 | 1/1000 |

**6.1.2** 空气处理机组检查项目：

1. 外观完好，无损伤变形
2. 内部清洁，无杂物
3. 风机转向正确
4. 表冷器、加热器完好
5. 过滤器安装正确
6. 减震装置安装牢固

**6.2 风机**

**6.2.1** 风机安装要求：

1. 风机叶轮旋转方向正确
2. 风机叶轮与机壳间隙均匀
3. 风机轴承温度≤70℃
4. 风机振幅≤0.05mm

**6.2.2** 风机试运转：

| 转速（rpm） | 运转时间（h） |
|-------------|----------------|
| ≤1500 | ≥2 |
| ＞1500 | ≥4 |

**6.3 过滤器**

**6.3.1** 初效过滤器**

1. 滤料：无纺布、玻璃纤维等
2. 效率：G1-G4级（计重效率）
3. 初阻力：≤50Pa
4. 容尘量：≥200g/m²

**6.3.2** 中效过滤器**

1. 滤料：无纺布、玻璃纤维等
2. 效率：F5-F9级（比色效率）
3. 初阻力：≤80Pa
4. 容尘量：≥150g/m²

**6.3.3** 高效过滤器（HEPA）

1. 滤料：玻璃纤维滤纸
2. 效率：H13-H14级（钠焰法≥99.97%）
3. 初阻力：≤220Pa@0.45m/s
4. 完整性测试：透过率≤0.01%

**6.3.4** 超高效过滤器（ULPA）

1. 滤料：玻璃纤维滤纸
2. 效率：U15-U17级（MPPS≥99.9995%）
3. 初阻力：≤250Pa@0.45m/s
4. 完整性测试：透过率≤0.001%

---

## 7 高效过滤器安装

**7.1 安装前检查**

**7.1.1** 高效过滤器安装前应检查：

1. 外观完好，无损伤
2. 框架无变形
3. 滤料无破损
4. 有出厂合格证和性能检测报告

**7.2 安装要求**

**7.2.1** 高效过滤器安装要求：

| 项目 | 允许偏差 |
|------|----------|
| 安装平整度 | ≤1mm/m |
| 对角线长度差 | ≤2mm |
| 安装缝隙 | ≤1mm |
| 密封条完整性 | 100% |

**7.2.2** 高效过滤器安装顺序：

1. 框架安装就位
2. 过滤器就位
3. 密封条安装
4. 压框固定
5. 完整性测试

**7.3 密封处理**

**7.3.1** 高效过滤器密封方法：

1. 槽边密封
2. 压紧密封
3. 液体密封

**7.3.2** 密封材料要求：

| 密封材料 | 适用场景 | 特点 |
|----------|----------|------|
| 海绵橡胶 | 一般洁净室 | 弹性好，密封可靠 |
| 硅橡胶 | 高温环境 | 耐高温，耐老化 |
| 凝胶密封 | 高级别洁净室 | 密封性能优异 |

## 8 消毒灭菌设备

**8.1 紫外线消毒装置**

**8.1.1** 紫外线消毒装置安装要求：

1. 安装位置：回风管或送风管
2. 紫外线强度：≥30W/cm²
3. 照射时间：≥3秒
4. 灯管寿命：≥8000小时

**8.2 臭氧消毒装置**

**8.2.1** 臭氧消毒装置要求：

1. 臭氧浓度：10-20mg/m³
2. 消毒时间：≥60分钟
3. 臭氧半衰期：约30分钟
4. 安全防护：防泄漏、自动停机

**8.3 气体过滤装置**

**8.3.1** 气体过滤装置要求：

1. 过滤精度：0.01μm
2. 过滤效率：≥99.999%
3. 压差：≤500Pa
4. 材质：316L不锈钢或PTFE

## 9 配电与照明

**9.1 配电系统**

**9.1.1** 洁净室配电要求：

1. 应采用TN-S接地系统
2. 插座应采用防水型
3. 线路应穿金属管保护
4. 设备应可靠接地

**9.2 照明系统**

**9.2.1** 照度要求：

| 区域 | 照度要求 |
|------|----------|
| 一般洁净区 | ≥300 Lux |
| 关键操作区 | ≥500 Lux |
| 精细操作区 | ≥800 Lux |
| 辅助区域 | ≥150 Lux |

**9.2.2** 照明灯具要求：

1. 应采用洁净室专用灯具
2. 灯具应密封安装
3. 灯具应易于清洁消毒
4. 灯具应防眩光

## 10 施工验收

**10.1 验收程序**

**10.1.1** 洁净室验收程序：

```
竣工验收准备 → 文件资料审查 → 外观检查 → 性能测试 → 综合评定
```

**10.2 文件资料审查**

**10.2.1** 应审查的文件资料：

1. 设计文件和变更文件
2. 施工组织设计和施工方案
3. 材料设备合格证明和检验报告
4. 施工记录和隐蔽工程记录
5. 调试记录和测试报告
6. 质量评定资料

**10.3 外观检查**

**10.3.1** 外观检查项目：

| 检查项目 | 检查内容 | 合格标准 |
|----------|----------|----------|
| 墙面 | 平整度、清洁度 | 无破损、无污染 |
| 地面 | 平整度、完整性 | 无开裂、无起鼓 |
| 吊顶 | 平整度、密封性 | 无松动、无渗漏 |
| 门窗 | 完整性、密封性 | 开闭灵活、密封良好 |
| 管道 | 安装质量、保温 | 布置合理、保温完好 |

**10.4 性能测试**

**10.4.1** 悬浮粒子测试**

| 洁净度级别 | 粒径（μm） | 最大允许浓度（个/m³） |
|------------|------------|------------------------|
| 5级（ISO 5） | ≥0.5 | 3520 |
| 6级（ISO 6） | ≥0.5 | 35200 |
| 7级（ISO 7） | ≥0.5 | 352000 |
| 8级（ISO 8） | ≥0.5 | 3520000 |

**10.4.2** 微生物测试**

| 洁净度级别 | 浮游菌（CFU/m³） | 沉降菌（CFU/皿·4h） |
|------------|------------------|---------------------|
| 5级 | ≤5 | ≤1 |
| 6级 | ≤100 | ≤3 |
| 7级 | ≤500 | ≤10 |
| 8级 | — | ≤25 |

**10.4.3** 压差测试**

| 区域 | 压差要求 |
|------|----------|
| 洁净区/非洁净区 | ≥10 Pa |
| 高级别/低级别 | ≥10 Pa |
| 相同级别相邻 | ≥5 Pa |
| 产尘房间 | 相对负压 |

**10.4.4** 风量测试**

1. 送风量：实测风量与设计风量偏差≤±10%
2. 回风量：实测风量与设计风量偏差≤±10%
3. 新风量：满足人员卫生要求和正压维持要求
4. 排风量：符合工艺和环保要求

**10.4.5** 换气次数测试**

| 洁净度级别 | 换气次数（次/h） |
|------------|-----------------|
| 5级 | 250-500 |
| 6级 | 150-250 |
| 7级 | 50-150 |
| 8级 | 10-50 |

**10.4.6** 气流流型测试**

1. 单向流：流向单一，流速均匀
2. 非单向流：涡流少，死区小
3. 气流组织：不产生交叉污染

**10.4.7** 温湿度测试**

| 区域 | 温度（℃） | 相对湿度（%） |
|------|-----------|----------------|
| 一般洁净区 | 18-26 | 45-65 |
| 特殊要求区域 | 按工艺要求 | 按工艺要求 |

**10.4.8** 噪声测试**

| 区域 | 噪声要求 |
|------|----------|
| 一般洁净区 | ≤65 dB(A) |
| 要求安静的区域 | ≤60 dB(A) |

**10.4.9** 照度测试**

见第9.2.1条。

**10.5 验收标准**

**10.5.1** 验收合格标准：

1. 文件资料齐全完整
2. 外观检查符合要求
3. 性能测试全部合格
4. 无严重质量问题

**10.5.2** 验收结论：

1. 合格：符合验收标准
2. 基本合格：有轻微缺陷，整改后合格
3. 不合格：存在严重缺陷，需返工后重新验收

## 11 维护管理

**11.1 日常维护**

**11.1.1** 日常维护项目：

| 维护项目 | 频率 | 维护内容 |
|----------|------|----------|
| 清洁消毒 | 每日 | 地面、墙面、设备表面 |
| 压差监测 | 每班 | 各区域压差 |
| 温湿度监测 | 每班 | 温度和相对湿度 |
| 粒子监测 | 按验证频率 | 悬浮粒子浓度 |
| 微生物监测 | 按验证频率 | 浮游菌、沉降菌 |

**11.2 定期维护**

**11.2.1** 过滤器更换周期：

| 过滤器类型 | 更换周期 | 更换条件 |
|------------|----------|----------|
| 初效过滤器 | 1-3个月 | 压差≥初阻力2倍 |
| 中效过滤器 | 6-12个月 | 压差≥初阻力2倍 |
| 高效过滤器 | 2-3年 | 压差≥初阻力2倍或泄漏超标 |

**11.2.2** 设备维护周期：

| 设备类型 | 维护周期 | 维护内容 |
|----------|----------|----------|
| 空气处理机组 | 每月 | 清洁、检查、润滑 |
| 风机 | 每月 | 检查、润滑、紧固 |
| 照明灯具 | 每月 | 清洁、检查 |
| 控制系统 | 每月 | 检查、校准 |
