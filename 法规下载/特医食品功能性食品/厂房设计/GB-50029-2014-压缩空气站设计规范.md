# 压缩空气站设计规范
## Design Code for Compressed Air Station

**标准号**: GB 50029-2014
**发布日期**: 2014年
**实施日期**: 2014年8月1日
**主管部门**: 中华人民共和国住房和城乡建设部
**归口单位**: 中国机械工业联合会

---

## 1 总则

### 1.0.1
为使压缩空气站设计能够保证安全生产、保护环境、节约能源、改善操作条件、提高供气质量、做到技术先进、经济合理，制定本规范。

### 1.0.2
本规范适用于新建、改建、扩建的压缩空气站设计。

### 1.0.3
压缩空气站设计除应符合本规范的规定外，尚应符合国家现行的有关标准规范的规定。

---

## 2 术语

### 2.0.1 压缩空气站
压缩空气的产生、处理、储存和分配的设施和场所。

### 2.0.2 工艺压缩空气
用于生产工艺过程的压缩空气。

### 2.0.3 仪表压缩空气
用于仪表控制系统的压缩空气。

### 2.0.4 无油压缩空气
不含油分的压缩空气。

### 2.0.5 干燥压缩空气
含水量低的压缩空气。

### 2.0.6 露点温度
空气中的水蒸气凝结成水的温度。

## 3 基本规定

### 3.0.1
压缩空气站的位置应根据下列因素综合确定：
1. 靠近负荷中心；
2. 交通运输方便；
3. 避开粉尘、有害气体等污染源；
4. 远离易燃易爆场所；
5. 有良好的自然通风条件。

### 3.0.2
压缩空气站的布置应符合下列要求：
1. 设备布置应合理、紧凑；
2. 便于操作、维护和检修；
3. 便于管道连接；
4. 符合防火、防爆要求。

### 3.0.3
压缩空气站的设备选型应根据下列因素确定：
1. 压缩空气的用量和压力；
2. 压缩空气的质量要求；
3. 供气的可靠性和安全性；
4. 能源消耗；
5. 环境保护要求。

---

## 4 工艺系统

### 4.1 压缩空气系统

### 4.1.1
压缩空气系统应根据用户对压缩空气的用量、压力和质量要求确定。

### 4.1.2
压缩空气系统应包括下列设备：
1. 空气压缩机；
2. 后冷却器；
3. 油水分离器；
4. 储气罐；
5. 干燥器；
6. 过滤器。

### 4.1.3
压缩空气系统应符合下列要求：
1. 系统应安全可靠；
2. 系统应便于操作和维护；
3. 系统应节约能源；
4. 系统应满足供气质量要求。

### 4.2 空气压缩机

### 4.2.1
空气压缩机的选型应根据下列因素确定：
1. 压缩空气的用量和压力；
2. 压缩空气的质量要求；
3. 能源消耗；
4. 环境保护要求；
5. 设备的可靠性和使用寿命。

### 4.2.2
空气压缩机的台数应根据下列因素确定：
1. 压缩空气的用量变化；
2. 供气的可靠性；
3. 设备的备用；
4. 能源消耗。

### 4.2.3
空气压缩机的布置应符合下列要求：
1. 设备之间应留有足够的通道；
2. 设备应便于操作、维护和检修；
3. 设备的布置应符合防火、防爆要求；
4. 设备的布置应满足通风要求。

### 4.3 干燥系统

### 4.3.1
压缩空气干燥系统的选型应根据下列因素确定：
1. 压缩空气的质量要求；
2. 压缩空气的用量；
3. 压缩空气的压力；
4. 能源消耗；
5. 环境保护要求。

### 4.3.2
压缩空气干燥系统应符合下列要求：
1. 系统应满足干燥要求；
2. 系统应安全可靠；
3. 系统应便于操作和维护；
4. 系统应节约能源。

### 4.4 过滤系统

### 4.4.1
压缩空气过滤系统的选型应根据下列因素确定：
1. 压缩空气的质量要求；
2. 压缩空气的用量；
3. 压缩空气的压力；
4. 过滤精度要求；
5. 过滤器更换周期。

### 4.4.2
压缩空气过滤系统应符合下列要求：
1. 系统应满足过滤要求；
2. 系统应安全可靠；
3. 系统应便于操作和维护；
4. 过滤器应便于更换。

## 5 设备布置

### 5.1 一般规定

### 5.1.1
压缩空气站的设备布置应符合下列要求：
1. 设备布置应合理、紧凑；
2. 便于操作、维护和检修；
3. 便于管道连接；
4. 符合防火、防爆要求。

### 5.1.2
压缩空气站的设备间距应符合下列要求：
1. 设备之间应留有足够的通道；
2. 通道宽度不应小于1.5m；
3. 设备与墙之间的距离不应小于1m。

### 5.2 空气压缩机布置

### 5.2.1
空气压缩机的布置应符合下列要求：
1. 空气压缩机应布置在通风良好的场所；
2. 空气压缩机应远离热源；
3. 空气压缩机应便于操作和维护；
4. 空气压缩机的布置应符合防火、防爆要求。

### 5.2.2
空气压缩机的基础应符合下列要求：
1. 基础应平整、坚实；
2. 基础应满足设备荷载要求；
3. 基础应有减振措施；
4. 基础应便于施工。

---

## 6 建筑结构

### 6.1 一般规定

### 6.1.1
压缩空气站的建筑应符合下列要求：
1. 建筑应满足设备布置要求；
2. 建筑应满足防火、防爆要求；
3. 建筑应满足通风要求；
4. 建筑应满足噪声控制要求。

### 6.1.2
压缩空气站的建筑结构应符合下列要求：
1. 结构应安全可靠；
2. 结构应满足设备荷载要求；
3. 结构应有减振措施；
4. 结构应便于施工。

### 6.2 防火防爆

### 6.2.1
压缩空气站的防火设计应符合下列要求：
1. 建筑的耐火等级不应低于二级；
2. 建筑应设置防火墙；
3. 建筑应设置安全出口；
4. 建筑应设置消防设施。

### 6.2.2
压缩空气站的防爆设计应符合下列要求：
1. 电气设备应采用防爆型；
2. 设备应有静电接地；
3. 管道应有防静电措施；
4. 设备应有安全泄压装置。

---

## 7 电气

### 7.1 一般规定

### 7.1.1
压缩空气站的电气设计应满足下列要求：
1. 供电应可靠；
2. 电气设备应安全可靠；
3. 电气设备应便于操作和维护；
4. 电气设备应符合防爆要求。

### 7.2 供配电

### 7.2.1
压缩空气站的供配电系统应符合下列要求：
1. 供电应可靠，应设置备用电源；
2. 配电系统应简单可靠；
3. 配电设备应便于操作和维护；
4. 配电设备应有安全保护措施。

### 7.2.2
压缩空气站的电气设备应符合下列要求：
1. 电气设备应选用节能型；
2. 电气设备应选用低噪声型；
3. 电气设备应选用防爆型；
4. 电气设备应有安全保护措施。

### 7.3 照明

### 7.3.1
压缩空气站的照明应符合下列要求：
1. 照度应满足操作要求；
2. 照度应均匀，避免眩光；
3. 照明灯具应易于清洁；
4. 照明灯具应便于维护。

### 7.3.2
压缩空气站的照度标准值应符合下表的规定：

| 房间名称 | 照度标准值 |
|---------|-----------|
| 压缩机机房 | 150～200 lx |
| 控制室 | 300～500 lx |
| 配电室 | 200～300 lx |
| 维修间 | 200～300 lx |

---

## 8 给水排水

### 8.1 一般规定

### 8.1.1
压缩空气站的给水排水设计应符合下列要求：
1. 满足生产、生活用水的需要；
2. 满足设备冷却用水的需要；
3. 排水系统应通畅；
4. 水质应符合要求。

### 8.2 给水

### 8.2.1
压缩空气站的给水系统应符合下列要求：
1. 给水系统应满足设备用水量要求；
2. 给水系统应满足水压要求；
3. 给水系统应便于维护；
4. 给水系统应有防冻措施。

### 8.3 排水

### 8.3.1
压缩空气站的排水系统应符合下列要求：
1. 排水系统应通畅，便于维护；
2. 排水管道应有防冻措施；
3. 排水管道应有防倒灌措施；
4. 排水应进行处理。

---

## 9 采暖通风

### 9.1 采暖

### 9.1.1
压缩空气站的采暖设计应符合下列要求：
1. 采暖系统应满足采暖要求；
2. 采暖系统应便于维护；
3. 采暖系统应节能；
4. 采暖系统应安全可靠。

### 9.2 通风

### 9.2.1
压缩空气站的通风设计应符合下列要求：
1. 通风系统应满足通风要求；
2. 通风系统应便于维护；
3. 通风系统应节能；
4. 通风系统应安全可靠。

### 9.2.2
压缩空气站的通风量应符合下列要求：
1. 应满足设备散热要求；
2. 应满足空气质量要求；
3. 应满足防爆要求；
4. 应满足节能要求。

---

## 10 节能

### 10.1 一般规定

### 10.1.1
压缩空气站的节能设计应符合下列要求：
1. 应选用节能型设备；
2. 应优化系统设计；
3. 应采用节能控制措施；
4. 应加强能源管理。

### 10.2 设备选型

### 10.2.1
压缩空气站的设备选型应符合下列节能要求：
1. 空气压缩机应选用节能型；
2. 干燥器应选用节能型；
3. 过滤器应选用低阻力型；
4. 电动机应选用高效型。

### 10.3 系统设计

### 10.3.1
压缩空气站的系统设计应符合下列节能要求：
1. 系统应优化设计，减少压力损失；
2. 系统应采用余热回收措施；
3. 系统应采用变频控制措施；
4. 系统应采用智能控制措施。

---

**备注**：本规范适用于压缩空气站的设计，特医食品生产企业的压缩空气系统可参考执行。
