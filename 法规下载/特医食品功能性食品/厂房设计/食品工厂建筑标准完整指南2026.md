# 食品工厂厂房建筑标准完整指南（2026版）

## 文档说明

**创建日期**: 2026-02-28
**适用范围**: 特医食品、功能性食品工厂厂房设计、施工、验收
**法规依据**: GB 50073、GB 50591、GB 50243、GB 50303等建筑标准

---

## 一、洁净厂房设计规范（GB 50073-2013）

### 1.1 适用范围

本规范适用于新建、改建、扩建的洁净厂房设计。

**对特医食品的参考价值**：
- 特医食品生产车间洁净度设计
- 无菌工艺车间设计
- 洁净室验收标准

### 1.2 洁净度分级

| 等级 | ≥0.5μm粒子数/m³ | ≥5μm粒子数/m³ | 微生物要求 | 适用区域 |
|------|----------------|---------------|-----------|----------|
| 1级 | ≤10 | - | - | - |
| 2级 | ≤100 | - | - | - |
| 3级 | ≤1000 | - | - | - |
| 4级 | ≤10000 | - | - | - |
| 5级 | ≤100000 | - | - | 高洁净度 |
| 6级 | ≤1000000 | ≤1000 | - | 一般洁净度 |
| 7级 | ≤10000000 | ≤10000 | - | 低洁净度 |
| 8级 | ≤100000000 | ≤100000 | - | - |
| 9级 | - | - | - | - |

**注**: 特医食品生产一般采用6-8级（对应GMP的C、D级）

### 1.3 厂址选择

#### 1.3.1 环境要求

1. **大气环境**：
   - 远离污染源
   - 空气质量良好
   - 远离机场、主干道

2. **周边环境**：
   - 无粉尘、烟雾
   - 无有害气体
   - 无强电磁场

3. **基础设施**：
   - 供水供电可靠
   - 排水通畅
   - 交通便利

#### 1.3.2 规划要求

1. **厂区布局**：
   - 生产区、辅助区、行政区分开
   - 人流、物流分开
   - 洁污分流

2. **绿化要求**：
   - 厂区绿化率≥20%
   - 种植不产生花絮的植物
   - 草坪为主

### 1.4 建筑设计

#### 1.4.1 平面布置

1. **分区原则**：
   - 生产区按工艺流程布置
   - 洁净区与非洁净区分开
   - 不同洁净级别分开

2. **人流组织**：
   - 更衣室→洁净区
   - 一更→二更→缓冲→洁净区
   - 避免交叉往返

3. **物流组织**：
   - 设专用出入口
   - 设传递窗
   - 设气闸室

#### 1.4.2 建筑构造

1. **墙体**：
   - 彩钢板/不锈钢板
   - 表面光滑、易清洁
   - 圆弧转角（R≥50mm）
   - 无死角、无积尘

2. **地面**：
   - 环氧树脂自流坪
   - 防滑、耐腐蚀
   - 无裂缝、不起砂
   - 地漏加防护罩

3. **吊顶**：
   - 彩钢板吊顶
   - 密封良好
   - 设检修口
   - 灯具嵌入式

4. **门窗**：
   - 密闭性好
   - 玻璃固定
   - 窗台斜坡
   - 门设闭门器

### 1.5 HVAC系统

#### 1.5.1 系统设计

1. **送风量**：
   | 洁净度级别 | 换气次数 |
   |-----------|----------|
   | 6级 | 50-80次/h |
   | 7级 | 15-25次/h |
   | 8级 | 10-15次/h |

2. **气流组织**：
   - 单向流（层流）：高级别
   - 非单向流（湍流）：低级别
   - 推流：避免死角

3. **压差控制**：
   - 不同级别≥5Pa
   - 洁净区与非洁净区≥10Pa
   - 产尘区保持负压

#### 1.5.2 过滤系统

1. **三级过滤**：
   - 初效（G3-G4）：新风
   - 中效（F5-F9）：回风
   - 高效（H13-H14）：末端

2. **HEPA过滤器**：
   - 效率：H13（99.97%@0.3μm）
   - 完整性测试：每年1次
   - 更换条件：阻力达到初阻力的2倍

#### 1.5.3 温湿度控制

1. **温湿度要求**：
   - 温度：18-26℃
   - 湿度：45-65%

2. **控制方式**：
   - 夏季：降温除湿
   - 冬季：加热加湿
   - 过渡季：全新风

### 1.6 照明设计

1. **照度要求**：
   - 主要工作室：≥300lx
   - 辅助工作室：≥200lx
   - 走廊：≥150lx

2. **灯具要求**：
   - 嵌入式安装
   - 密封良好
   - 易于清洁
   - 防眩光

## 二、洁净室施工及验收规范（GB 50591-2010）

### 2.1 施工要求

#### 2.1.1 建筑结构

1. **彩钢板安装**：
   - 板缝密封
   - 板块垂直
   - 接缝平整
   - 表面无损伤

2. **地面施工**：
   - 基层处理
   - 自流平施工
   - 厚度均匀
   - 无起泡开裂

3. **门窗安装**：
   - 密封条完整
   - 开启灵活
   - 锁具完好
   - 玻璃清洁

#### 2.1.2 风管系统

1. **风管制作**：
   - 材质：镀锌钢板/不锈钢
   - 清洁：无油污
   - 密封：法兰密封

2. **风管安装**：
   - 坡度：1-3‰
   - 支架：间距≤3m
   - 法兰：对接平整

3. **保温**：
   - 材料：橡塑/玻璃棉
   - 厚度：≥25mm
   - 保护层：铝箔

#### 2.1.3 设备安装

1. **空调机组**：
   - 基础平整
   - 减振安装
   - 接管正确
   - 便于检修

2. **高效过滤器**：
   - 框架密封
   * 压紧：均匀
   * 检漏：PAO扫描
   * 标记：合格证

### 2.2 验收要求

#### 2.2.1 外观检查

1. **表面**：
   - 光滑清洁
   - 无锈蚀
   - 无损伤

2. **接缝**：
   - 密封良好
   - 无可见缝隙
   - 无渗漏

3. **标识**：
   - 清晰完整
   - 规范统一

#### 2.2.2 性能检测

1. **洁净度**：
   - 粒子计数
   - 微生物检测
   - 符合设计级别

2. **压差**：
   - 所有房间压差
   - 符合设计要求
   - 稳定可控

3. **温湿度**：
   - 连续监测
   - 符合要求
   - 分布均匀

4. **照度**：
   - 工作面照度
   - 符合标准

5. **噪声**：
   - ≤65dB(A)
   - 符合要求

## 三、通风与空调工程施工质量验收规范（GB 50243-2016）

### 3.1 风管制作

#### 3.1.1 材料要求

1. **金属风管**：
   - 镀锌钢板
   - 厚度符合规范
   | 直径D | 板厚 |
   |-------|------|
   | D≤320 | 0.5mm |
   | 320<D≤630 | 0.6mm |
   | 630<D≤1000 | 0.75mm |
   | 1000<D≤2000 | 1.0mm |

2. **非金属风管**：
   - 玻璃钢/无机玻璃钢
   - 厚度≥5mm
   - 阻燃性能

#### 3.1.2 制作要求

1. **咬口**：
   - 平咬口/联合角咬口
   - 严密无缝隙
   - 无毛刺

2. **法兰**：
   - 角钢法兰
   - 钻孔准确
   - 密封面平整

### 3.2 风管安装

#### 3.2.1 安装要求

1. **水平度**：
   - 每3m偏差≤3mm
   - 总偏差≤20mm

2. **垂直度**：
   - 每3m偏差≤2mm
   - 总偏差≤20mm

3. **支吊架**：
   - 间距：水平≤3m，垂直≤4m
   - 牢固可靠
   * 防晃动

#### 3.2.2 风阀安装

1. **位置**：
   - 便于操作
   - 检修空间
   * 手动阀柄朝下

2. **方向**：
   - 标识箭头与气流一致
   - 开闭灵活

### 3.3 通风空调设备

#### 3.3.1 空调机组

1. **安装**：
   - 基础平整度≤2mm/m
   - 减振安装
   - 排水管坡度≥1%

2. **接管**：
   - 软连接
   - 无应力
   - 密封良好

#### 3.3.2 风机

1. **安装**：
   - 基础稳固
   * 减振
   * 对中良好

2. **试运行**：
   - 运行平稳
   - 无异常噪声
   * 电流正常

### 3.4 防腐与保温

#### 3.4.1 防腐

1. **除锈**：
   - Sa2.5级
   - 表面清洁

2. **涂漆**：
   - 底漆：防锈漆
   * 面漆：调和漆
   * 遍数：2遍

#### 3.4.2 保温

1. **材料**：
   - 橡塑/玻璃棉
   - 厚度≥25mm

2. **施工**：
   - 接缝严密
   - 保护层完整
   * 无可见缝隙

### 3.5 系统调试

#### 3.5.1 单机试运转

1. **风机**：
   - 运行时间≥2h
   - 滑动轴承温度≤70℃
   * 滚动轴承温度≤80℃

2. **水泵**：
   - 运行时间≥2h
   * 轴承温度≤80℃

#### 3.5.2 系统调试

1. **风量平衡**：
   - 各风口风量
   * 偏差≤15%

2. **温湿度**：
   - 符合设计要求

3. **压差**：
   - 符合设计要求

## 四、建筑电气工程施工质量验收规范（GB 50303-2015）

### 4.1 配线工程

#### 4.1.1 导线敷设

1. **材料**：
   - 铜芯绝缘线
   * 截面积符合要求

2. **敷设**：
   - 横平竖直
   * 固定点间距≤0.5m
   * 接头在接线盒内

3. **颜色**：
   - 相线：黄/绿/红
   * 零线：淡蓝
   * PE线：黄绿双色

#### 4.1.2 线管敷设

1. **金属线管**：
   - 镀锌钢管
   * 弯曲半径≥6D
   * 管口光滑

2. **PVC线管**：
   - 难燃型
   * 弯曲半径≥6D
   * 接头胶粘牢固

### 4.2 电气设备安装

#### 4.2.1 配电箱

1. **安装**：
   - 垂直度偏差≤1.5mm/m
   * 距地高度：明装1.2m，暗装1.4m

2. **接线**：
   - 排列整齐
   * 编号清晰
   * 压接牢固

#### 4.2.2 照明

1. **灯具**：
   - 安装牢固
   * 位置正确

2. **开关插座**：
   - 距地：开关1.3m，插座0.3m
   * 同一房间高度差≤5mm

### 4.3 防雷接地

#### 4.3.1 接地装置

1. **接地体**：
   - 角钢/扁钢
   * 埋深≥0.6m
   * 焊接牢固

2. **接地电阻**：
   - 共用接地：≤1Ω
   * 独立接地：≤4Ω

### 4.4 电气检测

1. **绝缘电阻**：
   - 各回路≥0.5MΩ

2. **通电试运行**：
   - 连续运行≥8h
   * 无故障

## 五、建筑给水排水设计标准（GB 50015-2019）

### 5.1 给水系统

#### 5.1.1 水质要求

1. **生活饮用水**：
   - 符合GB 5749
   * 定期检测

2. **生产用水**：
   - 根据工艺要求
   * 处理后使用

#### 5.1.2 管道材料

1. **生活给水**：
   - PPR管/不锈钢管
   * 热熔连接

2. **生产给水**：
   - 不锈钢管
   * 氩弧焊接

### 5.2 排水系统

#### 5.2.1 排水方式

1. **污废分流**：
   - 生活污水
   * 生产废水
   * 雨水

2. **排放要求**：
   - 达标排放
   * 或处理回用

#### 5.2.2 管道设计

1. **坡度**：
   - DN50：i=0.035
   * DN75：i=0.025
   * DN100：i=0.020

2. **通气管**：
   - 伸顶通气
   * 专用通气

### 5.3 消防给水

1. **室外**：
   - 消火栓：间距≤120m
   * 水泵接合器

2. **室内**：
   - 消火栓：间距≤30m
   * 自动喷淋：按规范

## 六、建筑设计防火规范（GB 50016-2014，2018年版）

### 6.1 厂房分类

| 类别 | 火灾危险性特征 | 示例 |
|------|----------------|------|
| 甲 | 易燃易爆 | 溶剂库 |
| 乙 | 可燃气体 | - |
| 丙 | 可燃固体 | 原料库 |
| 丁 | 难燃 | 一般车间 |
| 戊 | 不燃 | 办公楼 |

**特医食品工厂一般为丙、丁类**

### 6.2 防火分区

| 耐火等级 | 最多层数 | 最大允许面积 | 备注 |
|----------|----------|-------------|------|
| 一级 | 不限 | 8000㎡ | 设自喷加倍 |
| 二级 | 不限 | 6000㎡ | 设自喷加倍 |
| 三级 | 2层 | 3000㎡ | - |
| 四级 | 1层 | 1500㎡ | - |

### 6.3 安全疏散

1. **出口数量**：
   - 每防火分区≥2个
   * 疏散距离≤80m（设自喷加倍）

2. **疏散宽度**：
   - 百人宽度≥0.6m
   * 门宽≥0.9m

### 6.4 消防设施

1. **消火栓**：
   - 室内：水量10L/s，火灾时间2h
   * 室外：水量30L/s，火灾时间2h

2. **自动喷淋**：
   - 火灾危险等级：中危险级I级
   * 喷水强度：6L/min·m²
   * 作用面积：160㎡

3. **灭火器**：
   - A类火灾
   * 严重危险级
   * 保护距离≤15m

## 七、采暖通风与空气调节设计规范（GB 50019-2015）

### 7.1 采暖

1. **热媒**：
   - 95/70℃热水
   * 蒸汽

2. **散热器**：
   - 钢制/铜铝复合
   * 明装

### 7.2 通风

1. **全面通风**：
   - 换气次数
   * 车间：6-8次/h
   * 辅助：3-4次/h

2. **局部排风**：
   - 热源、尘源处
   * 罩口风速≥0.5m/s

### 7.3 空气调节

1. **负荷计算**：
   - 冷负荷
   * 热负荷
   * 湿负荷

2. **系统选择**：
   - 全空气系统
   * 空气-水系统
   * 制冷剂系统

## 八、压缩空气站设计规范（GB 50029-2014）

### 8.1 站房布置

1. **位置**：
   - 靠近负荷中心
   * 远离污染源

2. **布置**：
   - 设备间距≥1.5m
   * 检修通道≥1.2m

### 8.2 设备选型

1. **空压机**：
   - 螺杆式
   * 无油润滑

2. **储气罐**：
   - 容量：5-10min用气量
   * 压力：0.7-0.8MPa

### 8.3 气体处理

1. **干燥**：
   - 冷冻干燥
   * 露点≤-20℃

2. **过滤**：
   - 初效：3μm
   * 精效：0.01μm

## 九、应用指南

### 对特医食品工厂的设计建议

#### 9.1 洁净车间

1. **洁净度**：
   - 一般区域：8级（D级）
   - 关键区域：7级（C级）
   * 无菌操作：6级（B级背景）

2. **压差**：
   - 洁净区→一般区：≥10Pa
   * 高级别→低级别：≥5Pa
   * 产尘区：负压

#### 9.2 功能分区

1. **生产区**：
   - 原料处理
   * 配料
   * 包装
   * 成品

2. **辅助区**：
   - 更衣
   * 缓冲
   * 仓储
   * 质检

#### 9.3 暖通空调

1. **系统**：
   - 独立系统
   * 分区控制

2. **送风**：
   - 顶送侧下回
   * 均匀分布

#### 9.4 电气设计

1. **电源**：
   - 双路供电
   * 备用发电机

2. **照明**：
   - 洁净区：≥300lx
   * 一般区：≥200lx

#### 9.5 给排水

1. **给水**：
   - 生活用水
   * 生产用水
   * 纯化水

2. **排水**：
   - 污废分流
   * 清污分流

#### 9.6 消防

1. **系统**：
   - 消火栓
   * 自动喷淋
   * 灭火器

2. **疏散**：
   - 双出口
   * 距离控制
