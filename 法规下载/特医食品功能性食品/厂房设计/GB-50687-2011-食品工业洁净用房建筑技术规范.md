# 食品工业洁净用房建筑技术规范
## Technical Code for Construction of Clean Room for Food Industry

**标准号**: GB 50687-2011
**发布日期**: 2011年
**实施日期**: 2012年
**主管部门**: 中华人民共和国住房和城乡建设部
**归口单位**: 中国食品发酵工业研究院

---

## 1 总则

### 1.0.1
为在食品工业洁净用房设计和施工中贯彻国家有关方针政策，做到技术先进、经济合理、安全适用、确保质量，制定本规范。

### 1.0.2
本规范适用于食品工业洁净用房的设计、施工和验收。

### 1.0.3
食品工业洁净用房的设计和施工应符合下列要求：
1. 满足食品生产工艺要求；
2. 满足食品卫生要求；
3. 满足环境保护要求；
4. 满足节能要求。

### 1.0.4
食品工业洁净用房的设计和施工除应符合本规范的规定外，尚应符合国家现行有关标准规范的规定。

---

## 2 术语

### 2.0.1 食品工业洁净用房
空气中悬浮粒子浓度受控，用于食品生产的房间。

### 2.0.2 洁净度
根据单位体积空气中不同粒径粒子的数量，将洁净用房划分为不同的级别。

### 2.0.3 洁净区
洁净用房内部的空气悬浮粒子浓度受控的特定空间。

### 2.0.4 气闸室
设置在洁净用房出入口，用于分隔不同洁净级别区域，防止交叉污染的房间。

### 2.0.5 传递窗
设置在洁净用房隔墙上，用于传递物料和物品，防止交叉污染的装置。

### 2.0.6 人员净化用室
用于人员进入洁净用房前进行净化的房间，包括更衣室、淋浴室、风淋室等。

### 2.0.7 物料净化用室
用于物料进入洁净用房前进行净化的房间，包括缓冲间、气闸室等。

### 2.0.8 污染
任何影响食品质量、安全的不利物质或因素。

### 2.0.9 交叉污染
不同产品、不同工序之间的相互污染。

## 3 基本规定

### 3.1 洁净用房分级

### 3.1.1
食品工业洁净用房应根据食品生产工艺要求、产品特性和卫生要求，合理确定洁净度级别。

### 3.1.2
食品工业洁净用房的洁净度级别可分为以下四级：
- **Ⅰ级**：高洁净度级别，用于高风险食品的生产
- **Ⅱ级**：中高洁净度级别，用于较高风险食品的生产
- **Ⅲ级**：中等洁净度级别，用于一般风险食品的生产
- **Ⅳ级**：一般洁净度级别，用于低风险食品的生产

### 3.1.3
不同洁净度级别的要求应符合下表的规定：

| 洁净度级别 | ≥0.5μm悬浮粒子最大允许数/立方米 | ≥5μm悬浮粒子最大允许数/立方米 |
|-----------|-------------------------------|------------------------------|
| Ⅰ级 | ≤3520 | ≤20 |
| Ⅱ级 | ≤35200 | ≤293 |
| Ⅲ级 | ≤352000 | ≤2930 |
| Ⅳ级 | ≤3520000 | ≤29300 |

### 3.2 选址和总平面布置

### 3.2.1
食品工业洁净用房的选址应符合下列要求：
1. 应远离污染源；
2. 应远离有害气体、烟尘等污染源；
3. 应交通便利，便于物流运输；
4. 应有良好的给排水条件和能源供应条件。

### 3.2.2
食品工业洁净用房的总平面布置应符合下列要求：
1. 功能分区明确，人流、物流分开；
2. 生产区、辅助区、行政区应分开布置；
3. 洁净区与非洁净区应分开布置；
4. 应有足够的绿化隔离带。

### 3.3 工艺布局

### 3.3.1
食品工业洁净用房的工艺布局应符合下列要求：
1. 工艺流程应合理、顺畅；
2. 应避免人流、物流交叉；
3. 应避免交叉污染；
4. 应便于清洁和消毒。

### 3.3.2
食品工业洁净用房的工艺布局应符合下列原则：
1. 原料、半成品、成品应分开；
2. 生、熟食品应分开；
3. 清洁区与污染区应分开；
4. 不同洁净度级别的区域应分开。

## 4 建筑

### 4.1 一般规定

### 4.1.1
食品工业洁净用房的建筑设计应符合下列要求：
1. 满足生产工艺要求；
2. 满足卫生要求；
3. 满足防火要求；
4. 满足节能要求。

### 4.1.2
食品工业洁净用房的建筑围护结构应符合下列要求：
1. 气密性好；
2. 保温性能好；
3. 不易产生尘粒；
4. 易于清洁和消毒。

### 4.2 围护结构

### 4.2.1
食品工业洁净用房的墙、顶应符合下列要求：
1. 表面平整、光滑、不起尘；
2. 防霉、防潮、防菌；
3. 耐腐蚀、耐清洗；
4. 便于清洁和消毒。

### 4.2.2
食品工业洁净用房的地面应符合下列要求：
1. 表面平整、耐磨、耐腐蚀；
2. 防滑、防渗漏；
3. 不易产生尘粒；
4. 便于清洁和消毒。

### 4.3 门窗

### 4.3.1
食品工业洁净用房的门应符合下列要求：
1. 气密性好；
2. 易于清洁和消毒；
3. 不易产生尘粒；
4. 向洁净度级别高或压力高的方向开启。

### 4.3.2
食品工业洁净用房的窗应符合下列要求：
1. 气密性好；
2. 易于清洁和消毒；
3. 不易产生尘粒；
4. 与墙壁齐平。

### 4.4 人员净化用室

### 4.4.1
食品工业洁净用房应设置人员净化用室，并应符合下列要求：
1. 人员净化用室应包括更衣室、淋浴室、风淋室等；
2. 人员净化用室的设置应符合洁净度级别的要求；
3. 人员净化用室的面积应满足人员净化的需要。

### 4.4.2
食品工业洁净用房的人员净化用室应符合下列要求：
1. 更衣室应按"一更（脱外衣）—二更（穿洁净服）"的顺序设置；
2. 淋浴室应设置在更衣室之前；
3. 风淋室应设置在进入洁净区之前。

### 4.5 物料净化用室

### 4.5.1
食品工业洁净用房应设置物料净化用室，并应符合下列要求：
1. 物料净化用室应包括缓冲间、气闸室、传递窗等；
2. 物料净化用室的设置应符合洁净度级别的要求；
3. 物料净化用室的面积应满足物料净化的需要。

### 4.5.2
食品工业洁净用房的物料净化用室应符合下列要求：
1. 物料净化用室应设置在物料入口处；
2. 物料净化用室应与生产区相连；
3. 物料净化用室应便于物料的清洁和消毒。

## 5 空气净化

### 5.1 一般规定

### 5.1.1
食品工业洁净用房的空气净化系统应根据洁净度级别、生产工艺要求和室外气象条件等因素确定。

### 5.1.2
食品工业洁净用房的空气净化系统应符合下列要求：
1. 保证室内要求的空气洁净度；
2. 保证室内要求的温湿度；
3. 保证室内要求的压力和压力分布；
4. 保证室内新鲜空气量；
5. 排除室内产生的污染物。

### 5.2 气流组织

### 5.2.1
食品工业洁净用房的气流组织应符合下列要求：
1. 气流组织应满足工艺对空气洁净度的要求；
2. 气流组织应均匀，避免死角和涡流区；
3. 气流组织应有利于排除室内产生的污染物。

### 5.2.2
食品工业洁净用房的送风方式应符合下列要求：
1. 洁净度级别高的区域宜采用单向流或混合流；
2. 洁净度级别低的区域可采用非单向流；
3. 送风口宜布置在工艺设备的上方。

### 5.3 压差控制

### 5.3.1
食品工业洁净用房应保持一定的压差，并应符合下列要求：
1. 洁净区与非洁净区之间的压差不应小于10Pa；
2. 相邻不同级别洁净区之间的压差不应小于5Pa；
3. 高污染风险的区域应保持相对负压。

### 5.3.2
食品工业洁净用房的压差控制应符合下列要求：
1. 应采用压差控制装置；
2. 压差控制装置应可靠、灵敏、便于调节；
3. 压差控制装置应安装在便于观察和维护的位置。

## 6 给水排水

### 6.1 一般规定

### 6.1.1
食品工业洁净用房的给水排水设计应符合下列要求：
1. 满足生产、生活、消防用水的需要；
2. 水质应符合食品卫生要求；
3. 管道应便于清洁和消毒；
4. 排水系统应通畅。

### 6.1.2
食品工业洁净用房的给水排水管道应敷设在技术夹层或技术夹道内。

### 6.2 给水

### 6.2.1
食品工业洁净用房的用水水质应根据工艺要求确定，并应符合下列要求：
1. 生产用水应符合《生活饮用水卫生标准》GB 5749的有关规定；
2. 特殊工艺用水应符合相关标准的要求。

### 6.2.2
食品工业洁净用房的给水系统应符合下列要求：
1. 给水管道应耐腐蚀、不易产生污染物；
2. 给水管道应便于清洁和消毒；
3. 给水管道应有明显的标识。

### 6.3 排水

### 6.3.1
食品工业洁净用房的排水系统应符合下列要求：
1. 排水系统应通畅，便于清洁和维护；
2. 排水管道应有水封，防止臭气倒灌；
3. 排水管道应有防倒灌的装置。

### 6.3.2
食品工业洁净用房的地漏应符合下列要求：
1. 应采用洁净室专用地漏；
2. 应有水封，水封深度不应小于50mm；
3. 应便于清洁和消毒；
4. 应有防止倒灌的装置。

## 7 电气

### 7.1 一般规定

### 7.1.1
食品工业洁净用房的电气设计应满足生产、照明、空调、自动控制等的用电需要。

### 7.1.2
食品工业洁净用房的电气设计应符合下列要求：
1. 配电系统应可靠；
2. 照明应满足生产需要；
3. 防静电措施应有效；
4. 接地系统应可靠。

### 7.2 照明

### 7.2.1
食品工业洁净用房的照明应符合下列要求：
1. 照度应满足工艺要求；
2. 照度应均匀，避免眩光；
3. 照明灯具应易于清洁，不易产生尘粒；
4. 照明灯具宜采用吸顶式或嵌入式安装。

### 7.2.2
食品工业洁净用房的照度标准值应符合下表的规定：

| 房间名称 | 照度标准值 |
|---------|-----------|
| 生产车间 | 300～500 lx |
| 包装车间 | 300～500 lx |
| 检验室 | 500～750 lx |
| 辅助房间 | 200～300 lx |
| 走廊 | 150～200 lx |

## 8 验收

### 8.1 施工验收

### 8.1.1
食品工业洁净用房的施工验收应包括下列内容：
1. 材料和设备的验收；
2. 隐蔽工程的验收；
3. 分项工程的验收；
4. 分部工程的验收。

### 8.1.2
材料和设备的验收应符合下列规定：
1. 材料和设备的规格、型号应符合设计要求；
2. 材料和设备应有合格证明文件；
3. 进口材料和设备应有商检证明。

### 8.2 综合性能验收

### 8.2.1
食品工业洁净用房的综合性能验收应包括下列内容：
1. 空气洁净度检测；
2. 静压差检测；
3. 风速或风量检测；
4. 温湿度检测；
5. 照度检测；
6. 噪声检测；
7. 微生物检测。

### 8.2.2
空气洁净度的检测应符合下列规定：
1. 应采用粒子计数器进行检测；
2. 采样点应均匀布置；
3. 每个采样点的采样量应不小于1m³；
4. 检测应在静态或动态下进行。

### 8.2.3
微生物的检测应符合下列规定：
1. 应采用沉降菌法或浮游菌法进行检测；
2. 检测应在静态或动态下进行；
3. 检测结果应符合设计要求。

---

**备注**：本规范适用于食品工业洁净用房的设计、施工和验收，特医食品厂房建设可参考执行。
