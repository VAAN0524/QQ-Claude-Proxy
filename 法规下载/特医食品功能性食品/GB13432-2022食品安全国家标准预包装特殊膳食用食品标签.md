# GB 13432-2022 食品安全国家标准 预包装特殊膳食用食品标签

**标准编号**: GB 13432-2022
**实施日期**: 2023年7月1日
**替代标准**: GB 13432-2013
**发布部门**: 国家卫生健康委员会 国家市场监督管理总局

---

## 1. 范围

本标准适用于预包装特殊膳食用食品的标签。

**特殊膳食用食品**包括：
1. 婴幼儿配方食品
2. 婴幼儿辅助食品
3. 特殊医学用途配方食品
4. 除上述类别外的其他特殊膳食用食品

---

## 2. 术语和定义

### 2.1 特殊膳食用食品
为满足特殊的身体或生理状况，或满足疾病、紊乱等状态下的特殊膳食需求，专门加工或配方的食品。

### 2.2 营养素
食品中具有特定生理作用，能维持机体生长、发育、活动、繁殖以及正常代谢所需的物质。

### 2.3 推荐摄入量
基于科学证据，按照特定人群生理需要制定的摄入量标准。

---

## 3. 基本要求

### 3.1 强制标示内容

特殊膳食用食品必须标示：

| 标示项目 | 要求 |
|----------|------|
| 食品名称 | 必须标示"特殊膳食用食品"属性 |
| 配料表 | 按递减顺序排列 |
| 净含量 | 标明净含量/规格 |
| 生产者/经销者 | 名称、地址、联系方式 |
| 生产日期/保质期 | 清晰标示 |
| 贮存条件 | 明确贮存要求 |
| 食品生产许可证编号 | SC编号 |
| 产品标准代号 | 相应产品标准号 |
| 营养成分表 | **强制标示** |
| 适用人群 | **强制标示** |
| 食用方法 | **强制标示** |
| 推荐摄入量 | **强制标示** |

---

## 4. 营养成分表标示要求

### 4.1 能量和营养素

特殊膳食用食品必须标示"1+4"：

| 必标项目 | 单位 |
|----------|------|
| 能量 | kJ |
| 蛋白质 | g |
| 脂肪 | g |
| 碳水化合物 | g |
| 钠 | mg |

### 4.2 可选择性标示

根据产品特点可选择性标示：
- 维生素：A、D、E、K、C、B族等
- 矿物质：钙、铁、锌、硒等
- 其他营养素：胆碱、肌醇、牛磺酸等

### 4.3 标示值允许误差范围

| 营养素 | 允许误差范围 |
|--------|--------------|
| 蛋白质、脂肪、碳水化合物 | ≥80%标示值 |
| 维生素、矿物质 | ≥80%标示值 |
| 能量 | ≥80%标示值且≤120%标示值 |
| 钠 | ≤120%标示值 |

---

## 5. 特殊医学用途配方食品特殊要求

### 5.1 标示内容
特医食品必须标示：
1. **产品类别**：全营养配方食品/特定全营养配方食品/非全营养配方食品
2. **适用人群**：明确标注适用人群和特殊医学用途
3. **食用方法和食用量**：详细说明
4. **警示说明和注意事项**

### 5.2 警示语要求

必须在标签醒目位置标示：
```
"特殊医学用途配方食品，应在医生或临床营养师指导下使用"
"不适用于非目标人群使用"
```

### 5.3 特定全营养配方食品
应标明所针对的疾病类型或医学状况

### 5.4 非全营养配方食品
应标明配合全营养配方食品使用的说明

---

## 6. 婴幼儿配方食品特殊要求

### 6.1 必标内容

| 项目 | 要求 |
|------|------|
| 产品类型 | 婴儿配方食品/较大婴儿和幼儿配方食品 |
| 适用月龄 | 0-6月龄/6-12月龄/12-36月龄 |
| 冲调方法 | 详细说明冲调比例和温度 |
| 推荐摄入量 | 按月龄明确推荐 |

### 6.2 婴儿配方食品(0-6月龄)
- 必须标示："最理想的婴儿食品是母乳，在母乳不足或无母乳时可食用本产品"

### 6.3 较大婴儿和幼儿配方食品
- 必须标示："本品不能作为母乳替代品使用"

---

## 7. 婴幼儿辅助食品特殊要求

### 7.1 产品类别标示

| 类别 | 说明 |
|------|------|
| 婴幼儿谷类辅助食品 | 强化谷物基粉/面条/饼干等 |
| 婴幼儿罐装辅助食品 | 泥糊状/颗粒状/汁类辅助食品 |

### 7.2 必标内容
- 适用的月龄范围
- 食用方法
- 保质期内开封后的贮存条件

---

## 8. 营养声称和功能声称

### 8.1 营养声称

| 声称类型 | 含量要求 |
|----------|----------|
| 无/不含 | ≤限量的20% |
| 低 | ≤限量的40% |
| 来源/含有 | ≥NRV的15% |
| 高/富含 | ≥NRV的30% |

### 8.2 功能声称限制

**禁止**使用以下声称：
- 治疗疾病功能声称
- 预防疾病功能声称
- 替代药物功能声称

**允许**的功能声称：
- 有助于生长发育
- 有助于骨骼发育
- 有助于维持正常免疫功能
- 等基于科学证据的声称

---

## 9. 过敏原标示

### 9.1 过敏原清单

以下过敏原**必须标示**：
1. 含有麸质的谷物及其制品
2. 甲壳纲类动物及其制品
3. 鱼类及其制品
4. 蛋类及其制品
5. 花生及其制品
6. 大豆及其制品
7. 乳及乳制品
8. 坚果及其制品

### 9.2 标示方式
- 可在配料表中用**粗体**标示
- 可在配料表附近单独标示
- 推荐表述："含有xxx"或"此生产线也生产含xxx的产品"

---

## 10. 标签格式要求

### 10.1 字体字号

| 内容 | 最小字号 |
|------|----------|
| 食品名称 | ≥7号字 |
| 净含量、生产者信息 | ≥6号字 |
| 其他强制性标示 | ≥6号字 |
| 警示语 | ≥5号字，**加粗** |

### 10.2 标示位置

| 内容 | 位置要求 |
|------|----------|
| 食品名称 | 标签主要展示版面 |
| 净含量 | 与食品名称在同一版面 |
| 警示语 | 醒目位置 |
| 营养成分表 | 便于阅读的位置 |

---

## 11. 进口特殊膳食用食品

### 11.1 标签要求

进口特殊膳食用食品必须：
1. 有中文标签
2. 符合本标准要求
3. 标示原产地及境内代理商信息

### 11.2 注册备案要求

- 特医食品需取得注册证书
- 婴幼儿配方食品需取得注册证书
- 标签应标示注册号

---

## 12. 过渡期

| 时间节点 | 要求 |
|----------|------|
| 2023年7月1日前 | 按GB 13432-2013执行 |
| 2023年7月1日后 | 按GB 13432-2022执行 |
| 2024年7月1日前 | 过渡期，允许符合旧标准的标签 |
| 2024年7月1日后 | 必须符合新标准 |

---

## 13. 罚则

违反本标准要求的，按照《中华人民共和国食品安全法》处罚：

| 违法行为 | 处罚措施 |
|----------|----------|
| 标签不符合标准 | 责令改正，警告 |
| 无中文标签(进口食品) | 罚款，没收违法所得 |
| 虚假标示/夸大宣传 | 罚款，吊销许可证 |
| 虚假功能声称 | 罚款，追究刑事责任 |

## 附录C: 常见问题

### Q1: 特医食品与保健食品的区别？
- **特医食品**：用于疾病状态下的营养支持，属于食品管理
- **保健食品**：用于保健功能调节，属于特殊食品管理

### Q2: 能否声称"治疗"功效？
- **不能**。特殊膳食用食品不得声称治疗、预防疾病功能

### Q3: 婴儿配方食品能否声称"最接近母乳"？
- **不能**。GB 13432-2022禁止使用"最接近母乳""母乳化"等表述
