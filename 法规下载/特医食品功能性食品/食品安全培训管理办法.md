# 食品安全培训管理办法

## 办法基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家市场监督管理总局 |
| 发布年份 | 2023年 |
| 适用范围 | 食品生产经营企业从业人员培训 |
| 法规性质 | 部门规章 |

## 1. 总则

### 1.1 目的

为规范食品生产经营企业从业人员食品安全培训工作，提高从业人员食品安全意识和操作技能，保障食品安全，根据《中华人民共和国食品安全法》及相关法律法规，制定本办法。

### 1.2 适用范围

本办法适用于：
1. 食品生产企业从业人员
2. 食品经营企业从业人员
3. 餐饮服务从业人员
4. 特殊医学用途配方食品生产企业从业人员
5. 保健食品生产企业从业人员
6. 食品相关产品生产企业从业人员

### 1.3 培训原则

- **全员覆盖**：所有从业人员必须接受培训
- **分级分类**：根据岗位和职责分类培训
- **学时充足**：保证必要的学习时间
- **考核合格**：培训后必须考核合格
- **持续更新**：及时更新培训内容

1. **考试组织**：
   - 统一命题
   - 闭卷考试
   - 严格监考

2. **成绩管理**：
   - 登记建档
   - 不合格补考
   - 补考仍不合格调离岗位

3. **证书管理**：
   - 考核合格发培训合格证
   - 证书有效期1年
   - 到期前复训考核

## 7. 培训效果评估

### 7.1 评估层次

| 层次 | 评估内容 | 评估方法 |
|------|----------|----------|
| 反应层 | 学员满意度 | 问卷调查 |
| 学习层 | 知识掌握程度 | 考试 |
| 行为层 | 行为改变 | 观察、检查 |
| 结果层 | 绩效改善 | 数据分析 |

### 7.2 评估指标

| 指标 | 计算方法 | 目标值 |
|------|----------|--------|
| 培训覆盖率 | 参训人数/应训人数×100% | 100% |
| 考核合格率 | 合格人数/参训人数×100% | ≥95% |
| 知识保持率 | 复训成绩/初训成绩×100% | ≥90% |
| 违规下降率 | （培训前违规-培训后违规）/培训前违规×100% | 逐年下降 |

### 7.3 持续改进

1. 根据评估结果改进培训
2. 更新培训内容
3. 改进培训方式
4. 调整培训计划

## 8. 监督管理

### 8.1 企业内部监督

1. 定期检查培训完成情况
2. 抽查培训档案
3. 现场考核员工
4. 将培训纳入绩效考核

### 8.2 外部监督检查

| 检查类型 | 检查内容 | 检查频率 |
|----------|----------|----------|
| 日常检查 | 培训档案、人员能力 | 按监管计划 |
| 专项检查 | 专项培训落实 | 专项部署 |
| 飞行检查 | 实际能力 | 不定期 |

### 8.3 违规处理

| 违规行为 | 处理措施 |
|----------|----------|
| 未按规定培训 | 责令改正、警告 |
| 培训档案不实 | 责令改正、罚款 |
| 无证上岗 | 责令调离、罚款 |
| 培训不合格仍上岗 | 责令改正、罚款 |

## 9. 附则

### 9.1 实施

企业应制定培训管理制度实施细则，确保本办法落实。

### 9.2 解释

本办法由国家市场监督管理总局负责解释。

---

**办法说明**：
- 本办法为部门规章
- 各地可制定实施细则
- 企业应结合实际制定内部制度
