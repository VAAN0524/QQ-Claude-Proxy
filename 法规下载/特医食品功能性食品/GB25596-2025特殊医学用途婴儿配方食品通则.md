# GB 25596-2025 食品安全国家标准 特殊医学用途婴儿配方食品通则

## 标准基本信息

| 项目 | 内容 |
|------|------|
| **标准编号** | GB 25596-2025 |
| **标准名称** | 食品安全国家标准 特殊医学用途婴儿配方食品通则 |
| **发布日期** | 2025年3月27日 |
| **实施日期** | **2027年3月16日**（2年过渡期） |
| **替代标准** | GB 25596-2010 |
| **适用范围** | 0~12月龄婴儿食用的特殊医学用途婴儿配方食品 |

---

## 关于"2026"的说明

该标准发布于2025年3月，**正式实施日期为2027年3月16日**。设有2年过渡期，为企业产品换版留出时间。2026年是过渡期的重要阶段，预计2026年上半年将迎来申报高峰。

---

## 产品类别变化

### 原标准（6类）→ 新标准（12-13类）

**原有类别：**
1. 无乳糖配方或低乳糖配方
2. 乳蛋白部分水解配方
3. 乳蛋白深度水解配方
4. 氨基酸配方
5. 早产/低出生体重婴儿配方
6. 氨基酸代谢障碍配方
7. 母乳营养补充剂

**新增6个类别：**
1. **生酮配方** - 用于难治性癫痫治疗
2. **防反流配方** - 用于频发胃食管反流婴儿
3. **脂肪代谢异常配方** - 用于脂肪酸代谢吸收障碍
4. **高能量配方** - 用于生长发育迟缓、重度营养不良
5. **蛋白质组件** - 满足临床应用灵活性需求
6. **中链脂肪组件** - 同上

## 主要技术变化

### 1. 术语和定义调整
- 明确适用0~12月龄婴儿
- 删除月龄要求，使适用人群更清晰
- 不再限制产品状态（液体、粉剂等）

### 2. 原料要求变化
- 原料不应含有"麸质"（原为"谷蛋白"）
- 不应含有氢化油脂
- 不应使用经辐照处理的原料

### 3. 营养成分要求调整

| 变化项目 | 具体内容 |
|---------|---------|
| **胆碱** | 由可选择成分调整为必需成分 |
| **碳水化合物来源** | 首选乳糖，可添加葡萄糖聚合物，**不得使用蔗糖** |
| **营养素含量** | 调整维生素D、烟酸、胆碱、钠、钾等必需成分的含量范围 |
| **膳食纤维能量系数** | 调整为8 kJ/g，与GB 28050-2025保持一致 |

### 4. 安全性指标
- 污染物限量、真菌毒素限量、致病菌限量引用通用标准要求
- 需检测脲酶活性的产品由"含有大豆成分的产品"调整为"以大豆及大豆蛋白制品为主要蛋白来源的产品"

### 5. 标签标识要求
- 从"适用的特殊医学状况"修改为"适用的特殊医学状况人群"
- **所有产品类别均需注明渗透压**
- 新增防反流配方的标签标识要求（冲调温度、放置时间、稠度等）

### 6. 单体氨基酸要求
- 仅列出允许使用的单体氨基酸种类及化合物来源
- 删除氨基酸质量指标要求，改为应符合相应标准和有关规定

## 重要提醒

根据国家卫生健康委专家建议：
- **必须做到"精准对应"**：根据婴儿具体医学状况，在医生或临床营养师指导下选择合适产品
- **特医婴配食品不具备治疗疾病功能**，而是营养补充或替代方案
- **家长不可盲目选择或更换产品**
- 已按GB 25596-2010注册的产品需要在2027年3月16日前完成换版注册
