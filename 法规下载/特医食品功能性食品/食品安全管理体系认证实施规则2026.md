# 食品安全管理体系认证实施规则 2026

---

## 第一章 总则

### 第一条 目的依据

为规范食品安全管理体系认证工作，保证认证工作质量，促进食品生产经营者提高食品安全管理水平，根据《中华人民共和国认证认可条例》《食品安全法》等法律法规，制定本规则。

### 第二条 适用范围

本规则适用于在中华人民共和国境内开展的食品安全管理体系认证活动，包括：
1. ISO 22000食品安全管理体系认证
2. 食品安全全球标准（BRCGS）认证
3. 食品安全系统认证（FSSC 22000）
4. 其他食品安全管理体系认证

### 第三条 认证原则

1. **客观公正**原则
2. **科学规范**原则
3. **诚信守则**原则
4. **保密原则**

---

## 第二章 认证机构要求

### 第四条 认证机构资质

从事食品安全管理体系认证的机构应当具备以下条件：

1. **基本要求**：
   - 取得认证机构批准书
   - 具备法人资格
   - 注册资本不低于300万元
   - 固定办公场所和设施

2. **专业能力**：
   - 有10名以上符合要求的专职认证人员
   - 有完整的认证程序和规则
   - 有风险防范能力

### 第五条 认证人员要求

| 人员类别 | 资质要求 |
|----------|----------|
| 认证机构负责人 | 熟悉认证认可法律法规，5年以上认证管理工作经验 |
| 认证人员 | 经注册确认，具备相应专业能力 |
| 技术专家 | 具备相应领域专业知识和经验 |
| 审核员 | 经培训考核合格，注册确认 |

### 第六条 认证人员专业能力

1. **教育背景**：
   - 食品、医学、生物等相关专业
   - 或大专学历+4年以上食品行业工作经验

2. **工作经历**：
   - 4年以上食品安全管理、技术工作经历
   - 或本科+2年以上相关经历

3. **审核经历**：
   - 作为审核组成员完成至少4次完整体系审核
   - 累计审核时间不少于20天

## 第三章 认证程序

### 第七条 认证申请

#### 7.1 申请条件

申请食品安全管理体系认证的企业应当具备以下条件：

| 条件类别 | 具体要求 |
|----------|----------|
| 法律地位 | 取得营业执照等合法主体资格 |
| 生产许可 | 取得食品生产许可证 |
| 管理体系 | 建立并运行食品安全管理体系3个月以上 |
| 产品合规 | 产品符合法律法规和标准要求 |
| 无重大事故 | 申请前3年内无重大食品安全事故 |
| 无严重违法 | 申请前1年内未被撤销认证证书 |

#### 7.2 申请材料

1. **基本材料**：
   - 认证申请书
   - 营业执照复印件
   - 食品生产许可证复印件
   - 组织机构代码证（如需要）

2. **管理体系文件**：
   - 食品安全管理体系手册
   - 程序文件清单
   - HACCP计划（如适用）
   - 前提方案清单

3. **其他材料**：
   - 产品执行标准
   - 厂区布局图
   - 工艺流程图
   - 管理体系运行证明

### 第八条 申请受理

认证机构收到申请后，应当在10个工作日内完成以下审查：

1. **材料审查**：
   - 材料是否齐全
   - 是否符合受理条件

2. **受理决定**：
   - 材料齐全符合要求：受理申请
   - 材料不齐全：一次性告知补正
   - 不符合要求：不予受理并说明理由

### 第九条 认证审核

#### 9.1 审核策划

认证机构受理申请后，应当进行审核策划：

1. **审核组组成**：
   - 组长：1名（高级审核员）
   - 组员：1-3名（审核员）
   - 技术专家：0-2名（如需要）

2. **审核计划**：
   - 审核目的、范围
   - 审核准则
   - 审核组成员
   - 审核时间和日程
   - 审核语言

#### 9.2 文件审查

1. **审查内容**：
   - 管理体系手册
   - 程序文件
   - HACCP计划
   - 前提方案
   - 法律法规清单

2. **审查结果**：
   - 符合要求：安排现场审核
   - 不符合要求：要求修改后重新审查

#### 9.3 现场审核

**第一阶段审核**：
- 目的：了解企业基本情况，确认审核范围
- 内容：管理体系建立情况、生产现场情况
- 时长：一般1-2天

**第二阶段审核**：
- 目的：评价管理体系运行有效性
- 内容：全过程审核，包括管理层、生产现场、检验等
- 时长：根据企业规模确定

#### 9.4 审核内容

| 要素 | 审核要点 |
|------|----------|
| 管理职责 | 领导承诺、食品安全方针、职责权限 |
| 前提方案 | 厂房设施、设备、人员卫生、清洗消毒 |
| 危害分析 | 危害识别、可接受水平、危害评估 |
| HACCP计划 | 关键控制点、关键限值、监控程序、纠正措施 |
| 验证策划 | 验证方法、验证频次、验证人员 |
| 不合格控制 | 纠正、纠正措施、潜在不安全产品处置 |
| 可追溯性 | 批次追溯、召回演练 |
| 沟通 | 内部沟通、外部沟通 |

#### 9.5 不符合项

**不符合项分级**：

| 级别 | 定义 | 示例 |
|------|------|------|
| 严重不符合 | 体系失效或存在系统性问题 | 未进行危害分析、无HACCP计划 |
| 一般不符合 | 个别、孤立的问题 | 部分记录不完整、标识不规范 |

**不符合项处理**：
1. 审核组开具不符合项报告
2. 企业分析原因并采取纠正措施
3. 提交纠正措施验证材料
4. 审核组验证纠正措施有效性

### 第十条 认证决定

#### 10.1 审核报告

现场审核结束后，审核组应当提交审核报告，包括：

1. **基本信息**：企业名称、地址、审核范围
2. **审核概述**：审核过程、审核方法
3. **审核发现**：符合项、不符合项
4. **审核结论**：推荐认证/不推荐认证
5. **改进建议**：持续改进建议

#### 10.2 认证决定

认证机构根据审核报告作出认证决定：

| 审核结论 | 认证决定 |
|----------|----------|
| 无不符合项或仅有轻微不符合项 | 准予认证 |
| 存在一般不符合项，整改合格 | 准予认证 |
| 存在严重不符合项 | 不予认证 |

#### 10.3 认证证书

认证证书内容包括：

| 项目 | 内容 |
|------|------|
| 证书编号 | 统一编码 |
| 获证组织 | 与营业执照一致 |
| 地址 | 实际生产地址 |
| 认证范围 | 具体产品/过程 |
| 认证标准 | ISO 22000等 |
| 发证日期 | 证书签发日期 |
| 有效期 | 3年 |

### 第十一条 监督审核

#### 11.1 监督频次

| 时间 | 要求 |
|------|------|
| 首次监督 | 认证后6个月~12个月 |
| 后续监督 | 每年至少1次 |
| 两次监督间隔 | ≤12个月 |

#### 11.2 监督内容

1. **体系运行**：
   - 管理体系持续符合性
   - 不符合项整改情况
   - 内审和管理评审

2. **产品安全**：
   - 产品检验结果
   - 顾客投诉处理
   - 食品安全事件

3. **变化情况**：
   - 组织机构变化
   - 生产场所变化
   - 产品/工艺变化

### 第十二条 再认证

1. **再认证时间**：
   - 证书有效期届满前3个月申请
   - 到期前完成再认证审核

2. **再认证审核**：
   - 全面审核
   - 覆盖全部认证范围
   - 包括现场审核

3. **再认证决定**：
   - 符合要求：换发新证书
   - 不符合要求：暂停或撤销证书

## 第四章 认证要求

### 第十三条 ISO 22000标准要求

#### 13.1 关键要求

| 条款 | 要求 |
|------|------|
| 4.1 食品安全管理体系 | 建立、实施、保持和改进 |
| 4.2 文件要求 | 文件化、控制、记录 |
| 5.1 管理承诺 | 最高管理者承诺 |
| 5.2 食品安全方针 | 与宗旨相适应、沟通 |
| 5.3 职责权限 | 明确规定并沟通 |
| 5.4 食品安全小组 | 任命、具备专业知识 |
| 6.1 资源提供 | 及时提供所需资源 |
| 6.2 人力资源 | 能力意识培训 |
| 6.3 基础设施 | 工作环境 |
| 7.2 前提方案（PRP） | 建立实施PRP |
| 7.3 实施危害分析的预备步骤 | 产品特性、预期用途、流程图 |
| 7.4 危害分析 | 识别评估、控制措施 |
| 7.5 操作性前提方案 | 建立实施OPRP |
| 7.6 HACCP计划 | 建立HACCP计划 |
| 7.7 更新 | 信息更新 |
| 7.8 验证策划 | 验证方法、频次 |
| 7.9 可追溯性 | 建立可追溯系统 |
| 7.10 不符合控制 | 纠正纠正措施 |
| 8.2 验证 | 内部审核 |
| 8.3 单项验证结果评价 | 验证结果分析 |
| 8.4 食品安全管理体系更新 | 持续改进 |
| 8.5 管理评审 | 定期评审 |

### 第十四条 HACCP要求

#### 14.1 HACCP原理

| 原理 | 内容 |
|------|------|
| 1 | 进行危害分析 |
| 2 | 确定关键控制点 |
| 3 | 建立关键限值 |
| 4 | 建立监控程序 |
| 5 | 建立纠正措施 |
| 6 | 建立验证程序 |
| 7 | 建立记录保持程序 |

#### 14.2 HACCP计划内容

| 项目 | 要求 |
|------|------|
| 关键控制点 | 明确确定 |
| 显著危害 | 识别具体危害 |
| 关键限值 | 可测量可操作 |
| 监控程序 | 方法、频次、人员 |
| 纠正措施 | 偏离时采取的行动 |
| 验证程序 | 确认、验证、审核 |
| 记录保持 | 准确、及时、完整 |

### 第十五条 前提方案要求

#### 15.1 PRP类别

| 类别 | 内容 |
|------|------|
| 建筑物和设施 | 选址、布局、结构 |
| 设备和工器具 | 材质、设计、安装 |
| 人员卫生和培训 | 健康检查、培训、卫生规范 |
| 采购 | 供应商管理、验收检验 |
| 交叉污染控制 | 物理隔离、人流物流分开 |
| 清洗和消毒 | 清洗消毒程序、效果验证 |
| 虫害控制 | 防治计划、定期检查 |
| 化学品控制 | 清单、标识、储存使用 |
| 追溯和召回 | 批次追溯、召回演练 |
| 废弃物处理 | 分类、存放、处置 |

#### 15.2 GMP要求

1. **人员**：
   - 健康检查
   - 培训考核
   - 卫生规范

2. **厂房**：
   - 选址环境
   - 功能布局
   - 洁净分区

3. **设备**：
   - 材质要求
   - 清洗消毒
   - 维护保养

4. **生产过程**：
   - 工艺控制
   - 卫生控制
   - 记录完整

## 第五章 认证证书管理

### 第十六条 证书使用

1. **证书范围**：
   - 仅限证书规定的认证范围
   - 不得扩大使用范围

2. **标识使用**：
   - 按规定使用认证标志
   - 误导宣传

3. **证书宣传**：
   - 实事求是
   - 不得夸大

### 第十七条 证书变更

1. **需要变更的情形**：
   - 企业名称、地址变更
   - 认证范围扩大或缩小
   - 认证标准变更

2. **变更程序**：
   - 提交变更申请
   - 审核评估
   - 换发证书

### 第十八条 证书暂停

暂停认证证书的情形：

| 情形 | 暂停期限 |
|------|----------|
| 监督审核不合格 | 3-6个月 |
| 未按规定接受监督审核 | 暂停至接受审核 |
| 发生食品安全事故 | 视情况确定 |
| 持证人请求暂停 | ≤12个月 |
| 其他违反认证规则 | 视情况确定 |

### 第十九条 证书撤销

撤销认证证书的情形：

| 情形 | 处理 |
|------|------|
| 暂停期间未整改 | 撤销证书 |
| 发生重大食品安全事故 | 撤销证书 |
| 提供虚假材料骗取证书 | 撤销证书 |
| 超范围使用认证标志 | 撤销证书 |
| 拒绝接受监督审核 | 撤销证书 |
| 证书暂停期满未恢复 | 撤销证书 |

### 第二十条 证书注销

注销认证证书的情形：
1. 认证证书有效期届满，未申请延续
2. 持证人申请注销
3. 企业依法终止

## 第六章 认证收费

### 第二十一条 收费原则

1. **公开透明**：收费标准公开
2. **合理收费**：符合规定
3. **自愿选择**：企业自愿选择

### 第二十二条 收费项目

| 项目 | 说明 |
|------|------|
| 申请费 | 申请时收取 |
| 审核费 | 按人天计算 |
| 批准注册费 | 认证决定时收取 |
| 年金（监审费）| 每年收取 |
| 其他 | 差旅费等 |

### 第二十三条 收费标准

1. **审核人天**：
   - 第一阶段审核：根据企业规模
   - 第二阶段审核：根据企业规模
   - 监督审核：初审的1/3~1/2

2. **人日数计算**：
   - 员工人数
   - 生产过程复杂程度
   - 产品风险等级

## 第七章 附则

### 第二十四条 争议处理

1. **申诉**：
   - 对认证决定不服
   - 向认证机构申诉
   - 认证机构调查处理

2. **投诉**：
   - 对认证机构或人员不满
   - 向认可机构投诉
   - 认可机构调查处理

### 第二十五条 认可要求

认证机构应当获得国家认可机构的认可：
1. 认可依据：GB/T 27021、ISO/TS 22003
2. 认可范围：食品安全管理体系认证
3. 定期评审：确保持续符合认可要求

### 第二十六条 国际互认

中国认证机构获得的认可加入国际互认体系：
1. IAF MLA（国际认可论坛多边互认协议）
2. 与其他国家的认证证书互认

## 附录B：常见不符合项

### B.1 严重不符合项示例

1. 未进行系统的危害分析
2. 未确定关键控制点
3. HACCP计划未有效实施
4. 无可追溯系统
5. 无内审或管理评审

### B.2 一般不符合项示例

1. 部分记录不完整
2. 标识不规范
3. 设备维护记录缺失
4. 人员培训记录不完整
5. 验证活动未按计划执行
