# 特医食品临床试验质量管理规范（2026版）

## 规范概述

**规范名称**：特殊医学用途配方食品临床试验质量管理规范（GCP）
**发布机构**：国家市场监督管理总局
**最新版本**：2026版
**适用范围**：特殊医学用途配方食品（特医食品）的临床试验

---

## 一、总则

### 1.1 目的

为保证特殊医学用途配方食品临床试验数据的真实、准确、完整和可靠，保障受试者权益和安全，规范临床试验行为，根据《中华人民共和国食品安全法》《特殊医学用途配方食品注册管理办法》等法律法规，制定本规范。

### 1.2 适用范围

本规范适用于在中国境内开展的特医食品临床试验。

### 1.3 基本原则

| 原则 | 内容 |
|------|------|
| **伦理原则** | 遵守《赫尔辛基宣言》，保障受试者权益 |
| **科学原则** | 试验设计科学合理，数据真实可靠 |
| **合规原则** | 遵守法律法规和GCP要求 |
| **风险控制原则** | 将受试者风险降至最低 |

## 二、临床试验的类型

### 2.1 按产品分类

| 产品类型 | 试验要求 |
|---------|----------|
| **特殊医学用途婴儿配方食品** | 必须进行临床试验 |
| **全营养配方食品** | 必须进行临床试验 |
| **非全营养配方食品** | 根据产品特点决定是否需要临床试验 |
| **特定全营养配方食品** | 必须进行临床试验 |

### 2.2 按试验阶段分类

| 阶段 | 目的 | 要求 |
|------|------|------|
| **探索性试验** | 初步评价安全性和耐受性 | 小样本（20-30例） |
| **确证性试验** | 证实安全性和有效性 | 大样本（100例以上） |
| **上市后研究** | 监测安全性和有效性 | 按需进行 |

## 三、临床试验设计

### 3.1 设计原则

#### 3.1.1 随机对照试验（RCT）

| 设计要素 | 要求 |
|---------|------|
| **随机化** | 采用适当的随机化方法 |
| **对照** | 设置安慰剂对照或阳性对照 |
| **盲法** | 推荐双盲设计，至少单盲 |
| **样本量** | 具有统计学效力 |

#### 3.1.2 特殊设计

| 设计类型 | 适用情况 |
|---------|----------|
| **交叉设计** | 适用于慢性病稳定期患者 |
| **析因设计** | 研究多个因素 |
| **非劣效性设计** | 与已上市产品比较 |
| **适应性设计** | 根据中期结果调整 |

### 3.2 试验分期

#### 3.2.1 Ⅰ期临床试验

| 项目 | 内容 |
|------|------|
| **目的** | 初步评价安全性和耐受性 |
| **受试者** | 健康志愿者或目标患者 |
| **样本量** | 20-30例 |
| **观察期** | 短期（1-4周） |

#### 3.2.2 Ⅱ期临床试验

| 项目 | 内容 |
|------|------|
| **目的** | 评价有效性和安全性，探索剂量 |
| **受试者** | 目标疾病患者 |
| **样本量** | 50-100例 |
| **观察期** | 中期（4-12周） |

#### 3.2.3 Ⅲ期临床试验

| 项目 | 内容 |
|------|------|
| **目的** | 确证有效性和安全性 |
| **受试者** | 目标疾病患者 |
| **样本量** | 100-300例 |
| **观察期** | 长期（12-52周） |

### 3.3 主要评价指标

#### 3.3.1 安全性指标

| 指标类别 | 具体指标 |
|---------|----------|
| **生命体征** | 体温、血压、心率、呼吸频率 |
| **血液学** | 血常规、凝血功能 |
| **生化指标** | 肝功能、肾功能、电解质、血糖、血脂 |
| **免疫功能** | 免疫球蛋白、淋巴细胞亚群 |
| **不良反应** | 不良事件类型、发生率、严重程度 |

#### 3.3.2 有效性指标

| 产品类型 | 主要有效性指标 |
|---------|---------------|
| **全营养配方食品** | 营养状况改善（体重、BMI、白蛋白等） |
| **糖尿病配方食品** | 血糖控制（HbA1c、空腹血糖等） |
| **肾病配方食品** | 肾功能改善（血肌酐、尿素氮等） |
| **肝病配方食品** | 肝功能改善（ALT、AST、白蛋白等） |
| **肿瘤配方食品** | 营养状况维持、生活质量改善 |
| **吸收障碍综合征** | 营养物质吸收率、粪便脂肪排出量 |
| **手术后配方食品** | 氮平衡、伤口愈合情况 |

### 3.4 次要评价指标

| 指标类别 | 具体指标 |
|---------|----------|
| **生活质量** | SF-36、疾病特异性生活质量问卷 |
| **患者满意度** | 产品口感、耐受性、便利性 |
| **依从性** | 实际摄入量与要求摄入量的比值 |
| **经济学指标** | 医疗资源使用、成本效益 |

## 四、受试者权益保护

### 4.1 知情同意

#### 4.1.1 知情同意书内容

| 内容 | 要求 |
|------|------|
| **试验目的** | 清楚说明试验目的和意义 |
| **试验过程** | 详细说明试验流程和持续时间 |
| **预期受益** | 客观说明可能的受益 |
| **潜在风险** | 详细说明所有已知和潜在风险 |
| **替代治疗** | 说明可用的替代治疗方案 |
| **自愿原则** | 强调参与完全自愿，可随时退出 |
| **补偿** | 说明提供的补偿和保险 |

#### 4.1.2 知情同意过程

| 步骤 | 要求 |
|------|------|
| **告知** | 研究者充分告知试验信息 |
| **理解确认** | 确认受试者已理解所有信息 |
| **自愿同意** | 受试者自愿签署同意书 |
| **副本留存** | 受试者和研究者各持一份 |

### 4.2 伦理审查

#### 4.2.1 伦理委员会职责

| 职责 | 内容 |
|------|------|
| **试验方案审查** | 审查试验科学性和伦理性 |
| **知情同意书审查** | 审查知情同意书内容 |
| **研究者资格审查** | 审查研究者的资质和经验 |
| **安全性审查** | 审查安全性监测计划 |
| **持续监督** | 持续监督试验进行情况 |

#### 4.2.2 审查要求

| 审查类型 | 时机 |
|---------|------|
| **初始审查** | 试验开始前 |
| **修正审查** | 方案修改时 |
| **年度审查** | 每年至少一次 |
| **严重不良事件审查** | 发生严重不良事件时 |

### 4.3 受试者安全保障

| 保障措施 | 内容 |
|---------|------|
| **纳入排除标准** | 严格筛选受试者 |
| **安全性监测** | 定期进行安全性评估 |
| **中止机制** | 制定明确的中止标准 |
| **医疗救治** | 提供必要的医疗救治 |
| **保险** | 为受试者购买临床试验保险 |

## 八、数据管理

### 8.1 数据收集

| 要求 | 内容 |
|------|------|
| **原始记录** | 及时、准确、完整记录原始数据 |
| **病例报告表（CRF）** | 与原始数据一致 |
| **电子数据** | 电子系统应符合ALCOA原则 |
| **数据核查** | 定期进行数据核查 |

### 8.2 ALCOA原则

| 原则 | 含义 |
|------|------|
| **A（Attributable）** | 可归因性：数据可追溯到产生者 |
| **L（Legible）** | 清晰性：数据清晰可读 |
| **C（Contemporaneous）** | 同时性：数据及时记录 |
| **O（Original）** | 原始性：保留原始记录 |
| **A（Accurate）** | 准确性：数据准确无误 |

### 8.3 数据存储与保管

| 要求 | 保存期限 |
|------|----------|
| **原始数据** | 试验结束后10年 |
| **CRF** | 试验结束后10年 |
| **知情同意书** | 试验结束后10年 |
| **其他文件** | 试验结束后10年 |

## 九、统计分析

### 9.1 分析数据集

| 数据集 | 定义 |
|--------|------|
| **全分析集（FAS）** | 至少使用一次试验产品的受试者 |
| **符合方案集（PPS）** | 符合方案要求、完成试验的受试者 |
| **安全性数据集（SS）** | 至少使用一次试验产品且有安全性数据的受试者 |

### 9.2 统计方法

| 指标类型 | 统计方法 |
|---------|----------|
| **计量资料** | t检验、方差分析、非参数检验 |
| **计数资料** | 卡方检验、Fisher精确检验 |
| **等级资料** | 秩和检验 |
| **生存资料** | Kaplan-Meier法、Log-rank检验 |
| **重复测量** | 重复测量方差分析、混合效应模型 |

### 9.3 统计学显著性

| 情况 | 要求 |
|------|------|
| **优效性试验** | P < 0.05（双侧） |
| **非劣效性试验** | 非劣效界值可信区间上限 < 0 |
| **等效性试验** | 等效界值可信区间完全在等效范围内 |

## 十、安全性监测与报告

### 10.1 不良事件（AE）

| 定义 | 在临床试验中，受试者发生的任何不利医学事件，不一定与试验产品有因果关系。 |

### 10.2 不良反应（ADR）

| 定义 | 与试验产品有肯定或可疑因果关系的不良事件。 |

### 10.3 严重不良事件（SAE）

| 标准 | 符合以下任一标准：
- 导致死亡
- 危及生命
- 需要住院或延长住院时间
- 导致永久或显著残疾/功能丧失
- 先天性异常/出生缺陷
- 其他重要医学事件 |

### 10.4 报告时限

| 事件类型 | 报告时限 |
|---------|----------|
| **严重不良事件** | 24小时内报告申办者和伦理委员会 |
| **死亡事件** | 7日内报告国家监管部门 |
| **其他安全性信息** | 定期汇总报告 |

### 10.5 安全性报告内容

| 内容 | 说明 |
|------|------|
| **患者信息** | 年龄、性别等 |
| **事件描述** | 发生时间、严重程度、转归 |
| **因果关系判断** | 与试验产品的相关性 |
| **处理措施** | 采取的处理措施 |

## 十二、试验终止与提前结束

### 12.1 终止条件

| 情况 | 处理 |
|------|------|
| **安全性问题** | 出现严重安全性问题 |
| **伦理问题** | 伦理问题无法解决 |
| **科学问题** | 试验设计存在严重缺陷 |
| **申办者决定** | 申办者因商业原因终止 |

### 12.2 终止程序

| 步骤 | 内容 |
|------|------|
| **通知** | 通知研究者和伦理委员会 |
| **受试者处理** | 妥善处理在试受试者 |
| **数据收集** | 收集已获得的数据 |
| **报告** | 向监管部门报告终止原因 |
| **后续处理** | 按要求处理试验样品和数据 |

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **特医食品** | 特殊医学用途配方食品，用于特殊医学状况下人群的营养支持 |
| **临床试验** | 在人体（患者或健康志愿者）进行产品的系统性研究 |
| **GCP** | Good Clinical Practice，临床试验质量管理规范 |
| **知情同意** | 向受试者告知试验信息并获得其自愿同意的过程 |
| **伦理委员会** | 负责审查临床试验伦理性的独立委员会 |
| **不良事件（AE）** | 在临床试验中受试者发生的任何不利医学事件 |
| **严重不良事件（SAE）** | 导致死亡、危及生命等严重后果的不良事件 |
