# 药品经营质量管理规范(GSP)现场检查指导原则

**文件编号**: 国家药监局公告2024年第XX号
**实施日期**: 2024年XX月XX日
**最新修订**: 2026年1月更新

---

## 第一章 总则

### 第一条 检查目的
为规范药品经营企业现场检查工作，保证药品经营质量，根据《中华人民共和国药品管理法》《药品经营质量管理规范》（以下简称《规范》）等法律法规，制定本指导原则。

### 第二条 适用范围
本指导原则适用于药品监督管理部门对药品经营企业（含药品批发、零售企业）的许可检查、常规检查、有因检查等各类现场检查。

### 第三条 检查原则
现场检查应当遵循以下原则：
- **风险管控原则**：以风险为核心，聚焦关键环节和突出问题
- **问题导向原则**：针对企业实际情况，精准发现问题
- **客观公正原则**：依据事实，依法依规进行检查
- **科学高效原则**：运用科学方法，提高检查效能

---

## 第二章 检查机构和人员

### 第四条 检查机构
药品监督管理部门负责组织实施现场检查工作，可委托第三方专业机构参与。

### 第五条 检查人员
检查人员应当具备以下条件：
1. 具有药学、医学、生物学、化学等相关专业背景
2. 具有三年以上药品监督管理或药品经营质量管理经验
3. 经过专门的检查培训并考核合格
4. 遵守廉洁自律规定

### 第六条 检查组组成
检查组一般由2-3名检查人员组成，设组长1名。根据检查需要，可邀请技术专家参与。

---

## 第三章 检查内容和标准

### 第七条 质量管理体系
**检查要点**：
1. 质量管理制度健全性
2. 组织机构设置合理性
3. 岗位职责明确性
4. 质量管理体系文件完整性

**检查标准**：
- 建立完善的质量管理体系文件
- 明确企业负责人、质量负责人、质量管理部门职责
- 质量管理制度能够有效执行并持续改进

### 第八条 机构与人员
**检查要点**：
1. 企业负责人资质
2. 质量负责人资质
3. 质量管理人员配备
4. 人员培训与健康

**检查标准**：
- 企业负责人具有大专以上学历或中级以上专业技术职称
- 质量负责人具有大学本科以上学历、执业药师资格
- 质量管理人员数量与经营规模相适应
- 每年开展质量培训并建立档案

### 第九条 设施与设备
**检查要点**：
1. 营业场所面积与布局
2. 仓储设施条件
3. 温湿度监测设备
4. 冷链设备配置

**检查标准**：
- 批发企业仓库面积不小于500平方米
- 零售企业营业面积不小于40平方米
- 配备符合要求的温湿度监测系统
- 冷链设备符合药品储存要求

### 第十条 采购与验收
**检查要点**：
1. 供应商审核
2. 首营品种审核
3. 验收程序执行
4. 进货查验记录

**检查标准**：
- 对供应商进行资质审核并建立档案
- 首营品种审核资料完整
- 验收记录真实、完整、可追溯

### 第十一条 储存与养护
**检查要点**：
1. 药品分类储存
2. 温湿度控制
3. 近效期药品管理
4. 药品养护检查

**检查标准**：
- 按包装标示的温度要求储存
- 常温库：10-30℃，阴凉库：不超过20℃，冷库：2-10℃
- 每月进行养护检查并记录
- 近效期药品有明显标识

### 第十二条 销售与运输
**检查要点**：
1. 销售对象审核
2. 销售记录完整性
3. 运输过程控制
4. 冷链药品运输

**检查标准**：
- 只向合法的药品经营企业或医疗机构销售
- 销售记录真实完整
- 运输过程中采取措施保证药品质量
- 冷链药品运输全程温度监控

---

## 第四章 冷链药品管理专项要求

### 第十三条 冷链设施设备
**检查要点**：
1. 冷库配置与验证
2. 冷链车/冷藏箱配置
3. 温湿度监测设备
4. 备用电源配置

**检查标准**：
- 冷库经过验证并定期再验证
- 配备与经营规模相适应的冷藏车
- 温湿度监测设备能够自动记录和报警
- 配备双路供电或备用发电机

### 第十四条 冷链储存管理
**检查要点**：
1. 温湿度控制
2. 温湿度监测
3. 偏离处理
4. 设备维护

**检查标准**：
- 冷库温度控制在2-10℃
- 自动记录温湿度，至少每小时1次
- 温湿度超标时及时处理并记录
- 冷链设备定期维护保养

### 第十五条 冷链运输管理
**检查要点**：
1. 运输前准备
2. 运输过程监控
3. 交接验收
4. 应急处理

**检查标准**：
- 使用经过验证的冷藏车或保温箱
- 运输全过程温度监控并记录
- 交接时核对温度记录
- 制定应急预案并定期演练

### 第十六条 冷链药品收货验收
**检查要点**：
1. 温度检查
2. 运输记录查验
3. 外观检查
4. 不合格品处理

**检查标准**：
- 收货时检查运输温度记录
- 温度不符合要求的拒收
- 运输过程温度记录完整保存
- 建立冷链药品验收记录

## 第五章 检查程序

### 第十七条 检查准备
1. 制定检查方案
2. 组建检查组
3. 准备检查文件
4. 通知被检查企业

### 第十八条 现场检查
1. 首次会议
2. 现场检查
3. 资料审查
4. 人员访谈
5. 末次会议

### 第十九条 检查方法
1. **查阅资料**：检查质量管理体系文件、记录档案
2. **现场查看**：检查设施设备、储存条件
3. **人员考核**：询问岗位人员、抽查操作
4. **追踪检查**：选取品种进行全过程追踪

### 第二十条 缺陷分类
**严重缺陷**：
- 涉嫌虚假资料、记录
- 经营假药、劣药
- 温湿度控制严重不符合要求
- 冷链设施未经验证投入使用

**主要缺陷**：
- 质量管理制度不健全
- 人员资质不符合要求
- 记录不完整、不规范
- 温湿度监测不符合要求

**一般缺陷**：
- 记录填写不规范
- 标识不清晰
- 个别记录缺失

## 第六章 检查结果处理

### 第二十一条 结果判定
| 检查结果 | 判定标准 |
|---------|----------|
| **通过** | 0项严重缺陷，且主要缺陷≤10% |
| **整改后通过** | 0项严重缺陷，且10%<主要缺陷≤30% |
| **不通过** | 存在严重缺陷或主要缺陷>30% |

### 第二十二条 整改要求
1. 明确整改期限（一般不超过30天）
2. 制定整改方案
3. 实施整改措施
4. 提交整改报告
5. 现场复核验证

### 第二十三条 不合格处理
检查不通过的企业，按以下情形处理：
1. 暂停经营活动
2. 责令限期整改
3. 整改后重新检查
4. 仍不合格的，吊销经营许可证

---

## 第七章 附则

### 第二十四条 术语定义
- **冷链药品**：指需要冷藏储存运输的药品，包括生物制品、疫苗、胰岛素等
- **验证**：确认设施设备能够达到预期要求的一系列活动
- **偏差**：温湿度超出规定范围的情况

### 第二十五条 解释权
本指导原则由国家药品监督管理局负责解释。

### 第二十六条 施行日期
本指导原则自发布之日起施行。
