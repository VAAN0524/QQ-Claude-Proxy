# 特殊医学用途配方食品GMP实施细则详细版（2026）

## 法规信息
- **发布机构**：国家市场监督管理总局
- **实施日期**：2026年1月1日
- **适用范围**：特殊医学用途配方食品生产企业
- **依据标准**：GB 29923-2023 特殊医学用途配方食品良好生产规范

---

## 第一章 总则

### 第一条 目的
为规范特殊医学用途配方食品（以下简称"特医食品"）生产质量管理，根据《中华人民共和国食品安全法》及其实施条例、《特殊医学用途配方食品注册管理办法》等法律法规，制定本细则。

### 第二条 适用范围
本细则适用于特医食品的生产许可审查、生产过程质量管理和监督检查。

### 第三条 基本原则
特医食品生产企业应当：
1. 建立完善的质量管理体系
2. 配备与生产规模相适应的专业技术人员
3. 具有与生产相适应的厂房、设施、设备
4. 建立严格的原料采购、验收、使用制度
5. 建立完善的生产过程控制制度
6. 建立严格的产品检验和出厂检验制度

---

## 第二章 机构与人员

### 第四条 组织机构
企业应当建立独立的质量管理部门，直属企业主要负责人领导，并明确规定其职权和职责。

### 第五条 人员资质
1. **企业负责人**：应当熟悉特医食品相关法律法规，具备特医食品生产质量管理知识
2. **质量负责人**：应当具有食品、医药、生物等相关专业本科以上学历，3年以上特医食品或药品质量管理经验
3. **生产负责人**：应当具有食品、医药、生物等相关专业大专以上学历，3年以上食品生产管理经验
4. **检验人员**：应当具有食品检验相关专业中专以上学历，经培训合格后上岗

### 第六条 人员培训
1. 新入职人员应当经过岗前培训，考核合格后方可上岗
2. 在岗人员每年应当进行继续教育培训，培训时间不少于40小时
3. 培训内容包括：法律法规、标准规范、操作规程、卫生知识等
4. 建立培训档案，记录培训内容、时间、考核结果

### 第七条 人员健康
1. 建立从业人员健康管理制度
2. 每年进行健康检查，取得健康证明后方可上岗
3. 患有国务院卫生行政部门规定的有碍食品安全疾病的人员，不得从事接触直接入口食品的工作

---

## 第三章 厂房与设施

### 第八条 选址要求
1. 厂区周围不得有粉尘、有害气体、放射性物质和其他扩散性污染源
2. 厂区应当保持环境整洁，路面平整硬化，无积水
3. 厂区周围应当有适当的隔离带，防止外界污染

### 第九条 厂区布局
1. 厂区布局应当合理，各功能区划分明确
2. 生产区、辅助生产区、行政生活区应当分开
3. 人流、物流应当分开，原料、半成品、成品应当分开
4. 厂区道路应当采用混凝土或沥青硬化，防止积水积尘

### 第十条 车间布局
1. 车间内部布局应当合理，工艺流程应当顺畅，避免交叉污染
2. 车间应当根据生产工艺划分洁净区、一般生产区
3. 洁净区应当设置独立的空气净化系统
4. 原料处理、配料、混合、成型、干燥、包装等工序应当有效隔离

### 第十一条 洁净区要求
1. **洁净度级别**：
   - 100,000级：配料、混合、成型工序
   - 10,000级：无菌灌装工序
   - 100级：关键操作区域（如灌装点）

2. **环境监控**：
   - 悬浮粒子：每班次监测
   - 微生物：每月监测
   - 温湿度：连续监测
   - 压差：连续监测

### 第十二条 仓储设施
1. 原料库、成品库应当分开设置
2. 仓库应当具备良好的通风、防潮、防虫、防鼠设施
3. 冷库温度应当控制在2-8℃
4. 危险品应当单独存放，符合安全管理规定

---

## 第四章 设备

### 第十三条 设备选型
1. 设备应当与生产规模相适应
2. 设备材质应当符合食品级要求，不与产品发生化学反应
3. 设备应当便于清洗、消毒、维护
4. 关键设备应当具备自动化控制功能

### 第十四条 设备安装
1. 设备安装应当符合工艺要求，便于操作和维护
2. 设备与地面、墙壁之间应当保持适当距离，便于清洁
3. 管道排列整齐，标识清晰，无死角
4. 电气设备应当符合防爆、防潮要求

### 第十五条 设备维护
1. 建立设备台账，记录设备型号、安装日期、维护记录
2. 制定设备维护保养计划，定期执行
3. 关键设备应当制定预防性维护计划
4. 设备维修后应当重新验证，确认合格后方可使用

---

## 第五章 卫生管理

### 第十六条 卫生制度
1. 建立完善的卫生管理制度和操作规程
2. 划分卫生责任区，明确责任人
3. 定期进行卫生检查，记录检查结果
4. 发现卫生问题应当及时整改

### 第十七条 清洁消毒
1. 制定详细的清洁消毒规程
2. 不同洁净区的清洁工具应当专用，不得混用
3. 清洁剂、消毒剂应当经检验合格，不影响产品质量
4. 清洁消毒后应当进行验证

### 第十八条 虫害控制
1. 建立虫害控制程序
2. 定期检查厂区及周边环境
3. 设置防鼠板、防蝇灯等设施
4. 发现虫害应当及时采取控制措施

---

## 第六章 原料管理

### 第十九条 原料采购
1. 建立供应商审核制度，选择合格供应商
2. 原料应当从具有合法资质的供应商采购
3. 采购的原料应当符合食品安全标准
4. 进口原料应当具有检验检疫证明

### 第二十条 原料验收
1. 建立原料验收标准
2. 每批原料应当进行验收检验
3. 验收内容包括：外观、标识、检验报告等
4. 不合格原料应当拒收，并做好记录

### 第二十一条 原料储存
1. 原料应当分类存放，标识清晰
2. 做到先进先出，防止过期
3. 特殊原料应当按规定条件储存
4. 定期检查库存原料质量

---

## 第七章 生产过程控制

### 第二十二条 生产工艺
1. 建立标准的生产工艺规程
2. 工艺参数应当经过验证
3. 关键工艺参数应当实时监控
4. 工艺变更应当经过验证和审批

### 第二十三条 生产操作
1. 生产人员应当严格按照操作规程进行操作
2. 生产过程中应当做好生产记录
3. 发现异常情况应当及时报告并处理
4. 生产记录应当真实、完整、可追溯

### 第二十四条 污染控制
1. 生产过程中应当防止交叉污染
2. 不同产品、不同批次之间应当彻底清洁
3. 生产废弃物应当及时清理
4. 生产设备应当定期清洁消毒

---

## 第八章 质量检验

### 第二十五条 检验机构
1. 建立独立的质量检验机构
2. 配备与生产规模相适应的检验人员和设备
3. 检验人员应当经过培训，具备相应的资格
4. 检验设备应当定期校准和维护

### 第二十六条 检验要求
1. 建立完善的产品标准和检验规程
2. 原料、半成品、成品应当按规定检验
3. 每批产品应当进行出厂检验
4. 检验记录应当完整、准确、可追溯

### 第二十七条 不合格品处理
1. 建立不合格品处理程序
2. 不合格原料不得投入使用
3. 不合格半成品不得流入下道工序
4. 不合格成品不得出厂

---

## 第九章 产品追溯与召回

### 第二十八条 追溯系统
1. 建立完善的产品追溯系统
2. 产品应当具有唯一标识
3. 批记录应当完整保存至产品有效期后1年
4. 应当能够追溯到原料来源

### 第二十九条 召回制度
1. 建立产品召回制度
2. 发现产品存在安全隐患应当立即召回
3. 召回产品应当按规定处理
4. 召回情况应当向监管部门报告

---

## 第十章 文件管理

### 第三十条 文件体系
1. 建立完善的文件管理体系
2. 文件应当经过起草、审核、批准
3. 文件应当定期审核修订
4. 废止文件应当及时收回

### 第三十一条 记录管理
1. 建立完善的记录管理制度
2. 记录应当真实、完整、及时
3. 记录应当按规定期限保存
4. 记录填写应当清晰、不得涂改

---

## 第十一章 附则

### 第三十三条 本细则自2026年1月1日起施行

## 附录B：特医食品GMP检查要点

### B.1 机构与人员检查要点
- 质量管理部门独立性
- 人员资质符合性
- 培训档案完整性
- 健康证明有效性

### B.2 厂房设施检查要点
- 选址合规性
- 布局合理性
- 洁净区符合性
- 仓储设施适宜性

### B.3 设备检查要点
- 设备材质适宜性
- 安装规范性
- 维护保养完整性
- 清洁消毒有效性

### B.4 生产过程检查要点
- 工艺规程完整性
- 操作规范性
- 记录及时完整性
- 污染控制有效性
