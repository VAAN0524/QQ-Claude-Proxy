# 食品生产设备安装验收规范

## 规范基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 国家市场监督管理总局 |
| 发布年份 | 2022年 |
| 适用范围 | 食品生产设备安装验收 |
| 规范性质 | 推荐性技术规范 |

## 1. 总则

### 1.1 目的

规范食品生产设备的安装验收工作，确保设备满足生产要求和质量标准，保障食品生产安全。

### 1.2 适用范围

本规范适用于：
1. 新建食品生产企业的设备安装验收
2. 现有企业设备更新改造的安装验收
3. 特殊医学用途配方食品生产设备
4. 保健食品生产设备
5. 其他食品生产设备

### 1.3 基本原则

- **符合设计**：严格按照设计文件和合同要求验收
- **符合标准**：满足相关国家标准和行业标准
- **安全可靠**：确保设备运行安全可靠
- **文档完整**：验收资料齐全完整

## 2. 安装前准备

### 2.1 技术准备

#### 2.1.1 设计文件审查

| 文件类型 | 审查内容 | 要求 |
|----------|----------|------|
| 设备布置图 | 位置、间距、通道 | 符合工艺流程和GMP要求 |
| 工艺管道图 | 连接方式、坡度、支撑 | 便于清洗和消毒 |
| 电气图 | 供电、接地、联锁 | 符合电气规范 |
| 设备说明书 | 技术参数、安装要求 | 准确完整 |

#### 2.1.2 现场条件确认

| 检查项目 | 标准 | 方法 |
|----------|------|------|
| 基础验收 | 强度、平整度、预埋件 | 检查验收记录 |
| 环境条件 | 温度、湿度、洁净度 | 现场测量 |
| 公用工程 | 水、电、气、汽供应 | 试运行确认 |
| 运输通道 | 宽度、高度、承重 | 实地测量 |
| 吊装空间 | 高度、周围障碍物 | 现场确认 |

### 2.2 设备开箱验收

#### 2.2.1 外观检查

| 检查项目 | 合格标准 |
|----------|----------|
| 包装完整性 | 无破损、无水浸 |
| 设备外观 | 无变形、无锈蚀、无涂层脱落 |
| 铭牌信息 | 清晰、与合同一致 |
| 随机文件 | 齐全（说明书、合格证、图纸） |

#### 2.2.2 清点核对

| 核对内容 | 要求 |
|----------|------|
| 主机型号 | 与合同一致 |
| 附件清单 | 按装箱单清点 |
| 备品备件 | 按合同清点 |
| 专用工具 | 齐全完好 |

#### 2.2.3 缺损处理

- 发现缺件、损坏、锈蚀等，立即拍照记录
- 编制缺件/损坏清单
- 通知供应商协调处理
- 重大问题暂停安装

### 2.3 安装方案编制

安装方案应包含：

1. **工程概况**
   - 设备名称、型号、数量
   - 安装位置、环境要求
   - 工程量、工期要求

2. **施工组织**
   - 人员组织及职责分工
   - 施工进度计划
   - 施工机具准备

3. **技术措施**
   - 吊装方案
   - 找正找平方法
   - 精度控制措施

4. **安全措施**
   - 安全防护措施
   - 应急预案
   - 安全交底

5. **质量控制**
   - 质量标准
   - 检验方法
   - 记录要求

## 3. 安装施工

### 3.1 定位放线

| 项目 | 允许偏差 | 检验方法 |
|------|----------|----------|
| 纵向中心线 | ±2 mm | 拉线、钢卷尺 |
| 横向中心线 | ±2 mm | 拉线、钢卷尺 |
| 标高 | ±2 mm | 水准仪 |
| 间距 | ±3 mm | 钢卷尺 |

### 3.2 设备吊装

#### 3.2.1 吊装准备

1. 确认吊装设备和索具完好
2. 确认吊点位置正确
3. 清理吊装路线障碍物
4. 设置警戒区域

#### 3.2.2 吊装要求

| 要求 | 说明 |
|------|------|
| 平稳起吊 | 避免冲击、倾斜 |
| 保护设备 | 使用垫衬保护设备表面 |
| 严禁超载 | 不超过吊装设备额定载荷 |
| 信号统一 | 设专人指挥信号 |

### 3.3 找正找平

#### 3.3.1 找正标准

| 项目 | 精度要求 | 检测方法 |
|------|----------|----------|
| 水平度 | 0.05 mm/m | 水平仪 |
| 垂直度 | 0.1 mm/m | 垂线、水平仪 |
| 同轴度 | 0.05 mm | 百分表、激光对中仪 |
| 平行度 | 0.1 mm/m | 水平仪、塞尺 |

#### 3.3.2 找正方法

1. **粗找**：调整垫铁，使设备大致水平
2. **精找**：使用精密仪器精确调整
3. **复测**：调整后复测，确认合格
4. **固定**：合格后固定地脚螺栓

### 3.4 地脚螺栓安装

| 项目 | 要求 |
|------|------|
| 螺栓材质 | 符合设计要求 |
| 安装位置 | 偏差≤2 mm |
| 垂直度 | 偏差≤1° |
| 丝扣长度 | 露出螺母2-3扣 |
| 紧固力矩 | 符合技术文件要求 |

### 3.5 垫铁布置

| 布置原则 | 说明 |
|----------|------|
| 位置 | 地脚螺栓两侧 |
| 间距 | 500-700 mm |
| 叠放层数 | 不超过3块 |
| 接触面积 | ≥60% |
| 点焊固定 | 找平后点焊 |

### 3.6 二次灌浆

#### 3.6.1 灌浆材料

- 强度等级：≥C25
- 采用无收缩灌浆料
- 或采用细石混凝土（比基础高一级）

#### 3.6.2 灌浆要求

| 项目 | 要求 |
|------|------|
| 清理基础 | 清除杂物、油污，湿润 |
| 模板支设 | 严密不漏浆 |
| 灌浆顺序 | 一侧灌入，排除空气 |
| 养护 | 7天以上 |
| 拆模 | 强度达到75%以上 |

## 4. 管道安装

### 4.1 卫生级管道安装

#### 4.1.1 管道材质

- 与物料接触：316L不锈钢
- 其他：304不锈钢
- 管件：卫生级快装或焊接

#### 4.1.2 安装要求

| 项目 | 标准 |
|------|------|
| 坡度 | 顺坡≥0.3%，反坡≥0.5% |
| 支架间距 | ≤2 m |
| 对接偏差 | ≤1 mm |
| 内壁处理 | 钝化、抛光Ra≤0.8 μm |

#### 4.1.3 焊接要求

1. **氩弧焊**：全氩弧焊或氩弧焊打底
2. **保护气体**：高纯氩（99.99%）
3. **焊缝要求**：
   - 双面成型
   - 无裂纹、气孔、咬边
   - 内壁光滑无毛刺
   - 焊缝余高0-0.5 mm
4. **酸洗钝化**：焊接后酸洗钝化处理

### 4.2 管道支架

| 类型 | 适用场合 | 间距 |
|------|----------|------|
| 固定支架 | 固定点 | 按设计 |
| 滑动支架 | 热膨胀段 | ≤3 m |
| 导向支架 | 直管段 | ≤6 m |
| 弹簧支架 | 垂直管段 | 按设计 |

### 4.3 阀门安装

| 项目 | 要求 |
|------|------|
| 安装方向 | 阀体箭头与流向一致 |
| 操作空间 | 便于操作和维修 |
| 卫生阀门 | 无死角、易拆卸清洗 |
| 密封性 | 安装后进行密封试验 |

## 5. 电气安装

### 5.1 电缆敷设

| 项目 | 要求 |
|------|------|
| 电缆选型 | 符合负载要求，预留余量 |
| 敷设方式 | 桥架、穿管或直埋 |
| 弯曲半径 | ≥电缆外径的10倍 |
| 接头处理 | 中间接头尽量少，必要时做防爆处理 |
| 标识 | 两端挂标识牌 |

### 5.2 接地系统

| 接地类型 | 接地电阻 | 备注 |
|----------|----------|------|
| 设备保护接地 | ≤4 Ω | 金属外壳必须接地 |
| 控制系统接地 | ≤1 Ω | 独立接地 |
| 防静电接地 | ≤100 Ω | 易燃易爆场所 |
| 避雷接地 | ≤10 Ω | 建筑物避雷系统 |

### 5.3 电气柜安装

| 项目 | 要求 |
|------|------|
| 安装位置 | 干燥、通风、无腐蚀性气体 |
| 防护等级 | ≥IP54（一般场所） |
| 空间要求 | 前面≥800 mm，后面≥600 mm |
| 照明 | 柜内照明≥300 lux |
| 通风 | 强制通风或空调 |

## 6. 验收标准

### 6.1 静态验收

#### 6.1.1 外观检查

| 检查项目 | 合格标准 |
|----------|----------|
| 设备外观 | 完好无损，涂层完整 |
| 安装精度 | 符合技术文件要求 |
| 管道连接 | 正确、牢固、无渗漏 |
| 电气连接 | 正确、牢固、标识清晰 |
| 安全装置 | 齐全、安装正确 |

#### 6.1.2 尺寸检查

| 检查项目 | 允许偏差 | 检验方法 |
|----------|----------|----------|
| 设备定位 | ±3 mm | 钢卷尺 |
| 标高 | ±2 mm | 水准仪 |
| 水平度 | 0.05 mm/m | 水平仪 |
| 垂直度 | 0.1 mm/m | 垂线、水平仪 |
| 间距 | ±5 mm | 钢卷尺 |

#### 6.1.3 紧固检查

| 项目 | 要求 | 检验方法 |
|----------|------|----------|
| 螺栓紧固 | 按规定力矩 | 力矩扳手 |
| 防松措施 | 齐全有效 | 目视检查 |
| 地脚螺栓 | 均匀紧固 | 力矩扳手 |

### 6.2 动态验收

#### 6.2.1 空载试运行

| 测试项目 | 合格标准 | 检验方法 |
|----------|----------|----------|
| 启动性能 | 平稳启动，无异常声响 | 听觉、振动仪 |
| 运行稳定性 | 运行平稳，无异常振动 | 振动仪 |
| 温升 | 符合技术要求 | 温度计 |
| 噪声 | 符合技术要求 | 声级计 |
| 电流 | 空载电流正常 | 钳形电流表 |
| 控制功能 | 各功能正常 | 操作测试 |

#### 6.2.2 负载试运行

| 测试项目 | 合格标准 | 检验方法 |
|----------|----------|----------|
| 产能 | 达到设计产能 | 计量 |
| 产品质量 | 符合质量标准 | 检验 |
| 能耗 | 符合技术要求 | 计量 |
| 稳定性 | 连续运行稳定 | 计时 |
| 安全联锁 | 正常动作 | 模拟测试 |

### 6.3 特殊设备验收

#### 6.3.1 洁净室设备

| 检验项目 | 合格标准 | 检验方法 |
|----------|----------|----------|
| 洁净度 | 符合设计要求 | 颗粒计数器 |
| 压差 | 符合设计要求 | 压差计 |
| 气流流型 | 单向流/乱流符合设计 | 发烟测试 |
| 自净时间 | 符合设计要求 | 颗粒计数器 |
| 温湿度 | 符合设计要求 | 温湿度计 |

#### 6.3.2 制水设备

| 检验项目 | 合格标准 | 检验方法 |
|----------|----------|----------|
| 产水量 | 达到设计产能 | 流量计 |
| 水质 | 符合药典/标准 | 检验 |
| 电导率 | 符合标准 | 电导率仪 |
| 微生物 | 符合标准 | 微生物检验 |
| TOC | 符合标准 | TOC仪 |

#### 6.3.3 灭菌设备

| 检验项目 | 合格标准 | 检验方法 |
|----------|----------|----------|
| 灭菌效果 | 符合标准 | 生物指示剂 |
| 热分布 | F0≥12（湿热） | 热电偶 |
| 穿透速率 | 符合设计 | 热电偶 |
| D值验证 | 符合标准 | 生物指示剂 |

### 7.1 文件清单

#### 7.1.1 基础文件

| 序号 | 文件名称 | 要求 |
|------|----------|------|
| 1 | 设备采购合同 | 原件或复印件 |
| 2 | 设备技术文件 | 说明书、图纸、合格证 |
| 3 | 开箱验收记录 | 签字齐全 |
| 4 | 基础验收记录 | 签字齐全 |

#### 7.1.2 安装记录

| 序号 | 文件名称 | 要求 |
|------|----------|------|
| 1 | 安装施工方案 | 审批齐全 |
| 2 | 安装记录 | 签字齐全 |
| 3 | 隐蔽工程记录 | 签字、照片齐全 |
| 4 | 焊接记录 | 焊缝编号、探伤报告 |
| 5 | 试压/吹洗记录 | 签字齐全 |
| 6 | 绝缘测试记录 | 签字齐全 |
| 7 | 接地电阻测试记录 | 签字齐全 |

#### 7.1.3 验收记录

| 序号 | 文件名称 | 要求 |
|------|----------|------|
| 1 | 分项验收记录 | 签字齐全 |
| 2 | 空载试运行记录 | 数据完整 |
| 3 | 负载试运行记录 | 数据完整 |
| 4 | 性能测试报告 | 盖章齐全 |
| 5 | 验收报告 | 完整签署 |

### 7.2 归档要求

1. **纸质档案**：
   - 一式两份（使用单位、档案室各一份）
   - 装订成册
   - 编目清晰

2. **电子档案**：
   - 扫描存档
   - 备份保存
   - 便于检索

3. **保存期限**：
   - 设备寿命期+5年
   - 关键设备永久保存

## 8. 验收结论

### 8.1 结论分类

| 结论 | 说明 | 处理 |
|------|------|------|
| 合格 | 各项指标符合要求 | 投入使用 |
| 整改后合格 | 存在一般问题，可整改 | 整改后复验 |
| 不合格 | 存在重大问题 | 退回供应商或返工 |

### 8.2 验收签字

| 角色 | 职责 |
|------|------|
| 安装单位 | 确认安装完成，提交验收 |
| 使用单位 | 组织验收，提出意见 |
| 质量部门 | 质量把关，签署意见 |
| 设备部门 | 技术把关，签署意见 |
| 项目负责人 | 综合评价，签署结论 |
