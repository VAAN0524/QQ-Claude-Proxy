# GB 4789.7-2024 食品微生物学检验 副溶血性弧菌检验

## 标准信息

| 项目 | 内容 |
|------|------|
| **标准号** | GB 4789.7-2024 |
| **标准名称** | 食品微生物学检验 副溶血性弧菌检验 |
| **发布日期** | 2024年 |
| **实施日期** | 2024年 |
| **替代标准** | GB 4789.7-2013 |
| **适用范围** | 食品中副溶血性弧菌的检验 |

---

## 1. 范围

本标准规定了食品中副溶血性弧菌（Vibrio parahaemolyticus）的检验方法。

本标准适用于水产品、肉制品、蛋制品、乳制品等食品中副溶血性弧菌的检验。

---

## 2. 设备和材料

### 2.1 设备

- **恒温培养箱**：36℃±1℃、42℃±1℃
- **冰箱**：2℃~5℃
- **恒温水浴箱**：48℃±1℃
- **天平**：感量0.1g
- **均质器**
- **显微镜**：10×~100×
- **无菌吸管、试管、锥形瓶、培养皿**

### 2.2 培养基和试剂

- **3%氯化钠碱性蛋白胨水（APW）**
- **硫代硫酸盐柠檬酸盐胆盐蔗糖琼脂（TCBS）**
- **3%氯化钠胰酪大豆胨琼脂（TSA）**
- **3%氯化钠三糖铁琼脂**
- **嗜盐性试验培养基**
- **神奈川现象试验培养基（我妻氏琼脂）**
- **生化鉴定试剂盒**

---

## 3. 检验程序

```
样品 → 增菌（3%氯化钠APW，37℃，8h~18h）
    ↓
分离（TCBS，37℃，24h）
    ↓
挑选可疑菌落（绿色或蓝绿色菌落）
    ↓
初步生化鉴定（3%氯化钠TSI等）
    ↓
确认试验（嗜盐性、神奈川现象等）
    ↓
报告结果
```

---

## 4. 操作步骤

### 4.1 增菌

称取25g样品，加入225mL 3%氯化钠APW中，均质后于37℃培养8h~18h。

### 4.2 分离

取增菌液划线接种于TCBS平板，37℃培养24h。

### 4.3 菌落特征

**在TCBS上**：
- 副溶血性弧菌：绿色或蓝绿色菌落，直径2mm~3mm
- 霍乱弧菌：黄色菌落
- 其他弧菌：黄色或绿色菌落

### 4.4 初步鉴定

挑取绿色/蓝绿色菌落，进行：

| 试验项目 | 副溶血性弧菌 |
|----------|--------------|
| 氧化酶 | + |
| 3%氯化钠TSI | 底层黄色，斜面黄色，不产生硫化氢 |
| 无盐胨水 | - |
| 3%氯化钠胨水 | + |
| 6%氯化钠胨水 | + |
| 8%氯化钠胨水 | - |
| 10%氯化钠胨水 | - |

### 4.5 确认试验

#### 4.5.1 嗜盐性试验

接种于不同氯化钠浓度的胨水：

| 氯化钠浓度 | 生长情况 |
|------------|----------|
| 0% | - |
| 3% | + |
| 6% | + |
| 8% | - |
| 10% | - |

#### 4.5.2 神奈川现象

在我妻氏琼脂上：
- 阳性菌：菌落周围有β型溶血环
- 阴性菌：无溶血环

#### 4.5.3 生化鉴定

| 试验 | 结果 |
|------|------|
| 葡萄糖产酸 | + |
| 葡萄糖产气 | - |
| 蔗糖 | - |
| 乳糖 | - |
| 阿拉伯糖 | - |
| 靛基质 | + |
| V-P试验 | - |
| 尿素酶 | - |
| 赖氨酸脱羧酶 | + |
| 鸟氨酸脱羧酶 | + |
| 精氨酸双水解酶 | - |
| 明胶液化 | + |
| ONPG | - |

---

## 5. 结果报告

### 5.1 检出副溶血性弧菌
- 分离出符合菌落特征的菌株
- 生化反应符合副溶血性弧菌特性
- 嗜盐性试验阳性

### 5.2 未检出副溶血性弧菌
- 增菌后分离培养无可疑菌落
- 或可疑菌落经鉴定不符合副溶血性弧菌特性

---

## 6. 致病性

### 6.1 食物中毒

副溶血性弧菌是引起食物中毒的主要病原菌之一，主要通过：
- 海产品（鱼、虾、蟹、贝类）
- 交叉污染的肉类、蔬菜

### 6.2 临床表现

- 潜伏期：2h~48h，平均12h
- 症状：腹痛、腹泻（水样便或血水样便）、恶心、呕吐、发热
- 病程：自限性，2d~3d

### 6.3 毒力因子

- **耐热直接溶血毒素（TDH）**：神奈川现象阳性
- **耐热相关溶血毒素（TRH）**：神奈川现象阴性
- **III型分泌系统**

---

## 7. 质量控制

### 7.1 阳性对照
- 副溶血性弧菌ATCC 17802
- 副溶血性弧菌ATCC 33847

### 7.2 阴性对照
- 霍乱弧菌ATCC 14035（黄色菌落）
- 溶藻弧菌ATCC 17749

### 7.3 培养基质控
- TCBS应抑制大部分杂菌生长
- 3%氯化钠APW应促进副溶血性弧菌生长

---

## 8. 安全注意事项

### 8.1 生物安全
- 副溶血性弧菌为B类病原微生物
- 操作时应穿戴防护用品
- 废弃物高压灭菌处理

### 8.2 操作要点
- 样品应低温保存，尽快检验
- 使用含3%氯化钠的培养基
- 注意与其他弧菌的鉴别

---

## 9. 附录

### 9.1 副溶血性弧菌分型

**血清型**：
- O群：O1-O13
- K抗原：K1-K71

**常见流行株**：
- O3:K6
- O4:K8
- O1:K56

### 9.2 检验意义

- **食品安全监测**：海产品必检项目
- **食物中毒诊断**：临床样本和可疑食物检验
- **流行病学调查**：血清分型追踪污染源
