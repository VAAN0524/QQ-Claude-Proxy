# 食品从业人员健康检查规范

## 规范基本信息

| 项目 | 内容 |
|------|------|
| 文号 | GB 31604-2023 |
| 名称 | 食品安全国家标准 食品生产经营从业人员健康检查规范 |
| 发布机构 | 国家卫生健康委员会 |
| 发布日期 | 2023年XX月XX日 |
| 实施日期 | 2024年XX月XX日 |
| 适用范围 | 食品生产经营从业人员 |

## 1. 适用范围

本规范规定了食品生产经营从业人员健康检查的项目、方法、判定标准和管理要求。

适用于：
1. 食品生产企业从业人员
2. 食品经营企业从业人员
3. 餐饮服务从业人员
4. 食品相关产品生产企业从业人员
5. 特殊医学用途配方食品生产企业从业人员
6. 保健食品生产企业从业人员

## 2. 基本要求

### 2.1 检查机构

从业人员健康检查应由具备以下资质的医疗机构承担：
- 县级以上疾病预防控制中心
- 二级及以上综合医院
- 经卫生行政部门批准的专业体检机构

### 2.2 检查周期

| 类别 | 检查频率 |
|------|----------|
| 上岗前检查 | 必须进行 |
| 在岗期间检查 | 每年一次 |
| 患病后复岗 | 疾病治愈后进行 |
| 特殊岗位检查 | 按企业规定 |

## 3. 检查项目

### 3.1 必查项目

#### 3.1.1 传染病检查

| 检查项目 | 检查方法 | 目的 |
|----------|----------|------|
| 活动性肺结核 | 胸部X光检查 | 排除活动性肺结核 |
| 病毒性肝炎 | 肝功能+肝炎病毒标志物 | 排除甲型、戊型肝炎 |
| 伤寒、副伤寒 | 血培养/肥达反应 | 排除伤寒、副伤寒 |
| 细菌性和阿米巴性痢疾 | 粪便培养 | 排除痢疾 |
| 化脓性或渗出性皮肤病 | 体格检查 | 排除相关皮肤病 |

#### 3.1.2 肝功能检查

| 项目 | 正常值范围 | 临床意义 |
|------|------------|----------|
| 丙氨酸氨基转移酶（ALT） | 0-40 U/L | 肝功能损害指标 |
| 天门冬氨酸氨基转移酶（AST） | 0-40 U/L | 肝功能损害指标 |
| 总胆红素（TBIL） | 3.4-20.5 μmol/L | 胆红素代谢指标 |

#### 3.1.3 胸部X光检查

目的：排除活动性肺结核及其他肺部传染性疾病

### 3.2 选查项目

根据岗位需要，可增加以下检查项目：
- 肠道致病菌携带者检查
- 皮肤真菌检查
- 其他根据地方流行病学要求的检查项目

## 4. 禁入疾病判定标准

### 4.1 禁入疾病清单

从业人员患有以下疾病之一的，不得从事接触直接入口食品的工作：

| 疾病类别 | 具体疾病 | 判定依据 |
|----------|----------|----------|
| 呼吸道传染病 | 活动性肺结核 | 胸部X光+痰涂片/培养阳性 |
| 消化道传染病 | 病毒性肝炎（甲型、戊型） | 抗HAV-IgM阳性或抗HEV-IgM阳性 |
| 消化道传染病 | 伤寒、副伤寒 | 血培养/肥达反应阳性 |
| 消化道传染病 | 细菌性痢疾 | 粪便培养志贺菌阳性 |
| 消化道传染病 | 阿米巴性痢疾 | 粪便检查溶组织内阿米巴阳性 |
| 传染病 | 化脓性或渗出性皮肤病 | 体格检查确认 |
| 传染病 | 手部有开放性伤口未愈合 | 体格检查确认 |

### 4.2 判定细则

#### 4.2.1 活动性肺结核

**阳性判定**：
- 胸部X光显示活动性病变
- 痰涂片抗酸染色阳性
- 痰培养结核分枝杆菌阳性

**合格判定**：
- 胸部X光显示陈旧性病灶（钙化灶、纤维化灶）
- 无临床症状
- 痰菌检查阴性

#### 4.2.2 病毒性肝炎

**甲型肝炎**：
- 抗HAV-IgM阳性 → 禁入
- 抗HAV-IgG阳性，IgM阴性 → 合格

**戊型肝炎**：
- 抗HEV-IgM阳性 → 禁入
- 抗HEV-IgG阳性，IgM阴性 → 合格

**乙型、丙型肝炎**：
- 单纯HBsAg阳性或抗HCV阳性，肝功能正常 → 可从事食品工作
- 伴有肝功能异常 → 暂调离岗位，治疗复查

#### 4.2.3 伤寒、副伤寒

**阳性判定**：
- 血培养伤寒/副伤寒沙门菌阳性
- 肥达反应：O凝集效价≥1:80，H凝集效价≥1:160
- 持续发热伴典型临床表现

#### 4.2.4 痢疾

**细菌性痢疾**：
- 粪便培养志贺菌阳性 → 禁入
- 治疗后连续3次粪便培养阴性 → 合格

**阿米巴性痢疾**：
- 粪便检查发现溶组织内阿米巴滋养体 → 禁入
- 治疗后连续3次粪便检查阴性 → 合格

#### 4.2.5 皮肤病

**禁入范围**：
- 化脓性皮炎（脓疱疮、毛囊炎、疖肿等）
- 渗出性皮炎（湿疹、皮炎等）
- 手部银屑病活动期
- 手部真菌感染

**合格判定**：
- 皮损已愈合或得到有效控制
- 不在手部及暴露部位

## 5. 健康证明管理

### 5.1 健康证明格式

健康证明应包含以下信息：
- 从业人员基本信息（姓名、性别、身份证号）
- 检查日期
- 检查机构
- 检查结果
- 有效期限（1年）
- 发证日期和编号
- 检查机构印章

### 5.2 健康证明使用

1. 从业人员上岗必须持有有效健康证明
2. 健康证明应随身携带或在工作场所公示
3. 监管部门检查时应出示健康证明
4. 健康证明到期前1个月应进行复查

### 5.3 档案管理

企业应建立从业人员健康档案，包含：
- 健康证明复印件
- 检查报告存档
- 复查记录
- 患病调离记录
- 复岗评估记录

档案保存期：从业人员离职后不少于3年

## 6. 疾病报告与处理

### 6.1 主动报告

从业人员发现自己患有禁入疾病时，应立即向企业报告，停止接触直接入口食品的工作。

### 6.2 企业处理

企业发现从业人员患有禁入疾病时：
1. 立即调离接触直接入口食品的岗位
2. 安排治疗和休息
3. 治愈后经复查合格方可复岗
4. 做好记录和档案管理

### 6.3 疾病报告

医疗机构发现从业人员患有禁入疾病时：
1. 及时告知从业人员
2. 出具诊断证明
3. 告知从业人员应向企业报告
4. 按规定向疾病预防控制机构报告

## 7. 恢复工作标准

### 7.1 恢复工作条件

| 疾病 | 治愈标准 |
|------|----------|
| 活动性肺结核 | 症状消失，胸部X光病灶稳定或钙化，痰菌连续3次阴性 |
| 甲型/戊型肝炎 | 症状消失，肝功能正常，抗HAV-IgM/抗HEV-IgM转阴 |
| 伤寒/副伤寒 | 症状消失，体温正常，血培养连续3次阴性 |
| 细菌性痢疾 | 症状消失，粪便培养连续3次阴性 |
| 阿米巴痢疾 | 症状消失，粪便检查连续3次阴性 |
| 化脓性皮肤病 | 皮损愈合，无渗出 |

### 7.2 复岗程序

1. 治愈后进行健康复查
2. 取得复查合格证明
3. 企业评估健康状况
4. 办理复岗手续
5. 更新健康档案

## 8. 质量控制

### 8.1 检查机构质量控制

1. 建立健康检查质量控制体系
2. 定期参加室间质评
3. 检验人员持证上岗
4. 建立检查结果申诉机制

### 8.2 企业监督管理

1. 建立从业人员健康管理制度
2. 定期检查健康证明有效期
3. 对新上岗人员进行健康检查
4. 对在岗人员进行年度复查
5. 建立患病人员报告和处理机制

## 9. 记录与报告

### 9.1 检查记录

医疗机构应保存检查记录至少5年，包括：
- 检查申请表
- 检查报告
- X光片/影像资料
- 实验室检验报告

### 9.2 统计报告

医疗机构应定期向卫生行政部门报告：
- 检查人数统计
- 禁入疾病检出情况
- 疾病谱分布
- 异常情况分析
