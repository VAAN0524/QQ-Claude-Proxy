## 附录1：无菌药品

### 一、范围

本附录适用于无菌制剂的制备，包括无菌原料药的粉碎、过筛、混合、分装等生产过程。

**对特医食品的参考价值**：
- 特医食品中的无菌工艺产品（如特殊医学用途婴儿配方食品）可参考本附录
- 洁净区设计和管理要求
- 无菌保证相关要求

### 二、原则

1. **无菌生产的目标**：最大限度地降低微生物、微粒和热原污染的风险
2. **质量风险管理**：采用质量风险管理方法评估和控制风险
3. **洁净度分级**：根据产品特性确定洁净度级别要求

### 三、洁净度级别

| 级别 | 尘粒最大允许数/立方米 | 微生物最大允许数 | 适用区域 |
|------|---------------------|-----------------|----------|
| ≥0.5μm | ≥5μm | 浮游菌/立方米 | 沉降菌/皿 |
| A级 | 3520 | 20 | <1 | <1 | 关键操作区（无菌灌装、无菌装配） |
| B级 | 352000 | 2900 | 10 | 5 | 无菌背景区 |
| C级 | 3520000 | 29000 | 100 | 50 | 辅助区 |
| D级 | 35200000 | 290000 | 200 | 100 | 一般生产区 |

### 四、厂房与设施

#### 4.1 总体要求

1. **洁净区布局**：
   - 洁净区与非洁净区分开
   - 不同洁净级别区域分开
   - 人流、物流分开

2. **压差控制**：
   - 洁净区与非洁净区之间≥10Pa
   - 不同级别洁净区之间≥5Pa
   - 产尘房间保持相对负压

3. **气闸设计**：
   - 设置气闸室防止污染
   - 风淋室、缓冲间设置
   - 联锁控制

#### 4.2 HVAC系统

1. **送风要求**：
   - A级区：单向流，风速0.36-0.54 m/s
   - B级区：单向流或湍流，换气次数≥40次/h
   - C级区：湍流，换气次数≥20次/h
   - D级区：湍流，换气次数≥12次/h

2. **空气过滤**：
   - 初效→中效→高效三级过滤
   - A级区必须使用终端高效过滤器（HEPA）
   - 定期进行完整性测试

3. **温湿度控制**：
   - 温度：18-26℃
   - 相对湿度：45-65%

#### 4.3 设备与管道

1. **设备设计**：
   - 易于清洁和灭菌
   - 无盲管、死角
   - 材质符合要求（316L不锈钢等）

2. **CIP/SIP系统**：
   - 在线清洗系统设计
   - 在线灭菌系统设计
   - 验证要求

### 五、人员要求

#### 5.1 培训

1. **无菌操作培训**：
   - 无菌技术基础培训
   - 更衣程序培训
   - 洁净区行为规范培训

2. **微生物知识培训**：
   - 微生物污染源和控制
   - 无菌操作关键点
   - 污染监测和处理

#### 5.2 健康与卫生

1. **健康检查**：
   - 每年体检
   - 传染病患者调离岗位

2. **着装要求**：
   - A/B级区：无菌更衣程序
   - C/D级区：一般更衣程序
   - 定期更换

### 六、灭菌与消毒

#### 6.1 灭菌方法

1. **湿热灭菌**：
   - 121℃ 30min（过度杀灭法）
   - 验证要求（F0值计算）

2. **干热灭菌**：
   - 160-170℃ 2-4h
   - 用于去热原

3. **辐射灭菌**：
   - 钴-60 γ射线
   - 电子束

#### 6.2 消毒方法

1. **化学消毒剂**：
   - 75%乙醇
   - 0.2%过氧乙酸
   - 2%戊二醛

2. **轮换使用**：防止耐药性产生

### 七、无菌工艺模拟

1. **培养基灌装**：
   - 频率：每半年一次
   - 批量：与生产批量相当
   - 结果：不得有污染

2. **模拟条件**：
   - 最差条件
   - 包括所有操作人员
   - 包括换班

## 附录10：确认与验证

### 一、范围

本附录适用于药品生产设施的确认、设备和工艺的验证。

**对特医食品的参考价值**：
- 清洁验证方法
- 工艺验证要求
- 设备确认程序

### 二、验证生命周期

#### 2.1 验证主计划（VMP）

内容包括：
1. 验证方针
2. 组织机构与职责
3. 设施、系统、设备清单
4. 验证进度计划
5. 文件管理要求

#### 2.2 用户需求说明书（URS）

应包括：
1. 功能需求
2. 性能需求
3. 安全需求
4. 法规符合性要求

### 三、确认类别

#### 3.1 设计确认（DQ）

内容：
1. 符合GMP要求
2. 满足URS要求
3. 材质证明
4. 图纸审核

#### 3.2 安装确认（IQ）

内容：
1. 设备安装位置
2. 公用设施连接
3. 仪器仪表校准
4. 文件完整性

#### 3.3 运行确认（OQ）

内容：
1. 功能测试
2. 报警测试
3. 安全测试
4. 操作参数验证

#### 3.4 性能确认（PQ）

内容：
1. 负载测试
2. 最差条件测试
3. 连续运行验证
4. 工艺一致性

### 四、工艺验证

#### 4.1 验证方法

1. **前验证**：
   - 新产品、新工艺
   - 采用三个连续批次

2. **同步验证**：
   - 特殊情况
   - 加强监控

3. **回顾性验证**：
   - 历史数据分析
   - 至少10-30批次

#### 4.2 验证方案

必须包括：
1. 验证目的
2. 设备和工艺描述
3. 关键质量属性（CQA）
4. 关键工艺参数（CPP）
5. 取样计划
6. 接受标准

### 五、清洁验证

#### 5.1 原则

1. 证明清洁程序的有效性
2. 防止交叉污染
3. 确保产品安全

#### 5.2 最差条件

考虑因素：
1. 溶解性
2. 毒性
3. 活性成分含量
4. 清洁难度

#### 5.3 取样方法

1. **淋洗水取样**：
   - 适用于大面积
   - 回收率验证

2. **棉签擦拭法**：
   - 适用于局部
   - 规范操作

#### 5.4 接受标准

1. **目测清洁**：无可见残留
2. **化学残留**：
   - MACO（最大允许残留量）
   - 限度：≤10ppm或≤日剂量的0.1%
3. **微生物限度**：符合规定

### 六、再验证

#### 6.1 变更后验证

重大变更后必须验证：
1. 原料变更
2. 工艺变更
3. 设备变更
4. 设施变更

#### 6.2 定期再验证

周期：
1. 关键工艺：每年
2. 重要工艺：每两年
3. 一般工艺：每三年

## 附录11：质量控制实验室

### 一、范围

本附录适用于质量控制实验室的管理。

**对特医食品的参考价值**：
- 检验方法验证
- 实验室管理要求
- 稳定性考察

### 二、人员

1. **资质要求**：
   - 专业背景
   - 培训记录
   - 上岗考核

2. **职责**：
   - 检验操作
   - 结果审核
   - 仪器维护

### 三、设施与设备

#### 3.1 实验室布局

1. **功能分区**：
   - 仪器室
   - 理化检验室
   - 微生物检验室
   - 标准品室
   - 留样室

2. **环境要求**：
   - 温湿度控制
   - 防尘措施
   - 照明要求

#### 3.2 仪器设备

1. **校准**：
   - 周期校准
   - 校准记录
   - 校准标识

2. **维护**：
   - 预防性维护
   - 维护记录
   - 备用设备

### 四、标准品与对照品

1. **来源**：
   - 国家标准品
   - 工作对照品

2. **管理**：
   - 专人保管
   - 储存条件
   - 使用记录

### 五、检验方法验证

#### 5.1 验证项目

| 类型 | 鉴别 | 杂质测定 | 含量测定 | 溶出度 |
|------|------|----------|----------|--------|
| 准确度 | - | + | + | + |
| 精密度 | - | + | + | + |
| 专属性 | + | + | + | + |
| 检出限 | - | + | - | - |
| 定量限 | - | + | - | - |
| 线性 | - | + | + | + |
| 范围 | - | + | + | + |
| 耐用性 | + | + | + | + |

#### 5.2 方法转移

1. **转移类型**：
   - 实验室间转移
   - 仪器间转移

2. **转移方案**：
   - 对比试验
   - 接受标准

### 六、稳定性考察

#### 6.1 考察类型

1. **影响因素试验**：
   - 高温
   - 高湿
   - 光照

2. **加速试验**：
   - 40℃/75%RH
   - 6个月

3. **长期试验**：
   - 25℃/60%RH
   - 根据有效期

#### 6.2 考察项目

根据产品特性确定，一般包括：
1. 外观
2. 含量
3. 有关物质
4. 微生物限度

### 七、取样

1. **取样计划**：
   - 统计抽样
   - 代表性

2. **取样方法**：
   - 无菌取样
   - 密闭容器

3. **取样记录**：
   - 时间、地点
   - 取样人

### 八、数据完整性

1. **ALCOA原则**：
   - A：Attributable（可归属）
   - L：Legible（清晰）
   - C：Contemporaneous（同时）
   - O：Original（原始）
   - A：Accurate（准确）

2. **数据管理**：
   - 电子数据备份
   - 访问权限
   - 审计追踪

## 附录：计算机化系统

### 一、范围

适用于药品生产使用的计算机化系统。

### 二、风险分级

| 级别 | 风险 | 示例 | 要求 |
|------|------|------|------|
| 1 | 高 | 关键工艺控制、电子记录 | 完全验证 |
| 2 | 中 | 监测系统、数据采集 | 部分验证 |
| 3 | 低 | 辅助系统 | 基础验证 |
| 4 | 无 | 标准软件 | 供应商评估 |
| 5 | 无 | 基础设施 | 不验证 |

### 三、验证要求

1. **硬件验证**：
   - 安装确认
   - 运行确认

2. **软件验证**：
   - 设计阶段验证
   - 测试
   - 用户验收测试

### 四、数据安全

1. **备份**：
   - 定期备份
   - 异地存储
   - 恢复测试

2. **权限**：
   - 分级管理
   - 密码保护
   - 审计追踪

## 附录：确认与验证补充

### 一、清洁验证补充

#### 1.1 清洁剂选择

1. 水
2. 稀酸/稀碱
3. 有机溶剂
4. 洗涤剂

#### 1.2 清洁方法

1. 人工清洁
2. CIP（在线清洁）
3. COP（定位清洁）

#### 1.3 验证方案要素

1. 产品/设备信息
2. 清洁程序描述
3. 取样位置图
4. 分析方法
5. 接受标准
6. 再验证周期

### 二、工艺验证补充

#### 2.1 关键质量属性（CQA）

指产品质量必须控制的属性：
1. 含量
2. 纯度
3. 杂质
4. 溶出度
5. 微生物限度

#### 2.2 关键工艺参数（CPP）

影响CQA的工艺参数：
1. 温度
2. 压力
3. 时间
4. 速度
5. pH值

#### 2.3 设计空间

在设计的空间内，工艺参数变化不影响产品质量：
1. 设计空间开发
2. 控制策略
3. 生命周期管理

### 三、运输验证

1. **验证内容**：
   - 温度分布
   - 极端条件
   - 运输时间

2. **监控方法**：
   - 温度记录仪
   - 定期监测
