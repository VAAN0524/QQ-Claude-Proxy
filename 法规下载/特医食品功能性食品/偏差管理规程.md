# 偏差管理规程

## 规程基本信息

| 项目 | 内容 |
|------|------|
| 制定机构 | 企业质量管理部门 |
| 适用范围 | 食品生产全过程偏差管理 |
| 规程性质 | 企业内部管理规程 |

## 1. 总则

### 1.1 目的

规范偏差管理流程，确保所有偏差得到及时调查、处理和纠正，防止偏差再次发生，保障产品质量和安全。

### 1.2 适用范围

本规程适用于：
1. 生产过程偏差
2. 检验偏差
3. 设备偏差
4. 物料偏差
5. 质量偏差
6. 其他偏差

### 1.3 基本原则

| 原则 | 说明 |
|------|------|
| **及时报告** | 发现偏差立即报告 |
| **彻底调查** | 彻底调查根本原因 |
| **有效纠正** | 采取有效纠正措施 |
| **预防再发** | 采取预防措施 |
| **闭环管理** | 偏差处理闭环 |

## 2. 偏差分类

### 2.1 按严重程度分类

| 级别 | 定义 | 示例 | 响应时间 |
|------|------|------|----------|
| **一级** | 严重影响产品质量安全 | 微生物超标、异物混入 | 立即 |
| **二级** | 可能影响产品质量安全 | 参数超限、记录错误 | 24小时内 |
| **三级** | 轻微偏差，不影响产品质量 | 文件小错误、标签小瑕疵 | 3天内 |

### 2.2 按来源分类

| 来源 | 说明 | 示例 |
|------|------|------|
| **人员偏差** | 人员操作失误 | 投料错误、记录错误 |
| **设备偏差** | 设备故障 | 设备停机、参数异常 |
| **物料偏差** | 物料问题 | 原料不合格、包装破损 |
| **工艺偏差** | 工艺参数偏离 | 温度超限、时间不足 |
| **环境偏差** | 环境条件异常 | 洁净度超标、温湿度异常 |
| **文件偏差** | 文件问题 | 记录不全、文件错误 |

### 2.3 按发现时机分类

| 类型 | 说明 |
|------|------|
| **即时偏差** | 操作过程中发现 |
| **事后偏差** | 记录审核时发现 |
| **客户投诉** | 客户反馈发现 |
| **监管检查** | 检查时发现 |

```
    ↓

```

## 4. 偏差调查

### 4.1 调查组织

| 人员 | 职责 |
|------|------|
| 调查组长 | 组织调查、协调资源 |
| 质量代表 | 质量评估 |
| 生产代表 | 生产情况说明 |
| 技术代表 | 技术支持 |
| 其他 | 根据需要 |

### 4.2 调查方法

#### 4.2.1 5Why分析法

| 步骤 | 问题 | 示例 |
|------|------|------|
| 1Why | 为什么发生 | 温度超限 |
| 2Why | 为什么温度超限 | 传感器故障 |
| 3Why | 为什么传感器故障 | 未定期校准 |
| 4Why | 为什么未校准 | 校准计划未执行 |
| 5Why | 为什么计划未执行 | 人员培训不足 |

#### 4.2.2 鱼骨图分析

| 类别 | 常见原因 |
|------|----------|
| 人员 | 培训不足、疲劳、疏忽 |
| 设备 | 故障、老化、维护不当 |
| 物料 | 不合格、错误 |
| 方法 | 规程不完善、不明确 |
| 环境 | 温湿度、照明、噪音 |
| 管理 | 监督不足、资源不足 |

#### 4.2.3 头脑风暴

| 原则 | 说明 |
|------|------|
| 延迟评判 | 不批评任何想法 |
| 自由发挥 | 鼓励创意 |
| 追求数量 | 想法越多越好 |
| 综合改进 | 在他人想法上改进 |

### 4.3 数据收集

| 监控数据 | 查阅监控记录 |

## 5. 风险评估

### 5.1 风险评估方法

#### 5.1.1 FMEA（失效模式与影响分析）

| 评分维度 | 1分 | 2分 | 3分 | 4分 | 5分 |
|----------|-----|-----|-----|-----|-----|
| 严重度 | 轻微 | 一般 | 严重 | 很严重 | 灾难性 |
| 发生概率 | 极低 | 低 | 中 | 高 | 很高 |
| 可检测性 | 肯定检测 | 很可能 | 可能 | 不太可能 | 不能检测 |

风险优先数（RPN）= 严重度 × 发生概率 × 可检测性

| RPN值 | 风险等级 | 处理要求 |
|-------|----------|----------|
| 1-50 | 低风险 | 可接受，常规监控 |
| 51-100 | 中风险 | 需采取措施降低 |
| 101-125 | 高风险 | 必须采取措施 |
| >125 | 极高风险 | 立即停产 |

### 5.2 影响评估

| 评估项目 | 说明 |
|----------|------|
| 产品质量 | 对产品质量的影响 |
| 产品安全 | 对产品安全的影响 |
| 合规性 | 对法规符合性的影响 |
| 客户 | 对客户的影响 |
| 成本 | 经济损失 |
| 声誉 | 对企业声誉的影响 |

## 6. 纠正和预防措施（CAPA）

### 6.1 CAPA分类

| 类型 | 说明 | 示例 |
|------|------|------|
| **纠正** | 立即纠正偏差 | 重新生产、返工 |
| **纠正措施** | 消除偏差原因 | 修改规程、培训人员 |
| **预防措施** | 防止再发 | 完善体系、优化流程 |

### 6.2 CAPA制定

#### 6.2.1 制定原则

| 原则 | 说明 |
|------|------|
| 针对性 | 针对根本原因 |
| 有效性 | 措施有效 |
| 可行性 | 可实施 |
| 及时性 | 及时实施 |
| 验证性 | 可以验证效果 |

#### 6.2.2 CAPA内容

| 项目 | 内容 |
|------|------|
| 措施描述 | 详细描述措施 |
| 责任人 | 明确责任人 |
| 完成期限 | 明确完成时间 |
| 资源需求 | 所需资源 |
| 验证方法 | 如何验证效果 |

### 6.3 CAPA示例

| 根本原因 | 纠正措施 | 预防措施 |
|----------|----------|----------|
| 传感器故障 | 更换传感器 | 增加校准频率 |
| 培训不足 | 培训操作人员 | 完善培训体系 |
| 规程不明确 | 修订规程 | 加强规程审核 |
| 维护不当 | 加强维护 | 优化维护计划 |

## 7. 偏差关闭

### 7.1 关闭条件

| 条件 | 说明 |
|------|------|
| 调查完成 | 调查报告完成 |
| 原因明确 | 根本原因明确 |
| CAPA完成 | 纠正预防措施完成 |
| 效果验证 | CAPA效果验证 |
| 文件更新 | 相关文件已更新 |
| 培训完成 | 相关人员已培训 |

### 7.2 关闭流程

```
调查完成
    ↓
制定CAPA
    ↓
实施CAPA
    ↓
验证效果
    ↓
更新文件
    ↓
完成培训
    ↓
偏差关闭
```

### 7.3 效果验证

| 验证方法 | 说明 |
|----------|------|
| 定期检查 | 定期检查措施执行情况 |
| 数据分析 | 分析相关数据趋势 |
| 现场审核 | 现场审核措施效果 |
| 重新发生 | 是否再次发生类似偏差 |
