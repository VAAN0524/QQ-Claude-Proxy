# 特医食品生产验证管理指南2026版

## 发布信息

- **编制机构**：国家市场监督管理总局食品审评中心
- **发布日期**：2026年X月X日
- **适用范围**：特殊医学用途配方食品生产企业
- **文件编号**：CFSA-SV-2026-001

---

## 前言

本指南为特殊医学用途配方食品生产企业提供验证管理的标准化方法，确保厂房设施、设备、工艺和检验方法能够持续满足预定用途。

本指南依据《特殊医学用途配方食品生产许可审查细则》《药品生产质量管理规范》等相关法规标准编制。

---

## 第1章 总则

**1.1 验证目的**

1. 证明厂房设施、设备、工艺和检验方法能够满足预定用途
2. 确保产品质量的一致性和可靠性
3. 降低产品质量风险
4. 满足法规要求

**1.2 验证原则**

1. 验证应当有科学依据
2. 验证应当有明确的接受标准
3. 验证应当有完整的记录
4. 验证结论应当有数据支持

**1.3 验证分类**

| 验证类型 | 验证对象 | 验证周期 |
|----------|----------|----------|
| 前验证 | 新设施、新设备、新工艺 | 使用前 |
| 回顾性验证 | 已有设施、设备、工艺 | 定期进行 |
| 再验证 | 变更后的验证 | 变更后 |
| 同步验证 | 正常生产中验证 | 特殊情况 |

## 第3章 厂房设施验证

**3.1 厂房验证**

**3.1.1** 验证项目：

| 验证项目 | 验证内容 | 接受标准 |
|----------|----------|----------|
| 压差验证 | 相邻区域压差 | ≥10 Pa |
| 气流流型 | 气流方向和均匀性 | 符合设计要求 |
| 悬浮粒子 | 粒子浓度 | 符合洁净度级别 |
| 沉降菌 | 菌落数 | 符合规定要求 |
| 照度 | 工作面照度 | ≥300 Lux |
| 噪声 | 生产区噪声 | ≤65 dB(A) |
| 温湿度 | 温度和相对湿度 | 符合设计要求 |

**3.1.2** 验证方法：

1. **压差验证**：使用微压计测量
2. **气流流型**：使用发烟剂观察
3. **悬浮粒子**：使用粒子计数器
4. **沉降菌**：使用沉降法
5. **照度**：使用照度计
6. **噪声**：使用声级计
7. **温湿度**：使用温湿度记录仪

**3.2 洁净室验证**

**3.2.1** 验证项目：

1. 空气过滤器完整性
2. 气流速度和均匀性
3. 换气次数
4. 自净时间
5. 密闭性测试

**3.2.2** 接受标准：

| 验证项目 | 接受标准 |
|----------|----------|
| 高效过滤器完整性 | 泄漏率≤0.01% |
| 单向流风速 | 0.3-0.5 m/s（ISO 5级） |
| 换气次数 | 符合设计要求 |
| 自净时间 | ≤15分钟（ISO 5级） |
| 密闭性 | 压差恢复时间≤10分钟 |

**3.3 水系统验证**

**3.3.1** 验证项目：

| 验证项目 | 验证内容 | 接受标准 |
|----------|----------|----------|
| 制水设备 | 产水水质 | 符合药典标准 |
| 储配水系统 | 水质稳定性 | 符合药典标准 |
| 分配系统 | 微生物控制 | ≤100 CFU/100mL |
| 纯蒸汽 | 蒸汽质量 | 符合设计要求 |

**3.3.2** 验证方法：

1. **化学指标**：取样检验
2. **微生物指标**：膜过滤法
3. **内毒素**：鲎试验法
4. **电导率**：在线监测
5. **TOC**：在线监测

**3.4 蒸汽系统验证**

**3.4.1** 验证项目：

1. 蒸汽质量
2. 蒸汽纯度
3. 蒸汽压力稳定性
4. 凝水质量

**3.4.2** 接受标准：

| 验证项目 | 接受标准 |
|----------|----------|
| 蒸汽纯度 | 符合EN 285标准 |
| 蒸汽压力 | 波动≤±10% |
| 凝水电导率 | ≤5 μS/cm |
| 凝水TOC | ≤500 ppb |

## 第4章 设备验证

**4.1 设备验证（IQ）

**4.1.1** 验证内容：

1. 设备型号规格
2. 设备安装位置
3. 设备校准状态
4. 设备文件资料
5. 设备配套设施
6. 设备安全装置

**4.1.2** 验证文件：

1. 设备采购合同
2. 设备技术文件
3. 设备安装图纸
4. 设备校准证书
5. 设备操作手册
6. 设备维护手册

**4.2 运行验证（OQ）

**4.2.1** 验证内容：

1. 设备运行参数
2. 设备运行稳定性
3. 设备安全功能
4. 设备报警功能
5. 设备操作功能
6. 设备记录功能

**4.2.2** 验证方法：

1. 空载运行试验
2. 负载运行试验
3. 极限条件试验
4. 安全功能试验
5. 报警功能试验

**4.3 性能验证（PQ）

**4.3.1** 验证内容：

1. 实际生产条件运行
2. 产品质量检验
3. 工艺参数验证
4. 设备能力验证
5. 持续运行验证

**4.3.2** 验证方法：

1. 连续3批产品
2. 对比验证数据
3. 统计分析数据
4. 评估设备能力

**4.4 清洁验证**

**4.4.1** 验证原则：

1. 最差条件原则
2. 残留限度原则
3. 可回收性原则
4. 安全性原则

**4.4.2** 验证方法：

1. **目视检查**：无可见残留物
2. **化学残留**：擦拭法或淋洗法
3. **微生物残留**：接触碟法或棉签擦拭法

**4.4.3** 接受标准：

| 残留类型 | 接受标准 |
|----------|----------|
| 目视检查 | 无可见残留物 |
| 化学残留 | ≤10 ppm或≤10 μg/100 cm² |
| 微生物残留 | ≤25 CFU/棉签或≤25 CFU/接触碟 |

## 第5章 工艺验证

**5.1 工艺验证设计**

**5.1.1** 验证策略：

| 验证阶段 | 批次要求 | 验证目的 |
|----------|----------|----------|
| 开发阶段 | 1-3批 | 确定工艺参数范围 |
| 试生产阶段 | 3批 | 确认工艺可行性 |
| 商业生产 | 3批 | 证明工艺稳健性 |

**5.1.2** 验证批次：

1. 连续生产3批
2. 使用商业生产设备
3. 按照正常生产条件
4. 按照正常生产人员

**5.2 工艺参数验证**

**5.2.1** 关键工艺参数（CPP）：

| 工艺步骤 | 关键参数 | 目标范围 | 验证方法 |
|----------|----------|----------|----------|
| 原料处理 | 投料量、混合时间 | ±2% | 称重、计时 |
| 制粒工艺 | 搅拌速度、制粒时间 | ±5% | 现场监测 |
| 干燥工艺 | 干燥温度、干燥时间 | ±3℃、±5% | 温度记录、计时 |
| 混合工艺 | 混合速度、混合时间 | ±5% | 现场监测 |
| 包装工艺 | 封口温度、包装速度 | ±5℃、±5% | 温度记录、计数 |

**5.2.2** 关键质量属性（CQA）：

| 产品类型 | 关键质量属性 | 规格标准 | 检验方法 |
|----------|--------------|----------|----------|
| 粉状产品 | 粒度分布 | D50: 100-300 μm | 筛分法 |
| 颗粒产品 | 含量均匀度 | RSD ≤5% | 含量测定 |
| 液体产品 | 悬浮稳定性 | 沉降体积比 | 仪器法 |
| 胶囊产品 | 溶出度 | Q≥80% | 溶出仪 |

**5.3 统计分析**

**5.3.1** 工艺能力分析：

1. 计算工艺能力指数（Cpk）
2. 评估工艺能力
3. 确定工艺参数范围
4. 建立控制策略

**5.3.2** 接受标准：

| 工艺能力 | Cpk值 | 结论 |
|----------|-------|------|
| 充分能力 | Cpk ≥1.33 | 接受 |
| 基本能力 | 1.00 ≤ Cpk < 1.33 | 有条件接受 |
| 能力不足 | Cpk < 1.00 | 不接受 |

## 第6章 检验方法验证

**6.1 验证原则**

**6.1.1** 验证要求：

1. 所有检验方法使用前应当验证
2. 验证应当证明方法适用
3. 验证应当有完整记录
4. 验证应当定期回顾

**6.2 验证项目**

**6.2.1** 鉴别试验：

| 验证指标 | 要求 | 方法 |
|----------|------|------|
| 专属性 | 能区分目标物质 | 阴性对照试验 |
| 检出限 | 能检出低浓度 | 系列稀释试验 |
| 耐用性 | 能检测不同样品 | 不同样品测试 |

**6.2.2** 杂质检查：

| 验证指标 | 要求 | 方法 |
|----------|------|------|
| 检出限 | 能检出杂质 | 系列稀释试验 |
| 定量限 | 能定量杂质 | 精密度试验 |
| 准确度 | 回收率符合要求 | 加样回收试验 |
| 专属性 | 能分离杂质 | 分离度试验 |

**6.2.3** 含量测定：

| 验证指标 | 要求 | 方法 |
|----------|------|------|
| 准确度 | 回收率98%-102% | 加样回收试验 |
| 精密度 | RSD ≤2% | 重复试验 |
| 线性 | r ≥0.999 | 线性范围试验 |
| 范围 | 覆盖规定浓度 | 线性范围试验 |
| 耐用性 | 抗干扰 | 干扰试验 |
| 稳健性 | 小变化不影响 | 参数变化试验 |

**6.3 微生物检验方法验证**

**6.3.1** 验证项目：

| 验证项目 | 要求 | 方法 |
|----------|------|------|
| 专属性 | 能检出目标菌 | 阳性对照试验 |
| 检出限 | 能检出低菌量 | 系列稀释试验 |
| 计数准确度 | 回收率≥70% | 加样回收试验 |
| 重复性 | RSD ≤35% | 重复试验 |
| 稳健性 | 小变化不影响 | 参数变化试验 |

## 第7章 运输验证

**7.1 运输验证原则**

**7.1.1** 验证目的：

1. 证明运输过程不影响产品质量
2. 确定运输条件和监控方法
3. 评估运输风险
4. 制定运输规程

**7.1.2** 验证场景：

| 运输场景 | 验证重点 | 验证周期 |
|----------|----------|----------|
| 常规运输 | 温度、湿度 | 季节性验证 |
| 极端天气 | 高温、低温 | 每年验证 |
| 长途运输 | 时间、震动 | 按需要验证 |
| 国际运输 | 多环节 | 按需要验证 |

**7.2 验证方法**

**7.2.1** 验证工具：

1. 温湿度记录仪
2. 震动记录仪
3. GPS跟踪器
4. 数据记录仪

**7.2.2** 验证步骤：

1. 安装监控设备
2. 模拟实际运输
3. 记录运输数据
4. 检验产品质量
5. 评估运输效果

**7.3 接受标准**

**7.3.1** 温度要求：

| 产品类型 | 运输温度 | 偏离限度 |
|----------|----------|----------|
| 特医食品 | 0-25℃ | 短时±5℃ |
| 低温产品 | 2-8℃ | 短时±2℃ |
| 冷冻产品 | -18℃以下 | 短时±5℃ |

**7.3.2** 湿度要求：

| 产品类型 | 运输湿度 | 偏离限度 |
|----------|----------|----------|
| 普通产品 | 35%-75% | 短时±10% |
| 吸湿产品 | 35%-65% | 短时±5% |
| 防潮产品 | 45%以下 | 短时±5% |

## 第9章 再验证

**9.1 再验证触发条件**

**9.1.1** 应当进行再验证的情况：

1. 设施、设备重大变更
2. 工艺、配方重大变更
3. 产品质量异常
4. 验证周期到期
5. 监管要求变化

**9.2 再验证策略**

**9.2.1** 部分再验证：

| 变更类型 | 再验证范围 |
|----------|------------|
| 设备更新 | 设备验证 |
| 工艺调整 | 工艺验证 |
| 场地变更 | 设施验证 |
| 方法更新 | 方法验证 |

**9.2.2** 全面再验证：

1. 设施验证
2. 设备验证
3. 工艺验证
4. 方法验证
5. 运输验证

**9.3 再验证周期**

| 验证类型 | 再验证周期 |
|----------|------------|
| 洁净室验证 | 每年1次 |
| 关键设备验证 | 每2-3年 |
| 工艺验证 | 每3-5年 |
| 检验方法 | 每3-5年 |
| 运输验证 | 按需要 |
