# ISO 22000 食品安全管理体系标准

> 标准号：ISO 22000:2018
> 标准名称：Food safety management systems — Requirements for any organization in the food chain
> 中文名称：食品安全管理体系 食品链中各类组织的要求
> 发布机构：国际标准化组织（ISO）
> 发布日期：2018年6月
> 替代版本：ISO 22000:2005

---

## 一、标准概述

### 1.1 标准背景
ISO 22000:2018是国际标准化组织发布的食品安全管理体系国际标准，适用于食品链中各类组织，确保食品安全控制体系的一致性和有效性。

### 1.2 标准目的
- 确保食品安全危害得到识别、评估和控制
- 增强食品安全管理体系的透明度和可信度
- 促进食品链中各类组织的协调合作
- 满足食品安全相关的法律法规要求

### 1.3 适用范围
本标准适用于食品链中各类组织，无论其规模大小，包括：
- 饲料生产者
- 初级食品生产者
- 食品制造者
- 运输和贮存经营者
- 零售商
- 餐饮服务提供者
- 其他相关服务提供者

### 1.4 与中国标准的对应
中国国家标准GB/T 22000-2006《食品安全管理体系 食品链中各类组织的要求》等同采用ISO 22000:2005。
目前最新的GB/T 22000正在修订中，拟等同采用ISO 22000:2018。

---

## 二、核心原则

### 2.1 PDCA循环
ISO 22000:2018采用PDCA（计划-执行-检查-改进）循环与HACCP原理相结合的模式：

```
计划（Plan）
├── 食品安全方针和目标
├── 危害分析和控制措施
└── 操作性前提方案（OPRP）和HACCP计划

执行（Do）
├── 实施控制措施
├── 监视和测量
└── 信息沟通

检查（Check）
├── 验证控制措施
├── 内部审核
└── 管理评审

改进（Act）
├── 持续改进
├── 纠正措施
└── 更新体系
```

### 2.2 食品安全管理体系框架

| 要素 | 内容 |
|------|------|
| 管理职责 | 领导作用、食品安全方针、职责权限 |
| 资源管理 | 人力资源、基础设施、工作环境 |
| 安全产品实现 | 前提方案、危害分析、HACCP计划 |
| 验证分析改进 | 监视测量、验证、内部审核、改进 |

## 三、标准主要条款

### 第4章：组织环境
**4.1 理解组织及其环境**
组织应确定与其目标和战略方向相关并影响其实现食品安全管理体系预期结果的能力的内部和外部问题。

**4.2 理解相关方的需求和期望**
组织应确定：
- 与食品安全管理体系有关的相关方
- 这些相关方的有关要求和期望
- 这些要求和期望中哪些是合规义务

**4.3 确定食品安全管理体系的范围**
组织应确定食品安全管理体系的边界和适用性，以确定其范围。

**4.4 食品安全管理体系**
组织应按照本标准的要求，建立、实施、保持和持续改进食品安全管理体系。

### 第5章：领导作用
**5.1 领导作用和承诺**
最高管理者应通过以下方式证明其对食品安全管理体系的领导作用和承诺：
- 对食品安全管理体系的有效性负责
- 确保制定食品安全方针和目标
- 确保食品安全管理体系要求融入组织的业务过程
- 确保获得食品安全管理体系所需的资源

**5.2 食品安全方针**
最高管理者应制定、实施和保持食品安全方针，方针应：
- 与组织的宗旨和战略方向相适应
- 包含符合食品安全法律法规的承诺
- 包含持续改进食品安全管理体系的承诺
- 提供制定和评审食品安全目标的框架

**5.3 组织岗位、职责和权限**
最高管理者应确保组织内相关岗位的职责、权限得到分派、沟通和理解。

### 第6章：策划
**6.1 应对风险和机遇的措施**
组织应策划：
- 应对风险和机遇的措施
- 如何在食品安全管理体系过程中整合并实施这些措施
- 如何评价这些措施的有效性

**6.2 食品安全目标及其策划**
组织应在相关职能、层次和过程上建立食品安全目标，目标应：
- 与食品安全方针保持一致
- 可测量
- 考虑适用的要求
- 被监视
- 被沟通
- 适时更新

**6.3 变更的策划**
当组织确定需要对食品安全管理体系进行变更时，变更应按策划的方式实施。

### 第7章：支持
**7.1 资源**
组织应确定并提供建立、实施、保持和持续改进食品安全管理体系所需的资源。

**7.2 能力**
组织应：
- 确定在其控制下工作的人员所需具备的能力
- 确保这些人员是胜任的
- 适用时，采取措施获得所需能力

**7.3 意识**
在组织控制下工作的人员应：
- 知晓食品安全方针
- 知晓他们对食品安全管理体系有效性的贡献
- 知晓不符合食品安全管理体系要求的后果

**7.4 沟通**
组织应确定与食品安全管理体系相关的内部和外部沟通。

**7.5 成文信息**
组织应控制和保持食品安全管理体系所需的成文信息。

### 第8章：运行
**8.1 运行策划和控制**
组织应策划、实施和控制满足食品安全产品和服务要求所需的过程。

**8.2 前提方案（PRP）**
组织应建立、实施和保持前提方案，以助于：
- 控制食品安全危害
- 通过工作环境控制产品和产品的食品安全性

**8.3 可追溯性**
组织应建立可追溯性系统，确保能够识别产品批次及其与原料批次、生产和交付记录的关系。

**8.4 应急准备和响应**
组织应识别和评估可能影响食品安全的紧急情况和事故，并建立应急准备和响应程序。

**8.5 危害控制**
- **危害分析**：组织应实施危害分析，以确定需要控制的危害
- **控制措施**：确定和评价控制措施
- **HACCP计划**：建立HACCP计划
- **操作性前提方案（OPRP）**：建立操作性前提方案

### 第9章：绩效评价
**9.1 监视、测量、分析和评价**
组织应监视、测量、分析和评价其食品安全管理体系的绩效和有效性。

**9.2 内部审核**
组织应按照策划的时间间隔进行内部审核，以提供食品安全管理体系是否符合本标准要求的信息。

**9.3 管理评审**
最高管理者应按照策划的时间间隔评审组织的食品安全管理体系。

### 第10章：改进
**10.1 不符合和纠正措施**
当出现不符合时，组织应：
- 对不符合做出响应，适用时采取措施控制和纠正不符合
- 评价消除不符合原因的措施的需求

**10.2 改进**
组织应持续改进食品安全管理体系的适宜性、充分性和有效性。

**10.3 更新**
组织应利用监视、测量、分析、验证、内部审核和管理评审的结果，更新食品安全管理体系。

## 四、关键概念

### 4.1 HACCP原理在ISO 22000中的应用

| HACCP原理 | ISO 22000对应条款 |
|-----------|------------------|
| 1. 进行危害分析 | 8.5.1 危害分析 |
| 2. 确定关键控制点（CCP） | 8.5.2 控制措施的选择和评价 |
| 3. 建立关键限值 | 8.5.3 确定关键控制点的关键限值 |
| 4. 建立监控程序 | 8.5.4 建立关键控制点的监视系统 |
| 5. 建立纠正措施 | 8.5.5 建立关键控制点的纠正措施 |
| 6. 建立验证程序 | 8.8 控制措施的确认和验证 |
| 7. 建立记录保持程序 | 7.1.5 成文信息 |

### 4.2 前提方案（PRP）
前提方案是为保障食品安全卫生而必须遵守的基本条件和活动，包括：
- 建筑物和设施的布局和构造
- 空气、水、能源和其他供应品的适宜性
- 设施的清洁、消毒和虫害控制
- 人员卫生和员工能力
- 交叉污染的预防
- 化学品的控制
- 医疗废弃物和废弃物处置
- 追溯和召回

### 4.3 操作性前提方案（OPRP）
操作性前提方案是通过危害分析确定的、必需控制食品安全危害的前提方案，其控制措施是通过监视获得可测量的结果。

### 4.4 关键控制点（CCP）
关键控制点是能够进行控制，并且该控制对防止、消除食品安全危害或将其降低到可接受水平所必需的步骤。

## 五、与其他标准的关系

### 5.1 ISO 9001质量管理体系
ISO 22000:2018与ISO 9001:2015采用相同的高层结构（HLS），便于组织整合质量管理和食品安全管理体系。

### 5.2 HACCP体系
ISO 22000包含了HACCP原理和步骤，同时将其整合在更广泛的管理体系框架中。

### 5.3 FSSC 22000食品安全体系认证
FSSC 22000是基于ISO 22000、ISO 22002-1（前提方案）和FSSC 22000附加要求的食品安全认证体系。

### 5.4 与中国法规的对应

| 中国法规/标准 | ISO 22000对应关系 |
|---------------|------------------|
| 食品安全法 | 法律要求 |
| GB 14881 食品生产通用卫生规范 | 前提方案（PRP） |
| GB/T 22000-2006 | 等同采用ISO 22000:2005 |
| HACCP体系认证实施规则 | HACCP原理应用 |
| 食品安全管理体系认证实施规则 | 认证要求 |

## 六、实施步骤

### 6.1 建立食品安全管理体系

```
第一步：准备阶段
├── 获得管理层承诺
├── 成立食品安全小组
└── 制定实施计划

第二步：现状评估
├── 诊断现有体系
├── 识别差距
└── 制定改进方案

第三步：体系设计
├── 确定范围
├── 制定食品安全方针
├── 确定食品安全目标
└── 编制体系文件

第四步：体系实施
├── 培训员工
├── 运行体系
└── 记录运行数据

第五步：体系验证
├── 内部审核
├── 管理评审
└── 采取纠正措施

第六步：认证审核
├── 选择认证机构
├── 申请认证
└── 通过审核获得证书
```

### 6.2 文件结构要求

| 文件层次 | 内容 | 示例 |
|---------|------|------|
| 一级文件 | 食品安全手册 | 描述体系整体框架 |
| 二级文件 | 程序文件 | 描述各项活动的程序 |
| 三级文件 | 作业指导书 | 具体操作方法 |
| 四级文件 | 记录表格 | 活动的证据 |

## 七、认证要求

### 7.1 认证依据
在中国，食品安全管理体系认证的依据是：
- CNCA-N-007:2021《食品安全管理体系认证实施规则》

### 7.2 认证流程
1. 申请认证：提交申请材料
2. 评审受理：认证机构评审申请
3. 审核策划：制定审核计划
4. 现场审核：一阶段和二阶段审核
5. 审核决定：认证机构做出决定
6. 监督审核：年度监督审核
7. 复评：三年复评

### 7.3 证书有效期
- 证书有效期：3年
- 监督审核：每年1次
- 复评审核：3年期满前进行

## 八、在特医食品和保健食品中的应用

### 8.1 特殊医学用途配方食品
实施ISO 22000应重点关注：
- 产品配方的安全性和稳定性
- 生产过程的风险控制
- 追溯体系的完整性
- 医学用途的特殊要求

### 8.2 保健食品
实施ISO 22000应重点关注：
- 原料采购的安全控制
- 生产过程的交叉污染控制
- 功效成分的安全管理
- 标签标识的合规性

### 8.3 相关控制措施示例

| 危害类型 | 控制措施 | 控制类型 |
|---------|---------|---------|
| 生物危害（病原菌） | 杀菌温度/时间控制 | CCP |
| 化学危害（农残） | 原料验收检测 | OPRP/CCP |
| 物理危害（异物） | 金属检测、X光检测 | CCP |
| 过敏原 | 标签标识、生产线隔离 | OPRP |
| 交叉污染 | 人员卫生、环境控制 | OPRP |

## 十、最新动态

### 10.1 ISO 22000:2018主要变化
与ISO 22000:2005相比，主要变化包括：
1. 采用高层结构（HLS），与其他管理体系标准更兼容
2. 更强调领导作用和管理承诺
3. 引入基于风险的思维
4. 更注重PDCA循环
5. 强化可追溯性要求
6. 增加对组织环境的理解
7. 更新操作性前提方案的定义

### 10.2 中国标准修订进展
GB/T 22000正在修订中，拟等同采用ISO 22000:2018，预计2025-2026年发布新版。

## 附录A：术语和定义

| 术语 | 定义 |
|------|------|
| 食品安全 | 食品在按照预期用途进行制备和（或）食用时不会伤害消费者的保证 |
| 食品链 | 从初级生产直至消费的各环节和操作的顺序 |
| 食品安全危害 | 生物、化学或物理因素 |
| 控制措施 | 能够用于防止或消除显著食品安全危害或将其降低到可接受水平的行动或活动 |
| 关键控制点（CCP） | 能够进行控制，并且该控制对防止、消除食品安全危害或将其降低到可接受水平所必需的步骤 |
| 前提方案（PRP） | 为保障食品安全卫生而必须遵守的基本条件和活动 |
| 操作性前提方案（OPRP） | 通过危害分析确定的、必需控制食品安全危害的前提方案 |
