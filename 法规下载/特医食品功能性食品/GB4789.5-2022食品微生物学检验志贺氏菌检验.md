# GB 4789.5-2022 食品微生物学检验 志贺氏菌检验

## 标准信息

| 项目 | 内容 |
|------|------|
| **标准号** | GB 4789.5-2022 |
| **标准名称** | 食品微生物学检验 志贺氏菌检验 |
| **发布日期** | 2022年 |
| **实施日期** | 2022年 |
| **替代标准** | GB 4789.5-2012 |
| **适用范围** | 食品中志贺氏菌的检验 |

---

## 1. 范围

本标准规定了食品中志贺氏菌（Shigella）的检验方法。

本标准适用于食品中志贺氏菌的检验。

---

## 2. 设备和材料

### 2.1 设备

- **恒温培养箱**：36℃±1℃，42℃±1℃
- **冰箱**：2℃~5℃
- **恒温水浴箱**：48℃±1℃
- **天平**：感量0.1g
- **均质器**
- **显微镜**：10×~100×
- **无菌吸管**：1mL（具0.01mL刻度）、10mL（具0.1mL刻度）
- **无菌试管**：16mm×160mm
- **无菌锥形瓶**：500mL、250mL
- **无菌培养皿**：直径90mm
- **无菌毛细管**
- **无菌刀、剪**

### 2.2 培养基和试剂

- **志贺氏菌增菌肉汤-新生霉素**
- **麦康凯琼脂**
- **木糖赖氨酸脱氧胆盐琼脂（XLD）**
- **三糖铁琼脂（TSI）**
- **营养琼脂斜面**
- **半固体琼脂**
- **志贺氏菌显色培养基**
- **生化鉴定试剂盒**

---

## 3. 检验程序

志贺氏菌检验程序如下：

```
样品 → 增菌（志贺氏菌增菌肉汤-新生霉素，42℃±1℃，18h~24h）
    ↓
分离（麦康凯琼脂、XLD，36℃±1℃，24h）
    ↓
挑选可疑菌落（无色透明或粉红色菌落）
    ↓
生化试验（TSI、半固体、初步生化）
    ↓
血清学分型鉴定
    ↓
报告结果
```

---

## 4. 操作步骤

### 4.1 增菌

称取25g样品，加入225mL志贺氏菌增菌肉汤-新生霉素中，均质后于42℃±1℃培养18h~24h。

### 4.2 分离

取增菌液划线接种于麦康凯琼脂平板和XLD平板，36℃±1℃培养24h。

### 4.3 菌落特征

**在麦康凯琼脂上**：
- 志贺氏菌：无色透明或浅粉色菌落
- 其他肠道菌：粉红色或红色菌落

**在XLD上**：
- 志贺氏菌：红色菌落，无黑色中心
- 其他肠道菌：黄色菌落或带黑色中心

### 4.4 初步鉴定

挑取可疑菌落接种于：
- **TSI琼脂**：斜面红色，底层黄色（不产气或少量产气），不产生硫化氢
- **半固体琼脂**：无动力

### 4.5 生化鉴定

进行以下生化试验：

| 试验项目 | 志贺氏菌 |
|----------|----------|
| 氧化酶 | - |
| 过氧化氢酶 | + |
| 葡萄糖产酸 | + |
| 葡萄糖产气 | - |
| 乳糖 | -（除宋内志贺氏菌迟缓阳性）|
| 麦芽糖 | -/d |
| 甘露醇 | -/d |
| 蔗糖 | -/d |
| 靛基质 | -/d |
| 甲基红 | + |
| VP试验 | - |
| 柠檬酸盐利用 | - |
| 硫化氢 | - |
| 尿素酶 | - |
| 苯丙氨酸脱氨酶 | - |
| 赖氨酸脱羧酶 | - |
| KCN生长 | - |
| 水杨苷 | - |
| 七叶苷 | - |
| 明胶液化 | - |

注：+ 阳性；- 阴性；d 种间差异。

### 4.6 血清学分型

使用志贺氏菌诊断血清进行分型：

**血清群**：
- A群：痢疾志贺氏菌（S. dysenteriae）
- B群：福氏志贺氏菌（S. flexneri）
- C群：鲍氏志贺氏菌（S. boydii）
- D群：宋内志贺氏菌（S. sonnei）

---

## 5. 结果报告

**检出志贺氏菌**：
- 分离出符合生化反应特性的菌株
- 血清学分型确认

**未检出志贺氏菌**：
- 增菌后分离培养无可疑菌落
- 或可疑菌落经鉴定不符合志贺氏菌特性

---

## 6. 质量控制

### 6.1 阳性对照
使用已知志贺氏菌标准菌株，如福氏志贺氏菌ATCC 12022。

### 6.2 阴性对照
使用无菌水或已知不含志贺氏菌的样品。

### 6.3 培养基质量控制
每批培养基使用前应进行质量控制，确保其支持目标菌生长并抑制非目标菌。

---

## 7. 注意事项

### 7.1 安全防护
- 志贺氏菌为B类病原微生物，操作时应注意生物安全
- 在生物安全柜内进行可能产生气溶胶的操作
- 废弃物应经高压灭菌后处理

### 7.2 操作要点
- 样品处理应在取样后4h内完成
- 增菌温度和时间应严格控制
- 血清学分型时应使用新鲜培养物

### 7.3 干扰因素
- 大肠杆菌某些菌株在XLD上与志贺氏菌菌落相似
- 应结合生化试验和血清学试验进行综合判断

---

## 8. 附录

### 8.1 志贺氏菌分类

| 菌种 | 血清群 | 血清型 |
|------|--------|--------|
| 痢疾志贺氏菌 | A | 1-15 |
| 福氏志贺氏菌 | B | 1-6及亚型 |
| 鲍氏志贺氏菌 | C | 1-18 |
| 宋内志贺氏菌 | D | 1 |

### 8.2 致病性

志贺氏菌是细菌性痢疾的病原体，通过粪-口途径传播，可引起：
- 典型痢疾：发热、腹痛、里急后重、脓血便
- 轻型腹泻：水样便
- 慢性携带状态
