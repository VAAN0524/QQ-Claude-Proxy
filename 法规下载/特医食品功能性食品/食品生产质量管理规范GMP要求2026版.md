# 食品生产质量管理规范(GMP)要求（2026版）

**发布部门**: 国家市场监督管理总局
**适用范围**: 特医食品、保健食品、普通食品生产企业
**更新日期**: 2026年1月

---

## 一、GMP概述

### 1.1 GMP定义

GMP(Good Manufacturing Practice)，即良好生产规范，是食品生产企业必须遵守的基本生产准则，旨在确保食品在规定的条件下生产，保证产品质量安全。

### 1.2 GMP与HACCP的关系

| 项目 | GMP | HACCP |
|------|-----|-------|
| 性质 | 基础性、前提性 | 预防性、针对性 |
| 范围 | 全面的生产卫生管理 | 针对关键控制点的管理 |
| 作用 | 为HACCP提供基础 | 在GMP基础上实施 |
| 要求 | 必须实施 | 推荐实施 |

## 二、厂房设施要求

### 2.1 厂址选择

| 要求 | 说明 |
|------|------|
| 环境卫生 | 远离污染源，环境清洁 |
| 地势 | 地势干燥，不易积水 |
| 交通 | 便于原料和产品运输 |
| 水源 | 有充足、符合标准的水源 |
| 电力 | 电力供应稳定 |

### 2.2 厂区布局

```
厂区布局示例：

┌─────────────────────────────────────────────────────┐
│                    厂区围墙                          │
│  ┌──────────┐  ┌─────────┐  ┌──────────┐           │
│  │  办公区  │  │ 生产区  │  │ 辅助区   │           │
│  │          │  │         │  │          │           │
│  │ 实验室   │  │ 洁净车间│  │ 仓库     │           │
│  │ 更衣室   │  │ 一般车间│  │ 动力设施 │           │
│  │ 会议室   │  │         │  │ 废物处理 │           │
│  └──────────┘  └─────────┘  └──────────┘           │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │               人流物流分离通道                │  │
│  └──────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### 2.3 厂房设计原则

| 原则 | 要求 |
|------|------|
| 工艺流畅 | 按工艺流程顺序布局 |
| 分区明确 | 清洁区、一般区、污染区分开 |
| 人流物流分开 | 避免交叉污染 |
| 便于清洁 | 地面、墙面易清洁消毒 |
| 通风良好 | 有良好的通风设施 |

### 2.4 生产区域划分

| 区域 | 洁净级别 | 适用工序 |
|------|----------|----------|
| **一般生产区** | 无特殊要求 | 原料处理、外包装 |
| **洁净生产区** | 100000级(ISO 8) | 内包装、称量配料 |
| **高洁净生产区** | 10000级(ISO 7) | 暴露工序、关键工序 |

### 2.5 地面要求

| 项目 | 要求 |
|------|------|
| 材料 | 防滑、耐腐蚀、易清洗 |
| 坡度 | 1.5%-2%坡向地漏 |
| 地漏 | 有水封，便于排水 |
| 阴角 | 圆弧形，便于清洁 |

### 2.6 墙面和天花板要求

| 项目 | 要求 |
|------|------|
| 墙面 | 浅色、光滑、易清洗 |
| 墙角 | 圆弧形，无死角 |
| 天花板 | 防潮、防霉、易清洗 |
| 高度 | 满足生产工艺要求 |

### 2.7 门窗要求

| 项目 | 要求 |
|------|------|
| 门 | 不锈钢或铝合金，易清洗 |
| 窗 | 与生产区相连的窗不得开启 |
| 密封 | 良好的密封性 |
| 防虫 | 安装防虫纱窗、风幕机 |

## 三、设备设施要求

### 3.1 设备选型原则

| 原则 | 说明 |

| 固定 | 设备安装牢固 |
| 排水 | 设备底部有排水设施 |

### 3.3 生产设备清单

| 设备类型 | 具体设备 |
|----------|----------|
| 原料处理设备 | 筛选机、清洗机、粉碎机 |
| 配料设备 | 电子秤、配料系统 |
| 混合设备 | 混合机、均质机 |
| 加热设备 | 杀菌机、蒸煮锅 |
| 干燥设备 | 喷雾干燥机、流化床 |
| 包装设备 | 包装机、封口机 |
| 清洗设备 | CIP清洗系统 |

### 3.4 检验设备要求

| 设备类型 | 具体设备 |
|----------|----------|
| 微生物检验 | 超净工作台、培养箱、显微镜 |
| 理化检验 | 天平、分光光度计、液相色谱仪 |
| 快速检验 | 水分仪、蛋白质测定仪 |
| 计量器具 | 温度计、湿度计、压力表 |

### 3.5 供水系统

| 项目 | 要求 |
|------|------|
| 水源 | 符合GB 5749生活饮用水标准 |
| 制水 | 纯化水系统(如需要) |
| 管道 | 不锈钢管道 |
| 储存 | 密闭储水罐 |
| 消毒 | 定期消毒，有记录 |

### 3.6 供汽系统

| 项目 | 要求 |
|------|------|
| 锅炉 | 符合安全要求 |
| 管道 | 保温、标识 |
| 过滤 | 蒸汽过滤(直接接触用) |
| 排水 | 有冷凝水排放系统 |

### 3.7 通风空气净化系统

| 区域 | 换气次数 | 过滤要求 |
|------|----------|----------|
| 一般生产区 | 6-10次/h | 初效、中效 |
| 100000级区 | 15-20次/h | 初效、中效、高效 |
| 10000级区 | 20-30次/h | 初效、中效、高效 |

### 3.8 照明要求

| 区域 | 照度要求 |
|------|----------|
| 生产车间 | ≥300 lux |
| 检验室 | ≥500 lux |
| 精细操作区 | ≥800 lux |
| 仓库 | ≥150 lux |
