# 保健食品检测标准完整版（2026）

## 标准概述

保健食品是指声称具有特定保健功能或者以补充维生素、矿物质为目的的食品，适用于特定人群食用，具有调节机体功能，不以治疗疾病为目的，并且对人体不产生急性、亚急性或者慢性危害的食品。

## 三、标志性成分检测方法

### 3.1 辅酶Q10（GB/T 22252-2008）

#### 3.1.1 方法原理

试样中辅酶Q10用乙醇超声提取，经高效液相色谱分离，紫外检测器检测，外标法定量。

#### 3.1.2 色谱条件

| 参数 | 条件 |
|------|------|
| **色谱柱** | C18柱，250 mm × 4.6 mm，5 μm |
| **流动相** | 甲醇-无水乙醇（85:15） |
| **流速** | 1.0 mL/min |
| **检测波长** | 275 nm |
| **进样量** | 20 μL |
| **柱温** | 30°C |

#### 3.1.3 结果计算

```
X = (ρ × V × f) / m

式中：
X - 辅酶Q10含量，g/100g
ρ - 从标准曲线上查得的辅酶Q10浓度，μg/mL
V - 定容体积，mL
f - 稀释倍数
m - 试样质量，g
```

### 3.2 褪黑素（GB/T 22271-2008）

#### 3.2.1 方法原理

试样中的褪黑素用甲醇提取，经高效液相色谱分离，荧光检测器检测，外标法定量。

#### 3.2.2 色谱条件

| 参数 | 条件 |
|------|------|
| **色谱柱** | C18柱，250 mm × 4.6 mm，5 μm |
| **流动相** | 甲醇-水（55:45） |
| **流速** | 1.0 mL/min |
| **检测波长** | 激发波长278 nm，发射波长348 nm |
| **进样量** | 20 μL |
| **柱温** | 30°C |

### 3.3 总黄酮（GB/T 22291-2008）

#### 3.3.1 方法原理

试样中的总黄酮用乙醇提取，与铝离子形成黄色络合物，在特定波长下测定吸光度，以芦丁为标准品计算含量。

#### 3.3.2 测定步骤

```
样品提取 → 醇提 → 过滤 → 定容 →
加硝酸铝溶液 → 加氢氧化钠溶液 → 静置 → 测定吸光度（510 nm）→ 计算
```

### 3.4 总皂苷（GB/T 22290-2008）

#### 3.4.1 方法原理

试样中的总皂苷用甲醇提取，与香草醛-高氯酸反应显色，在特定波长下测定吸光度，以人参皂苷Rb1为标准品计算含量。

#### 3.4.2 测定步骤

```
样品提取 → 醇提 → 过滤 → 定容 →
加香草醛-高氯酸溶液 → 水浴加热 → 冷却 → 测定吸光度（560 nm）→ 计算
```

## 四、保健食品注册检验要求

### 4.1 注册检验项目

| 检验类别 | 检验项目 |
|---------|----------|
| **卫生学检验** | 铅、砷、汞等污染物；菌落总数、大肠菌群等微生物 |
| **功效成分/标志性成分检验** | 产品声称的功效成分或标志性成分 |
| **稳定性检验** | 加速试验和长期试验 |
| **毒理学检验** | 急性毒性、遗传毒性、28天经口毒性等 |
| **功能学检验** | 人体试食试验或动物试验 |

### 4.2 卫生学检验

#### 4.2.1 理化指标

| 项目 | 限量要求 |
|------|----------|
| **铅** | ≤ 0.5 mg/kg（一般产品） |
| **砷** | ≤ 0.3 mg/kg |
| **汞** | ≤ 0.3 mg/kg |
| **镉** | ≤ 0.3 mg/kg |
| **农药残留** | 符合GB 2763要求 |
| **兽药残留** | 符合相应要求 |

#### 4.2.2 微生物指标

| 项目 | 限量要求 |
|------|----------|
| **菌落总数** | ≤ 1000-30000 CFU/g（mL）（产品类型不同） |
| **大肠菌群** | ≤ 0.3-40 MPN/g（mL） |
| **霉菌和酵母** | ≤ 50 CFU/g（mL） |
| **金黄色葡萄球菌** | 不得检出 |
| **沙门氏菌** | 不得检出 |

### 4.3 功效成分/标志性成分检验

#### 4.3.1 检验原则

- 按照产品配方和申报的标志性成分进行检验
- 优先采用国家标准方法
- 无国家标准方法时，采用行业公认方法

#### 4.3.2 含量要求

| 成分类型 | 含量要求 |
|---------|----------|
| **维生素、矿物质** | 标示量的80%-150% |
| **功效成分** | 标示量的80%-150% |
| **其他成分** | 企业标准规定范围 |

## 五、检验方法选择指南

### 5.1 根据成分性质选择

| 成分类型 | 首选方法 | 备选方法 |
|---------|----------|----------|
| **脂溶性成分** | HPLC（反相C18） | GC |
| **水溶性成分** | HPLC | 分光光度法 |
| **挥发性成分** | GC | GC-MS |
| **大分子（多糖、蛋白）** | 分光光度法 | HPLC（体积排阻） |
| **无机元素** | 原子吸收 | ICP-MS |
| **微生物** | 平板计数法 | PCR法 |

### 5.2 根据含量选择

| 含量水平 | 推荐方法 |
|---------|----------|
| **高含量（>1%）** | 直接进样HPLC |
| **中含量（0.1%-1%）** | 液液萃取-HPLC |
| **低含量（<0.1%）** | 固相萃取-HPLC/MS |

## 六、常见问题解答

### 6.1 样品前处理

| 问题 | 解决方案 |
|------|----------|
| **样品提取不完全** | 优化提取溶剂、时间、温度 |
| **基质干扰** | 采用固相萃取净化 |
| **成分不稳定** | 避光、低温、抗氧化剂保护 |
| **乳化** | 离心、加盐 |

### 6.2 仪器分析

| 问题 | 解决方案 |
|------|----------|
| **峰形差** | 调整流动相、色谱柱温度 |
| **分离度不够** | 更换色谱柱、调整梯度 |
| **响应低** | 调整检测波长、进样量 |
| **重现性差** | 检查进样器、色谱柱 |

### 6.3 结果计算

| 问题 | 解决方案 |
|------|----------|
| **回收率低** | 检查提取方法、添加回收试验 |
| **空白高** | 检查试剂纯度、器皿清洁 |
| **平行性差** | 检查操作一致性、仪器状态 |

## 附录：术语定义

| 术语 | 定义 |
|------|------|
| **保健食品** | 声称具有特定保健功能或者以补充维生素、矿物质为目的的食品 |
| **功效成分** | 经动物和/或人体功能试验证实具有保健功能的成分 |
| **标志性成分** | 与保健功能相关的、需要在产品标签上标示的成分 |
| **卫生学检验** | 对产品中污染物、微生物等进行的检验 |
| **毒理学检验** | 评价产品安全性的检验 |
| **功能学检验** | 评价产品保健功能的检验 |
