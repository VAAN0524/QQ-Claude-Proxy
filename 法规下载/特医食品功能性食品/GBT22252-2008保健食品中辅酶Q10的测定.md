# GB/T 22252-2008 保健食品中辅酶Q10的测定

**标准编号**: GB/T 22252-2008
**实施日期**: 2008年11月1日
**发布部门**: 中华人民共和国国家质量监督检验检疫总局 中国国家标准化管理委员会

---

## 1. 范围

本标准规定了保健食品中辅酶Q10的测定方法。

本标准适用于以辅酶Q10为原料或功效成分的保健食品中辅酶Q10含量的测定。

**测定范围**：样品中辅酶Q10含量≥10mg/100g

---

## 2. 原理

试样用无水乙醇或正己烷-异丙醇混合溶液超声提取，经高效液相色谱柱分离，用紫外检测器（波长275nm）检测，外标法定量。

**辅酶Q10**：
- 化学名称：2-[(全-E)3,7,11,15,19,23,27,31,35,39-十甲基-2,6,10,14,18,22,26,30,34,38-四十碳十烯-1-基]-5,6-二甲氧基-3-甲基-p-苯醌
- 分子式：C59H90O2
- 相对分子质量：863.36
- 最大吸收波长：275nm

---

## 3. 试剂和材料

### 3.1 试剂

| 试剂 | 规格 | 说明 |
|------|------|------|
| 无水乙醇 | 色谱纯 | 提取溶剂 |
| 正己烷 | 色谱纯 | 提取溶剂 |
| 异丙醇 | 色谱纯 | 提取溶剂 |
| 乙腈 | 色谱纯 | 流动相 |
| 甲醇 | 色谱纯 | 流动相 |

### 3.2 标准品

**辅酶Q10标准品**：
- 纯度：≥98%
- 储存条件：避光，-20℃保存
- 有效期：按说明书要求

### 3.3 标准溶液配制

**标准储备液（1mg/mL）**：
```
准确称取辅酶Q10标准品25mg（精确至0.0001g），
置于25mL棕色容量瓶中，用无水乙醇溶解并定容至刻度，
摇匀，置于冰箱中保存，有效期3个月。
```

**标准工作液系列**：
```
分别吸取标准储备液0.5、1.0、2.0、5.0、10.0mL，
置于10mL容量瓶中，用流动相定容至刻度，
得到浓度为50、100、200、500、1000μg/mL的标准工作液。
```

## 4. 仪器和设备

| 仪器设备 | 规格要求 |
|----------|----------|
| 高效液相色谱仪 | 配紫外检测器或二极管阵列检测器 |
| 色谱柱 | C18柱，250mm×4.6mm，5μm |
| 超声波清洗器 | 功率≥250W |
| 离心机 | 转速≥4000r/min |
| 分析天平 | 感量0.0001g |
| 容量瓶 | 10mL、25mL、50mL |
| 移液器 | 100-1000μL |
| 棕色容量瓶 | 用于标准溶液配制 |
| 0.45μm滤膜 | 有机系 |

---

## 5. 分析步骤

### 5.1 试样处理

#### 5.1.1 固体试样（片剂、胶囊内容物、粉末等）

```
1. 取试样约1g（精确至0.0001g），置于50mL容量瓶中
2. 加入约30mL无水乙醇
3. 超声提取30分钟（期间摇动2-3次）
4. 取出冷却至室温
5. 用无水乙醇定容至刻度，摇匀
6. 取部分溶液离心10分钟（4000r/min）
7. 取上清液过0.45μm滤膜，待测
```

#### 5.1.2 液体试样（口服液、饮料等）

```
1. 取试样适量（含辅酶Q10约10mg），置于50mL容量瓶中
2. 加入20mL正己烷-异丙醇（1:1）混合溶液
3. 振摇萃取5分钟
4. 用正己烷-异丙醇（1:1）定容至刻度
5. 取部分溶液离心10分钟
6. 取上清液过0.45μm滤膜，待测
```

#### 5.1.3 软胶囊试样

```
1. 取软胶囊10粒，剪开囊皮
2. 将内容物全部挤入烧杯中，用正己烷洗涤囊皮2-3次
3. 将内容物和洗涤液合并，转移至50mL容量瓶中
4. 用正己烷-异丙醇（1:1）定容至刻度
5. 取适量体积稀释至适当浓度
6. 过0.45μm滤膜，待测
```

### 5.2 色谱参考条件

| 参数 | 设定值 |
|------|--------|
| 色谱柱 | C18柱，250mm×4.6mm，5μm |
| 流动相 | 乙腈-甲醇（50:50）或甲醇-异丙醇（70:30） |
| 流速 | 1.0 mL/min |
| 检测波长 | 275 nm |
| 柱温 | 30℃ |
| 进样量 | 20 μL |
| 运行时间 | 20 min |

### 5.3 标准曲线绘制

```
1. 将标准工作液系列按浓度由低到高进样
2. 记录色谱图
3. 以峰面积为纵坐标，浓度为横坐标
4. 绘制标准曲线
5. 计算回归方程和相关系数
```

**相关系数要求**：r ≥ 0.999

### 5.4 试样测定

```
1. 将处理后的试样溶液进样
2. 记录色谱图
3. 测定辅酶Q10峰面积
4. 根据标准曲线计算含量
```

---

## 6. 结果计算

### 6.1 计算公式

```
         X = (A × V × C₀) / (A₀ × m)

其中：
X — 试样中辅酶Q10含量，mg/100g 或 mg/100mL
A — 试样溶液中辅酶Q10的峰面积
A₀ — 标准溶液中辅酶Q10的峰面积
V — 试样溶液定容体积，mL
C₀ — 标准溶液浓度，μg/mL
m — 试样质量或体积，g 或 mL
```

### 6.2 结果表示

- 保留两位有效数字
- 报告结果时注明检测方法
- 结果低于检出限时报告"未检出"

## 9. 色谱图参考

### 9.1 辅酶Q10保留时间

- 在C18柱上，辅酶Q10保留时间约为8-12分钟
- 具体保留时间根据色谱柱和流动相条件而定

### 9.2 分离度要求

- 辅酶Q10峰与相邻杂质峰的分离度≥1.5
- 拖尾因子：0.9-1.2
- 理论塔板数≥5000

---

## 10. 注意事项

### 10.1 样品处理

1. **避光操作**：辅酶Q10对光敏感，全过程应避免强光直射
2. **低温保存**：提取液和标准溶液应低温避光保存
3. **快速分析**：提取后应尽快分析，避免降解
4. **容器选择**：使用棕色玻璃器皿

### 10.2 色谱条件

1. **柱温控制**：保持柱温恒定，避免保留时间漂移
2. **流动相脱气**：使用前应充分脱气
3. **系统适用性**：每批样品分析前应进行系统适用性试验

### 10.3 干扰排除

1. **辅料干扰**：某些辅料可能干扰测定，需优化提取条件
2. **异构体**：辅酶Q10与辅酶Q9等需良好分离
3. **氧化产物**：避免样品氧化产生降解产物干扰

## 12. 质量控制

### 12.1 空白试验

每批样品应进行空白试验，确认无干扰峰。

### 12.2 平行样

每批样品应做平行试验，相对偏差应满足精密度要求。

### 12.3 加标回收

每批样品应进行加标回收试验，回收率应在要求范围内。

### 12.4 质控样

每批样品应同时测定质控样，结果应在给定范围内。

---

## 13. 常见问题处理

### 13.1 峰形异常

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 拖尾严重 | 色谱柱污染 | 清洗或更换色谱柱 |
| 前延 | 溶剂效应 | 调整流动相比例 |
| 分离度差 | 流动相不合适 | 优化流动相组成 |

### 13.2 保留时间漂移

| 原因 | 解决方法 |
|------|----------|
| 柱温波动 | 检查柱温控制系统 |
| 流动相变化 | 重新配制流动相 |
| 色谱柱老化 | 更换新色谱柱 |
