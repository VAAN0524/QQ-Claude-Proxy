# 特殊医学用途配方食品注册申请服务指南（2026版）

**发布部门**: 国家市场监督管理总局食品审评中心
**适用范围**: 特殊医学用途配方食品（特医食品）注册申请
**更新日期**: 2026年1月

---

## 一、注册申请概述

### 1.1 特医食品定义

特殊医学用途配方食品是指为了满足进食受限、消化吸收障碍、代谢紊乱或特定疾病状态人群对营养素或膳食的特殊需要，专门加工配制而成的配方食品。

### 1.2 产品类别

| 类别 | 定义 | 示例 |
|------|------|------|
| **全营养配方食品** | 可作为单一营养来源满足目标人群营养需求的特殊医学用途配方食品 | 氨基酸代谢障碍全营养配方食品 |
| **特定全营养配方食品** | 可作为单一营养来源能够满足特定疾病状态人群的营养需求的特殊医学用途配方食品 | 糖尿病全营养配方食品、肿瘤全营养配方食品 |
| **非全营养配方食品** | 可满足目标人群部分营养需求的特殊医学用途配方食品，不适用于作为单一营养来源 | 营养素组件、电解质补充配方食品 |

### 1.3 注册管理框架

```
注册流程图：
[产品研发] → [临床试验(如需要)] → [注册申请] → [技术审评] → [现场核查] → [注册审批] → [证书颁发]
```

## 二、注册申请条件

### 2.1 申请人资格

| 条件 | 要求 |
|------|------|
| 主体资格 | 中国境内登记的企业 |
| 生产能力 | 具备相应的生产条件和能力 |
| 研发能力 | 具备相应的研发机构和人员 |
| 质量管理 | 建立完善的食品安全管理体系 |
| 不良反应监测 | 具备不良反应监测和报告能力 |

### 2.2 产品条件

| 条件 | 要求 |
|------|------|
| 安全性 | 产品安全可靠 |
| 营养充足性 | 能满足目标人群营养需求 |
| 临床效果 | 有临床试验支持(特定全营养) |
| 配方科学 | 配方设计科学合理 |
| 工艺可行 | 生产工艺成熟稳定 |

### 3.2 核心材料详细要求

#### 3.2.1 产品研发报告

**必须包含以下内容**：

| 章节 | 内容 |
|------|------|
| 产品研发背景 | 市场需求、临床需求 |
| 产品配方研发 | 配方设计、依据 |
| 生产工艺研发 | 工艺选择、验证 |
| 产品标准制定 | 标准制定依据 |
| 临床试验(如需) | 临床试验方案和报告 |
| 产品稳定性研究 | 稳定性试验数据 |
| 标签说明书研发 | 标签说明书制定依据 |

#### 3.2.2 产品配方及依据

**配方表格式**：

| 原料 | 使用量 | 功能 | 质量标准 |
|------|--------|------|----------|
| 乳清蛋白粉 | XX g/100g | 蛋白质来源 | 符合GB 11674 |
| 麦芽糊精 | XX g/100g | 碳水化合物来源 | 符合GB/T 20884 |
| ... | ... | ... | ... |

**配方设计依据**：
- 参考国内外相关标准
- 参考医学指南和专家共识
- 参考科学文献

#### 3.2.3 生产工艺材料

**必须包含**：
- 完整的工艺流程图
- 关键工艺参数
- 生产设备清单
- 生产车间洁净级别

**工艺流程图示例**：

```
[原料验收] → [预处理] → [配料] → [混合] → [杀菌] → [浓缩] → [干燥] → [包装] → [检验] → [入库]
```

#### 3.2.4 三批试生产报告

**报告内容**：

| 项目 | 内容 |
|------|------|
| 批次信息 | 批号、生产日期 |
| 原料信息 | 原料批号、厂家 |
| 生产记录 | 完整的生产记录 |
| 质量检验 | 全项目检验结果 |
| 结论 | 产品符合要求 |

## 四、注册申请流程

### 4.1 申请流程

```
┌─────────────────────────────────────────────────────────────┐
│                     特医食品注册申请流程                      │
└─────────────────────────────────────────────────────────────┘

1. 【准备阶段】
   ├─ 产品研发
   ├─ 临床试验(特定全营养配方食品)
   ├─ 三批试生产
   └─ 申请材料准备

2. 【网上申报】
   ├─ 登录特殊食品注册管理信息系统
   ├─ 填写申请表
   ├─ 上传电子材料
   └─ 提交申请

3. 【材料受理】
   ├─ 形式审查(5个工作日)
   ├─ 补正通知(如需要)
   └─ 受理决定

4. 【技术审评】
   ├─ 专家评审(60个工作日)
   ├─ 补充材料通知(如需要)
   ├─ 补充材料评审(30个工作日)
   └─ 审评意见

5. 【现场核查】
   ├─ 生产现场核查
   ├─ 试验数据核查
   └─ 核查报告

6. 【审批决定】
   ├─ 审批(20个工作日)
   ├─ 证书颁发
   └─ 公告

总时限: 最多约200个工作日(不含补充材料时间)
```

### 4.2 受理审查要点

| 审查项目 | 要求 |
|----------|------|
| 材料完整性 | 按要求提供全部材料 |
| 材料规范性 | 符合格式要求 |
| 材料真实性 | 真实有效 |
| 产品类别 | 明确产品类别 |
| 申请人资格 | 符合条件 |

### 4.3 技术审评要点

| 审评项目 | 审评内容 |
|----------|----------|
| 产品配方 | 配方科学性、合理性 |
| 生产工艺 | 工艺可行性、稳定性 |
| 产品标准 | 标准适用性、完整性 |
| 安全性评估 | 安全性评价数据 |
| 营养充足性 | 营养成分充足性 |
| 临床效果 | 临床试验数据(特定全营养) |

### 4.4 现场核查要点

| 核查项目 | 核查内容 |
|----------|----------|
| 生产条件 | 是否具备生产条件 |
| 设备设施 | 设备是否满足生产 |
| 质量管理 | 质量管理体系是否完善 |
| 检验能力 | 是否具备检验能力 |
| 人员资质 | 人员是否具备相应资质 |
| 生产记录 | 生产记录是否完整真实 |

## 五、临床试验要求

### 5.1 需要临床试验的产品

| 产品类别 | 是否需要临床试验 |
|----------|------------------|
| 全营养配方食品 | 不需要 |
| 特定全营养配方食品 | 需要 |
| 非全营养配方食品 | 不需要 |

### 5.2 临床试验基本要求

| 项目 | 要求 |
|------|------|
| 试验设计 | 随机、对照、双盲(如可能) |
| 试验例数 | 每组不少于100例 |
| 试验期限 | 根据产品特点确定 |
| 观察指标 | 主要指标、次要指标 |
| 安全性观察 | 不良事件记录 |

### 5.3 临床试验报告要求

**报告必须包含**：
1. 试验方案
2. 研究者手册
3. 知情同意书
4. 病例报告表
5. 统计分析报告
6. 试验总结报告

### 7.1 审批时限

| 环节 | 时限 |
|------|------|
| 材料受理 | 5个工作日 |
| 技术审评 | 60个工作日(可延长30个工作日) |
| 现场核查 | 根据实际需要 |
| 审批决定 | 20个工作日 |

### 7.2 注册证书

**证书内容**：
- 注册人名称
- 产品名称
- 注册号
- 产品类别
- 适用人群
- 注册有效期
- 发证日期

**注册号格式**：
```
国食注字TY+4位年号+4位顺序号
示例：国食注字TY20260001
```

**证书有效期**：
- 5年
- 到期前6个月可申请延续

## 十、常见问题

### Q1: 特医食品注册需要多长时间？
A: 从申请到获得注册证，通常需要12-18个月，不包括补充材料时间。

### Q2: 全营养配方食品是否需要临床试验？
A: 全营养配方食品不需要临床试验，但需要提供充分的科学依据。

### Q3: 特医食品能否在普通食品中销售？
A: 不能。特医食品必须在医生或临床营养师指导下使用。

### Q4: 进口特医食品如何注册？
A: 进口特医食品与国产特医食品注册要求相同，但申请人需为境外企业指定的中国境内办事机构。

### Q5: 特医食品注册能否委托办理？
A: 可以委托专业代理机构办理，但申请人仍需对申请材料的真实性负责。
