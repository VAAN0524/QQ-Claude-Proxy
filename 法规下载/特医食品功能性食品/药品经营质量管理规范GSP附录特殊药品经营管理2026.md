## 前言

本附录是《药品经营质量管理规范》（GSP）的配套文件，规定了特殊药品（包括特殊医学用途配方食品、药品类易制毒化学品、麻醉药品、精神药品等）经营管理的特殊要求。

---

## 第1章 总则

**第一条** 为加强特殊药品经营管理，确保特殊药品质量安全，根据《药品管理法》《药品管理法实施条例》和《药品经营质量管理规范》，制定本附录。

**第二条** 本附录适用于特殊药品的经营企业，包括：

1. 特殊医学用途配方食品经营企业
2. 药品类易制毒化学品经营企业
3. 麻醉药品和精神药品经营企业
4. 医疗用毒性药品经营企业
5. 放射性药品经营企业

**第三条** 特殊药品经营企业应当：

1. 具有特殊药品经营资质
2. 建立专门的质量管理体系
3. 配备专门的人员和设施
4. 执行更严格的管理制度

## 第2章 人员管理

**第四条** 企业负责人和质量管理负责人

| 岗位 | 学历要求 | 工作经验 | 专业要求 |
|------|----------|----------|----------|
| 企业负责人 | 大专以上 | 3年以上药品经营 | 药学或相关专业 |
| 质量负责人 | 本科以上 | 5年以上质量管理 | 执业药师或药师 |
| 特殊药品管理人员 | 中专以上 | 2年以上 | 相关专业 |

**第五条** 专门人员要求

1. **特殊药品采购员**
   - 熟悉特殊药品法律法规
   - 经过专门培训考核
   - 无不良记录

2. **特殊药品销售员**
   - 熟悉特殊药品适应症
   - 经过专门培训考核
   - 无不良记录

3. **特殊药品保管员**
   - 熟悉特殊药品储存要求
   - 经过专门培训考核
   - 责任心强

**第六条** 培训要求

1. **岗前培训**：新员工必须经过岗前培训考核
2. **定期培训**：每年至少进行12小时专门培训
3. **继续教育**：保持专业知识更新
4. **考核评估**：定期进行考核评估

## 第3章 设施与设备

**第七条** 仓储设施

**7.1 专用仓库**

1. 特殊药品应当实行专库或专柜管理
2. 仓库应当符合GSP要求
3. 仓库应当配备安全监控设施
4. 仓库应当设置警示标识

**7.2 仓库区域划分**

| 区域 | 面积要求 | 设施要求 |
|------|----------|----------|
| 待验区 | ≥20平方米 | 防潮、防鼠、防虫 |
| 合格区 | 按储存量确定 | 阴凉、干燥、通风 |
| 发货区 | ≥15平方米 | 防潮、防鼠、防虫 |
| 不合格区 | ≥10平方米 | 专柜上锁、标识醒目 |
| 待处理区 | ≥10平方米 | 专柜上锁、标识醒目 |

**第八条** 储存设备

**8.1 温湿度控制**

| 药品类型 | 温度要求 | 湿度要求 |
|----------|----------|----------|
| 特医食品 | 0-25℃ | 45%-75% |
| 易制毒化学品 | 0-30℃ | 45%-75% |
| 麻醉药品 | 2-8℃或常温 | 45%-75% |
| 精神药品 | 2-8℃或常温 | 45%-75% |

**8.2 监测设备**

1. 温湿度自动监测系统
2. 温湿度记录仪
3. 报警装置
4. 备用电源

**第九条** 安全设施

**9.1 物理安全**

1. 防盗门窗
2. 保险柜（特殊药品专用）
3. 监控摄像头（全覆盖）
4. 红外报警器

**9.2 技术防范**

1. 电子门禁系统
2. 视频监控系统（保留90天）
3. 入侵报警系统
4. 消防设施

## 第4章 采购管理

**第十条** 供应商审核

**10.1 审核内容**

1. 营业执照
2. 生产许可证/经营许可证
3. GMP证书/GSP证书
4. 产品批准证明文件
5. 产品质量标准

**10.2 现场审计**

对高风险特殊药品供应商应当进行现场审计：

| 审计项目 | 审计要点 | 评分标准 |
|----------|----------|----------|
| 质量管理体系 | 文件、人员、设施 | 0-20分 |
| 生产条件 | 厂房、设备、工艺 | 0-25分 |
| 质量控制 | 检验能力、检验记录 | 0-25分 |
| 产品追溯 | 批次管理、追溯系统 | 0-20分 |
| 合规性 | 法规符合、证照齐全 | 0-10分 |

**第十一条** 采购合同

采购合同应当明确：

1. 产品名称、规格、批号
2. 质量标准和验收要求
3. 供货时间和方式
4. 运输条件和要求
5. 质量责任和违约责任

**第十二条** 进货验收

**12.1 验收要求**

| 验收项目 | 验收内容 | 验收标准 |
|----------|----------|----------|
| 随货同行单 | 单货相符 | 100%相符 |
| 产品包装 | 完整性、标签标识 | 符合规定 |
| 产品合格证 | 检验报告 | 齐全有效 |
| 运输条件 | 温度、湿度、时限 | 符合要求 |

**12.2 验收记录**

验收记录应当包括：

1. 供货单位
2. 产品名称、规格、批号
3. 生产日期、有效期
4. 验收数量、验收结果
5. 验收人、验收日期

---

## 第5章 验收与检验

**第十三条** 验收管理

**13.1 验收程序**

```
核对单证 → 检查包装 → 清点数量 → 检查外观 → 记录登记 → 入库存储
```

**13.2 验收要点**

1. 随货同行单与实物相符
2. 包装完好、标签清晰
3. 在有效期内
4. 储存条件符合要求

**第十四条** 检验管理

**14.1 检验要求**

1. 首营品种必须检验
2. 怀疑质量问题的品种必须检验
3. 近效期品种加强检验
4. 定期抽检

**14.2 检验项目**

| 检验类型 | 检验项目 | 检验方法 |
|----------|----------|----------|
| 外观检查 | 性状、颜色、气味 | 目测、鼻嗅 |
| 理化检查 | 鉴别、含量、有关物质 | 药典方法 |
| 微生物检查 | 细菌、真菌、控制菌 | 药典方法 |
| 包装检查 | 密封性、标签标识 | 目测、试验 |

---

## 第6章 储存与养护

**第十五条** 储存管理

**15.1 分类储存**

1. 按剂型分类存放
2. 按药理作用分类存放
3. 按储存条件分类存放
4. 特殊药品专柜（库）存放

**15.2 堆码要求**

1. 货垛间距：不小于100cm
2. 墙距：不小于30cm
3. 顶距：不小于50cm
4. 地距：不小于10cm

**第十六条** 养护管理

**16.1 养护检查**

| 检查类型 | 检查周期 | 检查内容 |
|----------|----------|----------|
| 定期检查 | 每月 | 质量状况、储存条件 |
| 重点检查 | 每周 | 特殊药品、近效期药品 |
| 季节检查 | 换季前 | 防潮、防霉、防虫 |

**16.2 养护记录**

养护记录应当包括：

1. 检查日期
2. 检查人
3. 检查品种和数量
4. 检查结果
5. 处理措施

---

## 第7章 销售管理

**第十七条** 销售资质审核

销售特殊药品应当审核购货单位以下资质：

1. 营业执照
2. 生产许可证或经营许可证
3. GMP证书或GSP证书
4. 授权委托书（如有）
5. 采购人员身份证明

**第十八条** 销售记录

销售记录应当包括：

1. 购货单位
2. 产品名称、规格、批号
3. 销售数量、销售日期
4. 发货人、收货人
5. 运输方式和条件

**第十九条** 禁止销售

以下情况禁止销售：

1. 超出经营范围
2. 无相关资质
3. 疑似质量问题
4. 来源不明

## 第8章 运输管理

**第二十条** 运输要求

**20.1 运输工具**

| 运输方式 | 运输工具 | 温控要求 |
|----------|----------|----------|
| 常温运输 | 密封货车 | 10-20℃ |
| 低温运输 | 冷藏车 | 2-8℃ |
| 冷冻运输 | 冷冻车 | -15℃以下 |

**20.2 运输监控**

1. 温度实时监控
2. GPS定位跟踪
3. 到货确认签收
4. 异常情况记录

**第二十一条** 运输交接

**21.1 交接内容**

1. 核对单证
2. 检查包装
3. 清点数量
4. 检查温度
5. 签字确认

**21.2 交接记录**

交接记录应当包括：

1. 交接时间
2. 交接地点
3. 交接品种和数量
4. 运输温度记录
5. 交接人签字

## 第10章 不合格品处理

**第二十四条** 不合格品管理

**24.1 不合格品识别**

以下情形应当判定为不合格：

1. 包装破损、污染
2. 标签标识不清
3. 超过有效期
4. 质量检验不合格
5. 存储不当

**24.2 不合格品处理程序**

```
发现不合格 → 隔离存放 → 标识醒目 → 记录登记 → 评审处置 → 监督销毁
```

**24.3 处置方式**

| 不合格原因 | 处置方式 | 审批权限 |
|------------|----------|----------|
| 轻微缺陷 | 降级使用 | 质量负责人 |
| 严重缺陷 | 销毁处理 | 企业负责人 |
| 疑似假药 | 报告药监部门 | 企业负责人 |

## 第11章 投诉与召回

**第二十五条** 投诉处理

**25.1 投诉受理**

1. 建立投诉受理渠道
2. 记录投诉内容
3. 及时调查处理
4. 反馈处理结果

**25.2 投诉记录**

投诉记录应当包括：

1. 投诉人信息
2. 投诉产品信息
3. 投诉内容
4. 调查结果
5. 处理措施

**第二十六条** 召回管理

**26.1 召回分级**

| 召回级别 | 定义 | 召回时限 |
|----------|------|----------|
| 一级召回 | 存在严重安全隐患 | 24小时 |
| 二级召回 | 存在安全隐患 | 48小时 |
| 三级召回 | 存在质量风险 | 72小时 |

**26.2 召回程序**

```
启动召回 → 通知客户 → 召回产品 → 评估处理 → 记录报告
```
