## 已知问题和限制

### 1. 向量编码占位符
- **问题**: `_encode_image_clip()` 和 `_encode_text()` 使用随机向量
- **影响**: 图像和文本检索不准确
- **优先级**: 高
- **解决方案**: 集成实际的 CLIP 和 sentence-transformers 模型

### 2. zhipu-vision 调用失败处理
- **问题**: 实际 API 调用可能失败（API key、网络等）
- **影响**: 图像分析失败时只有错误信息
- **优先级**: 中
- **解决方案**: 添加重试机制和更好的错误处理

### 3. 公式解析缺失
- **问题**: 无法处理 LaTeX 公式
- **影响**: 数学文档无法完整解析
- **优先级**: 低
- **解决方案**: 使用 MathPix 或类似服务
