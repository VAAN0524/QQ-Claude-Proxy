## 核心问题和解决方案

### 问题 1: 配图风格不符合要求 ❌

**表现**:
- 用户反馈："配图还是稀烂，都是抽象画"
- 使用智谱 CogView-3-flash 生成的都是抽象插画
- 图片尺寸小（80KB），艺术风格浓重
- 不适合医疗健康、美食等需要真实感的主题

**根本原因**:
- 没有根据文章类型选择合适的模型
- 智谱 CogView 是艺术插画模型，无法生成写实照片

**解决方案** ✅:
1. **明确模型选择规则**:
   ```
   写实照片需求 → ModelScope Qwen-Image-2512
   抽象插画需求 → 智谱 CogView-3-flash

   判断标准：
   - 医疗健康、美食、产品展示、生活场景 → 写实照片
   - 科技概念、数据可视化、思维导图 → 抽象插画
   ```

2. **实现 ModelScope 异步调用**:
   ```python
   # 关键配置
   headers = {
       "Authorization": f"Bearer {api_key}",
       "Content-Type": "application/json",
       "X-ModelScope-Async-Mode": "true"  # ⚠️ 必须启用异步模式
   }

   # 轮询结果
   for attempt in range(max_retries):
       result_response = requests.get(
           f"{BASE_URL}v1/tasks/{task_id}",
           headers={
               "Authorization": f"Bearer {api_key}",
               "X-ModelScope-Task-Type": "image_generation"
           }
       )
       task_status = result_data.get("task_status")
       if task_status == "SUCCEED":
           # 获取图片 URL
   ```

3. **兼容两种响应格式**:
   ```python
   # ⚠️ ModelScope API 响应格式不一致
   output_images = result_data.get("output_images", [])
   if isinstance(output_images[0], dict):
       image_url = output_images[0].get("url")
   else:
       image_url = output_images[0]  # 直接是字符串
   ```

**效果对比**:
| 维度 | 智谱 CogView | ModelScope Qwen |
|------|-------------|----------------|
| 图片风格 | 抽象插画 | 写实照片 ✅ |
| 文件大小 | 80 KB | 2.6 MB |
| 适用场景 | 科技概念 | 医疗健康 ✅ |
| 用户满意度 | ❌ 不符合要求 | ✅ 符合要求 |

### 问题 4: 图片路径未正确替换 ❌

**表现**: 章节配图显示为本地路径 `images/xxx.png`，微信无法访问

**解决方案** ✅:
1. **先上传所有图片到素材库**（必须第一步）
2. **获取每张图片的微信 URL**
3. **使用正则表达式替换所有格式**:

```python
import re

# 支持所有格式的图片引用
pattern = r'!\[([^\]]*)\]\(images/' + re.escape(img_file) + r'\)'
replacement = f'<p style="text-align:center;"><img src="{url}" style="max-width:100%;"/></p>'
html = re.sub(pattern, replacement, md_content)

# 支持格式：
# ✅ ![描述](images/chapter01.png)
# ✅ ![](images/chapter01.png)
# ✅ ![描述](images/chapter01.png "title")
```

## 经验教训

### 1. 模型选择是关键

**教训**: 不是所有图像生成模型都适合所有场景

**规则**:
- 写实照片 → ModelScope Qwen-Image-2512
- 抽象插画 → 智谱 CogView-3-flash

### 2. 异步 API 调用要注意

**教训**: ModelScope Qwen 必须使用异步模式

**要点**:
- 设置 `X-ModelScope-Async-Mode: true`
- 实现轮询机制
- 兼容多种响应格式

### 3. 编码问题要重视

**教训**: 中文发布必须正确处理 UTF-8 编码

**要点**:
- `json.dumps(data, ensure_ascii=False).encode('utf-8')`
- 设置 `Content-Type: application/json; charset=utf-8`

### 4. 顺序不能错

**教训**: 上传图片 → 替换路径 → 创建草稿

**原因**:
- 微信只能访问素材库中的图片
- 本地路径无法在微信中显示
