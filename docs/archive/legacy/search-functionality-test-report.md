## 📝 使用建议

### 日常搜索场景

- **中文信息搜索**: 优先使用 `zhipu-search` skill
- **英文技术搜索**: 使用 `Exa 搜索` 或 `OpenWebSearch (Bing)`
- **网页内容提取**: 使用 `Jina Reader`
- **通用搜索**: 使用 `智能搜索 V2` 自动路由

### 配置优化

1. **启用 Tavily 搜索**:
   ```bash
   # 访问 https://tavily.com 申请 API Key
   # 在 .env 文件中添加：
   TAVILY_API_KEY=tvly-your-api-key-here
   ```

2. **设置代理** (如果需要):
   ```bash
   # 在 .env 文件中添加：
   HTTP_PROXY=http://127.0.0.1:7890
   HTTPS_PROXY=http://127.0.0.1:7890
   ```

3. **修复 YouTube/Bilibili 搜索**:
   - 检查网络连接
   - 增加超时时间（当前 30s）
   - 配置 Cookies 绕过反爬限制
