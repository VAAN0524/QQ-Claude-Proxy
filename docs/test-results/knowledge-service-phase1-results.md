## ⚠️ 已知限制

### Phase 1 功能范围

1. **没有智能检索**：
   - 当前使用简单的 LIKE 搜索
   - 没有向量嵌入和语义相似度
   - 没有任务类型识别
   - 这是 Phase 2 的功能

2. **没有经验自动提取**：
   - 需要手动保存知识
   - Claude 不会自动建议保存
   - 这是 Phase 3 的功能

3. **没有加密和脱敏**：
   - 数据以明文存储
   - 没有敏感信息检测
   - 这是 Phase 4 的功能

4. **搜索性能**：
   - sql.js 比 better-sqlite3 慢
   - 大量数据时可能需要优化
   - 这是后续优化项

## 🐛 问题记录

### 开发过程中遇到的问题

1. **better-sqlite3 编译失败**
   - **原因**: 需要 Visual Studio 构建工具
   - **解决**: 改用 sql.js（纯 JavaScript 实现）
   - **影响**: 性能略差，但符合零依赖要求

2. **TypeScript 类型错误**
   - **原因**: SearchContext 没有 tags 属性
   - **解决**: 在内存中过滤结果
   - **影响**: 无，功能正常

3. **重复函数定义**
   - **原因**: 添加了新的 sendMessage 方法
   - **解决**: 使用原有的方法，更新调用方式
   - **影响**: 无
