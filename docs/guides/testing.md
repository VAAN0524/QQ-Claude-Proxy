## ⚠️ 故障排查

### 问题 1: Bot 无响应
**可能原因**:
- QQ Bot 连接断开
- Claude Code CLI 未启动
- 网络问题

**解决方法**:
1. 检查 `logs/app.log` 中的错误信息
2. 重启应用：`npm run dev:win`
3. 验证 Claude Code CLI: `claude --version`

---

### 问题 2: 响应时间过长
**可能原因**:
- 网络延迟
- API 限流
- 复杂任务处理

**解决方法**:
1. 查看 Dashboard 的任务状态
2. 检查网络连接
3. 简化测试用例

---

### 问题 3: 图片/文件处理失败
**可能原因**:
- 存储目录权限问题
- 文件格式不支持
- 磁盘空间不足

**解决方法**:
1. 检查 `uploads/` 目录权限
2. 验证文件格式
3. 清理磁盘空间
